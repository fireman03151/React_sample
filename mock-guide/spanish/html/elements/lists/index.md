## Liza

Hay tres tipos de listas en HTML. Todas las listas deben contener uno o más elementos de la lista.

### Lista HTML desordenada

La lista desordenada comienza con la etiqueta `<ul>` y los elementos de la lista comienzan con la etiqueta `<li>` . En listas desordenadas, todos los elementos de lista marcados con viñetas de forma predeterminada.

\`\` \`

*   ít.
*   ít.
*   ít.

\`\` \` Salida:

*   ít.
*   ít.
*   ít.

### Lista de HTML ordenados

La lista ordenada comienza con la etiqueta `<ol>` y los elementos de la lista comienzan con la etiqueta `<li>` . En las listas ordenadas todos los elementos de la lista están marcados con números.

\`\` \`

1.  Primer elemento
2.  Segundo artículo
3.  Tercer artículo

\`\` \` Salida:

1.  Primer elemento
2.  Segundo artículo
3.  Tercer artículo

### Lista de descripciones HTML

La lista de descripción contiene elementos de lista con sus definiciones. Usamos la etiqueta `<dl>` para iniciar la lista, la etiqueta `<dt>` para definir un término y la etiqueta `<dd>` para describir cada término.
