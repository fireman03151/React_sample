## Calculadora de Sub-rede Ip

As calculadoras de sub-redes IP ajudam a dividir as redes IP em sub-redes, calculando os endereços de rede apropriados, as máscaras de sub-rede, os endereços de broadcast e os intervalos de IP do host. Para redes simples (como uma LAN doméstica), pode ser muito fácil identificar os valores apropriados, mas para uma sub-rede mais complexa, uma Calculadora de sub-rede IP é uma excelente ferramenta.

Aqui estão algumas calculadoras de sub-rede IP on-line:

*   http://www.subnet-calculator.com/
*   http://jodies.de/ipcalc
*   https://www.iplocation.net/subnet-calculator
*   http://www.calculator.net/ip-subnet-calculator.html
