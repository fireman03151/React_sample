# Editor de imagem

Desenvolvedores / artistas usam editores de imagem para manipular ou transformar imagens. _Cortar, redimensionar, corrigir cores, compor (combine muitas imagens em uma) ou remova objetos indesejados._ Alguns são mais populares que outros, mas todos eles têm recursos de edição semelhantes. As seções a seguir abrangem algumas de suas diferenças.

### Editores de Imagem do Web App

Os Editores de Imagem do Web App estão disponíveis no navegador. Em outras palavras, você não precisa deixar seu navegador para trabalhar em uma imagem. Exemplo: a capacidade de abrir imagens de uma pasta / url local.

### Editores de código aberto

Disponível como downloads gratuitos e aberto a contribuições do usuário para o desenvolvimento do software.

### Editores pagos

Os editores pagos precisam de um pagamento único ou são baseados em assinatura. Mas versões de avaliação estão disponíveis para testar o software antes de se comprometer a comprar. Se acontecer de você ser um estudante, muitas vezes, há descontos disponíveis através de um e-mail ou ID do aluno.
