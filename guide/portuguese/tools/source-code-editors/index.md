## Editores de Código Fonte

Editores de código fonte são os programas que permitem a criação de código e  
edição. Qualquer editor de texto pode ser usado para escrever código. Mas código dedicado  
os editores oferecem muitos recursos úteis, incluindo: conclusão de texto,  
marcação de marcação e separadores / painéis para edição lado a lado. Esses recursos podem  
ser útil mesmo para novos usuários que não conhecem o caminho de todos os  
opções avançadas.

Existem muitos editores disponíveis para download ou uso  
online, pago e gratuito. Eles oferecem muitos dos mesmos recursos principais, mas  
cada um tem suas próprias sutilezas. Outras pesquisas podem ajudar a encontrar o que é  
certo para você. Os seguintes editores são todos de plataforma cruzada e  
livre para usar ou avaliar.

## [Texto Sublime](https://www.sublimetext.com/) ![Ícone de texto sublime](//discourse-user-assets.s3.amazonaws.com/original/2X/1/1249e18855b9c9ef177e636e6c4e308b4877bf2d.png)

![Sublime Text Screenshot](//discourse-user-assets.s3.amazonaws.com/original/2X/6/63f774c31a7b1a439bcef9d11e95463b6998310e.png)

O Sublime Text é um editor muito popular que existe [desde 2008](https://www.sublimetext.com/blog/articles/one-point-oh) . Lá  
há muitas opções e [extensões](https://packagecontrol.io/) disponíveis,  
e muita [ajuda disponível](http://stackoverflow.com/search?q=sublime+text)  
online para ajudar  
[configurá-lo](http://www.websightdesigns.com/wiki/Setting_up_Sublime_Text_3) .  
(A licença para uso continuado custa US $ 70. No entanto, o Sublime Text é gratuito para  
baixar e avaliar, com um pop-up de tela.)

## [Colchetes](http://brackets.io/) ![Ícone de suportes](//discourse-user-assets.s3.amazonaws.com/original/2X/d/df30640741461569071875cf4ff6cad9924f8b32.png)

![Captura de tela de colchetes](//discourse-user-assets.s3.amazonaws.com/original/2X/2/2f395a58d24ed551ed8440a10ae640777f1143b6.png)

Brackets é um editor de código aberto relativamente novo da Adobe. É muito usuário  
amigável, especialmente para pessoas que não estão acostumadas a interfaces de linha de comando  
ou configurações / preferências de estilo JSON. Extensões e temas são rápidos e fáceis  
para localizar e instalar através do Extension Manager.

## [Átomo](https://atom.io/) ![Ícone do átomo](//discourse-user-assets.s3.amazonaws.com/original/2X/7/7f6ba67d0ea310e1c1024f83c9acc1e0ecf255a8.png)

![Screenshot do Atom](//discourse-user-assets.s3.amazonaws.com/original/2X/8/8016e7c0937e8ce6fcb6104c9aee0026443912f5.png)

Atom é um editor de [código aberto](https://github.com/atom/atom) , desenvolvido  
pelo [GitHub](https://github.com/) . Como o Sublime Text, o Atom é bastante popular.  
É muito configurável ("hackable", como dizem), mas recursos avançados podem  
ser esmagadora para novos usuários. Há também muita [documentação](http://flight-manual.atom.io/) [útil](http://readwrite.com/2014/05/20/github-atom-5-tips-getting-started-tutorial-corey-johnson/)

[disponível](http://stackoverflow.com/search?q=atom) online.

## [Código do Visual Studio](https://code.visualstudio.com/) ![Ícone de código do Visual Studio](https://cdn-media-1.freecodecamp.org/imgr/b4vFsKa.png)

![Screenshot do código do Visual Studio](https://code.visualstudio.com/home/home-screenshot-mac-lg.png)

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft em 2015 e agora é um dos editores de código mais conhecidos para desenvolvimento web. Ele suporta muitas linguagens de programação e, como as outras, você pode baixar plugins ou extensões para torná-lo mais configurável para o seu estilo de codificação. Suas principais características são:

*   IntelliSense - Para realce de sintaxe, autocomplete, definições de funções e smart complete.
*   Depuração - Ser capaz de depurar usando pontos de interrupção, pilhas de chamadas e um console interativo.
*   Build-in Git - Revise as diferenças, os arquivos de estágio, faça commits, push e pull, todos disponíveis no editor.
*   Extensões - Instale extensões para adicionar novos idiomas, temas, depuradores e conectar serviços adicionais.
