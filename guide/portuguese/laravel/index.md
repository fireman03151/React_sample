# Laravel

O Laravel é um framework web PHP livre e de código aberto disponível no [GitHub](https://github.com/laravel/laravel) e licenciado sob os termos da Licença MIT. Foi criado por Taylor Otwell, que o projetou com o objetivo de permitir o desenvolvimento de aplicações web seguindo o padrão arquitetural model-view – controller (MVC) e agora é a estrutura PHP mais estrelada no GitHub. Alguns dos principais recursos encontrados no Laravel são maneiras diferentes de acessar bancos de dados relacionais, utilitários que auxiliam na implantação e manutenção de aplicativos e, por último, mas não menos importante, sistema de empacotamento modular com um gerenciador de dependências dedicado.

Como o Laravel é de código aberto, a comunidade em torno dele é muito forte, assim como a documentação onde você pode aprender como fazer quase tudo, tente dar uma olhada na [Documentação](https://laravel.com/docs/5.7/) do [Laravel](https://laravel.com/docs/5.7/) para obter um vislumbre do poder do código aberto!
