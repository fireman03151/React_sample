# jQuery Hover Method

O método jquery pairar é uma combinação do `mouseenter` e `mouseleave` eventos. A sintaxe é esta:
```
$(selector).hover(inFunction, outFunction); 
```

A primeira função, inFunction, será executada quando o evento `mouseenter` ocorrer. A segunda função é opcional, mas será executada quando o evento `mouseleave` ocorrer. Se apenas uma função é especificada, a outra função será executado tanto para o `mouseenter` e `mouseleave` eventos.
