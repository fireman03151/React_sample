O Python é conhecido por sua bela sintaxe e facilidade de leitura. Essa linguagem de script pode ser usada para uma prototipagem rápida e suja da sua ideia de um aplicativo da web. Ele também pode ser usado para criar grandes aplicativos da Web escalonáveis ​​e de fácil manutenção.

#### Frameworks da Web mais populares em Python:

*   Django
*   Balão
*   Garrafa
*   Pirâmide
*   Web2py

Então, vamos começar com alguns frameworks web e aprender como eles ajudam a construir aplicativos web em mãos. Mas antes disso, precisaríamos configurar o ambiente de desenvolvimento concluindo a [instalação de estruturas da Web no Python](https://guide.freecodecamp.org/python/setting-up-python-web-framework-django-and-flask) .
