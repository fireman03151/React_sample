---
title: Python More Built in Types
localeTitle: Python Mais Construído em Tipos
---
As seções a seguir descrevem os tipos padrão que são construídos no interpretador.

Os principais tipos internos são números, seqüências, mapeamentos, classes, instâncias e exceções.

Algumas classes de coleções são mutáveis. Os métodos que adicionam, subtraem ou reorganizam seus membros e não retornam um item específico, nunca retornem a instância de coleção em si, mas None.
