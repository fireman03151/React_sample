title: Python Numeric Types
localeTitle: Tipos Numéricos em Python
---
Os [tipos numéricos](https://docs.python.org/3/library/stdtypes.html#numeric-types-int-float-complex) para o Python são:

*   inteiros ( `int` )

#### Exemplos:

```py
print(1) 
 print(100) 
 print(8435367) 
```

*   números de ponto flutuante ( `float` )
    *   [construtor](https://docs.python.org/3/library/functions.html#float)

#### Exemplos:

```py
print(1.5) 
 print(46.84) 
 print(84357.435) 
```

*   números complexos
    *   [construtor](https://docs.python.org/3/library/functions.html#complex)

A biblioteca padrão adiciona tipos numéricos para

*   [frações](https://docs.python.org/3/library/fractions.html#module-fractions)
*   [decimais](https://docs.python.org/3/library/decimal.html#module-decimal)

Objetos numéricos são criados a partir de literais ou como resultado de funções e operadores. A sintaxe para literais numéricos está bem [documentada](https://docs.python.org/3/reference/lexical_analysis.html#numeric-literals) .
