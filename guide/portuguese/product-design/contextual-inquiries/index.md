## Consultas Contextuais

A Inquérito Contextual é um método imersivo para descobrir como um design funciona no contexto de seu uso. Designers podem usar a investigação contextual para descobrir aspectos de um design que os usuários podem não perceber conscientemente. A investigação contextual envolve tanto entrevistar usuários quanto observá-los usando o design em seu próprio ambiente.

Há quatro coisas para ter em mente:

**O contexto** deve ser mantido - o designer deve procurar entender a experiência contínua do usuário com o design e, para isso, é crucial acompanhar o usuário no seu dia-a-dia com o design.

**Parceria** - o designer deve fazer perguntas perspicazes para aprofundar o entendimento do usuário sobre o design. As respostas de um usuário serão mais informativas e mais completas se responderem enquanto estiverem usando o design.

**Interpretação** - o designer deve tentar interpretar os dados coletados de entrevistas e observações enquanto eles ainda estiverem no local com o usuário. É fundamental tentar verificar a precisão de quaisquer hipóteses sobre os dados com o usuário.

**Foco** - o designer deve se concentrar no modelo mental do usuário de como um design funciona. Em situações em que um designer pode ficar confuso ou surpreso com as ações de um usuário ou com as respostas às perguntas, o designer pode aproveitar essa oportunidade para tentar entender melhor a perspectiva do usuário e incorporá-la a iterações mais claras.
