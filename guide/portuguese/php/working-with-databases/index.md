## Quais opções estão disponíveis para o PHP se conectar a um banco de dados?

O PHP pode se conectar a uma variedade de bancos de dados diferentes, incluindo MongoDB, MsSQL e MySQL. Tanto o PHP quanto o MySQL são muito populares e fornecem sites fáceis, gratuitos e de código aberto. para ser criado e são frequentemente encontrados juntos para produzir sites de todos os tipos. O PHP e o MySQL podem ser dimensionados para suportar um grande número de usuários.

O PHP ainda suporta mais de uma maneira de lidar com conexões com o MySQL, incluindo MySQLi Procedural, PHP Data Objects (PDO) e MySQLi Object Orientated juntamente com o agora obsoleto MySQL Connect. Com o PHP, existem muitos recursos integrados à funcionalidade principal da linguagem que criam links para um banco de dados simples e fácil.
