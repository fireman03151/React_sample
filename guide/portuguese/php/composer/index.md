## Compositor

O Composer é um gerenciador de pacotes para pacotes PHP. Você usa um arquivo `composer.json` para configurar os pacotes para um projeto PHP, semelhante ao arquivo `package.json` em projetos NodeJS.

### Instalar o Composer

Para instalar o Composer, você primeiro precisa baixá-lo em [getcomposer.org](https://getcomposer.org/download/) .

Você pode então instalar o Composer local ou globalmente.

### Instalar pacotes

Instalar pacotes com a `composer install` . O Composer instalará os pacotes listados no arquivo `composer.json` no fornecedor / pasta.

```shell
    composer install 
```
