## Operadores

O PHP contém todos os operadores normais que se esperaria encontrar em uma linguagem de programação.

Um único "=" é usado como o operador de atribuição e um duplo "==" ou triplo "===" é usado para comparação.

Os usuais “<” e “>” também podem ser usados ​​para comparação e “+ =” pode ser usado para adicionar um valor e atribuí-lo ao mesmo tempo.

O mais notável é o uso do "." Para concatenar strings e ". =" Para acrescentar uma string ao final de outra.
