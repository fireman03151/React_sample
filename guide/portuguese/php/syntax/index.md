## Sintaxe

PHP é uma linguagem que permite incluir conteúdo dinâmico em páginas da Web estáticas. Ele é escrito em linha com html, basta renomear um arquivo .html para .php (web-server deve ter o php instalado) para começar.

```PHP
<!DOCTYPE html> 
 <html> 
 <head> 
 <title>PHP Example</title> 
 </head> 
 <body> 
 <h1>PHP Example</h1> 
 <?php 
  echo "Hello World from PHP!"; 
 ?> 
 </body> 
 </html> 
```
