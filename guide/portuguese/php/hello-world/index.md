## PHP - Olá Mundo

Scripts PHP são executados no servidor.

Antes de continuar, você deve ter uma compreensão básica do seguinte:

### HTML

### CSS

### JavaScript

Arquivos PHP podem conter texto, HTML, CSS, JavaScript e código PHP. Um script PHP é executado no servidor e o resultado HTML simples é enviado de volta ao navegador.

Um script PHP começa com `<?php` e termina com `?>` :

```php
<?php 
 // PHP code goes here 
 ?> 
```

ou você também pode escrever Um script PHP começa com `<?php` e termina sem `?>` :

```php
<?php 
 // PHP code goes here 
```

Abaixo, temos um exemplo de um arquivo PHP simples, com um script PHP que usa uma função PHP integrada "echo" para gerar o texto "Hello World!" em uma página da web:

```php
<!DOCTYPE html> 
 <html> 
 <body> 
 
 <?php 
 echo "My first PHP script!"; 
 ?> 
 
 </body> 
 </html> 

```
