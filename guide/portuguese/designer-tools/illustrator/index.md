## Ilustrador

O Illustrator é um programa da Adobe para criar e editar gráficos vetoriais. É uma ferramenta versátil usada por designers gráficos e ilustradores para ilustração 2D, design de layout, rastreamento e tipografia. É um software líder para designers, principalmente devido às suas funções versáteis e disponibilidade.

A principal característica que separa o Adobe Illustrator do Photoshop é o uso de gráficos vetoriais. Os gráficos vetoriais constroem imagens usando declarações matemáticas que permitem que as imagens aumentem e diminuam de forma nítida o tamanho sem a redução da qualidade da imagem. O Adobe Photoshop usa gráficos de varredura que se concentram em uma tela de tamanho de pixel definido. Se fosse para escalar uma imagem raster para cima ou para baixo, os resultados se tornariam pixelados, causando bordas borradas ou ásperas. É por esse motivo que o Adobe Illustrator é normalmente usado para designs grandes e pequenos, desde ícones de software a outdoors.
