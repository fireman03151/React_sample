## Banco de dados de gráficos

Um banco de dados gráfico é um banco de dados que usa estruturas gráficas para consultas semânticas com nós, arestas e propriedades para representar e armazenar dados. Um conceito-chave do sistema é o gráfico (ou borda ou relacionamento), que relaciona diretamente os itens de dados na loja. Os relacionamentos permitem que os dados na loja sejam vinculados diretamente e, em muitos casos, recuperados com uma operação.
