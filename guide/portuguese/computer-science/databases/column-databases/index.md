## Bancos de dados da coluna

Bancos de dados orientados a colunas são projetados para retornar dados com eficiência para um número limitado de colunas. Ele faz isso armazenando todos os valores de uma coluna juntos. Um banco de dados orientado por coluna será excelente em operações de leitura em um número limitado de colunas, no entanto, a operação de gravação será dispendiosa em comparação com bancos de dados orientados a linhas.
