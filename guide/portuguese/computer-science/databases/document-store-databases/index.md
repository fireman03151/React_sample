## Bancos de Dados do Armazenamento de Documentos

Um banco de dados de armazenamento de documentos é considerado mais um tipo de banco de dados NoSQL que é muito semelhante a um banco de dados Key Value. Um registro em um armazenamento de documentos representa um único documento estruturado que pode ser diferente de todos os outros documentos no banco de dados. Ao contrário de um banco de dados relacional, não há mapeamento de objetos para várias tabelas no banco de dados. Em vez disso, podemos pegar o objeto inteiro e escrevê-lo diretamente em um banco de dados de armazenamento de documentos.

Por exemplo, poderíamos ter o seguinte objeto dentro do nosso código.

```json
{ 
    "name": "freeCodeCamp", 
    "job": "contributor" 
 } 
```

Esse objeto inteiro pode ser gravado diretamente em um banco de dados de armazenamento de documentos sem análise adicional.
