### Instalando o Go no Mac OS X usando o Package Installer

Na [página de download do golang](https://golang.org/dl/) , obtenha o instalador do pacote Mac (terminando em .pkg) e execute-o.

![screenshot of golang's download page as of this writting, highliting link](https://raw.githubusercontent.com/AlexandroPerez/resources/master/img/mac_package_installer.jpg "Link do instalador de pacote do Mac")

#### Verifique a instalação e a versão do go

Para verificar se foi instalado com sucesso, abra seu terminal e use:

```shell
$ go version 
```

Isso deve imprimir no console a versão do go e, ao mesmo tempo, garantir que a instalação corra bem.
