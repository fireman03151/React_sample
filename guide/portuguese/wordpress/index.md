# WordPress

WordPress é um sistema de gerenciamento de conteúdo livre e de código aberto baseado em PHP e MySQL. Os recursos incluem uma arquitetura de plug-in e um sistema de modelos. É mais associado a blogs, mas suporta outros tipos de conteúdo da Web, incluindo listas de discussão e fóruns mais tradicionais, galerias de mídia e lojas on-line.

WordPress está alimentando quase 30% de todo o site e atualmente está dominando o CMS Market Share. De fato, mais de 60 milhões de websites contam com o Wordpress. Apoiado por uma enorme comunidade, esta plataforma de código aberto alimenta uma economia multimilionária com temas / plug-ins e software personalizado.

Wordpress oferece uma solução fácil de usar para ambos programadores e não-programadores para criação de sites.

Algumas vantagens do Wordpress:

- Usuários podem administrar seus sites em Wordpress de qualquer computador
- Tem um blog embutido e pronto executar sempre que aplicado
- Há plugins que extendem as funcionalidades dos sites em Wordpress
