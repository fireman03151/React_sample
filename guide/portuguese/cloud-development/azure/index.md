## Microsoft Azure

O Microsoft Azure é a oferta de nuvem pública da Microsoft que permite que indivíduos e organizações criem, implantem e operem aplicativos baseados em nuvem e serviços de infraestrutura. A plataforma de nuvem pública do Microsoft Azure oferece serviços de IaaS, PaaS e SaaS para organizações em todo o mundo e suporta muitas linguagens de programação, ferramentas e estruturas diferentes, incluindo software e sistemas específicos da Microsoft e de terceiros.

O Microsoft Azure também pode ser usado em conjunto com várias soluções da Microsoft, como o Microsoft System Center, e pode ser usado para estender o datacenter atual de uma organização em uma nuvem híbrida que amplia a capacidade e fornece recursos além do que poderia ser fornecido apenas ponto de vista das instalações.

### Usos para o Microsoft Azure

O Microsoft Azure oferece muitos serviços e ofertas de recursos. Por exemplo, você pode usar os serviços de computação do Azure Virtual Machines para criar uma rede de servidores virtuais para hospedar um aplicativo, banco de dados ou solução personalizada, que seria uma oferta baseada em IaaS.

Outros serviços podem ser categorizados como PaaS porque você pode usá-los sem manter os sistemas operacionais subjacentes. Por exemplo, quando você executa um site no Azure Web Apps ou um banco de dados SQL nos Bancos de Dados SQL do Azure, não é necessário garantir que você esteja usando a versão mais recente dos Serviços de Informações da Internet (IIS) ou SQL e tenha os patches e atualizações mais recentes instalado, pois isso é responsabilidade da plataforma Microsoft Azure.

Exemplo para o serviço SaaS no Microsoft Azure seria o Operations Management Suite, que é configurado e acessado por meio dos serviços do Operational Insights. Aqui, você apenas configura suas ferramentas de gerenciamento e conecta-se aos serviços que deseja gerenciar por meio do Microsoft Azure, portanto, nenhuma infraestrutura local é necessária ou precisa ser gerenciada.

### Ambientes

O Azure fornece uma grande variedade de diferentes tipos de ambiente, incluindo plataformas de sistema operacional como o Windows Server e o Ubuntu Server. Há várias variantes que podem ser escolhidas, como o Ubuntu 14.04 ou o Ubuntu 16.10, com a capacidade de usar esses ambientes por meio do acesso integrado à Web CLI ou SSH do Azure via seu cliente preferencial.

Existem vários níveis de “tamanhos” de máquina para escolher, desde simples máquinas de baixo nível de bar até caixas de rebatidas pesadas multi-GPU e multi-CPU.
