\## Rackspace Cloud

O Rackspace Cloud é um conjunto de produtos e serviços de computação em nuvem faturados em uma base de computação utilitária da empresa norte-americana Rackspace. As ofertas incluem hospedagem de aplicativos da Web ou plataforma como um serviço ("Cloud Sites"), Cloud Storage ("Cloud Files"), servidor virtual privado ("Cloud Servers"), balanceadores de carga, bancos de dados, backup e monitoramento. #### Mais Informações:
