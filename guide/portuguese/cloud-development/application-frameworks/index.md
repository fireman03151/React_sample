## Frameworks de Aplicação

Um Application Framework é um software pronto para uso para execução de aplicativos. Os frameworks de aplicativos lidam com tarefas comuns que a maioria dos aplicativos faz. Os Frameworks de Aplicativos economizam em tempo de desenvolvimento e manutenção, porque vários fornecedores de aplicativos agrupam recursos para uma causa comum. Um Application Framework é mais que uma coleção de seus componentes nomeados. Inclui software exclusivo que integra seus componentes a um paradigma de sistemas que simplifica tarefas comuns, difíceis ou repetitivas usando bibliotecas de código reutilizável. Cada estrutura de aplicativo resolve problemas com um conjunto exclusivo de componentes ou de uma maneira única - fornecendo gerenciamento de recursos, por exemplo.
