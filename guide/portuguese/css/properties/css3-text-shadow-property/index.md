## Propriedade de sombra de texto CSS3

A propriedade de sombra de texto adiciona sombra ao texto. Cada sombra é a combinação de deslocamentos X e Y do elemento original, raio de desfoque e cor. \`\` \`css Sintaxe Css

text-shadow: cor do raio de desfoque do deslocamento X com desvio X
```
**Example :** 
```

css p { text-shadow: 2px 2px 8px # FF0000; } \`\` \`
