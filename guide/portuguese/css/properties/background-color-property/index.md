## Propriedade da cor de fundo

Você usa a propriedade `background-color` para definir a cor de fundo de um elemento. Você pode usar um valor de cor (nome da cor, valor hexadecimal, valor RGB / RGBA, valor HSL / HSLA) ou a palavra-chave `transparent` .

**Exemplo:**

```css
body { 
    background-color: crimson; 
 } 
 
 div { 
    background-color: #ffffff; 
 } 
 
 .myClass { 
    background-color: rgba(0, 0, 0, 0.5); 
 } 
```

#### Valores de propriedade:

`background-color: color | transparent | initial | inherit;`

`color` - Especifica a cor de fundo (nome da cor, valor hexadecimal, valor RGB / RGBA, valor HSL / HSLA).

`transparent` - o valor padrão. Define a cor do plano de fundo como transparente.

`initial` - Define essa propriedade para seu valor inicial (o valor padrão).

`inherit` - Herda essa propriedade de seu elemento pai.
