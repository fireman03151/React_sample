## Propriedade Float

A propriedade `float` permite que um elemento flutue à esquerda ou à direita de outro elemento. Um uso primário do `float` é ter uma imagem à esquerda ou à direita de um bloco de texto.

```css
.my-element { 
  float: left; 
  float: right; 
  float: inline-start; 
  float: inline-end; 
  float: none; 
 } 
```
