## CSS3 Múltiplas Colunas

O **layout** de várias colunas do CSS estende o modo de _layout de bloco_ para permitir a fácil definição de várias colunas de texto. As pessoas têm dificuldade em ler o texto se as linhas forem muito longas; Se demorar muito para os olhos se moverem do fim da linha até o começo da próxima, eles perdem a noção de qual linha eles estavam. Portanto, para aproveitar ao máximo uma tela grande, os autores devem ter colunas de texto de largura limitada colocadas lado a lado, como fazem os jornais.

Infelizmente, isso é impossível de se fazer com CSS e HTML, sem forçar quebras de coluna em posições fixas ou restringir severamente a marcação permitida no texto, ou usar scripts heróicos. Essa limitação é resolvida adicionando novas propriedades CSS para estender o modo tradicional de layout de bloco.

Existem várias propriedades que permitem personalizar o layout de várias colunas:

*   contagem de colunas
*   lacuna de coluna
*   estilo de regra de coluna
*   coluna-regra-largura
*   coluna-regra-cor
*   regra de coluna
*   intervalo de coluna
*   largura da coluna
