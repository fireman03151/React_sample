## Alinhamento de texto

Esta propriedade CSS descreve o alinhamento horizontal do conteúdo embutido em seu elemento de bloco pai. `text-align` não controla o alinhamento de elementos de bloco, afeta apenas o conteúdo in-line.

### Valores:

A propriedade `text-align` é especificada como uma única palavra-chave escolhida na lista de valores abaixo:

`text-align: left;` alinha o texto à esquerda

`text-align: right;` alinha o texto à direita

`text-align: center;` alinha o texto para o centro

`text-align: justify;` faz com que as linhas tenham a mesma largura

`text-align: justify-all;` faz com que as linhas tenham a mesma largura, incluindo a última

`text-align: start;` alinha a última linha no começo da linha

`text-align: end;` alinha a última linha no final da linha

`text-align: match-parent;` calcule os valores inicial e final para a direção do pai e substitua pelo valor apropriado da esquerda ou da direita.

**Valores de alinhamento de bloco (sintaxe não padrão)** :

`text-align: -moz-center;`

`text-align: -webkit-center;`

**Valores Globais** :

`text-align: inherit;` herda de seu elemento pai

`text-align: initial;` valor padrão

`text-align: unset;` aplica o valor herdado ou inicial, dependendo das propriedades padrão do elemento
