## Direção do Flexbox

Agora que você sabe o que é o flexbox, é hora de saber como organizar o conteúdo em um contêiner flexível. Você pode definir isso via _flex-direction_

Existem 4 maneiras de definir a direção flexível

1.  Linha

![](https://css-tricks.com/wp-content/uploads/2013/04/flex-direction2.svg)

A direção flexível é definida como linha por padrão. Isso organiza todo o seu conteúdo em uma única linha **sem margens **.****

2.  Linha reversa

![](https://i-msdn.sec.s-msft.com/dynimg/IC681588.png)

Isso inverte seu conteúdo na direção oposta. Originalmente estava indo da esquerda para a direita, agora está indo da direita para a esquerda.

3.  coluna

![](https://i-msdn.sec.s-msft.com/dynimg/IC681589.png)

Isso organiza seu conteúdo em uma única coluna, de cima para baixo.

4.  coluna reversa

Isso organiza seu conteúdo em uma única coluna, de baixo para cima.
