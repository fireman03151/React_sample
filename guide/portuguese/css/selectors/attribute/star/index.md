## Estrela

O seletor de atributos Star CSS ou \* é usado para selecionar atributos que contêm um valor específico.

#### Exemplo

Se você deseja pesquisar todos os links de âncora que contêm "código" em qualquer lugar no valor da URL e torná-los amarelos, você pode fazer assim:

```css
a[href*="code"] { 
   color: yellow; 
 } 

```
