## Cifrão

O seletor de atributo CSS do sinal de dólar ou $ é usado para selecionar atributos que terminam com um valor específico.

#### Exemplo

Se você quiser pesquisar todos os links de âncora do tipo de arquivo .pdf e torná-los vermelhos, você pode fazer assim:

```css
a[href$=".pdf"] { 
   color: red; 
 } 

```
