## Tipo

O Seletor de tipos seleciona elementos por nome de nó (tag).

#### Sintaxe do Código

```css
* { 
    css declarations; 
 } 
```

#### Exemplo de código

```css
h1 { 
    background-color: red; 
 } 
```

seleciona todo o elemento h1 e define a cor de fundo em vermelho.

```css
div { 
    width: 100px; 
 } 
```

seleciona todo o elemento div e ajusta a largura do elemento 100px.
