\## Olá Mundo

Só temos que adicionar o método `print("Hello World")` (da biblioteca UIKit) para a função `viewDidLoad()` da classe \`ViewController:

\`\` \`Swift importar UIKit

classe ViewController: UIViewController {
```
override func viewDidLoad() { 
    super.viewDidLoad() 
    // Do any additional setup after loading the view, typically from a nib. 
 
    print("Hello World") 
 
 } 
```
