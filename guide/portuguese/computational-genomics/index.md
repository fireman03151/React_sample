# Genômica computacional

Um novo campo especializado no desenvolvimento de novos algoritmos para estudar o genoma humano.

Esta seção enfocará as várias estruturas de dados e algoritmos usados ​​e criados para essa finalidade, desde matrizes de sufixo até o alinhamento de seqüências.
