## Bulma

O Bulma é um framework CSS opensource, baseado no flexbox e usado por mais de 100.000 programadores.

Por que usar Bulma?
* 100% responsive.
* Modular
* Moderno.
* Grátis (Source code no Github).
* Comunidade crescente.
* Fácil de aprender.
* Personalização Rápida.

Apenas um comando para começar:
```terminal
$ npm install bulma
```
