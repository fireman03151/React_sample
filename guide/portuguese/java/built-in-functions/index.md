# Funções incorporadas

O Java também possui muitas funções internas ou pré-definidas que normalmente são armazenadas nos pacotes java.lang e java.io, que são importados automaticamente em editores como o Eclipse, IntelliJ, Netbeans e outros, ou podem ser importados manualmente usando a seguinte sintaxe:

```java
import java.util.*; // Nesse caso vai importar todas as classes contidas no pacote java.util  
import java.io.*; 

import java.util.Random; // Nesse caso vai importar apenas a classe Random do pacote java.util
```

Essas funções built-in facilitam bastante a vida do programador pois já são implementações que podemos usar bastante no dia a dia.
