## Formatação de texto em HTML

O HTML fornece muitos elementos úteis para formatação de texto. Permite fazer o seu texto: negrito, itálico, marca e muito mais. Mudar o estilo do seu texto não tem qualquer razão - o principal é apenas fazer o leitor dar uma olhada para algumas notas importantes.

### Negrito e Forte

Você pode facilmente alterar o seu significado de texto usando o elemento HTML `<b>` . Isso torna as palavras em negrito, cuja função é destacar o fragmento da sequência. Por exemplo:
```
The most important part of your code is <b>the end</b>, because if you <b>don't close</b> the element, it will affect to <b>everything</b>! 
```

Você também pode usar `<strong>` também - adiciona também importância "forte" semântica. Seu navegador não reconhece a diferença entre esses dois elementos, mas existe.

### Itálico e enfatizado

Geralmente usado quando citar algo ou colocar uma tradução da palavra em muitos artigos. Isso os torna em itálico - apenas imagine um pouco chutando nas letras certas. Por exemplo:
```
Theatre - <i>teatos</i>, <i>teates</i> and <i>teatron</i>. 
```

Você também pode usar `<em>` - adiciona também importância "enfatizada" semântica. Seu navegador não reconhece a diferença entre esses dois elementos, mas existe.

### Pequeno

Isso torna o texto menor que o tamanho normal da fonte usada. O significado deste elemento foi alterado em HTML5 - representa comentários secundários e letras pequenas.

`Normal, <small>small</small>, normal, <small>small</small>!`

### Marcado

O elemento `<mark>` torna o seu texto marcado - em outras palavras, ele destaca seu texto. Você pode usá-lo para dizer aos leitores que é uma das coisas importantes em seu artigo. Por exemplo:

`HTML is full of things and <mark>it's our journey</mark> to get known them better!`

### Excluído

O elemento `<del>` faz com que seu texto seja atingido no centro. É útil mostrar alterações nos seus documentos.

`WWI started in <del>1913</del> 1914 year.`

### Inserido

Tag `<ins>` faz com que seu texto seja inserido no artigo. Usando outras palavras que facilitam muito a compreensão - adicionado. Mostra uma linha sob o texto inserido.

`HTML isn't boring. <ins>You can make a lot of combinations of elements!</ins>`

### Inscrito

O uso do elemento `<sub>` fornece uma formatação útil como texto subscrito (mostrando-o menor na parte inferior). Existe um código de exemplo:

`This was in 2003 year <sub>(needs a link).`

### Sobrescrito

Se você quiser fazer um oposto ao texto subscrito, você pode facilmente usar o elemento `<sup>` . Mostra um texto menor no topo.

`10<sup>x+y</sup> = 1000`
