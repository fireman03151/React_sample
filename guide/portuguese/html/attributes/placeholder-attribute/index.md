## Atributo de espaço reservado | HTML5

Especifica uma dica curta que descreve o valor esperado de um controle de formulário `<input>` ou `<textarea>` .

O atributo de espaço reservado não deve conter retornos de carro ou feeds de linha.

Observação: não use o atributo de espaço reservado em vez de um elemento <label>, suas finalidades são diferentes. O atributo <label> descreve a função do elemento de formulário (isto é, indica que tipo de informação é esperado) e o atributo de espaço reservado é uma sugestão sobre o formato que o conteúdo deve tomar.

### Exemplo

Exemplo de entrada

```html

<form> 
    <input type="text" name="fname" placeholder="First Name"> 
    <input type="text" name="lname" placeholder="Last Name"> 
 </form> 
```

Exemplo de Textarea

```html

<textarea placeholder="Describe yourself here..."></textarea> 
```

### Compatibilidade

Este é um atributo HTML5.
