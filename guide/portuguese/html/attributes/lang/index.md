## Lang

Em HTML, a tag Lang é usada para declarar o idioma do todo ou parte de uma página da web.

### Exemplos

```html

<html lang="en"> 
 </html> 
```

O atributo lang também pode ser usado para especificar o idioma de um elemento específico:

```html

<p lang="hi"> 
  फ्री कोड कैंप 
 </p> 
```

No exemplo acima, "hi" denota a língua hindi. Da mesma forma, você pode usar "en" para inglês, "es" para espanhol, "fr" para francês e assim por diante.

Consulte a [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) para obter o código de idioma de dois dígitos apropriado.
