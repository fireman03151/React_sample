## Atributo de Função

O atributo `role` , descreve o papel de um elemento em programas que podem utilizá-lo, como leitores de tela ou lupas.

Exemplo de uso:

```html

<a href="#" role="button">Button Link</a> 
```

Os leitores de tela lerão esse elemento como "botão" em vez de "link".

Existem quatro categorias de papéis:

*   Papéis abstratos
*   Papéis de Widget
*   Funções da estrutura do documento
*   Papéis de Marco

Para obter uma lista completa das funções existentes, consulte a [documentação das funções árias](https://www.w3.org/TR/wai-aria/roles) .
