## Um atributo de destino

A `<a target>` atributo especifica onde abrir o documento vinculado em uma `a` tag (âncora).

#### Exemplos:

Um atributo de destino com o valor de "\_blank" abre o documento vinculado em uma nova janela ou guia.

```html

    <a href="https://www.freecodecamp.org" target="_blank">freeCodeCamp</a> 
```

Um atributo de destino com o valor de "\_self" abre o documento vinculado no mesmo quadro em que foi clicado (esse é o padrão e geralmente não precisa ser especificado).

```html

    <a href="https://www.freecodecamp.org" target="_self">freeCodeCamp</a> 
```

```html

    <a href="https://www.freecodecamp.org">freeCodeCamp</a> 
```

Um atributo de destino com o valor "\_parent" abre o documento vinculado no quadro pai.

```html

    <a href="https://www.freecodecamp.org" target="_parent">freeCodeCamp</a> 
```

Um atributo de destino com o valor de “\_top” abre o documento vinculado em todo o corpo da janela.

```html

    <a href="https://www.freecodecamp.org" target="_top">freeCodeCamp</a> 
```

Um atributo de destino com o valor de _"framename"_ Abre o documento vinculado em um quadro nomeado especificado.

```html

    <a href="https://www.freecodecamp.org" target="framename">freeCodeCamp</a> 
```
