## Tag de estilo

A tag de estilo é usada como um arquivo css, exceto dentro de um HTML, assim:
```
    <head> 
      <title>Your Title</title> 
      <style> 
        p { 
          color:red; 
        } 
      </style> 
    </head> 
```

Isso tornaria a cor da tag de parágrafo vermelha. É útil se você quer apenas colocar um pouco de código, mas se você quer uma folha de estilo muito longa, então eu recomendo usar apenas o `<link />` .

Mais informações:

A tag de estilo é usada para definir qualquer estilo CSS para página da Web dentro de um documento. A tag de estilo deve ser aninhada dentro da seção head do documento html:

```html

<head> 
  <style> 
  h1 { 
     text-align: center; 
     font-family: sans-serif; 
     } 
  </style> 
 </head> 
```

Você pode escrever qualquer código CSS dentro da tag style de acordo com sua sintaxe.
