## Tag B

A tag `<b>` é usada para especificar texto em negrito, sem transmitir qualquer importância ou relevância especial. Ao negrito de texto de importância ou relevância especial, é recomendável usar a tag `<strong>` .

### Exemplo:

```html

<b>This text is bold</b> 
```

Isso apareceria como:

**Este texto é ousado**
