## Tag U

A marca HTML `<u>` renderiza texto com um sublinhado.

O elemento `<u>` foi descontinuado no HTML 4.01. Em HTML5, a tag `<u>` foi redefinida para representar o texto que deveria ser estilisticamente diferente do texto normal. Isso inclui palavras com erros ortográficos ou substantivos próprios em chinês.

Em vez disso, para sublinhar o texto, recomenda-se que a tag `<span>` seja usada no lugar de `<u>` . Estilize suas tags `<span>` com a propriedade de `text-decoration` CSS com o valor `underline` .

### Exemplos:

\`\` \`html

Este parágrafo tem um sublinhado .
```
Underlining text with the `<span>` tag: 
```

html Todo mundo tem falado sobre freeCodeCamp ultimamente. \`\` \`
