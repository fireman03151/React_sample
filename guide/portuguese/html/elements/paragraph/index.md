## Parágrafo

O HTML

elemento representa um parágrafo de texto.

geralmente representa um bloco de texto separado de outros blocos pelo espaço em branco vertical.

### Exemplo

\`\` \`html

Este é um parágrafo

Este é outro parágrafo

\`\` \`
