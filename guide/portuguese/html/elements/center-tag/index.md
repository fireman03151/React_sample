## Tag Center

Em HTML, a tag `<center>` é usada para centralizar o texto em uma página. Você pode escrever uma tag `<center>` como `<center>My Text Here</center>` O recurso é obsoleto não recomendado para uso. A tag `<center>` foi descontinuada no HTML 4, o recurso pode ser removido dos navegadores da Web a qualquer momento. Recomenda-se usar o `text-align` CSS para o texto central.
