## Eu tag

O elemento `<i>` é usado para denotar o texto que é separado do texto ao redor de alguma forma. Por padrão, o texto rodeado por tags `<i>` será exibido em tipo itálico.

Nas especificações HTML anteriores, a tag `<i>` era usada apenas para denotar o texto a ser colocado em itálico. No HTML semântico moderno, no entanto, tags como `<em>` e `<strong>` devem ser usadas quando apropriado. Você pode usar o atributo "class" do elemento `<i>` para indicar por que o texto nas tags é diferente do texto ao redor. Você pode querer mostrar que o texto ou frase é de um idioma diferente, como no exemplo abaixo.

```HTML
<p>The French phrase <i class="french">esprit de corps</i> is often 
 used to describe a feeling of group cohesion and fellowship.</p> 
```
