## Tag Div

A tag `<div>` é um contêiner genérico que define uma seção em seu documento HTML. Um elemento `<div>` é usado para agrupar seções de elementos HTML e formatá-los com CSS. Um `<div>` é um elemento de nível de bloco. Isso significa que ele ocupa sua própria linha na tela. Elementos logo após o `<div>` serão empurrados para a linha abaixo. Para agrupamentos e estilos semelhantes que não sejam de nível de bloco, mas inline, você usaria a tag `<span>` vez disso. A tag é usada para agrupar elementos inline em um documento.

### Exemplo

Aqui está um exemplo de como exibir uma seção na mesma cor:

```html

<div style="color:#ff0000"> 
  <h3>my heading</h3> 
  <p>my paragraph</p> 
 </div> 
```
