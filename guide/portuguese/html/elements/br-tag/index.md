## Tag Br

A tag `<br>` produz uma quebra de linha em um texto. Isso é útil para poemas e endereços.

### Exemplo

```html

<!DOCTYPE html> 
 <html> 
  <body> 
    Mozilla Foundation<br> 
    1981 Landings Drive<br> 
    Building K<br> 
    Mountain View, CA 94043-0801<br> 
    USA 
  </body> 
 </html> 
```
