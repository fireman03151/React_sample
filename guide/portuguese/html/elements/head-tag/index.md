## Tag principal

A tag `<head>` contém informações sobre uma página da web. Juntamente com `<body>` , é um dos dois elementos necessários de um documento HTML. `<head>` deve ser o primeiro filho de um elemento `<html>` . Só pode haver um elemento `<head>` em uma página.

O elemento `<head>` contém informações sobre como uma página da Web deve ser exibida, também conhecida como metadados. O título do documento, links para folhas de estilo e tags `<script>` vinculados a arquivos JavaScript devem ser colocados no `<head>` . O `<head>` não deve conter nenhum conteúdo da página.

`html <html> <head> <title>Document Titles Belong in the Head</title> </head> <body> <p>This paragraph is content. It goes in the body!</p> </body> </html>`
