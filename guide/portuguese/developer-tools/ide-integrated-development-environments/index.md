## Ambientes de desenvolvimento integrados (IDEs)

Um Ambiente de Desenvolvimento Integrado (IDE) é um software que consolida as ferramentas básicas que os desenvolvedores precisam para escrever e testar software em um aplicativo.

IDEs oferecem uma interface de usuário gráfica central (GUI) que incorpora o seguinte: _um editor de código, um compilador ou interpretador, um depurador e ferramentas de automação de compilação._

Exemplos de IDEs são:

*   [NetBeans](https://netbeans.org/)
*   [Eclipse](https://www.eclipse.org/)
*   [IntelliJ IDEA](https://www.jetbrains.com/idea/)
*   [Estúdio visual](https://www.visualstudio.com/vs/)
*   [Xcode](https://developer.apple.com/xcode/)
*   [Estúdio Android](https://developer.android.com/studio/index.html)
*   [PyCharm](https://www.jetbrains.com/pycharm/)
