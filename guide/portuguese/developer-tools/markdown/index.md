## Markdown

O Markdown é uma linguagem de marcação leve com a sintaxe de formatação de texto simples. Ele é projetado para ser convertido em HTML e muitos outros formatos usando uma ferramenta com o mesmo nome. O Markdown costuma ser usado para formatar arquivos LEIA-ME, para gravar mensagens em fóruns de discussão on-line e para criar rich text usando um editor de texto simples. Você também pode escrever Markdown em seu editor de texto (como Sublime Text ou Visual Studio Code). O Markdown está ganhando visibilidade e popularidade através do surgimento de editores de texto simplificados, "livres de distrações".
