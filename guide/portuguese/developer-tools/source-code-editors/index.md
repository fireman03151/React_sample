## Editores de código fonte

Editores de código fonte são os programas que permitem a criação de código e  
edição. Qualquer editor de texto pode ser usado para escrever código. Mas código dedicado  
os editores oferecem muitos recursos úteis, incluindo: conclusão de texto,  
marcação de marcação e separadores / painéis para edição lado a lado. Esses recursos podem  
ser útil mesmo para novos usuários que não conhecem o caminho de todos os  
opções avançadas.

### Escolhendo um editor

Existem muitos editores disponíveis para download ou uso  
online, pago e gratuito. Eles oferecem muitos dos mesmos recursos principais, mas  
cada um tem suas próprias sutilezas. Outras pesquisas podem ajudar a encontrar o que é  
certo para você. Os seguintes editores são todos de plataforma cruzada e  
livre para usar ou avaliar.

### Texto sublime

![Ícone de texto sublime](https://logo.clearbit.com/sublimetext.com)

O Sublime Text é um editor muito popular que existe [desde 2008](https://www.sublimetext.com/blog/articles/one-point-oh) . Há muitas opções e [extensões](https://packagecontrol.io/) disponíveis e muita [ajuda disponível](http://stackoverflow.com/search?q=sublime+text) on-line para ajudar a [configurá-lo](http://www.websightdesigns.com/wiki/Setting_up_Sublime_Text_3) . (A licença para uso continuado custa US $ 70. No entanto, o Sublime Text é gratuito para download e avaliação, com um pop-up na tela.)

[sublimetext.com](https://www.sublimetext.com/)

![Sublime Text Screenshot](//discourse-user-assets.s3.amazonaws.com/original/2X/6/63f774c31a7b1a439bcef9d11e95463b6998310e.png)

### Colchetes

![Ícone de suportes](https://logo.clearbit.com/brackets.io)

Brackets é um editor de código aberto relativamente novo da Adobe. É muito amigável, especialmente para pessoas que não estão acostumadas com interfaces de linha de comando ou configurações / preferências de estilo JSON. Extensões e temas são rápidos e fáceis de encontrar e instalar através do Extension Manager.

[colchetes.io](http://brackets.io/)

![Captura de tela de colchetes](//discourse-user-assets.s3.amazonaws.com/original/2X/2/2f395a58d24ed551ed8440a10ae640777f1143b6.png)

### Átomo

![Ícone do átomo](https://logo.clearbit.com/atom.io)

Atom é um editor de [código aberto](https://github.com/atom/atom) , desenvolvido pelo [GitHub](https://github.com/) . Como o Sublime Text, o Atom é bastante popular. É muito configurável ("hackable", como dizem), mas os recursos avançados podem ser avassaladores para novos usuários. Há também muita [documentação](http://flight-manual.atom.io/) [útil](http://readwrite.com/2014/05/20/github-atom-5-tips-getting-started-tutorial-corey-johnson/) [disponível](http://stackoverflow.com/search?q=atom) on-line.

[atom.io](https://atom.io/)

![Screenshot do Atom](//discourse-user-assets.s3.amazonaws.com/original/2X/8/8016e7c0937e8ce6fcb6104c9aee0026443912f5.png)

### Código do Visual Studio

![Ícone de código do Visual Studio](https://logo.clearbit.com/code.visualstudio.com)

O Visual Studio Code (também conhecido como vsc ou vscode) é um editor de código aberto muito popular, desenvolvido pela [Microsoft](https://www.microsoft.com) e está se tornando amplamente usado por desenvolvedores de todos os idiomas. Assim como o Atom e o Sublime, o Visual Studio Code tem uma ampla variedade de [extensões](https://marketplace.visualstudio.com/VSCode) e [temas](https://marketplace.visualstudio.com/search?target=VSCode&category=Themes&sortBy=Downloads) disponíveis para ajudar a personalizar e aperfeiçoar sua experiência de uso, mas o VSC é altamente configurável "out-of-box". Há também uma equipe de desenvolvimento muito ativa por trás do VSCode, com novos recursos e correções de erros sendo adicionadas pela equipe do Visual Studio Code [todo mês](https://code.visualstudio.com/updates) .

[code.visualstudio.com](https://code.visualstudio.com/)

### Notepad ++

![Notepad ++ ícone](https://upload.wikimedia.org/wikipedia/commons/0/0f/Notepad%2B%2B_Logo.png)

O Notepad ++ é um editor de código livre e de código aberto que é executado no ambiente Microsoft Windows. Foi desenvolvido em 2003, por Dan Ho. Está escrito em [C ++](www.cplusplus.com) (o nome vem disso). O Notepad ++ suporta edição por abas, que permite trabalhar com múltiplos arquivos abertos em uma única janela, e também pode ser usado como um editor de texto simples.

[notepad-plus-plus.org](https://notepad-plus-plus.org)
