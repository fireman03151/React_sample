## Chão de matemática

`Math.floor()` é um método do objeto padrão Math que arredonda um determinado número para baixo, para o próximo inteiro. Observe que, para números negativos, isso significa que o número será arredondado para "longe de 0", em vez de para o número de menor valor absoluto, pois `Math.floor()` retorna o maior número inteiro menor ou igual ao número fornecido.

### Exemplos

```javascript
Math.floor(0.9)  //  0 
 Math.floor(1.3)  //  1 
 Math.floor(0.5)  //  0 
 Math.floor(-0.9) // -1 
 Math.floor(-1.3) // -2 
```
