## Math Pow

`Math.pow()` retorna o valor de um número para o poder de outro número.

#### Sintaxe

`Math.pow(base, exponent)` , onde `base` é o número base e o `exponent` é o número pelo qual elevar a `base` .

`pow()` é um método estático de `Math` , portanto, é sempre chamado de `Math.pow()` e não como um método em outro objeto.

#### Exemplos

```js
Math.pow(5, 2); // 25 
 Math.pow(7, 4); // 2401 
 Math.pow(9, 0.5); // 3 
 Math.pow(-8, 2); // 64 
 Math.pow(-4, 3); // -64 
```
