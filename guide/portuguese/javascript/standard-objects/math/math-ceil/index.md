## Teto Matemático

O `Math.ceil()` é um método do objeto padrão Math que arredonda um determinado número para cima, para o próximo inteiro. Tome nota que para números negativos isto significa que o número será arredondado "para 0" em vez do número de maior valor absoluto (ver exemplos).

### Exemplos

```javascript
Math.ceil(0.1)  //  1 
 Math.ceil(1.3)  //  2 
 Math.ceil(-0.9) // -0 
 Math.ceil(-1.5) // -1 
```
