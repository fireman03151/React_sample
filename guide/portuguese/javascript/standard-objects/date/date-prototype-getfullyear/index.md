## Date.prototype.getFullYear

O método Date.getFullYear () retorna o ano (quatro dígitos) da data especificada de acordo com a hora local.

### Sintaxe

```javascript
dateObj.getFullYear() 
```

### Exemplo

```javascript
var date = new Date(); 
 // creates a new Date() object with current date and time. 
 date.getFullYear() 
 // 2018 
```
