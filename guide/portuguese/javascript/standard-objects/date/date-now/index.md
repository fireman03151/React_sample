## Data agora

O método Date.now () retorna o número de milissegundos decorridos desde 1º de janeiro de 1970 às 00:00:00 UTC.

### Sintaxe

```js
var timeInMs = Date.now(); 
```

### Exemplo

```js
Date.now(); 
 // 1508783660969 
```
