## Matrizes JSON

O JSON Arrays não é diferente do objeto de array normal que você usa em Javascript. É um objeto de matriz que contém vários `JSON Objects` .

Aqui está um exemplo de uma matriz JSON:

```Javascript
    var aMyPlaylist = [{ 
        artist: "Ed Sheeran", 
        track: "Supermarket flowers", 
        myRating: 10 
    }, { 
        artist: "Tracy Chapman", 
        track: "Fastcar", 
        myRating: 9 
    }]; 
```

Esta é uma matriz JSON nomeada como `aMyPlaylist` . Todos os métodos de matriz como `map` , `filter` , `sort` etc. podem ser aplicados em qualquer JSON Array
