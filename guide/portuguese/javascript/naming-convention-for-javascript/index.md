Aqui você terá uma visão geral dos diferentes casos de código que são amplamente usados.

## camelCase

Na programação, a formatação do camelCase para nomes de variáveis ​​é assim:
```
var camelCase = "lower-case first word, capitalize each subsequent word"; 
```

## PascalCase

PascalCase (ou CamelCase) é uma variante do camelCase. Ele difere do camelCase capitalizando cada palavra, _incluindo_ a primeira palavra:
```
var PascalCase = "upper-case every word"; 
```

## snake\_case

Outro caso popular chamado snake\_case separa cada palavra com sublinhados dessa maneira:
```
var snake_case = "lower-case everything, but separate words with underscores"; 

```
