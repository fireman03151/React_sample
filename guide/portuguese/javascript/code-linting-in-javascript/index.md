Linting é o processo de executar um programa que analisa seu código para erros programáticos e estilísticos. Uma ferramenta de lintagem, ou um linter, marca ou sinaliza qualquer erro potencial no seu código, como erros de sintaxe ou nomes de variáveis ​​escritos incorretamente. Isso pode economizar tempo e ajudar você a escrever um código melhor.

Os linters podem ser instalados usando o `npm` ou outro gerenciador de pacotes. Linters podem ser usados ​​a partir da linha de comando, passando arquivos. Linters também estão disponíveis como plugins para ferramentas e, às vezes, são diretamente integrados aos editores.

Aqui estão alguns linters de Javascript populares:

[JSLint](http://www.javascriptlint.com/online_lint.php)

[JSHint](http://jshint.com)

[JSCS](http://jscs.info)

[ESLint](http://eslint.org)

#### Linters para editores:

*   Código Visual: Para instalar a extensão eslint, abra a paleta de comandos (View -> Command Palette .. ou cmd + shift + p) e execute o comando: ext instalar eslint
