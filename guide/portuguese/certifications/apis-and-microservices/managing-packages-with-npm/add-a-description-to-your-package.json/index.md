## Adicione uma descrição ao seu pacote.json

Para completar o desafio, siga os passos abaixo:

*   Vá para o link fornecido na página de introdução da estatística do currículo. ![Introdução ao freecodecamp apis-and-microservices](https://user-images.githubusercontent.com/15084301/45616545-65732480-ba8d-11e8-9893-412ff6c68be5.png) [Link do projeto Glitch Starter](https://glitch.com/#!/import/github/freeCodeCamp/boilerplate-npm)
    
*   Edite o arquivo package.json e atualize o autor
    
*   Em seguida, clique no link da mostra (seu aplicativo) na parte superior do projeto Glitch para obter o link público ![Glitch link do projeto público](https://user-images.githubusercontent.com/15084301/45616579-8176c600-ba8d-11e8-8a46-4ac930578e74.png)
    
*   Envie o link na solução e pronto. ![página de sucesso](https://user-images.githubusercontent.com/15084301/45616757-019d2b80-ba8e-11e8-8b5b-98dc6d02c235.png)
