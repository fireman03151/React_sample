## Obter entrada de parâmetro de rota do cliente

Se alguém lhe disser para construir um GET ou POST, você faria app.get (…) ou app.post (…) de acordo. A estrutura básica do desafio é:

```javascript
app.get("/:word/echo", function(req, res) { 
  // word = req.params.word; 
  // respond with the json object 
 }); 
```
