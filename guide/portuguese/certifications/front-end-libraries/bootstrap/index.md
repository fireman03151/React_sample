## Bootstrap

[O Bootstrap](https://www.getbootstrap.com/) é uma biblioteca front-end que fornece uma estrutura ou layout básico que um usuário pode usar diretamente ou modificar e, em seguida, implantar. Seu foco principal é criar uma estrutura compatível com dispositivos móveis, mas o Bootstrap também funciona com dispositivos de desktop.

Citado no site da Bootstrap:
```
Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery. 
```

O framework encontra empregabilidade em muitos projetos de front-end e é reconhecido como o framework HTML e CSS mais famoso.
