## Criar um componente funcional sem estado

### Sugestão 1

*   Funções sem estado são uma das maneiras de criar um componente React.
*   Essas funções podem receber dados e processá-los, mas não gerenciam nem rastreiam alterações nesses dados.
*   Para fazer o componente, um componente sem estado, uma função de retorno é uma das coisas necessárias e é isso que está faltando

### Sugestão 2

Uma função de retorno é adicionada assim

```jsx
return( 
    <return elements> 
 );
```
