## Use jQuery para modificar a página inteira

### Explicação do Problema

Adicione as classes `animated` e `hinge` ao elemento do `body` .

#### Links Relevantes:

*   [.addClass ()](https://api.jquery.com/addClass/e)

### Solução:

```javascript
<script> 
  $("body").addClass("animated hinge"); 
  }); 
 </script> 

```
