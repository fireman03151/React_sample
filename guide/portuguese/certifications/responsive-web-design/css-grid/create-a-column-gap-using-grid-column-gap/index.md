## Criar uma lacuna de coluna usando a lacuna da coluna de grade

Este desafio requer que você adicione uma lacuna de 20px entre todas as colunas do CSS Grid no .container.

### Sugestão

Para adicionar a lacuna entre as colunas de uma grade, usamos:

```css
grid-column-gap: parameter; 
```

onde parâmetro é o intervalo necessário.

### Solução

Como o desafio tem você adicionando uma lacuna de 20px, você adicionaria a seguinte linha ao codeblock CSS .container.

```css
grid-column-gap: 20px; 

```
