## Criar um intervalo de linhas usando intervalo de linhas de grade

Este desafio requer que você adicione uma lacuna de `20px` entre todas as linhas da CSS Grid em .container.

### Sugestão

Para adicionar uma lacuna entre as linhas em uma grade CSS, usamos a propriedade CSS `grid-row-gap` .

### Solução

Como o desafio tem você adicionando um intervalo de `5px` , você adicionaria a seguinte linha ao bloco de código CSS `.container` .

```css
grid-row-gap: 5px; 

```
