## Adicionar linhas com linhas de modelo de grade

Neste desafio, você precisa definir linhas para a grade CSS em `<div class="container">` .

### Sugestão

Você pode definir linhas usando a propriedade CSS `grid-template-rows` .

### Solução

Como o desafio requer que você crie duas linhas de `50px` cada, declare o seguinte em seu bloco de código CSS `.container` :

```css
grid-template-rows: 50px 50px; 

```
