## Adicionar colunas com colunas de modelo de grade

Neste desafio, você precisa definir colunas para a grade.

### Sugestão

Você pode definir colunas declarando o seguinte no seu codeblock CSS:

```css
grid-template-columns: parameters; 
```

onde parâmetros é o que você quer que seja.

### Solução

Como o desafio requer que você crie três colunas de 100px cada, declare o seguinte no seu codeblock CSS do container:

```css
grid-template-columns: 100px 100px 100px; 

```
