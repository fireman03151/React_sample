## Crie sua primeira grade CSS

### Sugestão 1

Para alterar a exibição de qualquer elemento, a seguinte sintaxe é usada:

```css
display: propertyName; 
```

### Sugestão 2

Altere a propriedade de exibição do elemento .container

### Dica 3

Definir a exibição do elemento .container para grade

### Solução

Para definir a exibição do elemento .container na grade:

```css
display: grid; 

```
