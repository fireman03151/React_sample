## Use unidades de grade CSS para alterar o tamanho das colunas e linhas

Esse desafio exige que você defina a largura das colunas do contêiner para as especificadas na descrição do desafio.

### Sugestão

Altere a propriedade `grid-template-columns` .

### Solução

Como o desafio requer que você defina a largura para `1fr` , `100px` e `2fr` , altere a propriedade `css grid-template-columns` de `.container` para:

```css
grid-template-columns: 1fr 100px 2fr; 
```
