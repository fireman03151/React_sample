## Criar uma consulta de mídia

Seguindo as instruções:

Adicione uma consulta de mídia para que a tag p tenha um tamanho de fonte de 10px quando a altura do dispositivo for menor ou igual a 800px.

a consulta de mídia é:

```css
  /* Add media query below */ 
  @media (max-height: 800px){ 
    p { 
    font-size: 10px; 
      } 
  } 
```

e o tamanho do texto será 10px quando a largura do dispositivo for menor ou igual a 800px.
