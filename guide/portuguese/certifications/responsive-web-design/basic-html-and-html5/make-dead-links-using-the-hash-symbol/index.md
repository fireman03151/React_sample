## Faça links mortos usando o símbolo Hash

Um link morto é apenas um elemento de ancoragem, sintaticamente correto, clicável que aponta para o topo da página quando clicado sem atualizar o conteúdo (você pode verificar essa questão no fórum da FCC para uma análise mais aprofundada: [O que o Dead Link está fazendo? HTML e CSS fazem?](https://forum.freecodecamp.org/t/what-does-dead-link-on-html-and-css-do/164550/7)

Para criar um link morto, primeiro você deve escrever um elemento de âncora normal (o desafio fornece um, não reescrevê-lo) e, em seguida, modificar o atributo `href` conforme indicado nas instruções.

Se você tiver excluído ou modificado algo no código original que não seja o atributo href do existente `a` tag lembrar que você pode clicar no `reset all code` botão para reiniciar com um código limpo.

Boa sorte!
