## Transforme uma imagem em um link

Em um desafio anterior, você já conseguiu a tarefa de aninhar um elemento HTML em outro (por exemplo, [aninhar um elemento âncora em um parágrafo](https://learn.freecodecamp.org/responsive-web-design/basic-html-and-html5/nest-an-anchor-element-within-a-paragraph) )

Agora você é solicitado a resolver o mesmo problema com tags diferentes: basta colocar a imagem (como está) no elemento âncora e ela se tornará o conteúdo clicável dessa âncora.

Boa sorte!
