## Definir a cabeça e o corpo de um documento HTML

O elemento `head` é usado para agrupar todos os metadados do documento, o `body` para exibir o conteúdo do documento.

Na `head` você pode encontrar qualquer tipo de tag usado para _descrever_ o documento: que linguagem ele irá usar, quais regras de estilo (através do link da folha de estilo), seu título, uma descrição literal,…  
De qualquer forma, nada dentro do elemento `head` será processado na página: você pode ver o título e o ícone normalmente na guia do navegador ou na barra de favoritos, mas essa não é a _página_ , é o elemento do navegador vinculado a ela.

O `body` pelo contrário, contém tudo que é apresentado na página: talvez não apenas isso, mas com certeza tudo isso.

O desafio fornece uma simples página html e pede que você insira a `head` e os elementos html do `body` no lugar certo; Em outras palavras, você deve quebrar com a tag `<body></body>` todos os elementos que você pode ver sendo renderizado no lado direito da página e envolver com a tag `<head></head>` tudo descritivo.

Lembre-se de que tanto a `head` quanto o `body` devem ser filhos do elemento `<html>` .
