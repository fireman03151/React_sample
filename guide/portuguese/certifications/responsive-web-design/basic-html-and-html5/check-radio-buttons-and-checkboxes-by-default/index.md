## Verificar botões de rádio e caixas de seleção por padrão

O atributo `checked` , exclusivamente utilizável com `type="radio"` e `type="checkbox"` , é um atributo booleano (você pode checar o desafio sobre o atributo `required` , booleano também, neste link: [use html5 para requerer um campo](https://learn.freecodecamp.org/responsive-web-design/basic-html-and-html5/use-html5-to-require-a-field/) ).

Como atributo booleano, você simplesmente tem que adicionar a palavra `checked` ao elemento HTML de `input` que deseja definir por padrão (o desafio requer que você defina por padrão _a primeira_ caixa de seleção e os botões de opção já presentes no código).

Boa sorte!
