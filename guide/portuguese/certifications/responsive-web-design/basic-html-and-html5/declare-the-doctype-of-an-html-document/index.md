## Declarar o Doctype de um documento HTML

A declaração do Doctype deve estar na primeira linha, pois fornece instruções ao navegador sobre como interpretar as seguintes linhas de código.

Pode ser seguido por um elemento `<html>` , mesmo que não seja obrigatório; este é o elemento raiz do documento e, se você não o escrever, a maior parte do navegador o inferirá. No entanto, é uma boa prática escrevê-lo explicitamente.

Todos os seus elementos html devem estar aninhados dentro do `<html>` : neste desafio você é solicitado a usar um elemento `<h1>` com o conteúdo de sua escolha.
