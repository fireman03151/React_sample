## Crie um campo de texto

`input` pode ser classificada como uma tag genérica: na verdade, ela é caracterizada por seu `type` atributo, que pode assumir uma faixa de valores diferentes e, com base nesse valor, a `input` será renderizada como um objeto diferente.

Alguns exemplos de valores de `type` diferentes:

valor do tipo | código - | - texto | `<input type="text" />` checkbox | `<input type="checkbox" />` rádio | `<input type="radio" />` botão | `<input type="button" />` …

O desafio quer que você estabeleça uma tag de `input` com o tipo set to `text` sem modificar o código existente; Se algo deu errado, verifique os erros da tag de verificação dupla (como a `input` é de fechamento automático, você precisa apenas de uma tag, e não de um par de open-close). Se você modificou ou excluiu algumas partes do código fornecido, você pode reiniciar usando o botão de `reset all code` .

Boa sorte!
