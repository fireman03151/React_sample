## Criar uma lista não ordenada com marcadores

Para passar o desafio, a primeira operação que você deve fazer é remover os elementos `p` com todo o seu conteúdo.

Depois disso, você terá que implementar a lista: ela deve conter pelo menos três elementos `li` dentro do elemento `ul` e esses `li` devem estar no mesmo nível, não aninhados uns nos outros:

Corrigir:
```
<ul> 
  <li></li> 
  <li></li> 
 </ul> 
```

Incorreto:
```
<ul> 
  <li> 
    <li> 
    </li> 
  </li> 
 </ul> 
```

Boa sorte!
