## Criar um conjunto de botões de rádio

Para resolver esse desafio, você pode dividi-lo em partes para entender melhor as etapas:

*   Crie os 'containers': estabeleça dois elementos de `label` e dê a eles o conteúdo de texto que o desafio lhe pede (é necessário ter o texto 'Indoor', um 'Outdoor', assunto de letras maiúsculas);
    
*   Crie o 'conteúdo': um botão de rádio é apenas um elemento de `input` com `type=radio` (crie dois deles);
    
*   Crie um _grupo_ de botões de opção: dê a ambos os botões de rádio o `name` do atributo definido para o mesmo valor (as instruções de desafio indicam que o grupo deve ser chamado de `indoor-outdoor` );
    
*   Aninhe cada botão de opção em um rótulo;
    
    Boa sorte!
