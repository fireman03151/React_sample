## Adicionar texto do marcador de posição a um campo de texto

`placeholder` é um atributo, não uma tag. Ele pode ser usado junto com o `text` do atributo da tag de `input` para criar um texto para visualizar quando a caixa de entrada estiver vazia.

Se você está preso, verifique estes problemas:

*   você deve adicionar o atributo de `placeholder` à tag de `input` já presente no código sem remover nada; Se por acaso você removeu ou modificou alguma coisa, lembre-se de que você pode reiniciar com um código limpo clicando no botão `reset all code`
*   a sintaxe é a mesma de todo atributo: `<tag attributeName="attributeValue" >` e o valor a ser inserido é indicado pelas instruções do desafio (verifique se há erros de digitação).

Boa sorte!
