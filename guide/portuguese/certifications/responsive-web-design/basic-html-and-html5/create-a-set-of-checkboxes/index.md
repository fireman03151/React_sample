## Criar um conjunto de caixas de seleção

O atributo `type="checkbox"` funciona basicamente como o atributo `type="radio"` : a principal diferença é que caixas de seleção são usadas para coletar uma ou mais opções, botões de rádio são usados ​​quando você quer escolhas exclusivas (uma ou outra).

O atributo `name` indica o grupo ao qual pertence a caixa de seleção e, para fazer parte de um grupo, será mais fácil gerenciar os dados no lado do servidor.

Para resolver esse desafio, você precisa criar três elementos HTML de `input` com o atributo `type="checkbox"` e com o atributo `name` definido como `"personality"` (assuntos de capitalização); o conteúdo do texto da etiqueta cabe a você.

Boa sorte!
