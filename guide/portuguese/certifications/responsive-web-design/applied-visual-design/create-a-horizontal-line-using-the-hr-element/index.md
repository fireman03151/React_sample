## Criar uma linha horizontal usando o elemento hr

A tag `hr` é um dos elementos HTML mais simples. Cria uma linha de linha horizontal fina em todo o elemento pai.

Esta lição usa a tag `hr` para separar o título e o conteúdo de um "cartão". Então, como você cria um elemento `hr` ?

```html

<!-- It's this easy --> 
 <hr> 
```

Este elemento é de fechamento automático, mas para ser amigável com analisadores e verificadores de estilo, você deve adicionar um marcador de fechamento explícito.

```html

<hr /> 
```

Qualquer que seja o formato escolhido, você criará uma linha horizontal, como [nesta lição,](https://learn.freecodecamp.org/responsive-web-design/applied-visual-design/create-a-horizontal-line-using-the-hr-element/) no [freeCodeCamp](https://www.freecodecamp.org/) .
