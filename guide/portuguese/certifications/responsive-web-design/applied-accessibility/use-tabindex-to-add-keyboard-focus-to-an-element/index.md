## Use tabindex para adicionar foco do teclado a um elemento

Seguindo as instruções:

Adicione um atributo tabindex à tag p e defina seu valor como "0".

a linha 16 \`\` \`css

Instruções: Preencha TODAS as suas informações e clique em **Enviar**
```
becomes: 
 ```css 
    <p tabindex="0">Instructions: Fill in ALL your information then click <b>Submit</b></p> 
```

Dessa forma, ele receberá o foco do teclado quando um usuário navegar por uma página.
