## Evitar problemas de daltonismo usando o contraste suficiente

Seguindo as instruções:

Aumente o valor de luminosidade da cor de fundo de 35% para 55% e diminua o valor de luminosidade de cor de 20% para 15%.

as linhas 4 e 5 tornam-se:

```css
    color: hsl(0, 55%, 15%); 
    background-color: hsl(120, 25%, 55%); 
```

Isso melhora o contraste para 5.9: 1.
