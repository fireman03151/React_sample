## Tornar Links Navegáveis ​​com Chaves de Acesso HTML

Seguindo as instruções: Adicione um atributo accesskey aos dois links e defina o primeiro para "g" (para Garfield) e o segundo para "c" (para Chuck Norris). as linhas 8 e 16 tornam-se:

```css
<h2><a id="first" accesskey="g" href="">The Garfield Files: Lasagna as Training Fuel?</a></h2> 
```

```css
<h2><a id="second" accesskey="c" href="">Is Chuck Norris a Cat Person?</a></h2> 
```

Desta forma, os links em torno dos dois títulos de artigos do blog para ter atalhos de teclado para que os usuários do seu site possam navegar rapidamente para a história completa.
