## Alinhar elementos usando a propriedade de itens de alinhamento

Embora seja encorajador tentar as diferentes opções para `align-items` , a configuração é o valor para o `center` é a única maneira de passar esse desafio.

Ao jogar com as diferentes opções, você perceberá que `strech` é o valor padrão da propriedade.
