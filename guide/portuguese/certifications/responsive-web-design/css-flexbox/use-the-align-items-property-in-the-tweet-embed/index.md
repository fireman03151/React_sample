## Use a propriedade de itens de alinhamento no Tweet Incorporar

Semelhante ao [desafio anterior](https://github.com/freecodecamp/guides/tree/master/src/pages/certifications/responsive-web-design/css-flexbox/align-elements-using-the-align-items-property/index.md) , você precisará definir o valor da propriedade para `center` no seletor direito.

Note que pode parecer que nada foi mudado, mas mais uma vez, quanto mais explícito estiver o seu estilo no início, menos efeitos colaterais você verá quando o CSS ficar mais complexo.
