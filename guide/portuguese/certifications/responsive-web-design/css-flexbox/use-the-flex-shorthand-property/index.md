## Use a propriedade flexível de taquigrafia

Enquanto você pode definir propriedades `flex-grow` , `flex-shrink` e `flex-basis` individualmente. Se você precisar ver os valores para todos eles, você pode fazê-lo em uma linha usando o seguinte formato `flex: flex-grow flex-shrink flex-basis;` .

Exemplo:

```css
#box-1 { 
    background-color: dodgerblue; 
    flex: 2 2 150px; 
    height: 200px; 
  } 

```
