## Use a propriedade flex-direction para criar uma coluna

Para empilhar os elementos filhos do seu contêiner flexível um sobre o outro, você mudaria a _direção flexionada_ da seguinte maneira:

```CSS
#main-container { 
    display: flex; 
    flex-direction: column; 
 } 

```
