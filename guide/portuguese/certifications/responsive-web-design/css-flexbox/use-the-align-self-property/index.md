## Use a propriedade align-self

O principal objetivo deste desafio deve ser o fato de que o `vertical-align` `float` , `clear` e `vertical-align` não funciona em itens flexíveis. É por isso que temos o `align-self` propriedade flex que aceita os mesmos valores que `align-items` e terá precedência sobre quaisquer valores definidos pelo último.

Isso significa `align-self: center;` funcionaria enquanto `align-items: center;` não faria.
