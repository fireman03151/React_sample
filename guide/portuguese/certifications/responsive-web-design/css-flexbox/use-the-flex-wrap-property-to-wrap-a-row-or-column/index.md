## Use a propriedade flex-wrap para agrupar uma linha ou coluna

Definir a propriedade de `wrap` `flex-wrap` para `wrap` fará com que as colunas retornem à largura especificada e as colunas gree e preta sejam colocadas no início do quadro.

Observe que configurá-lo para `wrap-reverse` não fará com que o espaço vazio cinza do `#box-container` fique na parte inferior enquanto altera a ordem das colunas.
