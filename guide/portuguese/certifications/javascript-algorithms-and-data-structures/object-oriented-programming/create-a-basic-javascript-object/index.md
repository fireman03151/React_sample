## Crie um objeto JavaScript básico

### Método:

A alma deste problema é mais ou menos idêntica ao exemplo dado. Dê ao objeto `dog` duas novas propriedades - `name` e `numLegs` - e configure-as para uma string e um número, respectivamente.

### Solução:

```javascript
let dog = { 
  name: "George", 
  numLegs: 4 
 }; 

```
