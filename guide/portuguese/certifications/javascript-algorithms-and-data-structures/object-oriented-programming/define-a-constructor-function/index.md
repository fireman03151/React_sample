## Definir uma função de construtor

### Método:

A função `Dog()` deve ser escrita exatamente no mesmo formato da função `Bird()` fornecida no exemplo. Simplesmente substitua `Bird` por `Dog` para passar todos os casos de teste.

### Solução:

```javascript
function Dog() { 
  this.name = "Geogre", 
  this.color = "White", 
  this.numLegs = 4; 
 } 

```
