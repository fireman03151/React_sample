## Escreva JavaScript reutilizável com funções

As funções permitem reutilizar o código repetidamente. Nossa tarefa é fazer um `reusableFunction()` simples `reusableFunction()` que imprime "Hi World" para o console (que você pode alcançar com **Ctrl + Shift + I** ).

Você começa usando a palavra-chave `function` e digitando o nome da função (que segue a formatação do Case Camel). Em seguida, digite the `()` e crie os `{}` colchetes. Igual a:

```javascript
function reusableFunction() {} 
```

Agora, sua função está pronta para ser digitada. Use o `console.log()` para imprimir uma mensagem no console. A solução básica de código da seguinte forma:

```javascript
function reusableFunction() { 
    console.log("Hi World"); 
 } 

```
