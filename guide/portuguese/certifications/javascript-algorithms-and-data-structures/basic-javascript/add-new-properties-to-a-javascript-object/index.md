## Adicionar novas propriedades a um objeto JavaScript

Aqui está o exemplo:

```js
var ourDog = { 
  "name": "Camper", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["everything!"] 
 }; 
 
 ourDog.bark = "bow-wow"; 
```

Aqui está uma solução:

```js
var myDog = { 
  "name": "Happy Coder", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["freeCodeCamp Campers"] 
 }; 
 
 // Only change code below this line. 
 myDog.bark = "woof"; 

```
