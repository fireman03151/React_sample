## Manipular matrizes com push ()

O método `push()` permite acrescentar (adicionar ao final) um elemento a um array. Igual a…

```javascript
var arr = [1, 2, 3, 4]; 
 arr.push(5); // Now, the array is [1, 2, 3, 4, 5] 

```
