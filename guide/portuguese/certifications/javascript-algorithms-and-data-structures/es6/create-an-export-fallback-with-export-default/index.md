## Criar um retorno de exportação com padrão de exportação

O que é um valor de fallback? É basicamente um padrão para o qual o código retornará se nada tiver sido definido. Por exemplo, as variáveis ​​têm o valor de fallback padrão de `undefined` . Dito isto, uma dica para este desafio!

## Sugestão 1:

Basta adicionar `export default` ao início da função.

## Alerta de Spoiler - Solução à frente!

## Solução:

```javascript
"use strict"; 
 export default function subtract(x,y) {return x - y;} 

```
