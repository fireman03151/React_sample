## Use \* para importar tudo de um arquivo

Se você quiser importar tudo o que for possível de um arquivo, use a sintaxe `import * as _ from` as\_, fornecida pelo ES6. Isso é exatamente o que você está encarregado de fazer neste desafio!

## Sugestão 1:

Preencha os espaços em branco: `import * as objName from "file-name"` . Você pode ser criativo com seu `objName` , mas seu nome de arquivo deve ser `capitalize_strings` .

## Alerta de Spoiler - Solução à frente!

## Solução:

```javascript
"use strict"; 
 import * as str from "capitalize_strings"; 

```
