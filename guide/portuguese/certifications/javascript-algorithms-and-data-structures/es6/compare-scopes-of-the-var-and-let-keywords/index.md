## ![:beginner:](https://forum.freecodecamp.com/images/emoji/emoji_one/beginner.png?v=3 ":principiante:") Solução básica de código:

```javascript
    function checkScope() { 
      "use strict"; 
      let i = "function scope"; 
      if (true) { 
        let i = "block scope"; 
        console.log("Block scope i is: ", i); 
      } 
    console.log("Function scope i is: ", i); 
    return i; 
    } 
```
