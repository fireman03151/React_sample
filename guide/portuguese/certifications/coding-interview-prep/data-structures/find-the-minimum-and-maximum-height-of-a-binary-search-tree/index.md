## Encontre a altura mínima e máxima de uma árvore de pesquisa binária

Todo navegador moderno inclui um poderoso conjunto de ferramentas para desenvolvedores. Essas ferramentas fazem várias coisas, desde inspecionar HTML, CSS e JavaScript atualmente carregados até mostrar quais recursos a página solicitou e quanto tempo demoraram para carregar. Algumas das principais características estão listadas abaixo:

*   Acesso a um console do navegador
*   Testar viewports responsivas e específicas do dispositivo
*   Edite o DOM
*   Depuração
*   Analisar atividade de rede
*   Entenda os problemas de segurança

e muito mais…

## Como abrir ferramentas de desenvolvimento

### Teclado
```
Ctrl + Shift + I 
```

### Barra de menu

#### Raposa de fogo
```
Tools ➤ Web Developer ➤ Toggle Tools 
```

#### Cromo / cromo
```
Tools ➤ Developer Tools 
```

#### Safári
```
Develop ➤ Show Web Inspector. 
```

Se você não puder ver o menu Desenvolver, vá para `Safari ➤ Preferences ➤ Advanced` e marque a caixa de seleção `Show Develop menu` na barra de menus.

#### Ópera
```
Developer ➤ Web Inspector 
```

### Menu contextual

`Right-click` um item em uma página da Web (pressione Ctrl e clique no Mac) e escolha `Inspect Element` no menu de contexto exibido.

Este método imediatamente destaca o código do elemento em que você clicou com o botão direito do mouse.
