## Comando Bash: mv

**Move arquivos e pastas.**
```
mv source target 
 mv source ... directory 
```

O primeiro argumento é o arquivo que você deseja mover e o segundo é o local para movê-lo.

Opções comumente usadas:

*   `-f` forçar movê-los e sobrescrever arquivos sem verificar com o usuário.
*   `-i` para solicitar confirmação antes de sobrescrever arquivos.
