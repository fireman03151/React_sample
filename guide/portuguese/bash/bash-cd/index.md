## Comando Bash: cd

**Altere o diretório** para o caminho especificado, por exemplo, `cd projects` .

Existem alguns argumentos realmente úteis para ajudar isso:

*   `.` refere-se ao diretório atual, como `./projects`
*   `..` pode ser usado para subir uma pasta, usar `cd ..` e pode ser combinado para subir vários níveis `../../my_folder`
*   `/` é a raiz do seu sistema para acessar pastas principais, como `system` , `users` etc.
*   `~` é o diretório inicial, geralmente o caminho `/users/username` . Volte para as pastas referenciadas em relação a esse caminho, incluindo-o no início do caminho, por exemplo, `~/projects` .
