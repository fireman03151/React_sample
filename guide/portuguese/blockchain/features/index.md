## Recursos da tecnologia Blockchain

Blockchain é quase sempre usado em vez de Bitcoin e cryptocurrency. No entanto, existem muitos outros lugares em que essa tecnologia pode ser usada. E estamos começando a arranhar a superfície dela. Com todo o hype em torno disso, sabemos, a Blockchain Technology (BlockTech) vai ser enorme. Mas o que o torna único?

Neste artigo, vamos explorar os principais recursos característicos da BlockTech.

#### _\* Sistema descentralizado_

> Blockchain é uma tecnologia descentralizada, por design.

Quando algo é controlado por uma autoridade central, onde o poder de tomar decisões está nas mãos do ápice da administração, esse sistema é chamado de Sistema Centralizado. Os bancos, por exemplo, são um sistema centralizado, onde é responsabilidade do governador tomar decisões.

Pelo contrário, quando o poder é investido nas mãos das pessoas ou dos usuários, esse sistema é considerado um Sistema Descentralizado. A rede peer to peer, Torrent, por exemplo, é um sistema descentralizado, onde o usuário tem controle completo.

![Sistemas descentralizados](https://raw.githubusercontent.com/Vagisha16/Hw3/master/blockchain_article_pic.jpg)

#### _\* Ledger distribuído_

> Os blockchains usam a tecnologia Ledger distribuída (DLT) para armazenar e acessar os dados.

Quando algo é armazenado em um Razão Distribuído, várias cópias dele são feitas na rede ao mesmo tempo. Ao contrário dos bancos de dados tradicionais, o ledger distribuído não possui um banco de dados central ou funcionalidade de administração.

![Ledger distribuído](https://qph.fs.quoracdn.net/main-qimg-2e24c4949a63eefa9bbab1773e185cdd)

Quando aplicado em um sistema descentralizado como o Blockchain, cada usuário possui uma cópia do ledger e participa da verificação da transação. Isso dá à Blockchain a propriedade de Imutabilidade e garante segurança. Desde então, os dados são distribuídos, não há versão centralizada dos dados para os hackers para corromper. Os dados e os registros são públicos e facilmente verificáveis. Isso também elimina o ponto único de falha.

#### _\* Ecossistema Seguro (Hash Criptográfico)_

A BlockTech usa os conceitos como prova de trabalho e criptografia Hash para garantir segurança e imutabilidade. A prova de trabalho envolve várias pessoas em todo o mundo usando algoritmo computacional para tentar encontrar o valor de hash apropriado que satisfaz uma condição predefinida em relação ao valor de hash.

![Hashing](https://raw.githubusercontent.com/Vagisha16/Hw3/master/Hash.png)

![Prova de Trabalho](https://qph.fs.quoracdn.net/main-qimg-098a67b40e4d0f625cf2cbbda2c95df0)

#### _\* Mineração_

Torrent é uma rede descentralizada peer-to-peer usada para compartilhar arquivos. A BlockTech usa tecnologia similar. O que diferencia os usuários é que, em Torrent, o sistema conta com o código de honra dos usuários para semear os arquivos. Considerando que, no blockchain, os usuários envolvidos na transação possuem incentivos econômicos. Esses usuários são chamados de "mineiros". Os mineiros gastam seus recursos de computação para resolver os hashes criptográficos e garantir a imutabilidade e a confiabilidade da transação. Toda solução bem-sucedida (decodificação) garante algum benefício econômico.

![Mineração](https://raw.githubusercontent.com/Vagisha16/Hw3/master/5f6609014470f4b0122de37eb09dbfc7.jpg)

#### _\* Cronológica e Hora carimbada_

Os blockchains, idealmente, são apenas listas vinculadas muito sofisticadas, em que cada bloco é um repositório que armazena informações pertencentes a uma transação e também vincula-se ao bloco anterior na mesma transação. Esses blocos são organizados em uma ordem e são marcados com o tempo durante a criação para garantir um registro justo.

#### _\* Baseado em consenso_

Consensus Based é uma abordagem para tomada de decisão. É uma maneira criativa e dinâmica de chegar a um acordo entre todos os membros de um grupo. Uma transação no Blockchain pode ser executada somente se todas as partes na rede aprovarem por unanimidade. No entanto, está sujeito a alterações para se adequar a várias circunstâncias.
