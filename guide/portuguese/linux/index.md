# Linux
![](https://upload.wikimedia.org/wikipedia/commons/3/35/Tux.svg)

Linux é um nome que denota amplamente uma família de distribuições de sistemas operacionais de software livre e de código aberto construídas em torno do kernel Linux. O kernel Linux foi lançado pela primeira vez em 17 de setembro. 1991 por Linus Torvalds. Muitas distribuições Linux usam a palavra "Linux" em seu nome. A Free Software Foundation usa o nome GNU/Linux para se referir à família de sistemas operacionais, bem como distribuições específicas, e enfatizar que a maioria das distribuições Linux não são apenas o kernel, mas também inúmeros utilitários e bibliotecas, uma grande proporção dos quais são do projeto [GNU](https://www.gnu.org/) (GNU is Not UNIX).

O desenvolvimento do Linux é um dos exemplos mais importantes de colaboração de software de código aberto no mundo. O código-fonte pode ser usado, modificado e distribuído, comercialmente ou não, por qualquer pessoa sob os termos de suas respectivas licenças, tais como como a Licença Pública Geral GNU, ou [GPL](https://www.gnu.org/licenses/gpl-3.0.html).

O Linux foi iniciado como um clone do sistema operacional MINIX. Posteriormente, diversas pessoas foram adicionando outros componentes de código aberto ao kernel, e expandiram sua compatibilidade de hardware. O fornecimento de hardware e o licenciamento gratuito do Linux tornaram o Linux uma escolha popular para a computação de desktops e servidores no mercado. meados dos anos 90. Hoje, o Linux é um dos sistemas operacionais mais populares do mundo. De acordo com [este artigo da Wired](https://www.wired.com/2016/08/linux-took-web-now-taking-world/), era responsável por quase 90% dos servidores do mundo em 20016.  O Linux é a base do popular sistema operacional Android do Google.

Há também muitas versões do Linux projetadas para serem executadas em computadores pessoais, como o Debian ou o Ubuntu. Mais importante, estas diferentes versões do Linux (chamadas distribuições) permitem que o usuário varie graus de personalização e controle dos sistema. Isso significa que os usuários podem escolher suas distribuições Linux com base em seus desejos e necessidades.

Há também o Kali Linux, que é usado para testes avançados de penetração e auditoria. Tem mais de 700 ferramentas. Para poder usá-lo com eficiência, você precisará saber dominar bastante a linha de comando. Não é recomendado para iniciantes.

Distribuições populares do Linux incluem:

*   Ubuntu;
*   Mint Linux;
*   CentOS;
*   Fedora;
*   Red Hat Enterprise Linux;
*   Arch Linux.

Os tipos de pacotes são divididos entre distribuições de alto nível, com o Debian (.deb) e Red Hat (.rpm) sendo dois dos mais usados. No entanto, existem outros, incluindo Pacman (usado pelo Arch Linux) e o PetGet (Puppy Linux).

As distribuições Linux também vêm em todos os formatos e tamanhos e a maioria, se não todas, oferecem a capacidade de executar diretamente via CD / DVD no que é conhecido como um ambiente 'Live CD', ou até mesmo diretamente via USB, se seu computador suportar a inicialização via USB.
