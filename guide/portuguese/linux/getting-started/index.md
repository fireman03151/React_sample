## Escolhendo uma distribuição.

Existem vários tipos de distribuições Linux para escolher hoje em dia, escolher uma é uma grande preocupação no mundo atual do Linux. Levando em consideração os sistemas operacionais fáceis de usar, os melhores cães nessa categoria são: Ubuntu - Derivado do sistema Debian estável, o Ubuntu prospera em um ambiente fácil de usar e atualizado. Mint - Baseado no Ubuntu e Debian, este sistema operacional tem ótimos recursos como a compatibilidade com o PPA do Ubuntu, o gerenciador de pacotes apt. A base de distribuição de hortelã lembrar do Windows, isso pode ser uma boa característica para os usuários que vêm do Windows. Trisquel - Uma distribuição derivada do Ubuntu que se concentra em apenas utilizar software totalmente livre, evitando software proprietário. O Trisquel é endossado pela Free Software Foundation (FSF).

## Instalando.

A maioria das distribuições Linux são extremamente fáceis de instalar, todas as informações são mostradas entre cada etapa da instalação.

Um passo importante a ter em atenção é quando decidir onde instalar a sua nova distribuição Linux. Para os primeiros usuários, é melhor escolher a opção "Guiada", pois ela fará todo o trabalho para você.

Lembre-se de ler todos os passos com cuidado e deixar algum espaço no seu disco rígido para a nova distro, eu diria que cerca de 30Gb no mínimo é um bom ter.

## O terminal temido.

O terminal do Linux não é para ser temido, na verdade, é bastante fácil de usar com alguma prática e pode tornar a nossa tarefa diária muito automatizada.

No Debian / Ubuntu e derivados, o atalho para abrir o cli (Comman Line Interface) é "Ctrl + Alt + T". Vamos abrir o terminal e tentar alguns comandos.

cd (Change Directory) - O comando cd é um dos comandos que você mais usará na linha de comando do linux. Ele permite que você mude seu diretório de trabalho. Você o usa para se movimentar dentro da hierarquia do seu sistema de arquivos.

```shell
cd 
```

Usando apenas o comando cd, o diretório atual será alterado para o diretório inicial do usuário, localizado em "/ home / username", como em "/ home / mark".

ls (List) - Este comando lista o conteúdo no diretório atual. Também pode ser usado para listar informações de arquivos.

```shell
ls 
```

Agora podemos ver nossos diretórios em nossa casa.
