## Introdução

Apartir do Godot 3.0 existe suporte para 4 linguagens de programação, para permitir uma maior flexibilidade ao programador que desejar desenvolver jogos no Godot.  As linguagens principais do Godot são GDScript e VisualScript pois têm um nivel de integração muito superior dentro do Godot e tornam a experiência com o motor mais suave e podem ser usadas com o editor integrado do Godot, enquanto que para o C# e o C++ é necessário usar um editor á parte.
