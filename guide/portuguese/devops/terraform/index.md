## Terraform

Terraform é uma infraestrutura como software de código da HashiCorp. Permite que os usuários definam um datacenter infra-estrutura em uma linguagem de configuração de alto nível, a partir da qual ele pode criar um plano de execução para construir a infra-estrutura em um provedor de serviços, como o OpenStack ou em um provedor de serviços, como IBM Cloud (anteriormente Bluemix), AWS, Microsoft Azure ou Google Cloud Platform. Infraestrutura é definida em uma sintaxe HCL Terraform ou formato JSON.
