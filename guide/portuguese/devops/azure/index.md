## DevOps do Azure

O DevOps Services do Azure é um serviço de nuvem para colaboração no desenvolvimento de código. Ele fornece um conjunto integrado de recursos que você acessa através do seu navegador da web ou cliente IDE, incluindo o seguinte:

*   Repositórios Git para controle de código-fonte do seu código
*   Crie e libere o gerenciamento para oferecer suporte à integração contínua e à entrega de seus aplicativos
*   Ferramentas ágeis para apoiar o planejamento e acompanhamento de seu trabalho, defeitos de código e problemas usando os métodos Kanban e Scrum
*   Uma variedade de ferramentas para testar seus aplicativos, incluindo testes manuais / exploratórios, testes de carga e testes contínuos
*   Painéis altamente personalizáveis ​​para compartilhar progresso e tendências
*   Wiki integrado para compartilhar informações com sua equipe

Além disso, o ecossistema DevOps do Azure fornece suporte para adicionar extensões, integrando-se a outros serviços populares, como: Campfire, Slack, Trello, UserVoice e muito mais, e desenvolver suas próprias extensões personalizadas.

Escolha os Serviços DevOps do Azure quando desejar configuração rápida, operações livres de manutenção, colaboração fácil entre domínios, dimensionamento elástico e segurança sólida. Você também terá acesso ao teste de carga da nuvem, aos servidores de criação de nuvem e aos insights de aplicativos.
