## Expoentes

Um expoente é uma abreviação para o número de vezes que um número é multiplicado por ele mesmo. Muitas vezes é denotado com um sobrescrito, karat, ou com "ao poder de" tais como:

*   2 3
*   2 ^ 3
*   2 ao poder de 3

Neste exemplo, 3 é o expoente.

Para calcular o valor de 2 3 , você multiplicaria 2 para si mesmo 3 vezes: 2 \* 2 \* 2. Isso é 8.

Expoentes comuns têm nomes especiais:

*   O expoente de 2 é geralmente chamado de quadrado. Então, 3 2 é referido como 3 ao quadrado, avaliando para 9.
*   O expoente de 3 é geralmente chamado de cubo. Então, 2 3 é referido como 3 cubos, avaliando a 8.

### Expoentes Negativos

Os expoentes negativos são calculados de forma semelhante, exceto que o valor é colocado como denominador abaixo de um numerador de 1.

Por exemplo, 2 \-2 = 1 / (2 \* 2) = 1/4
