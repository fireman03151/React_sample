## A fórmula quadrática

Esta é uma fórmula simples que podemos obter resolvendo a representação básica de uma equação quadrática para x:
```
      ax^2 + bx + c = 0 
```

onde a, b, c são os marcadores de coeficiente (ou as constantes em uma equação real) e x é a variável para a qual o valor precisa ser encontrado.

Resolvendo para x, obtemos a fórmula quadrática como:
```
    x = (-b +- sqroot(b^2 - 4ac)) / (2a) 
```

Isto é representado de uma maneira mais **clara** aqui: ![Aqui](https://wikimedia.org/api/rest_v1/media/math/render/svg/2a9804ca8ce019507e3199ca8fced800fb5b7d7c)

### Implicações da fórmula em encontrar soluções:

Apenas de relance, podemos concluir algumas instruções para qualquer equação quadrática em Domínio de Número Real e Intervalo:

Considere a expressão sob a raiz quadrada "b ^ 2 - 4ac" como E

1.  Se E for positivo, então teremos 2 soluções para x (propriedade de quadrados)
2.  Se E é zero, então existe uma e apenas uma solução para x
3.  Se E é negativo, então não existe **uma** solução **real** para x

A fórmula quadrática é uma ferramenta para resolver equações quadráticas. Uma equação quadrática é uma equação polinomial de grau dois. Um polinômio de grau dois é apenas um polinômio em que o maior expoente de _x_ é 2. A seguir, são apresentados exemplos de equações quadráticas.

*   ![x^2-5x+6=0](https://github.com/jasonu/freecodecamp-images/blob/master/quadratic_integer_roots.png "exemplo equação quadrática")
*   ![x^2+x-1=0](https://github.com/jasonu/freecodecamp-images/blob/master/quadratic_irrational_roots.png "exemplo equação quadrática")

A fórmula aplica-se apenas às equações que têm a forma acima, onde um polinômio é igual a zero. Em geral, a fórmula se aplica a equações que têm o formato:

![ax^2+bx+c=0](https://github.com/jasonu/freecodecamp-images/blob/master/quadratic_equation.png "equação quadrática geral")

Onde _a_ , _b_ e _c_ são coeficientes do polinômio. Neste caso, a equação terá solução (ões):

![quadratic formula](https://github.com/jasonu/freecodecamp-images/blob/master/quadratic_formula.png "Fórmula quadrática")

#### Exemplo:

Suponha que você deseja encontrar as soluções para: ![x^2-5x+6=0](https://github.com/jasonu/freecodecamp-images/blob/master/quadratic_integer_roots.png "exemplo equação quadrática") então, conectando _a = 1, b = -5, c = 6_ na fórmula quadrática, obtemos:

*   _x = 2_ ,
*   _x = 3_

#### Exemplo:

A solução para: ![x^2+x-1=0](https://github.com/jasonu/freecodecamp-images/blob/master/quadratic_irrational_roots.png "exemplo equação quadrática") é obtido colocando _a = 1, b = 1, c = -1_ na fórmula quadrática. Isso dá duas soluções ou raízes irracionais:

*   _x = (- 1 + √5) / 2_ ,
*   _x = (- 1-√5) / 2_ .

A fórmula quadrática pode ser usada para encontrar a solução de qualquer equação quadrática, e usar o determinante pode determinar quantas soluções estão presentes. Outros métodos, como fatorar, representar graficamente ou completar o quadrado, encontram a (s) solução (ões) de uma equação quadrática, mas a fórmula Quadrática é muito útil nos casos em que você não pode fatorar ou representar graficamente.

Ao escrever a equação quadrática como:

![ax ^ 2 + bx + c = 0](https://wikimedia.org/api/rest_v1/media/math/render/svg/70a0e43dfc81e6fea3be4fc96895a8f9ec2966ac/)

(x é a variável enquanto a, b e c são constantes)

A fórmula quadrática é:

![x = -b +/- sqrt (b ^ 2 - 4ac) em todo o 2a](https://wikimedia.org/api/rest_v1/media/math/render/svg/2a9804ca8ce019507e3199ca8fced800fb5b7d7c/)

### Discriminante

O discriminante é tudo sob o radical na fórmula quadrática. ![b ^ 2 - 4ac](http://www.katesmathlessons.com/uploads/1/6/1/0/1610286/what-is-the-discriminant_orig.png/)

Se o discriminante = 0, então o quadrático só tem uma solução. Graficamente, isso representa o vértice sendo colocado no eixo x.

Se o discriminante é positivo (> 0), então o quadrático tem duas soluções ou raízes reais. Isto representa a quadrática que intersecta o eixo x em dois lugares.

Se o discriminante é negativo (<0), o quadrático não tem soluções reais (duas soluções imaginárias). Isso é porque você não pode tirar a raiz quadrada de um negativo. Grapically, isso representa a função não passando pelo eixo x.

Mais frequentemente do que não, você será obrigado a memorizar a fórmula quadrática. Aqui estão alguns dispositivos mnemônicos úteis:

Além disso, criar uma história para lembrar a fórmula quadrática ajuda. Por exemplo: O garoto negativo estava inseguro (mais ou menos) para ir à festa radical, mas porque ele era muito quadrado, ele perdeu quatro garotas incríveis. A festa foi toda em 2Am.

### Erros comuns:

Muita gente esquece a Ordem de Operações e subtrai 4 antes de multiplicá-la para a e c.

Além disso, o 2a está embaixo da coisa toda, não apenas da raiz quadrada.

Certifique-se de ter cuidado para não deixar cair a raiz quadrada ou o "mais / menos" no meio de seus cálculos.

Lembre-se que "b ^ 2" significa "o quadrado de TODOS os b, incluindo o seu sinal", portanto não deixe b ^ 2 ser negativo.
