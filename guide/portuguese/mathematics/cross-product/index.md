## Produto cruzado

Na teoria dos conjuntos, um produto cruzado de dois conjuntos é uma operação binária executada em conjuntos, que gera um conjunto de resultados. É denotado pelo símbolo X. Às vezes também é chamado como produto cartesiano após o matemático René Descartes.
```

 AXB = {(a, b) | a &isin; A and b &isin; B} 
```

### Exemplos
```
let us have, 
  set A = {a, b} 
 & set B = {1, 2} 
 then AXB = {(a, 1), (a, 2), 
               (b, 1), (b, 2)} 
```

### Usar

É usado para gerar todas as combinações possíveis depois de combinar duas tabelas em um banco de dados.
