## Adicionar Frações com Denominadores Diferentes

Para adicionar frações com denominadores diferentes, você precisa reescrever as frações para que todas tenham o mesmo denominador. Uma maneira de fazer isso é multiplicar todos os denominadores diferentes para criar um denominador comum.
