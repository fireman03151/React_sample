## Área de um retângulo

A área de um retângulo pode ser calculada multiplicando-se os comprimentos dos dois lados perpendiculares (comprimento e altura).

`Area = a x b`

**Por exemplo** Para um retângulo onde _a = 3_e b = 4_

`Area = a x b = 3 x 4 = 12`
