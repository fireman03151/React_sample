## Factoring em álgebra

Factoring é como você toma um quadrático na forma padrão e o coloca em forma fatorada. A forma padrão para um quadrático é normalmente mostrada como ax 2 + bx + c, onde aeb são coeficientes e c é uma constante. A forma fatorada é normalmente mostrada como (x - d) (x - t) onde d e t são ambos coeficientes.

## Factoring em sua cabeça

Esse método de fatoração geralmente funciona melhor quando a = 1. Para fazer isso quando a = 1, você olha para o que dois números adicionam para igual a b, enquanto também multiplica para c. Esses dois números serão iguais a d e t.

#### Exemplo 1:

x 2 + 2x + 1 Aqui você tem que procurar dois números que se multiplicam para 1 e adicionar a 2. Esses dois números são 1 e 1. Então você coloca esses números na forma fatorada assim: (x + 1) (x + 1)

#### Exemplo 2:

x 2 + 7x + 12 Aqui você tem que procurar dois números que se multiplicam para 12 e somam 7. Esses dois números são 3 e 4. Então você coloca esses números na forma fatorada assim: (x + 7) (x + 4)
