## Regras de Integração

Assim como na diferenciação, usamos uma variedade de regras ao integrar funções. Abaixo estão alguns dos mais comuns.

**A Regra da Constante:** Dk dx = kx + C onde k é uma constante

**A Regra de Poder:** ∫ x n dx = + x n + 1 ⁄ n + 1 + C quando n ≠ 1

**Regra Exponencial:** ∫ e kx dx = 1 ⁄ kx e kx + C onde k é uma constante

**Regras Trigométricas:**

∫ cos (x) dx = sen (x) + C

∫ sen (x) dx = -cos (x) + C

**Regras de soma / diferença:**

∫ \[f (x) + g (x)\] dx = ∫f (x) dx + ∫g (x) dx

∫ \[f (x) - g (x)\] dx = ∫f (x) dx - ∫g (x) dx
