## Significar

A medida mais comum de tendência central, ou um valor usado para descrever um conjunto de números, é a média aritmética, ou média. Para encontrar a média ( _A_ ) de um conjunto de números, adicione seus valores e divida pelo número de valores ( _n_ ).

![Fórmula Média](https://wikimedia.org/api/rest_v1/media/math/render/svg/90330653b40adf032ea8e144f84d7eec1a88054d)

### Exemplos

*   **1, 4, 5, 2, 3**
    
    (1 + 4 + 5 + 2 +3) / 5 = 3
    
*   **20, 14, 23**
    
    (20 + 14 + 23) / 3 = 19
    
*   **\-12, -20, -16, -21**
    
    (-12, -20, -16, -21) / 4 = -17,25 \\
    

A média pode ser uma ótima maneira de descrever um conjunto de números. No entanto, é importante perceber que um valor que é muito maior ou menor que os outros valores, referido como um outlier, no conjunto pode fazer com que a média represente incorretamente o conjunto de números. Por exemplo, se você tiver um conjunto de números composto principalmente de números pequenos que contenham um número grande, a média será maior que a maioria dos números, retratando erroneamente o conjunto de números como sendo maior do que eles realmente são.

### Exemplos

*   **2, 7, -3, 1, 0, 2, 857**
    
    (2 + 7 + (-3) + 1 + 0 + 2 + 857) / 7 = 123,71
    
*   **1276, 1141, 1271, 1053, 1**
    
    (1276 + 1141 + 1271 + 1053 + 1) / 5 = 948,4
