## Sistema operacional BSD

![Sistema operacional BSD, mais notavelmente o freeBSD: o poder de servir](https://www.freebsd.org/layout/images/logo-red.png)

### Visão geral

O Berkeley Software Distribution (BSD) é um grupo de sistemas operacionais (SO) semelhantes a Unix de código-fonte aberto, com origens em versões anteriores do Research Unix no Bell Labs. O FreeBSD é o membro mais popular.

O BSD é configurado para hospedagem na Internet, hospedagem na web e hospedagem de vários servidores em um único sistema. É o primeiro sistema operacional que adicionou um protocolo de internet. Os sistemas operacionais BSD possuem um sistema de compartilhamento de tempo fortemente aplicado, o que os torna mais úteis quando os recursos são compartilhados entre os processos. Como comparação, o sistema operacional Linux é conhecido por ser preferido para processos de tarefa única, como supercomputadores e desktops. A parte efetiva do tempo de interrupção forçada multitarefa do BSD atrapalha os processos únicos dedicados.

O BSD inclui um sistema de 'Jails' que é um pouco análogo ao Linux Containers - exceto com segurança adicional e flexibilidade na implementação.
