## Ferramentas de Data Science

Nesta seção, teremos guias para uma ampla variedade de ferramentas usadas pelos cientistas de dados.

Os cientistas de dados são curiosos e frequentemente procuram novas ferramentas que os ajudem a encontrar respostas. Eles também precisam ser proficientes no uso das ferramentas do comércio, embora existam dezenas e dezenas delas. Em geral, os cientistas de dados devem ter um conhecimento prático de linguagens de programação estatística para a construção de sistemas de processamento de dados, bancos de dados e ferramentas de visualização. Muitos no campo também consideram um conhecimento de programação parte integrante da ciência de dados; entretanto, nem todos os estudantes de cientista de dados estudam programação, portanto, é útil estar ciente das ferramentas que contornam a programação e incluem uma interface gráfica amigável para que o conhecimento dos algoritmos dos cientistas de dados seja suficiente para ajudá-los a criar modelos preditivos.

R continua sendo a principal ferramenta, com 49% de participação, mas o Python cresce mais rápido e quase alcança o R. O RapidMiner continua sendo a plataforma mais popular da Data Science. Ferramentas de Big Data usadas por quase 40% e o uso de Deep Learning duplica. Data Science é OSEMN ( **O** btain, **S** crub, **M** odel, i **N** terpret) os Dados.
