Remover todos os seus desafios armazenados localmente resolverá muitos problemas relacionados ao travamento do navegador no FreeCodeCamp

No Chrome:

*   No [freecodecamp](https://freecodecamp.com) abra seu console
    *   Windows: `Ctrl` + `Shift` + `J`
    *   Mac OS: `Cmd` + `Opt` + `J`
*   Vá para a guia de recursos (chrome).
    *   Clique no link "Local Storage" na barra de navegação à direita.
*   Exclua todas as entradas no lado direito ou execute este comando no console do seu navegador para limpar todas as entradas do localStorage: `localStorage.clear();`
*   Veja se isso resolve seu problema

Alternativamente, se você está tendo problemas com um desafio específico para congelar seu navegador, [encontre o nome desse desafio no armazenamento local e exclua apenas esse](http://forum.freecodecamp.com/t/clear-specific-values-from-your-browser-local-storage/19128) .
