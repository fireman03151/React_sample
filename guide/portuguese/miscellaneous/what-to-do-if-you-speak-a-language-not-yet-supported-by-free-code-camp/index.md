---
title: What to Do if You Speak a Language not Yet Supported by Free Code Camp
localeTitle: O que fazer se você falar um idioma ainda não suportado pelo Free Code Camp
---
Tradução é uma proposta de tudo ou nada.

Não poderemos adicionar novos idiomas ao Free Code Camp até que todos os nossos  
desafios são traduzidos para esse idioma.

Além de traduzi-las inicialmente, também precisaremos manter  
a tradução como os desafios são gradualmente atualizados (usamos o  
sistema de emissão do github para manter registro).

Se você puder nos ajudar, junte-se ao [canal do gitter](https://gitter.im/FreeCodeCamp/Translators)  
e leia o [Guia de Tradução](http://forum.freecodecamp.com/t/guidelines-for-translating-free-code-camp-to-any-language/19111)
