[O Node.js](https://nodejs.org/) é um tempo de execução de JavaScript criado no [mecanismo JavaScript V8 do Chrome,](https://developers.google.com/v8/ "Motor V8 do Chrome") permitindo que o JavaScript seja usado para programação no lado do servidor.

O Node.js tornou-se uma parte essencial do 'JavaScript Everywhere', incentivando o desenvolvimento de aplicativos da Web a se unificar em torno de uma única linguagem de programação, em vez de depender de uma linguagem diferente para escrever scripts do lado do servidor.
