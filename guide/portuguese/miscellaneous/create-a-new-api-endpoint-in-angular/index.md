---
title: Create a New Api Endpoint in Angular
localeTitle: Criar um novo ponto final de API em angular
---
> > yo angular-fullstack: endpoint whatsit

Usando essa linha, você obtém o Yeoman para gerar automaticamente outro endpoint da API e um novo tipo de coleção para o seu banco de dados. Agora temos _coisas_ assim como _whatsits_ ! Sinta-se à vontade para abrir **/server/api/whatsit/whatsit.model.js** e definir o seu whatsit como quiser.
