---
title: What You Will Learn at Free Code Camp in Order
localeTitle: O que você aprenderá no Free Code Camp in Order
---
*   [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) - A estrutura das páginas da web.
*   [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) - O estilo visual das páginas da web.
*   [Bootstrap](http://getbootstrap.com/) - Uma ferramenta de "design responsivo" que ajuda seus sites a ficarem ótimos em tablets e telefones.
*   [jQuery](https://jquery.com/) e [Ajax](https://developer.mozilla.org/en-US/docs/AJAX) - ferramentas fáceis para controlar o conteúdo no navegador.
*   [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - A única linguagem de programação usada por todos os navegadores da web.
*   [Algoritmos](https://en.wikipedia.org/wiki/Algorithm) - Receitas passo-a-passo para fazer as coisas.
*   [Git](https://git-scm.com/) - Um sistema de controle de versão para salvar e compartilhar seus projetos.
*   [MongoDB](https://www.mongodb.org/) - Um banco de dados não-relacional popular.
*   [D3.js](http://d3js.org/) - Uma biblioteca JavaScript para produzir visualizações de dados dinâmicas e interativas em navegadores da web.
*   [Express.js](http://expressjs.com/) - Um poderoso framework de desenvolvimento web.
*   [Node.js](https://nodejs.org/) - Uma plataforma para criar aplicativos de rede escalonáveis.
*   [SASS](http://sass-lang.com/) - Uma linguagem de script que é interpretada em folhas de estilo em cascata (CSS).
*   [Desenvolvimento de APIs](https://en.wikipedia.org/wiki/Application_programming_interface) - Um conjunto de rotinas, protocolos e ferramentas para criar software e aplicativos.
*   [Ágil](https://en.wikipedia.org/wiki/Agile_software_development) - Um conjunto de princípios de desenvolvimento de software que focaliza o design e a produção de um projeto nas necessidades de seus usuários
