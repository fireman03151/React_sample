title: Why Free Code Camp Left Slack
localeTitle: Por que o Free Code Camp deixou folga
---
Slack é um serviço popular de sala de bate-papo que o Free Code Camp superou

![Estatísticas de uso do Free Code Camp's Slack](//discourse-user-assets.s3.amazonaws.com/original/2X/d/da98adf7a036ae5bd8410b09c64f394d014694a0.png)

Por mais que tenhamos gostado de usar esse serviço, é um ajuste inadequado para grandes comunidades de código aberto. Dito isto, muitas comunidades de código aberto ainda usam o Slack, e o Slack não faz nada para desencorajar essa prática.

Se você está curioso, pode ler sobre [as falsas propagandas do Slack e o desastre que isso causou em junho de 2015](http://blog.freecodecamp.com/2015/06/so-yeah-we-tried-slack-and-we-deeply-regretted-it.html) .

O Free Code Camp agora usa o [Gitter](https://github.com/FreeCodeCamp/freecodecamp/wiki/Gitter) .
