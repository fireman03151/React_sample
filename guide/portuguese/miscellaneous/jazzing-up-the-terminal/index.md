Fora da caixa, o terminal enviado com o Ubuntu é um pouco sem graça. Esta seção permitirá que você se torne um usuário avançado ![:muscle:](//forum.freecodecamp.com/images/emoji/emoji_one/muscle.png?v=2 ":músculo:") .

## Ferramentas:

#### o Exterminador do Futuro

[O Terminator](https://launchpad.net/terminator) permite que você organize vários terminais em uma interface parecida com uma grade.  
Para instalar o Terminator, digite `sudo apt-get install terminator` no terminal.

#### Oh meu ZSH!

Pré-requisitos:

*   `git` deve ser instalado

Para instalar o `ZSH` e o `Oh My ZSH!` use os seguintes comandos:
```
sudo apt install zsh && chsh -s $(which zsh) 
```

> Nota: você precisará sair e entrar novamente para usar o ZSH em vez de bash como seu shell padrão.
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)" 
```

Depois de reiniciar seu terminal, `Oh My ZSH!` deve ser instalado.

Finalize a [Documentação Oficial](https://github.com/robbyrussell/oh-my-zsh/wiki) para aprender como instalar plugins e temas.
