# O Gitter é um sistema de sala de bate-papo popular com projetos de código aberto.

![Uma captura de tela do Gitter](//discourse-user-assets.s3.amazonaws.com/original/2X/9/901b6ff18a83b746b1b7e96cb6907440d531d977.png)

[A Gitter.im](https://gitter.im) é uma startup em rápido crescimento, sediada em Londres.

O Free Code Camp tem centenas de quartos Gitter. Estes são acessíveis através do site da Gitter, bem como [aplicativos de desktop e móveis](https://gitter.im/apps) .

Você pode ver [todos os nossos quartos](https://github.com/FreeCodeCamp/FreeCodeCamp/wiki/Official-Free-Code-Camp-Chat-Rooms) e juntar-se aos que você gostaria.

Gitter permite que você compartilhe [códigos formatados](https://github.com/FreeCodeCamp/freecodecamp/wiki/Pasting-Code-in-Gitter) com outros campistas.
