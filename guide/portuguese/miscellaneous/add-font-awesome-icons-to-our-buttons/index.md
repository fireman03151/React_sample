## Adicionar ícones impressionantes da fonte aos nossos botões

Font Awesome é uma conveniente biblioteca de ícones. Esses ícones são gráficos vetoriais, armazenados no formato de arquivo `.svg` . Esses ícones são tratados como fontes. Você pode especificar seu tamanho usando pixels e eles assumirão o tamanho da fonte de seus elementos HTML pai.

Aqui está como você adiciona um ícone:

```html

<i class="fa fa-thumbs-up"></i> 
```

Aqui está como você iria inserir esse ícone em um botão:

```html

<button class="btn btn-block btn-primary"><i class="fa fa-thumbs-up"></i> Like</button> 
```
