## Mozilla Developer Network (MDN)

[O MDN](https://developer.mozilla.org/) é uma plataforma de aprendizado para tecnologias da Web e o software que impulsiona a Web.

O objetivo do MDN é fornecer aos desenvolvedores as informações necessárias para criar facilmente projetos na Web aberta. Se é uma tecnologia aberta exposta à Web, o MDN quer documentá-la.
