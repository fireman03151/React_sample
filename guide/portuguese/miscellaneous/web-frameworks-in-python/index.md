title: Web Frameworks in Python
localeTitle: Estruturas da Web em Python
---
Como a maioria dos domínios no desenvolvimento do Python, a comunidade Python trata o desenvolvimento da Web com muito carinho e nos oferece estruturas amadurecidas e bem mantidas; de complexidade variada.

Temos micro-frameworks como [Flask](http://flask.pocoo.org/) , [Bottle](http://bottlepy.org/docs/dev/index.html) . Também temos frameworks como o [Django](https://www.djangoproject.com/) , que vem com [baterias incluídas](https://github.com/rosarior/awesome-django) . Você pode encontrar uma lista completa de frameworks web, escritos em Python, [aqui](https://en.wikipedia.org/wiki/Comparison_of_web_frameworks#Python) .
