## SQL

SQL significa Linguagem de Consulta Estruturada. É a ferramenta mais comum usada para manipular e gerenciar dados em um banco de dados relacional (geralmente chamado de "banco de dados SQL").

SQL é comumente pronunciado "siːkwəl", a mesma pronúncia de *sequel* em inglês, ou mais comumente; SQL em português. Suas variantes mais populares são MySQL, PostgreSQL e SQLite - uma versão do SQL que é comumente usada para prototipagem. Introduziu o conceito de acessar muitos registros com um único comando, usando consultas SQL.
