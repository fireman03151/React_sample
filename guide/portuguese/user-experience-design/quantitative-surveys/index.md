## Pesquisas Quantitativas

**Pesquisas quantitativas** analisam a abordagem estatística da análise dos resultados da pesquisa. As perguntas são projetadas de maneira que as respostas sejam registradas em números. Então você olha para quantos usuários responderam certas questões.

#### Pesquisas Quantitativas nos falam sobre

*   Quantos usuários usam o produto?
*   Que tipo de pessoas usam o produto?
*   Qual é o mercado para o nosso produto?

É uma ótima abordagem para testar o mercado, ver tendências, padrões, base de usuários, ver o sucesso do produto etc.

#### Exemplos de perguntas

*   Você usa o campo de código livre?
    *   sim
    *   Não
*   Quantos desafios você terminou no campo de código livre?
    *   Menos de 100
    *   Menos do que 500
    *   Mais de 500
*   Quantos certificados você ganhou no campo de código livre no total?
    *   0
    *   1
    *   2
    *   3
