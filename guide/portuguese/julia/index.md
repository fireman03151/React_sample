## O que é Julia?

[Julia](https://julialang.org) é uma linguagem de programação de alto desempenho, opcionalmente voltada para computação científica e numérica. Possui bibliotecas para álgebra linear, otimização numérica, processamento de sinais e aprendizado de máquina, entre outros. Ele usa vários dispatch e possui um backend do LLVM e um compilador JIT. Julia é selecionada para ser usada no desenvolvimento do [sistema de prevenção de colisão aerotransportada de última geração, responsável](https://juliacomputing.com/case-studies/lincoln-labs.html) pela segurança [.](https://juliacomputing.com/case-studies/lincoln-labs.html)
