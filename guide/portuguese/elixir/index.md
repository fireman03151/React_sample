## Elixir

Elixir é uma linguagem de programação dinâmica e funcional criada para rodar na máquina virtual Erlang (BEAM). O Elixir é conhecido por ser uma linguagem concorrente e de propósito geral que é frequentemente aproveitada com o poderoso framework Phoenix. Esta linguagem foi criada por José Valim e tem sido influenciada por linguagens como Ruby, Erlang e Closure.
