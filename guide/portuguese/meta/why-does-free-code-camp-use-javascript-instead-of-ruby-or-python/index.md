---
title: Why Does Free Code Camp Use JavaScript Instead of Ruby or Python
localeTitle: Por que o Free Code Camp usa JavaScript em vez de Ruby ou Python?
---
Como o JavaScript, o Ruby e o Python são linguagens de script de alto nível que podem ser usadas para o desenvolvimento da Web com pilha completa.

Mas mesmo que você aprendesse esses idiomas, ainda precisaria aprender JavaScript. Isso porque o JavaScript é a única linguagem que é executada nos navegadores da web. O JavaScript existe há 20 anos e ainda está crescendo em popularidade.

Por causa disso, o JavaScript tem mais ferramentas e recursos de aprendizado online do que qualquer outro idioma.
