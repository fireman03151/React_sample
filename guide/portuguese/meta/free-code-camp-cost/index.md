title: Free Code Camp Cost
localeTitle: Custo do Camp Code Livre
---
Sempre seremos totalmente gratuitos para estudantes e organizações sem fins lucrativos.

Nosso nome é Free Code Camp. Nós somos um campo de código livre. Se você tivesse que nos pagar (ou assinar os ganhos futuros), teríamos que mudar nosso nome. E nós não vamos fazer isso.

Todos os que trabalham em nossa comunidade e nossos projetos de código aberto são voluntários.

Pretendemos, eventualmente, cobrir nossos custos operacionais, ganhando bônus de colocação de trabalho de empresas que contratam nossos graduados. Note-se que não vamos realmente garnish nenhum salário de nossos alunos - os empregadores já estão pagando milhares de dólares para encontrar candidatos bem sucedidos. Os empregadores simplesmente pagarão o bônus de recrutamento ao Acampamento de Código Livre em vez de pagar um recrutador.
