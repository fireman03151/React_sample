## Projeto Invisível na Internet

Este é um esboço. [Ajude nossa comunidade a expandi-lo](https://github.com/freecodecamp/guides/tree/master/src/pages/security/i2p/index.md) .

[Este guia de estilo rápido ajudará a garantir que sua solicitação de recebimento seja aceita](https://github.com/freecodecamp/guides/blob/master/README.md) .

O Invisible Internet Project (I2P) nasceu em 2003. Um grupo de hackers, desenvolvedores e arquitetos de software cujo objetivo principal era criar uma rede virtual privada, resistente a censura.
