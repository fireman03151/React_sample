## VPNs:
VPN (Virtual Private Network, ou, Rede Virtual Privada) é uma tecnologia que permite navegar na rede anonimamente e de uma forma segura. Essa tecnologia cria um caminho chamado túnel entre dois hosts usando diversos tipos de protocolos criptográficos, como SSL/TLS. Usuários podem trocar dados como se estivessem conectados diretamente (ninguém mais poderá ver os detalhes da conexão). VPN também pode ser usado para dar a volta, fazer um contorno por bloqueios geográficos ou censuras. VPN pode ser considerado uma alternativa para o TOR.

Exemplo de VPNs: OpenVPN, NordVPN, etc.
