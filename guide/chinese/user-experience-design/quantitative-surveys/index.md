## 定量调查

**定量调查**着眼于分析调查结果的统计方法。问题的设计方式使得答案以数字记录。然后你看看有多少用户回答了某些问题。

#### 定量调查告诉我们

*   有多少用户使用该产品？
*   什么样的人使用该产品？
*   我们的产品市场是什么？

这是一个测试市场的好方法，看趋势，模式，用户群，看到产品的成功等。

#### 示例问题

*   你使用Free code camp吗？
    *   是
    *   没有
*   你在免费代码阵营中完成了多少挑战？
    *   不到100
    *   不到500
    *   超过500
*   你在免费代码营中获得了多少张证书？
    *   0
    *   1
    *   2
    *   3
