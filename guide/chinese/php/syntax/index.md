## 句法

PHP是一种允许您在其他静态网页中包含动态内容的语言。它是用html内联编写的，只需将.html文件重命名为.php（web-server必须安装php）才能开始使用。

```PHP
<!DOCTYPE html> 
 <html> 
 <head> 
 <title>PHP Example</title> 
 </head> 
 <body> 
 <h1>PHP Example</h1> 
 <?php 
  echo "Hello World from PHP!"; 
 ?> 
 </body> 
 </html> 
```
