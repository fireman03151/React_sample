# 一次性学习

人类学习新概念时几乎不需要重复 - 例如，孩子可以概括这个概念 一本书中的一张“猴子”，但我们最好的深度学习系统需要数百或者一本 成千上万的例子可以掌握任何对象，甚至可以达到相当高的准确度。这激发了我们感兴趣的环境：“一次性”学习，其中 包括从单个（或很少）标记的示例中学习一个类。

一次性学习有各种方法，如[相似度函数](https://www.coursera.org/lecture/convolutional-neural-networks/one-shot-learning-gjckG) ， [贝叶斯的概率定理](https://www.youtube.com/watch?v=FIjy3lV_KJU) ，DeepMind使用One-Shot学习方法提出了它自己的神经网络版本！
