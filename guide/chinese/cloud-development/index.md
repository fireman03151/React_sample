## 云开发

云开发是在共享资源池中对基础架构组件使用软件工程实践。这可以通过多种方式在您的公司或组织自己的数据中心内完成，也可以在由第三方拥有和运营的“公共”共享数据中心内完成。

公共云提供商的热门选择是Amazon Web Services，Microsoft Azure和Google Compute Engine。这些商运营提供大规模数据中心集群，以满足成千上万的个人，公司和组织对资源的需求。操作云涉及为不同用户的使用，分区这些共享资源，跟踪资源的消耗（也称为计量），然后根据所消耗的各种计算，网络和存储资源的不同财务方案向用户收费。

据说许多现代网站和大数据工作都是在云中开发的，这意味着它们在一些大型共享数据中心中运行。
