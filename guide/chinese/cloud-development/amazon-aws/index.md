## 亚马逊网络服务

Amazon Web Services（简称AWS）是Amazon.com的子公司，为个人，公司和政府提供按需云计算平台。 付费订阅基础，免费等级选项可用12个月。它与微软的Azure和谷歌的云平台一起被认为是三大主流 公共云供应商。

该技术使订户可以随时使用通过互联网提供的完整虚拟计算机集群。 AWS的版本 虚拟计算机具有真实计算机的大部分属性，包括硬件（用于处理的CPU和GPU），本地/ RAM存储器，硬盘/ SSD存储器;一个选择 操作系统;联网;和预加载的应用程序软件，如Web服务器，数据库，CRM等。每个AWS系统还虚拟化其控制台I / O（键盘， 显示和鼠标），允许AWS用户使用现代浏览器从世界上任何地方连接到他们的AWS系统。浏览器充当了一个窗口 虚拟计算机，让订户登录，配置和使用他们的虚拟系统，就像他们真实的物理计算机一样。

订户可以选择部署他们的AWS系统，以便为他们自己和客户的利益提供基于互联网的服务。自个人或组织以来 使用AWS物理上不拥有AWS基础架构，他们不必处理与拥有物理数据中心相关的一些维护任务，例如 操作网络设备。

特别是AWS也在扩展他们的云产品。他们提供创新的服务，例如AWS Lambda，用户只需关注这些服务 编写代码而不是在将运行其代码的服务器上（产生新的_无服务器_运动）。由于这些好处，云计算 诸如AWS之类的平台在创业公司中很受欢迎，这些公司最初可能没有足够的资金投资于他们的信息技术或IT。相反，他们可以专注 通过AWS的随需应变模型，为用户构建优质产品并随着时间的推移发展业务。

流行的AWS服务包括：

*   通过EC2（弹性计算云）平台的虚拟机
*   数据库包括SQL和NoSQL
*   S3（简单存储服务）
*   SQS（简单队列服务）
*   通过Amazon Lumberyard游戏开发引擎和后续托管平台开发游戏。
*   DynamoDB（亚马逊Dynamo数据库）
*   CloudWatch（指标和警报）
*   CodeDeploy（部署）
*   Lambda（无服务器）
*   应用监控工具
*   移动开发工具和平台
*   亚马逊Alexa开发区

#### AWS认证

AWS为其从业者提供了许多不同的认证。 AWS证书和基于角色的认证有不同的层次。 等级是：

*   基础性
*   关联
*   专业的 每个角色都有这三个层次。

这些角色包括：

*   建筑师
*   开发人员
*   操作

AWS还提供与任何人相关的先决条件AWS Cloud Practitioner Essentials认证\*，无论他们的角色如何。 除标准角色外，还有高级网络，大数据和安全性的专业认证。
