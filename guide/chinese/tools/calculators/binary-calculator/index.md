## 二进制计算器

二进制计算器用于预先形成两个二进制数的加法，减法，乘法或除法。二进制计算器还可以将二进制值转换为十进制值，反之亦然。

例如： 10101010 + 11001100 = 0101110110（二进制）或374（十进制）

或者，10101010 x 11001100 = 01000011101111000（二进制）或34680（十进制）
