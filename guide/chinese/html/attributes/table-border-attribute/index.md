## 表边框属性

`<table>`标记边框属性是一种将边框排序为HTML表格的方法。支持的值为`0` （无边框）和`1` （边框）。 HTML 5中不支持border属性，即最新版本的HTML。建议使用CSS在HTML 5中为表添加边框。
