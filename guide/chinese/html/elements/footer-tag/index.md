## 页脚标记

`<footer>`标记表示HTML文档或部分的页脚。通常，页脚包含有关作者，版权信息，联系信息和站点地图的信息。 `<footer>`标记内的任何联系信息都应该放在`<address>`标记内。

### 例

```html

<html> 
 <head> 
  <title>Paragraph example</title> 
 </head> 
 <body> 
  <footer> 
    <p>&copy; 2017 Joe Smith</p> 
  </footer> 
 </body> 
 </html> 
```
