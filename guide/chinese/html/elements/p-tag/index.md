## P标签

`<p>`标记代表段落，这是用于在`<html>`文档中创建单词行的最常用标记。使用`<p>`在元素之间自动添加边距非常有用。 `<p>`属性也可以用于CSS样式。

### P标签与其他标签

`<p>`的使用与其他标签兼容，允许添加超链接（ `<a>` ），粗体（ `<strong>` ），斜体（ `<i>` ）等。

### 例

```html

<html> 
  <head> 
    <title>Paragraph example</title> 
  </head> 
  <body> 
    <p> 
      This <strong>sentence</strong> was created to show how the paragraph works. 
    </p> 
  </body> 
 </html> 
```

您还可以在段落中嵌套锚元素`<a>` 。

### 例

```html

<p>Here's a 
  <a href="http://freecodecamp.com">link to Free Code Camp.com</a> 
  for you to follow.</p> 
```
