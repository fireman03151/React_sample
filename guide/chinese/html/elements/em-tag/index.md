## Em Tag

`<em>`标记用于强调HTML文档中的文本。这可以通过将您想要强调的文本包装在`<em>`标记中来完成。大多数浏览器会将包含在`<em>`标记中的文本呈现为斜体。

注意： `<em>`标记不应用于样式化斜体文本。 `<em>`标签用于强调文本。

### 例：
```
<body> 
  <p> 
    Text that requires emphasis should go <em>here</em>. 
  </p> 
 </body> 
```
