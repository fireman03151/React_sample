## Td Tag

`<td>`标签定义HTML表格中的标准单元格。

HTML表有两种单元格：

*   标题单元格 - 包含标题信息（使用`<th>`元素创建）
*   标准单元格 - 包含数据（使用`<td>`元素创建）

`<th>`元素中的文本默认为粗体并居中。 `<td>`元素中的文本是常规的，默认情况下是左对齐的。

### 例

```html

<html> 
 <head> 
 <title>Td Tag Example</title> 
 </head> 
 <body> 
 <table> 
  <tr> 
    <th>Header1</th> 
    <th>Header2</th> 
  </tr> 
  <tr> 
    <td>Cell A</td> 
    <td>Cell B</td> 
  </tr> 
 </table> 
 </body> 
 </html> 

```
