## 评论标签

注释是用于在文档中留下注释的标签。只有开发人员才能访问评论，最终用户看不到，除非他们使用浏览器的Inspector元素进行检查。

当您使用某些代码时，将提示留给其他开发人员以了解其工作内容会很有帮助。

### 例

```html

<!-- This is a comment --> 
```

注释还可用于使代码处于非活动状态，而无需完全删除它。

### 例

```html

<!-- 
 <h1>Hello Campers</h1> 
 --> 
 <h2>Hello Friends</h2> 
 <!-- 
 <p>Hello Paragraph</p> 
 --> 

```
