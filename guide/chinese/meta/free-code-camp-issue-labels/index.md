localeTitle: 免费代码阵营问题标签

在本文中，您可以找到用于对提交给FCC的各种问题进行分类的标签（请参阅下表）以及说明（正在进行的工作）。标签范围从**错误**到**问题**到**课程要求** 。

您可以使用标签来帮助您选择要点击的问题，而不是随机点击以找到您感兴趣的问题。当然，您可以处理并提交没有标签的问题的PR。但是，如果您选择了一个问题，请务必在提交任何拉取请求之前阅读[“贡献指南”](https://github.com/FreeCodeCamp/FreeCodeCamp/blob/staging/CONTRIBUTING.md) 。

注意：标签**_被阻止_** ， **_确认_**和**_QA_**也与[Waffle](https://waffle.io/freecodecamp/freecodecamp) （FCC（GitHub）问题的管理系统）绑定，并由它自动添加和删除。

**标签：**说明

**可访问性：**该问题不一定会破坏网站，但会使其更难使用（例如，在您键入时用于评估的测试，这会导致整个屏幕闪烁）。该标签还标志着该网站对视力障碍或其他残疾人的可用性有所改善。

**帐户：**露营者的FCC帐户存在问题，例如登录问题，丢失进度或缺少某些解决方案的个人资料页面。

**beta：**该问题与FCC的测试版网站有关。

**自行车脱落：**请求是微不足道的或基本上与网站的目的无关。换句话说，那些没有实际后果的问题，但人们坚持开支的时间。

**阻止：**在等待创建者的响应或更多信息时，或者因为问题陈旧时不应该处理的问题。

**bug：** FCC代码中确认且可重现的错误，不仅发生在Internet Explorer中。

**chrome：**该问题只能在Google Chrome中重现。

**已确认：**已报告的问题已经过验证。 （它基本上说问题创建者已成功传达了如何复制问题，但没有人开始努力解决问题。）

**课程要求：**要求添加到当前课程中的任何内容 - 例如，请求新的挑战，为挑战添加更多解释等。

**特定于设备：**该问题仅适用于特定设备（智能手机，平板电脑，台式机等）。

**讨论：**此问题目前正处于讨论阶段。

**重复：**此问题重复另一个问题，很可能会被关闭。

**仅限第一次使用者：**这个问题并不重要，但营员们应该很容易在他们的前几个PR中修复。将这些视为露营者成为贡献者的门户。

**增强功能：**与功能请求类似，但更多与改进现有功能相关，而不是添加新功能。

**功能请求：**向FCC添加**功能的请求** 。例如，添加从露营者的个人资料页面编辑或删除算法的功能的请求。

**firefox：**该问题只能在Mozilla Firefox中重现。

**需要帮助：**业主或问题主持人需要营员帮助调查或解决问题。

**加息：**该问题与**加息** （目前课程中没有）有关。

**ie / edge：**该问题只能在Internet Explorer或Microsoft Edge中重现。 FCC官方仅支持Google Chrome。

**正在进行中：**有人正在解决此问题。

**必须尽快开始：**需要**尽快**审查/解决此问题。

**路线图：**通常，计划在将来的更新中实施或将在不久的将来实施的功能请求。

**质量保证:(**质量保证）已提交修复程序，需要在合并之前进行审核。

**问题：**问题是一个问题。

**重新解决：**问题与转向反应有关。

**已解决：**问题已解决/已解决。

**测试：**应该通过测试改进来解决问题，这意味着有人需要修复断言。

**todo：**将来需要做的事情。

**翻译：**该问题是翻译请求或与已实施的翻译相关。

**ux :(**用户体验）类似于可访问性，但更少关于某些事情是否难以使用，而不是关于露营者遇到导致负面用户体验的问题。例如，当某些内容可能过于复杂或难以理解某个功能的使用时，会使用此标记。

**wiki：**与FCC的GitHub Wiki文章有关。可能是创建一个请求。

**wontfix：**问题不可修复或不会在不久的将来修复。例如，添加对MS Edge浏览器的支持。
