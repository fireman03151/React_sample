我们的2015年10月3日峰会将于美国东部时间周六中午在[twitch.tv频道](http://twitch.tv/freecodecamp)开始，之后将在我们的[YouTube频道播出](https://www.youtube.com/channel/UC8butISFwT-Wl7EV0hUK0BQ?sub_confirmation=1) 。

在这1小时的直播中，我们将：

*   推出免费的前端和全栈开发证书课程
*   演示我们新的，更全面的投资组合页面
*   演示一种新的方式，我们所有的营员都可以立即开始帮助非营利组织
*   与露营者Blake Johnston交谈，他创建了Free Code Camp将用于我们的完整堆栈后端项目挑战的Clementine.js样板
*   在teespring.com上推出我们的首个T恤广告系列
