---
title: Why Does Free Code Camp Use JavaScript Instead of Ruby or Python
localeTitle: 为什么免费代码营使用JavaScript而不是Ruby或Python
---
与JavaScript一样，Ruby和Python是高级脚本语言，可用于完整的堆栈Web开发。

但即使您学习了这些语言，您仍然需要学习JavaScript。那是因为JavaScript是在Web浏览器中运行的唯一语言。 JavaScript已经存在了20年，并且它仍然越来越受欢迎。

因此，JavaScript拥有比任何其他语言更多的工具和在线学习资源。
