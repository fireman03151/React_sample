免费代码营学习小组代表了我们社区的本地化体验，它们完全由像您这样致力于学习如何编码和帮助其他人的人来运行和组织。他们的Facebook网站是组织咖啡和代码风格活动，在黑客会话中本地配对计划或分享您所在地区其他编码相关事件的新闻的地方。他们的目的是在15-20分钟内聚集人。

无论您是居住在大都市区还是小城镇，我们都鼓励您加入或创建自己的小组，以满足其他忙于学习编码的人。
