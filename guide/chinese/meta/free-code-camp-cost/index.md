---
title: Free Code Camp Cost
localeTitle: 免费代码营成本
---
对于学生和非营利组织，我们将永远免费。

我们的名字是免费代码营。我们是一个免费的代码阵营。如果您必须向我们付款（或签署未来收入），我们必须更改我们的名称。我们不打算这样做。

在我们的社区和我们的开源项目上工作的每个人都是志愿者。

我们计划通过从雇用我们毕业生的公司获得工作安置奖金来最终支付我们的运营成本。请注意，我们实际上不会从毕业生那里收取任何工资 - 雇主已经向招聘人员支付了数千美元来找到成功的候选人。雇主只需向Free Code Camp支付招聘奖金，而不是向招聘人员付款。
