## Bash命令：mv

**移动文件和文件夹。**
```
mv source target 
 mv source ... directory 
```

第一个参数是要移动的文件，第二个参数是将其移动到的位置。

**更改档名。**
```
mv old_name new_name
```

第一个参数是原来的名字，第二个参数是要改成的名字

常用选项：

*   `-f`强制移动它们并覆盖文件而不用与用户核对。
*   `-i`在覆盖文件之前提示确认。
