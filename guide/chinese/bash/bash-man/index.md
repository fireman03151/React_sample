Man， **man** ual的缩写，是一个bash命令，用于显示给定命令的在线参考手册。

Man显示给定命令的相关手册**页** （ **man** ual **页面的**缩写）。

### 用法

```bash
man [options] [command] 
```

最常用的选项：

*   `-f` ，打印给定命令的简短描述
*   `-a` ，连续显示手册中包含的所有可用的介绍手册页

### 例

显示ls的手册页：

```bash
man ls 
```
