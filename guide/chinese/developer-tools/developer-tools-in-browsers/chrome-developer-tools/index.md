## Chrome Firefox开发工具

Chrome和Firefox Developer工具可帮助Web开发人员分析网页上的HTML，CSS和Javascript。它们可用于调试，优化代码，检查元素等等。

**特征：**

*   **页面检查器：**查看和编辑页面内容和布局。
*   **Web控制台：**请参阅javascript console.log（）消息和错误
*   **JavaScript调试器：**查看和修改页面上运行的JavaScript
*   **网络监视器：**查看加载页面时发出的网络请求
*   **性能工具：**分析站点使用的计算机资源

### 访问开发工具

1.  在屏幕顶部的菜单栏中，选择View> Developer> Developer Tools。这些工具将显示在底部，侧面或弹出窗口中。
2.  在网页上右键单击元素。出现菜单时，选择“检查”。
