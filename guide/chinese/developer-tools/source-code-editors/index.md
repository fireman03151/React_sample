## 源代码编辑

源代码编辑器是允许代码创建的程序  
编辑。任何文本编辑器都可用于编写代码。但专用代码  
编辑提供了许多有用的功能，包括：文本完成，  
标记着色，以及用于并排编辑的选项卡/窗格。这些功能可以  
即使对那些不了解所有人的新用户也很有用  
高级选项。

### 选择一个编辑器

有许多编辑可供下载或使用  
在线，有偿和免费。它们提供了许多相同的核心功能，但是  
每个都有自己的细微差别。进一步的研究可能有助于找到那个  
适合你。以下编辑器都是跨平台的  
免费使用或评估。

### 崇高的文字

![崇高的文本图标](https://logo.clearbit.com/sublimetext.com)

Sublime Text是一个非常受欢迎的编辑器， [自2008年以来](https://www.sublimetext.com/blog/articles/one-point-oh)一直存在。有许多选项和[扩展](https://packagecontrol.io/)可用，很多[可用的帮助下](http://stackoverflow.com/search?q=sublime+text)在网上帮[其设置](http://www.websightdesigns.com/wiki/Setting_up_Sublime_Text_3) 。 （继续使用的许可费用为70美元。但是，Sublime Text可以免费下载和评估，并带有唠叨屏幕弹出窗口。）

[sublimetext.com](https://www.sublimetext.com/)

![Sublime Text截图](//discourse-user-assets.s3.amazonaws.com/original/2X/6/63f774c31a7b1a439bcef9d11e95463b6998310e.png)

### 括号

![括号图标](https://logo.clearbit.com/brackets.io)

Brackets是Adobe的一个相对较新的开源编辑器。它非常用户友好，特别是对于不习惯命令行界面或JSON样式设置/优先级的人。通过Extension Manager可以快速轻松地查找和安装扩展和主题。

[brackets.io](http://brackets.io/)

![Brackets截图](//discourse-user-assets.s3.amazonaws.com/original/2X/2/2f395a58d24ed551ed8440a10ae640777f1143b6.png)

### 原子

![原子图标](https://logo.clearbit.com/atom.io)

Atom是一个由[GitHub](https://github.com/)开发的[开源](https://github.com/atom/atom)编辑器。像Sublime Text一样，Atom很受欢迎。它是非常可配置的（正如他们所说的那样“可攻击”），但高级功能对于新用户来说可能是压倒性的。网上还有大量[有用的](http://readwrite.com/2014/05/20/github-atom-5-tips-getting-started-tutorial-corey-johnson/) [文档](http://flight-manual.atom.io/) [可供使用](http://stackoverflow.com/search?q=atom) 。

[atom.io](https://atom.io/)

![Atom截图](//discourse-user-assets.s3.amazonaws.com/original/2X/8/8016e7c0937e8ce6fcb6104c9aee0026443912f5.png)

### Visual Studio代码

![Visual Studio代码图标](https://logo.clearbit.com/code.visualstudio.com)

Visual Studio Code（又名vsc或vscode）是一个非常流行的开源编辑器，由[Microsoft](https://www.microsoft.com)开发，并迅速被所有语言的开发人员广泛使用。与Atom和Sublime一样，Visual Studio Code提供了广泛的[扩展](https://marketplace.visualstudio.com/VSCode)和[主题，](https://marketplace.visualstudio.com/search?target=VSCode&category=Themes&sortBy=Downloads)可帮助您定制和完善使用它的体验，但VSC是高度可配置的“开箱即用”。 VSCode背后还有一个非常活跃的开发团队，每个月Visual Studio Code团队[都会](https://code.visualstudio.com/updates)添加新功能和错误修复。

[code.visualstudio.com](https://code.visualstudio.com/)

### 记事本+ +

![记事本++图标](https://upload.wikimedia.org/wikipedia/commons/0/0f/Notepad%2B%2B_Logo.png)

Notepad ++是一个免费的开源代码编辑器，可在Microsoft Windows环境中运行。它由Dan Ho于2003年开发。它是用[C ++](www.cplusplus.com)编写的（名称来自于此）。 Notepad ++支持选项卡式编辑，允许在单个窗口中处理多个打开的文件，也可以用作纯文本编辑器。

[notepad-plus-plus.org](https://notepad-plus-plus.org)
