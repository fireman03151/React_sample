## 类型

类型选择器按节点（标记）名称选择元素。

#### 代码语法

```css
* { 
    css declarations; 
 } 
```

#### 代码示例

```css
h1 { 
    background-color: red; 
 } 
```

选择所有h1元素并将背景颜色设置为红色。

```css
div { 
    width: 100px; 
 } 
```

选择所有div元素并设置元素宽度100px。
