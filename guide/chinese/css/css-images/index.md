## CSS图像

这有助于向网站添加图像。

#### 代码示例

`<img src="picture.jpg" alt="Picture" width="100" height="100">`

*   **src：**它包含所需图像的路径值
*   **alt：**如果未显示图像，则可以使用alt属性显示备用文本。
*   **width：**指定图像的宽度（可以是百分比或px或auto）
*   **height：**指定图像的高度（可以是百分比或px或auto）
