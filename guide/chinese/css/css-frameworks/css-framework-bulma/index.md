# CSS框架Bulma

Bulma是一个基于[Flexbox](http://forum.freecodecamp.com/t/css-flexbox-tips-and-tricks)的现代CSS框架，使用[Sass](http://sass-lang.com/) （SCSS）进行高度自定义。

## 入门

这是一个简单的HTML模板，其中包含Bulma库的最新编译和缩小的CSS。

```html

    <!DOCTYPE html> 
    <html lang="en"> 
      <head> 
        <meta charset="utf-8"> 
        <meta name="viewport" content="width=device-width, initial-scale=1"> 
        <title>Bulma Template</title> 
        <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css"> 
        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/bulma/0.0.23/css/bulma.min.css"> 
      </head> 
      <body> 
        <h1>Hello World</h1> 
        <!-- Add all HTML Code here --> 
      </body> 
    </html> 
```

我们在此示例中使用了CDN，但您可以[在此处查看](http://bulma.io/documentation/overview/start/)安装Bulma的其他方法。
