## CSS3颜色

CSS中的颜色用于着色网页中的元素。有许多方法可以为元素指定颜色。您可以使用颜色的实际名称，RGB值或十六进制值。在CSS3中，hsl（色调饱和度 - 亮度）已添加到规范中。

### 颜色名称

目前，HTML支持140种颜色名称，只需键入其名称即可在CSS规则中指定。例如：

### 句法
```
p { 
  color: green; 
 } 
```

绿色的元素。 有关完整的140色列表，请访问：https：//www.w3schools.com/colors/colors\_names.asp

### RGB值

RGB代表“红色”，“绿色”和“蓝色”，我们还可以通过在规则中键入RGB值来指定颜色。 RGB值如下所示：rgb（255,0,0），其中每个数字定义最终混合中每种颜色的多少。

值的范围从0到255，在我们的示例中，我们看到只有第一个值是255而其他两个设置为0.这意味着我们的值中只有红色，相应的元素将显示为红色。 RGB值（0,0,0）将给出黑色，值（255,255,255）将给出白色。

不可能尝试记住每个颜色代码，因此有很多工具可以在线选择您想要的项目颜色。例如：https：//www.w3schools.com/colors/colors\_picker.asp或http://htmlcolorcodes.com/color-picker/。

```css
p { 
  color: rgb(0, 255, 0); 
 } 
```

此规则将所有p元素的字体颜色更改为绿色，就像上面一样。

### 十六进制值

十六进制值是另一种在CSS中定义颜色的方法，它们与RGB值的工作方式非常相似。

一个随机的十六进制代码是这样的： `#29432b` ，其中散列后面的前两个字符表示混合中的RED量，后两个代表绿色量，后两个代表蓝色量。

`#000000`和`#ffffff`的值分别代表黑色和白色。 您可以使用为RGB值提到的相同工具找到所需的特定十六进制颜色。

### 句法
```
p { 
  color: #00fe00; 
 } 
```

此规则再次将所有p元素的字体颜色更改为绿色。

### HSL

HSL有三个值。 第一个是**Hue** ，以度为单位。所以0（或360）表示红色，120表示绿色，240表示蓝色。 第二个是**饱和度** ，其百分比值范围为0到100％。 第三个是**Lightness** ，其百分比值也在0到100％之间。 0％为深黑色，50％为平均值，100％为白色。

### 句法
```
p { 
  color: hsl(0, 100%, 50%); 
 } 
```

### 为什么要使用RGB或HEX值？

颜色名称只有140个值，而RGB和HEX值有16,777,216种可能的组合。因此，如果您希望项目看起来完全符合您的设想，则应使用后两个选项并保留颜色名称值以用于模型和测试目的。
