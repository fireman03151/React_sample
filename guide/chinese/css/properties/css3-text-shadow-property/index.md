## CSS3文本阴影属性

文本阴影属性为文本添加阴影。 每个阴影都是原始元素的X和Y偏移，模糊半径和颜色的组合。 \`\`\`css Css语法

text-shadow：X偏移Y偏移模糊半径颜色
```
**Example :** 
```

CSS p { text-shadow：2px 2px 8px＃FF0000; } \`\`\`
