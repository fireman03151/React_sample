## 字体样式属性

`font-style`属性允许您更改字体样式，这意味着您可以使字体变为斜体。此属性接受以下值：

*   `normal` - 默认值。
*   `italic` - 字体通过选择字体系列中分类为斜体的字体来显示斜体。
*   `oblique` - 在其字体系列中被分类为倾斜的字体被使用。

`Italic`和`oblique`可能看起来相似。据[MDN](https://developer.mozilla.org/en-US/) _倾斜，只是倾斜版的常规脸_ 。  
您还可以在提供`oblique`值时指定倾斜度字体的角度。如果未提供角度，则默认使用14度。
