## 字体重量属性

使用`font-weight`属性可以设置更高或更低的`font-weight` 。可用于`font-weight`的值是`normal` ， `bold` ， `lighter` ， `bolder`和numeric值。

```css
  font-weight: lighter; 
  font-weight: bolder; 
  font-weight: bold; 
  font-weight: light; 
  font-weight: 100; 
  font-weight: 900; 
```

较粗和粗的值可增加字体和光线的重量，减轻重量。
