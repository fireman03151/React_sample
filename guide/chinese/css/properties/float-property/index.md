## 浮动物业

`float`属性允许元素浮动到另一个元素的左侧或右侧。 `float`的主要用途是在文本块的左侧或右侧显示图像。

```css
.my-element { 
  float: left; 
  float: right; 
  float: inline-start; 
  float: inline-end; 
  float: none; 
 } 
```
