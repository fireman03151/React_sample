title: Add Free Code Camp in Linkedin
localeTitle: 在Linkedin中添加免费代码营
---
LinkedIn将Free Code Camp视为一所大学。

您可以[在这里](https://www.linkedin.com/edu/school?id=166029)探索我们的校友活动。

您应该将Free Code Camp添加到您的LinkedIn个人资料。这是如何做：

*   点击此链接： [添加免费代码营](https://www.linkedin.com/profile/edit-education?school=Free+Code+Camp)
*   请遵循以下说明： ![](//discourse-user-assets.s3.amazonaws.com/original/2X/8/8b138a334f9cfc075fee7379b515a173a154934b.png)

一旦您完成我们的课程（或完成至少一个非营利项目，然后获得编码工作），我们将向您发放官方的Full Stack Web Development证书，您也可以将其添加到您的LinkedIn个人资料中。
