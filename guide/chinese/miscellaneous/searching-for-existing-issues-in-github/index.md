---
title: Searching for Existing Issues in Github
localeTitle: 在Github中搜索现有问题
---
如果在获得Gitter帮助后仍然看到问题，您将需要尝试查看是否有其他人发布了类似问题。

![gif遍历后续步骤以搜索GitHub的问题](//discourse-user-assets.s3.amazonaws.com/original/2X/3/3577718dd9fe14fbe80b203bc3cc56cdb0d9c3af.gif)

1.  转到FreeCodeCamp的[Github问题](https://github.com/FreeCodeCamp/FreeCodeCamp/issues)页面。
    
2.  使用搜索栏搜索可能与您的问题相关的已归档问题。
    
    *   如果找到一个，请阅读！您可以通过单击侧栏中的“ `Subscribe`来订阅以获取有关该特定问题的更新。如果您要添加某些内容，也可以对此问题发表评论。
        
    *   如果您找不到任何相关问题，您应该创建一个新的Github问题 。
