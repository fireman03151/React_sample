## 解决使FreeCodeCamp.com变得如此缓慢的瓶颈

*   我们的开源社区正在快速增长，本周我们的网站在高峰时段已经放慢了速度。
*   我们经历过，发现了几个瓶颈并修复了它们。免费代码营现在应该比以往更快。
*   非常感谢[@berkeleytrue](https://gitter.im/berkeleytrue)领导这项工作，以及帮助质量保证这些改进的许多营员。

## 介绍我们的新Wiki

*   我们的wiki现在在FreeCodeCamp.com上（而不是我们的主项目的GitHub Wiki）。
*   我们的维基像我们的聊天和地图一样弹出。我们欢迎有关此新功能的反馈。热键，上下文感知维基文章和其他功能即将推出。
*   非常感谢[@saintpeter](https://gitter.im/saintpeter) ， [@ hallaathrad](https://gitter.im/hallaathrad)和[@awesomeaniruddh](https://gitter.im/awesomeaniruddh)以及其他许多露营者为此付出的努力！

## 打开我们的商店

*   我们有一个新的T恤运动一直持续到2月15日。快点购买其中一个非常棒的免费代码营T恤： [点击这里查看我们的美国活动](https://teespring.com/free-code-camp-feb) | [点击这里查看我们的欧盟活动](https://teespring.com/free-code-camp-t-shirt-eu-shop)
*   我们创建了一个新的商店页面。它现在只链接到我们的T恤运动，但我们很快就会有笔记本电脑贴纸和其他很酷的装备。

## 退休露营新闻

*   在过去的一年里，Camper News取得了一定的成功。但我们的subreddit也是如此，它允许评论，审核和许多其他功能。所以我们正在转向Reddit进行新闻分享。
*   如果您还没有[加入我们的subreddit，](https://www.reddit.com/r/freecodecamp)我们建议您创建一个Reddit帐户。您现在可以与我们整个社区分享链接，以及您的想法和问题。
*   我们将保留Camper News一段时间，但我们将从导航菜单中删除它的链接，并删除引入它的挑战。

## 更新挑战

*   许多露营者要求我们在“构建个人投资组合网页”之前创建一个更容易的，以HTML为重点的项目。所以我们创建了一个新项目，只需使用名为“Build a Tribute Page”的HTML和Bootstrap即可完成。
*   我们还对我们的前端项目进行了重新排序。以下是他们现在的结构：
    *   基本前端项目
        *   建立致敬页面
        *   建立个人投资组合网页
    *   中级前端项目
        *   建立一个随机报价机
        *   显示当地天气
        *   构建Wikipedia查看器
        *   使用Twitch.tv JSON API
    *   先进的前端项目
        *   构建JavaScript计算器
        *   建立一个番茄钟
        *   建立一个Tic Tac Toe游戏
        *   建立一个西蒙游戏
*   你会注意到，自从我们退休Camper News以来，我们已经删除了“建立一个Camper News Viewer”项目。如果您之前已完成该挑战，则应在地图上为您标记“建立致敬页面”。
*   此外，对于“构建Tic Tac Toe游戏”，您不再需要使其无与伦比。 （如果你愿意，你当然可以 - [这就是方法](http://neverstopbuilding.com/minimax) 。虽然这样做很有用，但编写人工智能与项目的重点不同，即前端开发。
*   我们添加了三个新的可选挑战，帮助您加入我们：
    *   [版（Subreddit）](https://reddit.com/r/freecodecamp)
    *   [中等出版物](https://medium.freecodecamp.com)
    *   [Twitch.tv频道](https://twitch.tv/freecodecamp)
