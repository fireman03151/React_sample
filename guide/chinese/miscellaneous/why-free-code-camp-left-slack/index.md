title: Why Free Code Camp Left Slack
localeTitle: 为什么免费代码营左边松弛
---
Slack是Free Code Camp的一个流行的聊天室服务

![免费Code Camp的Slack使用情况统计](//discourse-user-assets.s3.amazonaws.com/original/2X/d/da98adf7a036ae5bd8410b09c64f394d014694a0.png)

尽管我们喜欢使用这项服务，但它很适合大型开源社区。这就是说，许多开源社区仍然使用Slack，而Slack并没有采取任何措施阻止这种做法。

如果你很好奇，你可以阅读[Slack的虚假广告以及2015年6月引发的失败](http://blog.freecodecamp.com/2015/06/so-yeah-we-tried-slack-and-we-deeply-regretted-it.html) 。

免费代码营现在使用[Gitter](https://github.com/FreeCodeCamp/freecodecamp/wiki/Gitter) 。
