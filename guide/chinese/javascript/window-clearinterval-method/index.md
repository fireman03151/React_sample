## Window clearInterval方法

'clearInterval（）'方法用于清除使用'setInterval（）'方法设置的计时器。

```js
    clearInterval(setInteval_ID); 
```

为了能够使用'clearInterval（）'方法，必须使用全局变量。

```js
    myID = setInterval(function, milliseconds); 
```
