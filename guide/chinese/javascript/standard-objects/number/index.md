## 数

`Number` Javacript对象表示任何类型的数字。 JavaScript不区分不同类型的数字，例如整数或浮点数。它们都以统一的64位浮点格式存储，符合国际IEEE 754标准1 。

新的`Number`对象是这样创建的： `var num = new Number(value)`尽管最常见的是它们可以通过为变量赋值来创建： `var num = 1.616;` 。在非构造函数上下文中（即，没有`new`运算符）， `Number`可用于执行类型转换。如果参数无法转换为数字，则返回[`NaN`](https://guide.freecodecamp.org/javascript/standard-objects/number/number-nan) 。
