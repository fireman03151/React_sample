## Date.prototype.getFullYear

Date.getFullYear（）方法根据本地时间返回指定日期的年份（四位数）。

### 句法

```javascript
dateObj.getFullYear() 
```

### 例

```javascript
var date = new Date(); 
 // creates a new Date() object with current date and time. 
 date.getFullYear() 
 // 2018 
```
