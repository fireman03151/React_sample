## 基本日志规则和扩展日志表达式

以下是常用的基本日志规则：

###### 对数乘积规则

log b （x \* y）= log b （x）+ log b （y）

###### 对数商数规则

log b （x / y）= log b （x） - log b （y）

###### 对数幂规则

log b （x y ）= y \* log b （x）

###### 负数的对数

当x <0时， log b （x）未定义

###### 对数为0

log b （0）未定义

lim x→0+ log b （x）= -infinity

###### 对数为1

log b （1）= 0

###### 基数的对数

log b （b）= 1

###### 无限的对数

lim x→无穷远 log b （x）=无穷大

###### 对数基本切换规则

log b （c）= 1 / log c （b）

###### 对数基数变化规则

log b （x）= log c （x）/ log c （b）
