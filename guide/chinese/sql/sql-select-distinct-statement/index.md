### 获取学习领域列表

```sql
SELECT DISTINCT programOfStudy FROM student; 
```

```text
+------------------+ 
 | programOfStudy   | 
 +------------------+ 
 | Literature       | 
 | Programming      | 
 | Computer Science | 
 +------------------+ 
 3 rows in set (0.00 sec) 
```
