## 使用CSS网格单位更改列和行的大小

此挑战要求您将容器列的宽度设置为质询描述中指定的宽度。

### 暗示

更改`grid-template-columns`属性。

### 解

由于挑战要求您将宽度设置为`1fr` ， `100px`和`2fr` ，因此`2fr`的`css grid-template-columns`属性`.container`为：

```css
grid-template-columns: 1fr 100px 2fr; 
```
