## 使用grid-template-rows添加行

在此挑战中，您需要在`<div class="container">`为CSS网格定义行。

### 暗示

您可以使用`grid-template-rows` CSS属性定义行。

### 解

由于挑战要求您创建两行，每行`50px` ，请在`.container` CSS代码块中声明以下内容：

```css
grid-template-rows: 50px 50px; 

```
