## 使用Tweet Embed中的align-items属性

与上[一个挑战](https://github.com/freecodecamp/guides/tree/master/src/pages/certifications/responsive-web-design/css-flexbox/align-elements-using-the-align-items-property/index.md)类似，您需要将属性值设置为右侧选择器的`center` 。

请注意，它可能看起来没有任何改变，但再一次，你的风格在早期就越明确，当你的CSS变得更复杂时你会看到的副作用越少。
