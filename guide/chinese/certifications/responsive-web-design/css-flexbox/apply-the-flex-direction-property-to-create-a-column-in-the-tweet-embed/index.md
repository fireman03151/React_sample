## 应用flex-direction属性在Tweet Embed中创建一个列

通过将属性显式添加到css，将flex方向从其默认行更改为将使得句柄显示在名称旁边。

```css
header .profile-name { 
    display: flex; 
    flex-direction: column; 
    margin-left: 10px; 
  } 

```
