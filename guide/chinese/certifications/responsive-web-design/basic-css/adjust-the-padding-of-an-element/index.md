## 调整元素的填充

填充是元素的内容和边界之间的间距。

•顶部，底部，左侧和右侧之间有间距。

•的间距是由被写为像素单位增加或减少的数值`px` 。

要调整元素的填充，请使用：

```css
padding: 10px; 
```

要将Padding创建为特定Element的类，请使用：

```css
.example-Padding { 
  padding: 10px 
 { 
```

要将此类应用于适当的字段，请使用：

```html

<div class="example-Padding"> 
 <p> Example Text </p> 
 </div> 
```

**要调整元素的填充** ，请使用：

填充： `10`像素;
