## 创建项目符号无序列表

要通过挑战，您应该执行的第一个操作是删除包含其所有内容的`p`元素。

之后你必须实现列表：它必须在`ul`元素中包含至少三个`li`元素，并且这些`li`必须在同一级别上，而不是彼此嵌套：

正确：
```
<ul> 
  <li></li> 
  <li></li> 
 </ul> 
```

不正确：
```
<ul> 
  <li> 
    <li> 
    </li> 
  </li> 
 </ul> 
```
