## 找到二叉搜索树的最小和最大高度

每个现代Web浏览器都包含一套强大的开发人员工具。这些工具可以执行各种操作，从检查当前加载的HTML，CSS和JavaScript到显示页面请求的资产以及加载时间。下面列出了一些主要功能：

*   访问浏览器控制台
*   测试响应和设备特定的视口
*   编辑DOM
*   调试
*   分析网络活动
*   了解安全问题

以及更多…

## 如何打开开发工具

### 键盘
```
Ctrl + Shift + I 
```

### 菜单栏

#### 火狐
```
Tools ➤ Web Developer ➤ Toggle Tools 
```

#### 铬/铬
```
Tools ➤ Developer Tools 
```

#### 苹果浏览器
```
Develop ➤ Show Web Inspector. 
```

如果您看不到“开发”菜单，请转到 `Safari ➤ Preferences ➤ Advanced` 并选中菜单栏中的`Show Develop menu`复选框。

#### 歌剧
```
Developer ➤ Web Inspector 
```

### 上下文菜单

`Right-click`网页上的项目（在Mac上按住Ctrl键并单击），然后从出现的上下文菜单中选择“ `Inspect Element` ”。

此方法直接突出显示右键单击元素的代码。
