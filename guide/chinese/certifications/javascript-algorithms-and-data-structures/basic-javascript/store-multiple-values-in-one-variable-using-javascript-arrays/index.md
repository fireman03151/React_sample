## 使用JavaScript数组在一个变量中存储多个值

数组允许您在单个变量中存储大量不同的数据。您所要做的就是将数据放入数组中，用逗号分隔。像这样......

```javascript
var arr = ["Code for change", 123]; 
```

请记住，数组被定义为一行。您将不同的数据存储在单个“行”（变量）中。
