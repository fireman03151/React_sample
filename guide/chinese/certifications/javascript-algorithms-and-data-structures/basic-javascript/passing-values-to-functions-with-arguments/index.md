## 将值传递给带参数的函数

我们的任务是创建一个具有**参数**的函数。这些是确定函数输出的输入。你将参数放在`()` ，如下所示：

```javascript
function functionWithArgs(one, two) { 
  console.log(one + two); 
 } 
```

我们现在必须在括号内添加代码。我们的任务是添加`one`和`two` ，并将总和打印到控制台。这是基本的代码解决方案：

```javascript
functionWithArgs(7, 3); 
 //This will console log 10. 

```
