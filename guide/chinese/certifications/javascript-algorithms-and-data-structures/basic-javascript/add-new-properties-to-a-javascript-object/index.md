## 将新属性添加到JavaScript对象

这是一个例子：

```js
var ourDog = { 
  "name": "Camper", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["everything!"] 
 }; 
 
 ourDog.bark = "bow-wow"; 
```

这是一个解决方案：

```js
var myDog = { 
  "name": "Happy Coder", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["freeCodeCamp Campers"] 
 }; 
 
 // Only change code below this line. 
 myDog.bark = "woof"; 

```
