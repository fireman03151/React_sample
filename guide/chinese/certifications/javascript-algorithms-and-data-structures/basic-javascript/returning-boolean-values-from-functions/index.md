## 从函数返回布尔值

我们可以使用比较运算符和最小代码在return语句中直接执行，而不是使用if / else块来比较变量。

### 问题解释

_修复函数`isLess`以删除`if...else`语句。_

```js
// Fix this code 
  if (a < b) { 
    return true; 
  } else { 
    return false; 
  } 
```
