## 创建一个Redux商店

这个挑战有三个部分。

1.  声明一个`store`变量。
2.  将其分配给`createStore()`方法。
3.  传递`reducer`作为参数。

### 步骤1.声明变量。

```javascript
const yourVariableName; 
```

### 第2步。将变量分配给方法。

```javascript
const yourVariableName = yourMethodName(); 
```

提示：请记住，Redux对象中提供了`createStore()`方法。例如： `Redux.createStore()`

### 第3步。将参数传递给您的方法。

```javascript
const yourVariableName = yourMethodName(yourArgumentName); 
```
