## Azure DevOps

Azure DevOps Services - это облачный сервис для совместной разработки кода. Он предоставляет интегрированный набор функций, которые вы получаете через ваш веб-браузер или клиент IDE, включая следующие:

*   Репозитории Git для контроля версий вашего кода
*   Создание и выпуск руководств для поддержки непрерывной интеграции и доставки ваших приложений
*   Гибкие инструменты для поддержки планирования и отслеживания вашей работы, дефектов кода и проблем с использованием методов Kanban и Scrum
*   Различные инструменты для тестирования ваших приложений, включая ручные / поисковые испытания, нагрузочные испытания и непрерывное тестирование
*   Высоко настраиваемые информационные панели для обмена достижениями и тенденциями
*   Встроенная вики для обмена информацией с вашей командой

Кроме того, экосистема Azure DevOps обеспечивает поддержку для добавления расширений, интеграцию с другими популярными сервисами, такими как: Campfire, Slack, Trello, UserVoice и т. Д., А также разработка собственных пользовательских расширений.

Выберите Azure DevOps Services, если вам нужна быстрая настройка, техобслуживание, простота совместной работы по доменам, эластичная шкала и надежная защита. У вас также будет доступ к облачным нагрузкам, серверам облачных вычислений и знаниям приложений.
