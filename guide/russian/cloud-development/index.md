## Разработка облаков

Разработка облака - это использование практик разработки программного обеспечения для компонентов инфраструктуры в общем пуле ресурсов. Это можно сделать несколькими способами, в вашей компании или в организации, имеющем собственный центр обработки данных, или в «общедоступном» совместном центре обработки данных, принадлежащем и управляемом третьей стороной.

Популярным выбором для поставщиков Public Cloud являются Amazon Web Services, Microsoft Azure и Google Compute Engine. Эти поставщики используют кластеры массивных центров обработки данных, в соответствии с которыми тысячи различных лиц, компаний и организаций потребляют эти ресурсы. Управление облаком предполагает включение разделов этих общих ресурсов для использования этих разных пользователей. Потребление ресурсов отслеживается (также упоминается как измеренное), а затем возвращается пользователю в зависимости от различной финансовой схемы для различных вычислительных ресурсов, сети и ресурсов хранения.

Многие современные веб-сайты и большие усилия данных, как говорят, разрабатываются в Облаке, то есть они работают в некоторых из этих массивных общих центров обработки данных.
