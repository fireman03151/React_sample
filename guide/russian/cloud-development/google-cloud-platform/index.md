## Виртуальная платформа Google

Виртуальная платформа Google, предлагаемая Google, представляет собой набор облачных сервисов, которые работают на той же инфраструктуре, которую Google использует внутри своих продуктов для конечных пользователей, таких как Google Search и YouTube. \[1\] Наряду с набором инструментов управления, он предоставляет ряд модульных облачных сервисов, включая вычисления, хранение данных, аналитику данных и машинное обучение.

Некоторые известные веб-службы включают

*   Google Compute Engine - IaaS, предоставляющий виртуальные машины.
*   Google App Engine - PaaS для хостинга приложений.
*   Bigtable - масштабируемая база данных NoSQL с масштабируемой масштабируемостью IaaS.
*   BigQuery - крупномасштабная аналитика базы данных SaaS.
*   Облачные функции Google. По состоянию на август 2017 года в бета-тестировании. FaaS предоставляет безсерверные функции для запуска облачных событий.
*   Google Cloud Datastore - DBaaS, обеспечивающий документированную базу данных.
*   Cloud Pub / Sub - служба публикации и подписки на потоки данных и сообщения. Приложения могут общаться через Pub / Sub, без непосредственной интеграции между самими приложениями. \[5\]
*   Хранилище Google - IaaS, предоставляющее онлайн-хранилище файлов и объектов RESTful.
