## Microsoft Azure

Microsoft Azure - это публичное облачное предложение Microsoft, которое позволяет отдельным лицам и организациям создавать, развертывать и управлять облачными приложениями и инфраструктурными службами. Общая облачная платформа Microsoft Azure предлагает услуги IaaS, PaaS и SaaS для организаций по всему миру и поддерживает множество различных языков, инструментов и фреймворков программирования, включая программное обеспечение и системы, специфичные для Microsoft и сторонних разработчиков.

Microsoft Azure также может использоваться в сочетании с различными решениями Microsoft, такими как Microsoft System Center, и может использоваться совместно для расширения текущего центра данных организации в гибридное облако, которое расширяет возможности и предоставляет возможности, выходящие за рамки того, что может быть поставлено исключительно на основе on- помещения.

### Использование для Microsoft Azure

Microsoft Azure предлагает множество услуг и предложений по ресурсам. Например, вы можете использовать вычислительные службы Azure Virtual Machines для создания сети виртуальных серверов для размещения приложения, базы данных или настраиваемого решения, которое будет предлагаться на основе IaaS.

Другие сервисы могут быть отнесены к категории PaaS, потому что вы можете использовать их, не поддерживая базовые операционные системы. Например, при запуске веб-сайта в Azure Web Apps или базе данных SQL в базе данных SQL Azure нет необходимости гарантировать, что вы используете последнюю версию IIS или SQL и имеете последние исправления и обновления поскольку это ответственность платформы Microsoft Azure.

Пример для службы SaaS на Microsoft Azure - это Operations Management Suite, который настроен и доступен через службы Operational Insights. Здесь вы просто настраиваете свои инструменты управления и подключаетесь к службам, которым хотите управлять через Microsoft Azure, поэтому никакая локальная инфраструктура не требуется или не нуждается в управлении.

### Среды

Azure предоставляет множество различных типов среды по всему миру, включая платформы ОС, такие как Windows Server и Ubuntu Server, существует ряд вариантов, которые можно выбрать, например, Ubuntu 14.04 или Ubuntu 16.10, с возможностью использования указанных сред либо через встроенный Azure Web CLI или SSH доступ через вашего предпочтительного клиента.

Существуют различные уровни размеров машины на выбор, начиная от простых barebone low spec машин и заканчивая многопроцессорными и многопроцессорными сильными ударными ящиками.
