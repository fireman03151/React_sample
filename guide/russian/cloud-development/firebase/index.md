## Firebase

![Логотип Firebase](https://firebase.google.com/_static/558bc0d91d/images/firebase/lockup.png)

### обзор

Firebase, в ее самой базовой форме, была приобретена Google в октябре 2014 года. С тех пор Google приобрела дополнительные компании, которые дополняли оригинальный продукт Firebase. Этот набор программных средств теперь составляет текущий набор инструментов Firebase, которые предлагаются сегодня.

### Основные функции Firebase

Основные функции Firebase сгруппированы в 3 категории:

1.  **развивать**
    
    *   Аутентификация
    *   База данных
    *   Место хранения
    *   Хостинг
    *   функция
    *   Лаборатория испытаний
    *   Отчеты о сбоях
    *   Представление
2.  **расти**
    
    *   Уведомления
    *   Удаленная настройка
    *   Динамические ссылки
3.  **зарабатывать**
    
    *   AdMob
