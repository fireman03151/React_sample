\## Heroku

Heroku - это облачная платформа как услуга (PaaS), поддерживающая несколько языков программирования, которые используются в качестве модели развертывания веб-приложений. Heroku, одна из первых облачных платформ, находится в разработке с июня 2007 года, когда она поддерживает только язык программирования Ruby, но теперь поддерживает Java, Node.js, Scala, Clojure, Python, PHP и Go. \[1\] \[ 2\] По этой причине Heroku считается платформой polyglot, поскольку он позволяет разработчику создавать, запускать и масштабировать приложения аналогичным образом на всех языках. Heroku была приобретена Salesforce.com в 2010 году за 212 миллионов долларов
