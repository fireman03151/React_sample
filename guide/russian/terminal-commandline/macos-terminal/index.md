# Использование терминала в Mac OS

В большинстве случаев пользователи взаимодействуют через графический интерфейс пользователя для взаимодействия с компьютером. Вы используете мышь, чтобы указать и щелкнуть, чтобы открыть, переместить или создать новые файлы или открыть приложения. Но вы также можете использовать приложение терминала для взаимодействия с вашей машиной с помощью письменных команд. Когда вы используете терминал, он позволяет вам копать глубже и настраиваться таким образом, который невозможен благодаря графическому интерфейсу.

### Открытие терминала и навигационных каталогов

Ваш терминал существует в каталоге Applications. Откройте приложение «Терминал». В окне терминала вы увидите приглашение. у него есть имя компьютера (MacBook) ABC, за которым следует имя пользователя (ABC), а затем «$». Если вы находитесь в корневом каталоге, последний символ будет «#.»

Чтобы узнать, в каком каталоге вы работаете, просто введите команду

`pwd`

pwd означает «Print Working Directory». Справочник - другое слово для папки.

Если вы хотите перечислить содержимое своего каталога, используйте команду:

`ls`

Чтобы переключиться на новый каталог, вы используете следующую команду:

`cd`

который обозначает каталог изменений.

Вот список общих команд:

Команда | использование ------------ | ------------- pwd | Печать рабочего каталога (где я?) ls | Список содержимого текущего каталога mkdir | Создать новый каталог сенсорный | Создать новый файл ф | Скопировать файл rm | Удалить файл rm -rf | Удалить каталог

### Примеры использования

Некоторые из вышеупомянутых команд не ясны без примеров. Ниже приведены несколько примеров использования, которые помогут вам в некоторых контекстах.

#### Создание каталога

`mkdir #YOUR-NEW-FOLDER-NAME-HERE`

#### Создание файла

`touch YOUR-FILE-NAME.JS`

Вы можете сделать файл с любым расширением, которое вы выберете. До тех пор, пока он находится в формате, принятом папкой или машиной.

#### Копирование файла

Используйте следующий синтаксис для копирования файла с терминала:

**cp _источник_ _назначения_**

Например, если у нас есть файл _test.txt,_ который хранится в нашем каталоге _/ Desktop,_ и мы хотим скопировать его в папку _/ Documents_ , наша команда будет выглядеть так:
```
cp ~/Desktop/test.txt ~/Documents 
```

#### Удаление файла

Используйте следующий синтаксис для удаления файла

**гт _#PATH_ _ПОДАТЬ_**
