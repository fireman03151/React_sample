## Степени для радианов

Умножьте угол на pi / 180. Помните, что pi радианы охватывают половину круга, что составляет 180 градусов. Вот функция python для преобразования градусов в радианы. Наслаждайтесь!

импортная математика  
def градусов _к_ радианам (угол):  
угол возврата \* math.pi / 180;

#### Дополнительная информация:

Один полный цикл = 360 градусов = 2π Радианы

1 градус = 2π / 360 Радианы

p degree = p × 2π / 360 Радианы
