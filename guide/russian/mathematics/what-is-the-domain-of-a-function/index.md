## Что такое домен функции

Доменом функции является полный набор возможных значений независимой переменной. Другими словами, домен представляет собой набор всех возможных значений х, которые заставят функцию «работать» и будут выводить реальные значения y.

Например, для y = sqrt (x + 2) область этой функции равна x> = -2 .
