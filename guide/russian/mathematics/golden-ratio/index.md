## Золотое сечение

`PHI = φ = 1.61803398875...`

Две величины находятся в золотом соотношении, если их соотношение такое же, как отношение их суммы к более большому quanitity.

Учитывая a> b> 0, (a + b) / a = a / b = φ

Золотое соотношение иррационально и может быть представлено как (1 + √5) / 2.
