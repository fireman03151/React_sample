# Мощность BI:

Power BI является коммерчески доступным продуктом от Microsoft для предоставления информации и визуализации данных для любознательных данных.

## Основные особенности:

Power BI обладает множеством функций для приема, анализа и потребления данных:

*   Пакеты данных для ввода данных из многих источников, включая GitHub, Facebook, AWS, Azure и большинство баз данных SQL
*   Интуитивные инструменты моделирования данных и трансляции, которые могут быть автоматизированы для таблиц аналогичной структуры или контента
*   Фондовые графики и графики (включая направленную направленность!) Для отчета о перетаскивании и построения панели
*   R для расширенного статистического анализа
*   Встроенные возможности просмотра панелей мониторинга в разных бизнес-группах или для общественного потребления

## Как это используется для науки о данных:

Хотя это не так эффективно для сложного анализа вывода или уменьшения карты, он процветает в представлении данных в интерактивных информационных панелях для типичных потребителей данных бизнес-типа. PowerBI - это мост между учеными-данными и остальным бизнес-миром, которые хотят получать данные для принятия решений в легко понятной визуализации. Используя среду Office 365, с которой знакомы многие потребители, Data Scientist может принести наиболее сложный пользовательский анализ или визуализацию через консоль R для взаимодействия кем угодно.
