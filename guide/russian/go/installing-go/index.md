# Установка Go

## Установка Go в Ubuntu

*   [Установка Go с помощью apt-get](https://guide.freecodecamp.org/go/installing-go/ubuntu-apt-get) . Это самый простой способ установки в ubuntu.
*   [Установка Go с помощью tarball](https://guide.freecodecamp.org/go/installing-go/ubuntu-tarball) . Используйте этот метод, если вы хотите установить последнюю стабильную версию go.

## Установка Go в Mac OS X

*   [Установка Go с помощью установщика пакетов](https://guide.freecodecamp.org/go/installing-go/mac-package-installer) . Самый простой способ установить последнюю стабильную версию go.
*   [Установка Go с помощью tarball](https://guide.freecodecamp.org/go/installing-go/mac-tarball) . Для тех, кто хочет использовать терминал.

## Установка Go в Windows

*   [Установка Go с помощью установщика MSI](https://guide.freecodecamp.org/go/installing-go/windows-installer)

## Установка Go из источника

*   Чтобы установить Go из исходной ссылки на [установку из исходной документации](https://golang.org/doc/install/source)
