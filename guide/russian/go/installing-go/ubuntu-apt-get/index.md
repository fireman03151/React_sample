### Установка Go в Ubuntu с помощью apt-get

Использование диспетчера исходных пакетов Ubuntu (apt-get) - это самый простой способ установки Go. Вы не получите последнюю стабильную версию, но для изучения этого должно быть достаточно.

> На момент написания этой статьи версия Ubuntu Xenial версии 1.6.1, а последняя версия стабильная версия - 1.9.1

```shell
$ sudo apt-get update 
 $ sudo apt-get install golang-go 
```

#### Проверьте установку и версию go

Чтобы проверить, успешно ли установлен go, используйте:

```shell
$ go version 
 > go version go1.9.1 linux/amd64 
```

Это выведет на консоль версию go, и в то же время убедитесь, что установка прошла гладко.
