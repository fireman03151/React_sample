## Нулевая стоимость

Понимание некоторых нулевых значений внутри Golang:

1.  false для булевых
2.  0 для целых чисел
3.  0.0 для поплавков
4.  "" для строк
5.  ноль для

*   указатели
*   функции
*   интерфейсы
*   ломтики
*   каналы
*   карты

Пожалуйста, используйте как можно более короткий оператор объявления.

*   Использовать var для:
    
*   нулевое значение
    
*   объем упаковки
