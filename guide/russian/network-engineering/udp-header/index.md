## Формат заголовка UDP

Протокол User Datagram Protcol (UDP) является одним из широко используемых протоколов в пакете Internet Protocol (IP) вместе с TCP и ICMP. Определяющие функции UDP - это быстрый протокол без подключения. Эти функции облегчаются с помощью заголовка UDP.

Заголовок UDP состоит из следующих четырех полей:

*   Порт источника (16 бит): Порт, используемый на компьютере отправителей для UDP-соединения, также упоминается как отправляющий порт. В сочетании с IP-адресом отправителей он позволяет посылающему компьютеру создавать несколько отдельных соединений (называемых мультиплексированием).
    
*   Порт назначения (16 бит): Порт, используемый на принимающем компьютере для UDP-соединения, также упоминается как принимающий порт. В сочетании с IP-ресиверами он позволяет мультиплексировать.
    
*   Длина (16 бит): Указывает длину заголовка и данных UDP.
    
*   Контрольная сумма (16 бит): Используется для проверки ошибок заголовка и данных. Необязательный.
