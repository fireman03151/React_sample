# Laravel

Laravel - это бесплатный и открытый фреймворк для разработки приложений на PHP. Полный исходный код проекта доступен на [GitHub](https://github.com/laravel/laravel) и лицензирован по лицензии MIT. Фреймворк был создан Тейлором Отуэлом, который разработал его с целью обеспечения разработки веб-приложений в соответствии с архитектурным шаблоном model-view-controller (MVC). На данный момент Laravel является самой из самых популярных фреймворков на PHP на GitHub. Некоторые из основных функций  Laravel: разные способы доступа к реляционным базам данных, утилиты, которые помогают в развертывании и обслуживании приложений и, не в последнюю очередь, модульная система поставки с выделенным менеджером зависимостей.

Поскольку Laravel является проектом с открытым исходным кодом, сообщество вокруг него очень обширное. У проекта есть подробная документация, в которой вы можете узнать, как делать почти все. Попробуйте взглянуть на [документацию Laravel,](https://laravel.com/docs/) чтобы получить представление!
