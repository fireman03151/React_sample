## Память только для чтения 

ROM (англ. Read Only Memory — постоянное запоминающее устройство) — энергонезависимая память, используется для хранения неизменяемых данных.Read Only Memory - это память, работающая только на считывание. Информация, находящаяся в такой памяти, заранее закладывается при ее изготовлении («зашивается») и при отключении питания не пропадает. 

Виды ROM-памяти:

BIOS-ROM<br>
В первые дни современных компьютеров одним из наиболее распространенных чипов ROM являлось ROM BIOS (Basic Input Output System). Чип содержит конкретные инструкции, необходимые компьютеру для запуска, и было крайне важно, чтобы информация не могла быть случайно изменена. Как только чип BIOS ROM был установлен (как правило, припаян к материнской плате), его нельзя было изменить. Ускоренная перемотка вперед к технологиям 2000-х и BIOS улучшился до того, что BIOS теперь может использовать EPROM (стираемый тип ПЗУ), где инструкции могут быть обновлены контролируемым образом производителем, но работа чипа будет по-прежнему иметь одинаковые поведение в режиме «только для чтения». Актом «обновления BIOS» для компьютера является обновление программного обеспечения EPROM Bios.

CD-ROM<br>
CD-ROM - это тип компакт-дисков, который требует использования оптического лазера для считывания данных с диска. Устройства, используемые для считывания данных, располагают лазерами и называются приводом CD-ROM. Сам диск, как правило, создается на заводе и после его изготовления читается только. Это контрастирует с носителем CD-R (компакт-дисков), который имеет дополнительную возможность хранения новых данных. Дальнейшим развитием CD-ROM стали DVD-ROM,BD-ROM.
