## Операционные системы реального времени

Операционные системы реального времени (RTOS) предназначены для обработки входов без буферизации. Буферизация - это процесс задержки входных данных в рамках системы управления ресурсами. Операционные системы Linux и BSD не предназначены для RTOS.
