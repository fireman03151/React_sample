## Вес и стиль шрифта

Шрифт-вес можно записать в виде текстовых значений:
```
font-weight: normal; 
 font-weight: bold; 
```

Или как числовое значение от `100` до `900` (кратным 100):
```
font-weight: 400;  /* equal to 'normal' above */ 
 font-weight: 700; /* equal to 'bold' above */ 
```

Численное значение и их общее описание

| Значение | Общее описание | | ----- | ---------------------------- | | 100 | Тонкая (волоса) | | 200 | Дополнительный свет (ультралегкий свет) | | 300 | Свет | | 400 | Нормальный | | 500 | Средний | | 600 | Полужирный (Деми Полужирный) | | 700 | Полужирный | | 800 | Extra Bold (Ultra Bold) | | 900 | Черный (тяжелый) |

Не все весы доступны для всех шрифтов. Некоторые специализированные шрифты могут быть доступны только в одном весе (обычно `normal` `400` ).

Шрифт-вес также может быть указан относительно родителя элемента (если шрифт имеет более одного веса):
```
font-weight: lighter; 
 font-weight: bolder; 

```
