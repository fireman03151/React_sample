## Типографическая иерархия
```
  A webpage itself has a hierarchy that not only is read, but contains interaction. The page as a whole must be designed in a way that clearly communicates to the user what actions are available and how to easily access the information they seek, how to purchase an item, etc. 
 
  For the purpose of this article, however, we're talking strictly about hierarchy as it applies to type. Luckily for us, we have our own handy HTML tag that lets us semantically establish typographic hierarchy into the websites we build. Heading tags (H tags) allow us to specify an order of importance into our content: H1 through H6, H1 being most important, H6 being least. Search engines use this data to interpret priority of content on a webpage.The H tag can be styled by using the following attributes. 
```

\-Размер Это самый простой и наиболее распространенный метод установления иерархии.

\-Вес Просто использование более смелого веса шрифта может помочь изолировать.

\-Цвет Цвет играет большую роль в том, что наш глаз видит как первичный и вторичный. Вообще говоря; теплые цвета поп, более холодные цвета отступают.

\-Позиция Там, где разделы информации расположены друг с другом, можно установить иерархию.

\-Тип контраста Отличным способом достижения иерархии является использование контрастных шрифтов.

Способы решения вопросов! Одним из наиболее важных понятий в типовом дизайне является интервал. Это одна из самых сложных концепций для начинающих дизайнеров, но она также одна из самых визуально очевидных. Правильный типографский интервал имеет решающее значение для установления иерархии; это может сделать разницу между путаницей и ясностью. Он используется в большинстве иерархических систем и присутствует во всех примерах этой статьи.
