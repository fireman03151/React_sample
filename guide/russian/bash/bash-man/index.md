## man(ман)

Большинство программ, предназначенных для использования в командной строке, предоставляют официальную документацию, которую называют страницей справочного руководства (man-страницу). Для просмотра этих страниц используется специальная программа постраничного просмотра man

### использование

```bash
man [опции] [название программы] 
```
### пример

Отобразите страницу man ls:

```bash
$ man ls 
```

или

```bash
man [раздел] [искомый_термин]
```
### пример

Вывести страницу с описанием формата файла /etc/passwd:

```bash
$ man 5 passwd
```

Командой `man` можно даже прочитать про ASCII-таблицу, наберите следующее:

```bash
$ man ascii
```

Наиболее часто используемые опции:

*   `-f` , ищет и выводит краткое описание всех man-страниц по ключевому слову,
*   `-a` , последовательно отображать все доступные внутренние справочные страницы, содержащиеся в руководстве
