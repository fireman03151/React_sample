## Введение

Если / Else является условным утверждением, в котором в зависимости от правдоподобности условия будут выполняться различные действия.

> **Примечание** . Скобки `{}` нужны только в том случае, если условие содержит более одного оператора действия.

## Если заявление
```
  if (condition){ 
    statement1; 
    statement2; 
  } 
```

> **Примечание.** Оператор `else` является необязательным.
> 
> ## Если / Else Statement
```
  if (condition){ 
    statement1; 
    statement2; 
  } 
  else{ 
    statement3; 
    statement4; 
  } 
```

## Если / Elseif / Else Statement
```
  if (condition){ 
    statement1; 
    statement2; 
  } 
  elseif{ 
    statement3; 
    statement4; 
  } 
  else 
    statement5; 
```

Для получения дополнительной информации ознакомьтесь со следующей ссылкой: [PHP Else](http://php.net/manual/en/control-structures.elseif.php)
