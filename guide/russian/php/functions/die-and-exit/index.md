## Умирать и выходить

Функции `die()` и `exit()` идентичны. Каждый из них принимает один аргумент (строку), содержащий сообщение об ошибке. После запуска они выводят сообщение и немедленно останавливают выполнение скрипта.

```PHP
<?php 
 die('Die() function was run'); 
```

```PHP
<?php 
 exit('Exit() function was run'); 
```
