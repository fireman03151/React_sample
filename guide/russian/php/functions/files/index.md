## файлы

PHP предоставляет несколько функций для работы с файлами. Эти функции позволяют разработчику включать загрузку пользовательских файлов, чтобы php мог читать и использовать данные из файла, и, наконец, PHP для записи данных в файл.

#### Дополнительная информация:

*   [Руководство пользователя php.net readfile ()](https://secure.php.net/manual/en/function.readfile.php)
*   [Руководство пользователя php.net fopen ()](https://secure.php.net/manual/en/function.fopen.php)
*   [Руководство по php.net fread ()](https://secure.php.net/manual/en/function.fread.php)
*   [Руководство пользователя php.net fclose ()](https://secure.php.net/manual/en/function.fclose.php)
*   [Руководство пользователя php.net fgets ()](https://secure.php.net/manual/en/function.fgets.php)
*   [Руководство пользователя php.net feof ()](https://secure.php.net/manual/en/function.feof.php)
*   [Руководство пользователя php.net fgetc ()](https://secure.php.net/manual/en/function.fgetc.php)
