## Какие параметры доступны для PHP для подключения к базе данных?

PHP может подключаться к различным базам данных, включая MongoDB, MsSQL и MySQL. И PHP, и MySQL очень популярны и предоставляют простые, бесплатные и открытые исходные тексты которые будут созданы и часто встречаются вместе для создания веб-сайтов всех типов. И PHP, и MySQL могут масштабироваться для поддержки большого числа пользователей.

PHP даже поддерживает более одного способа работы с подключениями к MySQL, включая MySQLi Processural, Объекты объектов PHP (PDO) и объект MySQLi Ориентированы вместе с устаревшим MySQL Connect. С PHP существует множество функций, встроенных в основные функциональные возможности языка, которые делают ссылки на база данных простая и простая.
