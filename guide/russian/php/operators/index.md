## операторы

PHP содержит все нормальные операторы, которые можно было бы найти на языке программирования.

В качестве оператора присваивания используется одиночный «=», а для сравнения используется двойное «==» или тройное «===».

Обычные «<» и «>» также могут использоваться для сравнения, а «+ =» может использоваться для добавления значения и назначения его одновременно.

Наиболее примечательным является использование «.» Для объединения строк и «. =» Для добавления одной строки в конец другой.
