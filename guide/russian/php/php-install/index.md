### Что мне нужно для установки?

Чтобы начать использовать PHP, вы можете:

Найдите веб-хостинг с поддержкой PHP и MySQL. Установите веб-сервер на свой ПК, а затем установите PHP и MySQL. Вы можете проверить следующие ссылки, чтобы получить PHP-сервер и MYSQL для создания локального envinonment для PHP для запуска: Веб-сервер: https://httpd.apache.org/download.cgi База данных: https://www.mysql.com/downloads/

### Установка PHP Parser

Прежде чем приступить к работе, важно убедиться, что у вас есть надлежащая настройка среды на вашем компьютере для разработки ваших веб-программ с использованием PHP.

Введите адрес в адресной строке вашего браузера.

```shell
http://127.0.0.1/info.php 
```

Если это отображает страницу с информацией о вашей установке, связанной с установкой PHP, значит, вы правильно установили PHP и Webserver.

### Настройка PHP на вашем собственном компьютере

Однако, если ваш сервер не поддерживает PHP, вы должны: установить веб-сервер установить PHP установите базу данных, такую ​​как MySQ.

### Больше информации

На официальном сайте PHP (PHP.net) есть инструкции по установке для PHP: http://php.net/manual/en/install.php Чтобы получить как веб-сервер, так и базу данных в одном пакете, используйте эту ссылку: https://www.mamp.info/en/
