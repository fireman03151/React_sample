## CSS Grid Layout

CSS Grid Layout - самая мощная система создания макетов, доступная в CSS. Это двухмерная система, то есть она может обрабатывать как столбцы, так и строки, в отличие от flexbox, которая в значительной степени является одномерной системой. Хотя Grid Layout не полностью поддерживается всеми браузерами, это самый продвинутый и удобный способ создавать макеты страниц.
