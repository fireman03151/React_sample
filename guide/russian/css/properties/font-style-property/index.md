## Свойство Стиль шрифта

Свойство `font-style` позволяет вам изменить стиль шрифта, что означает, что вы можете сделать курсивом курсивом. Это свойство принимает следующие значения:

*   `normal` - Значение по умолчанию.
*   `italic` - Шрифт выделяется курсивом, выбирая шрифт, который классифицируется как курсив в его семействе шрифтов.
*   `oblique` - Используется шрифт, который классифицируется как наклонный в его семействе шрифтов.

`Italic` и `oblique` могут выглядеть похожими. Согласно [MDN](https://developer.mozilla.org/en-US/) _косой - это только наклонная версия обычного лица_ .  
Вы также можете указать угол для скудной степени шрифта при предоставлении `oblique` значений. Если угол не предусмотрен, 14 градусов будут использоваться по умолчанию.
