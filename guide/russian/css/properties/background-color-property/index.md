## Цвет фона

Вы используете свойство `background-color` для установки цвета фона элемента. Вы можете использовать значение цвета (имя цвета, шестнадцатеричное значение, значение RGB / RGBA, значение HSL / HSLA) или ключевое слово `transparent` .

**Пример:**

```css
body { 
    background-color: crimson; 
 } 
 
 div { 
    background-color: #ffffff; 
 } 
 
 .myClass { 
    background-color: rgba(0, 0, 0, 0.5); 
 } 
```

#### Значения свойств:

`background-color: color | transparent | initial | inherit;`

`color` - указывает цвет фона (имя цвета, шестнадцатеричное значение, значение RGB / RGBA, значение HSL / HSLA).

`transparent` - значение по умолчанию. Устанавливает цвет фона как прозрачный.

`initial` - устанавливает это свойство в исходное значение (значение по умолчанию).

`inherit` - Наследует это свойство от его родительского элемента.
