## Свойство индекса Z

Свойство z-index указывает порядок стека элемента. Любые позиционированные элементы на веб-странице могут перекрывать друг друга в определенном порядке, имитируя третье измерение, перпендикулярное экрану. Элемент с большим порядком стека всегда находится перед элементом с более низким порядком стека. Порядок стека контролируется индексом z. Это свойство работает только для элементов, значение позиции которых установлено как абсолютное, фиксированное или относительное.

#### Синтаксис

z-index: auto | number | initial | inherit;

#### пример
```
div { 
  position: absolute; 
  z-index: -1; 
 } 
```

#### Значения

Авто: Устанавливает порядок стека, равный его родителям. Это значение по умолчанию. Number: Устанавливает порядок стека элемента. Отрицательные числа разрешены. Чем выше значение, тем выше элемент. При равном значении z-индекса элемент, описанный в HTML-коде ниже, будет впереди. Initial: Устанавливает это свойство по умолчанию. Inherit: Наследует это свойство от его родительского элемента.
