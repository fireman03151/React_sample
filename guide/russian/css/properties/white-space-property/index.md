## Белое пространство

Свойство white-space устанавливает, как отображать пробелы между словами.

#### Синтаксис

white-space: normal | nowrap | pre | pre-line | предварительная упаковка | унаследовать

#### пример
```
p { 
  white-space: nowrap; 
 } 
```

#### Значения

Обычный: значение по умолчанию. Последовательности пробелов рухнут в единую пробел. При необходимости текст будет завернут. Nowrap: Последовательности пробелов рухнут в одно пробелы. Текст не будет переноситься на следующую строку. Pre: пробел сохраняется браузером. Текст будет переноситься только на разрывы строк. Действует как

 тег в HTML. 
 Pre-line: Последовательности пробелов рушится в единую пробел. При необходимости текст будет завернут. 
 Предварительная упаковка: пробел сохраняется браузером. При необходимости текст будет завернут. 
 Initial: Устанавливает это свойство по умолчанию. 
 Inherit: Наследует это свойство от его родительского элемента.
