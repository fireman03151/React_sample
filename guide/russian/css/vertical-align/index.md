## Вертикально-Align

`vertical-align` - свойство CSS, используемое для вертикального выравнивания изображения.

Например, вы можете быть `vertical-align` чтобы выравнивать изображение:
```
img { 
    vertical-align: top; 
 } 
```

Это допустимые значения для `vertical-align` :

*   `baseline` - Значение по умолчанию. Он выравнивает элемент к базовой линии родительского элемента
*   `length` - эта привязка базовой линии этого элемента на заданной длине выше или ниже базовой линии родителя. Вы можете использовать пиксели, em,% и т. Д.
*   `sub` - Это выравнивает элемент так, как если бы он был индексом родительского элемента
*   `super` - Это выравнивает элемент так, как если бы он был надстрочным индексом родительского элемента
*   `top` - выравнивает элемент таким образом, чтобы вершина этого элемента находилась в соответствии с верхней частью самого высокого элемента на линии
*   `bottom` - это выравнивает элемент так, чтобы нижняя часть этого элемента была выровнена с нижней частью нижнего элемента на линии
*   `middle` - выравнивает элемент так, чтобы он находился в середине родительского элемента
*   `text=top` - выравнивает верхнюю часть этого элемента так, чтобы он был совмещен с верхней частью верхнего элемента родительского элемента
*   `text-bottom` - выравнивает нижнюю часть этого элемента так, чтобы он был выровнен с нижней частью текста родительского элемента
