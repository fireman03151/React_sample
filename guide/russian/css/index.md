### Каскадные таблицы стилей (CSS)

CSS является аббревиатурой Cascading Style Sheets. Каскадные таблицы стилей были впервые изобретены в 1996 году и теперь являются стандартной функцией всех основных веб-браузеров.

CSS позволяет разработчикам контролировать, как выглядят веб-страницы, путем «стилизации» структуры HTML этой страницы.

Спецификации CSS поддерживаются [консорциумом World Wide Web (W3C)](https://www.w3.org/) .

Вы можете создавать довольно интересные вещи с использованием только CSS, например, [сапер на чистом CSS](https://codepen.io/bali_balo/pen/BLJONk), в котором не используется JavaScript.

### Популярные CSS фреймворки в 2018 году
Фреймворки созданы для того, чтобы делать сложные части CSS более простыми и эффективными для разработчиков. Некоторые популярные CSS фреймворки 2018 года:
Bootstrap, Foundation, Bulma, uikit, Semantic UI, mini.css, Materialize, Material Design Lite, Spectre and Kube

#### Рекомендуемая литература:
Хорошим началом является учебная программа freeCodeCamp. [Введение в базовый CSS](https://learn.freecodecamp.org/responsive-web-design/basic-css) .

Еще одно предложение для начинающих - W3C. [Введение в HTML + CSS](https://www.w3.org/Style/Examples/011/firstcss) учит, как создать таблицу стилей.

Сайт [CSS Zen Garden](http://www.csszengarden.com/) - отличный пример того, как один и тот же HTML можно создать, чтобы выглядеть совершенно уникальным.

Для демонстрации силы CSS [Species In Pieces](http://species-in-pieces.com/#).

Некоторые интересные статьи о Front End разработке [CSS-Tricks](https://css-tricks.com)
