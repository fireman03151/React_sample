## Тип

Селектор типов выбирает элементы по имени узла (тега).

#### Синтаксис кода

```css
* { 
    css declarations; 
 } 
```

#### Пример кода

```css
h1 { 
    background-color: red; 
 } 
```

выбирает весь элемент h1 и устанавливает цвет фона красного цвета.

```css
div { 
    width: 100px; 
 } 
```

выбирает весь элемент div и устанавливает ширину элемента 100px.
