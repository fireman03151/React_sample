## звезда

Селектор атрибутов Star CSS или \* используется для выбора атрибутов, которые содержат определенное значение.

#### пример

Если вы хотите найти все привязывающие ссылки, содержащие «код» в любом месте URL-адреса, и сделать их желтыми, вы можете сделать это следующим образом:

```css
a[href*="code"] { 
   color: yellow; 
 } 

```
