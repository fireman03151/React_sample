## SQL

SQL обозначает язык структурированных запросов. Это самый распространенный инструмент, используемый для манипулирования данными и управления ими в реляционной базе данных (часто называемой «базой данных SQL»).

SQL обычно произносится как «сиквел». Его наиболее популярными вариантами являются MySQL, PostgreSQL и SQLite - версия SQL, которая обычно используется для прототипирования. В нем была введена концепция доступа к многим записям с помощью одной команды с использованием SQL-запросов.
