## Руководство SQL - ALTER TABLE

## Введение

Это руководство познакомит вас и попытается объяснить некоторые основы функций таблицы SQL alter в реляционной базе данных. **ВАЖНЫЙ Совет по безопасности: ВСЕГДА создавайте резервные копии данных перед внесением изменений!**

Мы будем использовать MySQL для всех примеров в этом руководстве freeCodeCamp SQL. Причины выбора MySQL: 1) он очень часто используется на веб-сайтах для бэкэнд-базы данных, 2) он бесплатный, и он забавна и проста в использовании.

## В этом руководстве

Мы будем использовать таблицы, созданные в руководстве «CREATE TABLE». Не стесняйтесь просмотреть это руководство, если вы не знакомы с созданием таблицы.

*   Изменение созданной таблицы изменит ее несколькими способами.
*   Мы изменим его имя и изменим столбцы
*   Добавить столбцы (при добавлении столбцов мы также рассмотрим несколько наиболее важных типов столбцов и их использование).
*   Удалить столбцы (это означает удаление столбца).
*   Создание таблицы путем импорта CSV-файла и изменения этой таблицы.
*   Создание и изменение таблиц с помощью инструментов Workbench MySQL.

Большая часть этого будет выполнена с использованием SQL-инструкций в инструменте сценариев Workbench MySQL, но мы также рассмотрим, как изменить таблицу с помощью интерфейса workbench, а не с помощью операторов SQL.
