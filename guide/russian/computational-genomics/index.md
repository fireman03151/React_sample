# Вычислительная геномика

Новое поле, которое специализируется на разработке новых алгоритмов для изучения генома человека.

Этот раздел будет посвящен различным структурам данных и алгоритмам, которые используются и создаются для этой цели из массивов суффиксов для выравнивания последовательностей.
