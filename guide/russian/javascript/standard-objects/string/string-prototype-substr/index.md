## String.prototype.substr

Метод substr () извлекает части строки, начиная с символа в указанной позиции, и возвращает указанное количество символов.

#### Синтаксис

```JavaScript
  string.substr(start, length); 
```

#### Значения параметров

| Параметр | Описание | | : ------------- | : ------------- | | начало | **Необходимые.** Позиция, в которой следует начать извлечение. Первый символ имеет индекс 0. Если _начало_ положительно и больше или равно длине строки, substr () возвращает пустую строку. Если _начало_ отрицательное, substr () использует его как индекс символа от конца строки. Если _начало_ отрицательно или больше длины строки, start устанавливается в 0 | | длина | **Необязательно** . Количество символов для извлечения. Если он опущен, он извлекает остальную часть строки |

#### Примеры:

```JavaScript
let str = "Hello world!"; 
 let res = str.substr(1, 4); 
```

Результатом res будет:
```
ello 
```
