## Массив из

Метод Array.from () создает новый экземпляр массива из объекта, подобного массиву или итерации.

### Синтаксис:

«»» Array.from (arrayLike \[, mapFn \[, thisArg\]\]) «»»

### Пример:

«»» Array.from ( 'Foo'); // \["f", "o", "o"\] «»»
