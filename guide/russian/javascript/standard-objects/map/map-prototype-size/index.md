## Map.prototype.size

Возвращает количество элементов в объекте `Map` .

## Синтаксис

```javascript
myMap.size(); 
```

## пример

```javascript
const myMap = new Map(); 
 myMap.set('foo',1); 
 myMap.set('bar',2); 
 myMap.set('baz',3); 
 
 
 myMap.size(); // 3 

```
