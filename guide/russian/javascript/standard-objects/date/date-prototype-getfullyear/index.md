## Date.prototype.getFullYear

Метод Date.getFullYear () возвращает год (четыре цифры) указанной даты в соответствии с местным временем.

### Синтаксис

```javascript
dateObj.getFullYear() 
```

### пример

```javascript
var date = new Date(); 
 // creates a new Date() object with current date and time. 
 date.getFullYear() 
 // 2018 
```
