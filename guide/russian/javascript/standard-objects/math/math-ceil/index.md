## Math Ceil

`Math.ceil()` - это метод стандартного объекта Math, который округляет заданное число до следующего целого числа. Обратите внимание, что для отрицательных чисел это означает, что число будет округлено «на 0» вместо числа большего абсолютного значения (см. Примеры).

### Примеры

```javascript
Math.ceil(0.1)  //  1 
 Math.ceil(1.3)  //  2 
 Math.ceil(-0.9) // -0 
 Math.ceil(-1.5) // -1 
```
