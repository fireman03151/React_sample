## Определение функции конструктора

### Метод:

Функция `Dog()` должна быть записана в том же формате, что и функция `Bird()` приведенная в примере. Просто замените `Bird` with `Dog` на все тестовые примеры.

### Решение:

```javascript
function Dog() { 
  this.name = "Geogre", 
  this.color = "White", 
  this.numLegs = 4; 
 } 

```
