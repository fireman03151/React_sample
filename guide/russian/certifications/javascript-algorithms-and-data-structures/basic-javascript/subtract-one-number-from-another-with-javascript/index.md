## Вычитайте один номер из другого с помощью JavaScript

Используя оператор вычитания '-', вы можете получить разницу в двух числах ...
```
var diff1 = 30 - 14; // Difference is 16 
 var diff2 = 90 - 60; // Difference is 30 

```
