## Обновление свойств объекта

## Подсказка:

Используйте точку \*\*. \*\* для доступа к объекту.

## Предупреждение о спойлере: решение впереди!

## Решение:

```javascript
// Setup 
 var myDog = { 
  "name": "Coder", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["freeCodeCamp Campers"] 
 }; 
 
 // Only change code below this line. 
 
 myDog.name = "Happy Coder"; // Solution 

```
