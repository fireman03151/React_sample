## Добавить новые объекты в объект JavaScript

Вот пример:

```js
var ourDog = { 
  "name": "Camper", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["everything!"] 
 }; 
 
 ourDog.bark = "bow-wow"; 
```

Вот решение:

```js
var myDog = { 
  "name": "Happy Coder", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["freeCodeCamp Campers"] 
 }; 
 
 // Only change code below this line. 
 myDog.bark = "woof"; 

```
