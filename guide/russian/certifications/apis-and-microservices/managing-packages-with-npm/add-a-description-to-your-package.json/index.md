## Добавьте описание в свой пакет package.json

Чтобы завершить задачу, выполните следующие шаги:

*   Перейдите по ссылке, приведенной на странице с инструкциями по программированию. ![Введение в freisececamp apis-and-microservices](https://user-images.githubusercontent.com/15084301/45616545-65732480-ba8d-11e8-9893-412ff6c68be5.png) [Ссылка проекта стартера](https://glitch.com/#!/import/github/freeCodeCamp/boilerplate-npm)
    
*   Отредактируйте файл package.json и обновите автора
    
*   Затем нажмите ссылку на шоу (ваше приложение) вверху проекта Glitch, чтобы получить общедоступную ссылку ![Ссылка на публичный проект Glitch](https://user-images.githubusercontent.com/15084301/45616579-8176c600-ba8d-11e8-8a46-4ac930578e74.png)
    
*   Отправьте ссылку в решении и все. ![страница sucess](https://user-images.githubusercontent.com/15084301/45616757-019d2b80-ba8e-11e8-8b5b-98dc6d02c235.png)
