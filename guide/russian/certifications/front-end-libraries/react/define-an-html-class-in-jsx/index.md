## Определить класс HTML в JSX

*   HTML и JSX могут показаться, что они точно такие же, но между ними существует несколько различий.
*   Одним из этих различий является соглашение об именах.
*   HTML-атрибуты и ссылки на события в JSX становятся camelCase (onclick => onClick).
*   Также могут быть некоторые слова, зарезервированные в JavaScript. Например, слово «класс» не может использоваться для определения классов HTML в JSX. Поэтому вместо использования класса вы можете использовать «className».

## Подсказка 1:

*   Вы можете изменить «класс» на «className».

## Решение

```javascript
const JSX = ( 
  <div className = "myDiv"> 
    <h1>Add a class to this div</h1> 
  </div> 
 ); 

```
