## Добавить комментарии в JSX

Вы можете комментировать, как вы делали с блоками кода в Javascript `/* some JS code */` но их нужно обернуть фигурными фигурными скобками для добавления комментариев в JSX: Например:

*   Однострочный комментарий:

```jsx
{/*<h1>Hanoi University of Science</h1>*/} 
```

*   Многострочные комментарии:

```jsx
{/* 
 <h1>Hanoi University of Science</h1> 
 <p>Falculty of Mathematics, Mechanics and Informatics</p> 
 */} 
```

Примечание. Вы не можете использовать комментарий HTML в JSX следующим образом:

```html

<!-- not working --> 

```
