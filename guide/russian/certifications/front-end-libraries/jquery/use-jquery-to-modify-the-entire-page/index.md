## Использование jQuery для изменения всей страницы

### Проблема Объяснение

Добавьте `animated` классы и `hinge` к элементу вашего `body` .

#### Полезные ссылки:

*   [.addClass ()](https://api.jquery.com/addClass/e)

### Решение:

```javascript
<script> 
  $("body").addClass("animated hinge"); 
  }); 
 </script> 

```
