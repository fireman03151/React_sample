## Создание магазина Redux

Для этой задачи есть три части.

1.  Объявите переменную `store` .
2.  Назначьте его `createStore()` .
3.  Перейдите в `reducer` в качестве аргумента.

### Шаг 1. Объявите переменную.

```javascript
const yourVariableName; 
```

### Шаг 2. Назначьте вашу переменную методу.

```javascript
const yourVariableName = yourMethodName(); 
```

Подсказка. Имейте в виду, что метод `createStore()` доступен из объекта Redux. Например: `Redux.createStore()`

### Шаг 3. Передайте аргумент вашему методу.

```javascript
const yourVariableName = yourMethodName(yourArgumentName); 
```

Удачи!
