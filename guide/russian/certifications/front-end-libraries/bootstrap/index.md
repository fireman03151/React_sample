## начальная загрузка

[Bootstrap](https://www.getbootstrap.com/) - это интерфейсная библиотека, которая обеспечивает базовую структуру или макет, которые пользователь может либо непосредственно использовать, либо модифицировать, а затем развертывать. Их основное внимание уделяется созданию мобильной среды, но Bootstrap также работает с настольными устройствами.

Цитируется с сайта Bootstrap:
```
Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery. 
```

Рамка находит возможность трудоустройства во многих интерфейсных проектах и ​​представляет собой ноу-хау, известную как самая известная структура HTML и CSS.
