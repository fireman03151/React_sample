## Используйте изображение Retina для дисплеев с более высоким разрешением

Следуя инструкциям:

Установите ширину и высоту тега img на половину их исходных значений. В этом случае как исходная высота, так и исходная ширина составляют 200 пикселей.

стиль 4 становится:

```css
  <style> 
  img { height: 100px; width: 100px; } 
  </style> 
```

обратите внимание, что самый простой способ сделать ваши изображения «сетчаткой» (и оптимизировать их для дисплеев сетчатки) - это определить их значения ширины и высоты как только половину исходного файла.
