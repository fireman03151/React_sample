## Создание медиа-запроса

Следуя инструкциям:

Добавьте медиа-запрос, чтобы тэг p имел размер шрифта 10px, когда высота устройства меньше или равна 800px.

медиа-запрос:

```css
  /* Add media query below */ 
  @media (max-height: 800px){ 
    p { 
    font-size: 10px; 
      } 
  } 
```

и размер текста будет 10 пикселей, когда ширина устройства меньше или равна 800 пикселей.
