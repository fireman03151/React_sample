## Добавьте тень к карточному элементу

Свойство box-shadow привязывает один или несколько теней к элементу.

h-offset: Требуется. Горизонтальное смещение тени. Положительное значение помещает тень в правую часть поля, отрицательное значение помещает тень в левую часть поля.

v-offset: Требуется. Вертикальное смещение тени. Положительное значение помещает тень под полем, отрицательное значение помещает тень над полем.
