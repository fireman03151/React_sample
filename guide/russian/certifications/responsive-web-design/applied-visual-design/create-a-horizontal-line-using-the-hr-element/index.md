## Создание горизонтальной линии с использованием элемента hr

Тег `hr` является одним из простейших элементов HTML. Он создает тонкую горизонтальную линию по его родительскому элементу.

Этот урок использует тег `hr` для разделения заголовка и содержимого «карты». Итак, как вы создаете элемент `hr` ?

```html

<!-- It's this easy --> 
 <hr> 
```

Этот элемент является самозакрывающимся, но чтобы быть дружелюбным к синтаксическим анализаторам и шагам стиля, вы должны добавить явный маркер закрытия.

```html

<hr /> 
```

Какой бы формат вы ни выбрали, он создаст горизонтальную линию, как в [этом уроке](https://learn.freecodecamp.org/responsive-web-design/applied-visual-design/create-a-horizontal-line-using-the-hr-element/) на [freeCodeCamp](https://www.freecodecamp.org/) . Вот скриншот из этого урока, показывающий элемент `hr` содержащийся в div.
