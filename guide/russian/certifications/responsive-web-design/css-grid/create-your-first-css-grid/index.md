## Создайте свою первую сетку CSS

### Подсказка 1

Чтобы изменить отображение любого элемента, используется следующий синтаксис:

```css
display: propertyName; 
```

### Подсказка 2

Измените свойство отображения элемента .container

### Подсказка 3

Установите отображение элемента .container в сетку

### Решение

Чтобы настроить отображение элемента .container в grid:

```css
display: grid; 

```
