## Добавление различной прокладки к каждой стороне элемента

Для настройки прокладки элемента используйте:

```css
.example { 
  padding: 10px; 
 } 
```

Чтобы указать размеры заполнения элемента на отдельных сторонах, мы можем использовать «padding-top», «padding-right», «padding-bottom» и «padding-left». Мы можем использовать любой из них в сочетании и в любом порядке. Например:

```css
.example { 
  padding-top: 5px; 
  padding-bottom: 0px; 
 } 
```

Или:

```css
.example { 
  padding-top: 20px; 
  padding-left: 25px; 
  padding-right: 5px; 
 } 

```
