## Отрегулируйте маржу элемента

Регулировка `Margin` `Element` означает увеличение или уменьшение количества интервалов между границами элемента и окружающими элементами.

• Поля корректируются с положительными и отрицательными целыми числами единицами пикселей ( `px` ).

• Поля могут быть созданы со всех сторон (сверху, снизу, слева, справа).

Это пример того, как это написано:

```css
margin: 20px; 
```

Вы можете легко изменить или отрегулировать `Margin` `Element` , изменив числовое значение:

**До** :

margin: `20` px;

**После** :

margin: `50` px;

Вы также можете создать собственный `class` с `margin` которую вы будете использовать повторно.

```css
  .style-margin { 
    margin: 20px; 
  } 
```

<h2 class="style-margin">Example</h2>
