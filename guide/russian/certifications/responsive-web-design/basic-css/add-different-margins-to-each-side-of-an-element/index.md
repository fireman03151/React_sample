## Добавление разных полей в каждую сторону элемента

Для настройки полей элемента используйте:

```css
.example { 
  margin: 10px; 
 } 
```

Чтобы указать размеры полей для элемента по отдельным сторонам, мы можем использовать «margin-top», «margin-right», «margin-bottom» и «margin left». Мы можем использовать любой из них в сочетании и в любом порядке. Например:

```css
.example { 
  margin-top: 5px; 
  margin-bottom: 0px; 
 } 
```

Или же:

```css
.example { 
  margin-top: 20px; 
  margin-left: 25px; 
  margin-right: 5px; 
 } 

```
