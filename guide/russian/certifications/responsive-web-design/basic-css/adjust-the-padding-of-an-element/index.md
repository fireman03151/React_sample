## Отрегулируйте прокладку элемента

Заполнение - это расстояние между содержимым и границей элемента.

• Расстояние сверху, снизу, слева и справа.

• Интервал - это числовое значение, которое увеличивается или уменьшается на единицы пикселей, которые записываются как `px` .

Чтобы настроить «Заполнение элемента», используйте:

```css
padding: 10px; 
```

Чтобы создать Padding как класс для определенного элемента, используйте:

```css
.example-Padding { 
  padding: 10px 
 { 
```

Чтобы применить этот класс к соответствующему полю, используйте:

```html

<div class="example-Padding"> 
 <p> Example Text </p> 
 </div> 
```

**Чтобы настроить прокладку элемента** , используйте:

заполнение: `10` пикселей;

Теперь измените числовое значение на нужную величину.

прокладка: `50` пикселей;
