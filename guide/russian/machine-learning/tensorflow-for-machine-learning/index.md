#### Что такое TensorFlow?

«TensorFlow - это библиотека обучения машинам с открытым исходным кодом для исследований и производства. TensorFlow предлагает API для начинающих и экспертов для разработки для настольных компьютеров, мобильных устройств, Интернета и облачных вычислений».

Tensorflow позволяет пользователям создавать графики потока данных. Это структуры, которые описывают, как данные перемещаются по графику. Эти узлы соединены друг с другом, узлы представляют математические операции и соединения, представляющие многомерные массивы или тензоры.

Tensorflow позволяет разработчикам больше сосредоточиться на логике приложения, чем застревать в сложных алгоритмах или выработать наиболее оптимальный способ реализации этих алгоритмов. При создании Deep Network также может оказаться утомительным, чтобы определить, к какому узлу необходимо подключиться. Tensorflow упрощает соединение между слоями. Таким образом, разработчикам становится легче сконцентрироваться на улучшении приложения.
