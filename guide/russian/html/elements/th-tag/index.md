## Th Tag

Тег `<th>` позволяет вам определять ячейку заголовка в таблицах HTML. По умолчанию большинство браузеров выделяют жирным шрифтом и центрируют данные в этих тегах.

### пример

```html

<table> 
 <tr> 
   <th>Movie</th> 
   <th>Rating</th> 
 </tr> 
 <tr> 
   <td>Die Hard</td> 
   <td>5 Stars</td> 
 </tr> 
 </table> 
```

Эта разметка создаст следующий результат:

Кино

Рейтинг

Живучи

5 звезд
