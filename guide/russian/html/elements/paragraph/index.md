## Параграф

HTML-код

Элемент представляет собой абзац текста.

обычно представляет собой блок текста, который он отделяет от других блоков по вертикальному пустому пространству.

### пример

\`\` \`html

Это абзац

Это еще один абзац

\`\` \`
