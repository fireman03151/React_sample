## Br Tag

Тег `<br>` создает разрыв строки в тексте. Это полезно для стихов и адресов.

### пример

```html

<!DOCTYPE html> 
 <html> 
  <body> 
    Mozilla Foundation<br> 
    1981 Landings Drive<br> 
    Building K<br> 
    Mountain View, CA 94043-0801<br> 
    USA 
  </body> 
 </html> 
```
