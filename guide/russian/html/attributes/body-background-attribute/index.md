## Атрибут Body Background

Если вы хотите добавить фоновое изображение вместо цвета, одним из решений является атрибут `<body background>` . Он указывает фоновое изображение для HTML-документа.

Синтаксис:

`<body background="URL">`

Атрибут:

`background - URL for background image`

Пример:

```html

<html> 
  <body background="https://assets.digitalocean.com/blog/static/hacktoberfest-is-back/hero.png"> 
  </body> 
 </html> 
```

## атрибут body-background устарел

атрибут body-background устарел в HTML5. Правильный способ это стилизовать тег `<body>` - с помощью CSS.

Существует несколько свойств CSS, используемых для установки фона элемента. Они могут использоваться для установки фона всей страницы.
