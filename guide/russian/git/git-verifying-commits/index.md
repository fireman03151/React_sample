## Git Verifying Commits

Когда вы создаете программное обеспечение для людей со всего мира, иногда важно проверить, что фиксации и теги поступают из идентифицированного источника. Git поддерживает подписание коммитов и тегов с GPG. GitHub показывает, когда подписанты и теги подписаны.

![Проверенные фиксации](https://cloud.githubusercontent.com/assets/25792/14253743/87b504be-fa41-11e5-9140-6dc8b7203c31.png)

Когда вы просматриваете подписанный фиксатор или тег, вы увидите значок, указывающий, может ли подпись быть проверена с использованием любого из ключей GPG вкладчика, загруженных в GitHub. Вы можете загрузить свои ключи GPG, перейдя на страницу настроек клавиш.

Многие проекты с открытым исходным кодом и компании хотят быть уверены, что фиксация происходит из проверенного источника. Проверка подписи GPG на коммитах и ​​тегах упрощает просмотр, когда фиксация или тег подписывается верифицированным ключом, о котором знает GitHub.

![Подтвержденная подпись](https://cloud.githubusercontent.com/assets/25792/14290042/5b27dab2-fb12-11e5-9ff9-44116a7780ea.png)
