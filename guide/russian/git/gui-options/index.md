## Параметры GUI для Git

Большинство людей предпочитают использовать Git в качестве [интерфейса командной строки (CLI)](https://en.wikipedia.org/wiki/Command-line_interface) , но есть много [графических интерфейсов (GUI),](https://en.wikipedia.org/wiki/Graphical_user_interface) которые позволяют использовать Git графически. Инструменты GUI могут оказаться полезными для лучшего представления состояния вашего репозитория Git, и некоторые из них могут оказаться более удобными в использовании, чем CLI. Тем не менее, важно знать, как использовать CLI, потому что он будет одним и тем же интерфейсом на каждой платформе, поскольку он считается более эффективным и потому что он требуется, как только вы хотите сделать что-то даже немного сложное ,

## Список решений на базе Git GUI

*   [GitKraken](https://www.gitkraken.com) - популярный графический интерфейс Git для Windows, Mac и Linux. Он является проприетарным, но бесплатным для некоммерческого использования.
*   [GitHub Desktop](https://desktop.github.com/) - это клиентское приложение Git, предоставляемое GitHub, что позволяет улучшить интеграцию с GitHub, чем другие решения. Он доступен для Windows и Mac, но еще не для Linux. Это бесплатный и открытый источник.
*   [SourceTree](https://www.sourcetreeapp.com/) - еще один графический интерфейс Git для Windows и Mac от Atlassian. Он имеет множество функций, таких как интерактивная перезагрузка, предназначенная для упрощения использования Git для новичков.
*   [Git Tower](https://www.git-tower.com/mac/) доступен для Mac и Windows.
*   [TortoiseGit](https://tortoisegit.org/) - это интерфейс Windows Shell для Git, основанный на TortoiseSVN. Он с открытым исходным кодом и может быть построен с бесплатным программным обеспечением.
*   [SmartGit](https://www.syntevo.com/smartgit/) - это бесплатный клиент Git для некоммерческого использования для Windows, Mac и Linux.
