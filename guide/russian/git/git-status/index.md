## Git Status

Команда `git status` отображает состояние рабочего каталога и проиндексированной области. Она отображает пути, которые имеют различия между `index` файлом и текущим `HEAD` коммитом, пути, которые имеют различия между рабочим деревом и `index` файлом, и пути в рабочем дереве, которые не отслеживаются в Git (и не игнорируются [gitignore](https://git-scm.com/docs/gitignore)

Вывод команды `git status` не показывает вам никакой информации о зафиксированой истории проекта. Для этого вам нужно использовать `git log`.

### Использование

```shell
git status 
```

Отображает, какие файлы проиндексированные, непроиндексированные и неотслеживаемые.
