---
title: Free Code Camp Completion Timeframe
localeTitle: Свободный период завершения лагеря для Code Code
---
Для завершения Free Code Camp требуется около 2080 часов.

Бесплатный кодовый лагерь полностью в сети, и на вашем уровне квалификации всегда будут другие люди, с которыми вы можете спарить программу, чтобы вы могли учиться в своем собственном темпе. Вот несколько примеров графиков кодирования:

Время в бюджете | Часы в неделю | Недели для завершения  
\--- | --- | ---  
Выходные | 10 часов в неделю | 208 недель (48 месяцев)  
Ночи и выходные | 20 часов в неделю | 104 недели (24 месяца)  
Полный рабочий день | 40 часов в неделю | 52 недели (12 месяцев)  
Традиционная стимуляция Bootcamp | 80 часов в неделю | 26 недель (6 месяцев)
