freeCodeCamp имеет несколько горячих клавиш, поэтому пользователи могут быстро обойтись.

## Как использовать горячие клавиши

Чтобы использовать эти горячие клавиши, вы должны нажать клавиши в том порядке, в котором они отображаются. Важно отметить, что вы не должны нажимать их одновременно.

### Переадресация горячих клавиш

g n n Переадресация на следующую задачу  
g n a Переадресация на страницу about  
g n m Переадресация на карту  
g n s Переадресация в магазин  
g n o Перенаправление настроек  
g n r Переадресация в хранилище Free Code Camp GitHub  
g n w Переадресация на форум

### Горячие клавиши редактора

Эти горячие клавиши должны быть нажаты одновременно, а не в порядке.

Ctrl + Enter Запускает тесты. Если в предыдущий раз тесты были успешными, это отправляет вызов и переходит к следующему  
Ctrl + / Комментарии или удаляет текущую строку или выбранный код

### Другие горячие клавиши

g m Переключить ящик карты  
g t n Переключение ночного режима  
g c Переключить главный чат (Gitter)

Esc Если вы нажмете эту кнопку, когда вход поиска находится в фокусе, он отключит вход
