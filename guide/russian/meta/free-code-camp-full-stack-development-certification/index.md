## Бесплатная сертификация разработки стека Code Code Camp

### обзор

Существует шесть сертификатов freeCodeCamp, которые заработаны, заполнив соответствующую учебную программу и требуемые проекты на сайте [learn.freecodecamp.org](https://learn.freecodecamp.org/) :

*   Отзывчивый веб-дизайн
*   Алгоритмы и структуры данных
*   Библиотеки фронт-энтов
*   Визуализация данных
*   API и микросервисы
*   Информационная безопасность и обеспечение качества

По завершении всех шести сертификатов выдается сертификат программы разработки FreeCodeCamp [Full Stack](https://guide.freecodecamp.org/meta/free-code-camp-full-stack-development-certification/) . Это заключительное различие означает завершение приблизительно 1800 часов кодирования с воздействием широкого спектра инструментов веб-разработки.

*   Если ваше имя не отображается в сертификате, убедитесь, что вы связали свой GitHub и что ваш профиль GitHub заполнен.

![Образ образца сертификата полной программы разработки стека](//discourse-user-assets.s3.amazonaws.com/original/2X/1/1150a2e2473730df906909f820e3d8fdf54d6c14.png)

Для получения дополнительной информации о новой программе сертификации см. Сообщение на форуме [здесь](https://www.freecodecamp.org/forum/t/freecodecamps-new-certificates-heres-how-were-rolling-them-out/141618) .

### Кодирование для некоммерческих организаций

Кодирование некоммерческих организаций теперь представляет собой отдельный процесс из программы сертификации. Доступ к открытым проектам, открытым для участия, можно получить в [Open Source for Good](https://www.freecodecamp.org/nonprofits/) .

Для получения дополнительной информации о новой программе сертификации см. Сообщение на форуме [здесь](https://www.freecodecamp.org/forum/t/freecodecamps-new-certificates-heres-how-were-rolling-them-out/141618) .
