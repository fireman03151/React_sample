title: Free Code Camp Deployment Workflow
localeTitle: Свободный рабочий процесс развертывания Code Camp
---
1.  Весь код написан и помещен в ветку на собственной вилке разработчиков Free Code Camp.
2.  Разработчики создают запросы на тяну, вытягивая код из своей ветки в промежуточную ветку Free Code Camp.
3.  Кто-то, кроме человека, который подал запрос на получение запроса на получение запроса на получение запроса и принял его, если все идеально. В противном случае они закрывают запрос на тяну с объяснением, почему он не идеален, и разработчик открывает еще один запрос на извлечение, как только они исправили ситуацию.
4.  Мастер периодически устанавливается на freecodecamp.com.

## Заметки:

*   Это может измениться после того, как мы получим лучшее покрытие тестирования и непрерывную интеграцию.
