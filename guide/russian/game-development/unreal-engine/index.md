## Разработка игр с помощью Unreal Engine

Unreal Engine - это кросс-платформенный игровой движок, разработанный Epic Games. Unreal Engine был первоначально разработан для названия FPS 1998 года Unreal, но впоследствии был использован для многих тысяч коммерческих и некоммерческих названий. Последняя версия движка Unreal Engine 4 предназначена для ПК, PlayStation 4, Xbox One, Mac OS X, iOS, Android, многих систем VR, Linux, SteamOS и HTML5, а редактор может работать в Windows, OS X и Linux.

Uscript - это родной язык сценариев, используемый для создания игрового кода и игровых событий до выхода Unreal Engine 4 и предназначен для программирования на высоком уровне. Сценарий был написан и запрограммирован Тимом Суини, также создателем другого языка сценариев ZZT-oop.

С 2015 года Unreal Engine был свободен в использовании, а Epic взимала 5% роялти за продажи титулов, произведенных с использованием двигателя. Epic делает большую часть своей кодовой базы свободно доступной через их Github, хотя источник для закрытых платформ, таких как Playstation 4 и Xbox One, доступен только для зарегистрированных разработчиков платформ.

### Неверные версии

Unreal провела 4 крупных пересмотра. Хотя некоторый код распространен между выпусками, каждая основная версия представляет собой отдельный движок, и проекты не могут перемещаться между ними. В каждом главном выпуске двигателя есть несколько второстепенных версий.

*   Unreal Engine 1 Выпущен в 1998 году и нацелен на ПК с ОС Windows, Linux, Mac, PlayStation 2 и Dreamcast. Двигатель был написан на C ++ и легко модифицируется с использованием языка Unreal Script.
    
*   Unreal Engine 2 Выпущен в 2002 году и предназначен для ПК с ОС Windows, Mac, Playstation 2 и Xbox. Двигатель был написан на C ++, использовался язык Unreal Script.
    
*   Unreal Engine 3 Выпущен в 2004 году и нацелен на ПК с ОС Windows, Linux, Mac, iOS, Playstation 3 и Xbox 360. Разработка разделена между Unreal Script и C ++, с дополнительным визуальным скриптовым интерфейсом Kismet.
    
*   Unreal Engine 4 Выпущен в 2015 году и нацелен на ПК, PlayStation 4, Xbox One, Mac OS X, iOS, Android, многие системы VR, Linux, SteamOS и HTML5, а редактор может работать в Windows, OS X и Linux. Unreal Script был удален и заменен динамически перезагруженными модулями C ++ и более продвинутым интерфейсом визуальных скриптов под названием Blueprints.
