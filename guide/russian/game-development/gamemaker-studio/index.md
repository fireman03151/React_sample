## GameMaker Studio

GameMaker Studio, ранее известная как GameMaker, представляет собой кросс-платформенное программное обеспечение для разработки игр, в первую очередь ориентированное на создание 2d-игр используя последовательности действий перетаскивания или язык сценариев, известный как Язык Game Maker (GML).

### Обзор:

GameMaker позволяет пользователям быстро создавать и прототипировать игры без необходимости изучать язык программирования. Игры GameMaker обычно состоит из по меньшей мере 3 вещей: спрайтов (изображений и анимаций), объектов (логика и интерактивность) и комнат (экранов и уровней). Каждой игре требуется по крайней мере одна комната, которая используется как для меню, так и для уровней, а также для перехода между ними.

GameMaker Studio 2 выпущена 2 ноября 2016 года.
