## Pygame

### Разработка игр с Pygame

Pygame - это библиотека с открытым исходным кодом, кроссплатформенная библиотека python, используемая для разработки игр, написанная Пит Синнерс. Документацию Pygame и дополнительную информацию можно найти на своем веб-сайте по адресу https://pygame.org.

### обзор

Проект начался в 2000 году в результате гибели PySDL. Библиотека Pygame версии 1.0 была выпущенный после шести месяцев развития в апреле 2001 года.
