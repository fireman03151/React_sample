Эти глобальные ярлыки управления для Mac сэкономит вам часы, ускорив ввод.

[https://player.vimeo.com/video/107073108](https://player.vimeo.com/video/107073108)

Эти глобальные ярлыки работают повсюду на Mac:

*   Контроль + F = Вперед
*   Контроль + B = назад
*   Control + N = следующая строка
*   Control + P = Предыдущая линия
*   Контроль + H = Backspace
*   Control + D = Удалить
*   Контроль + A = начало линии
*   Control + E = конец строки
*   Control + K = строка убийства
