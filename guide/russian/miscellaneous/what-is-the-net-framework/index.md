[.NET Framework](https://www.microsoft.com/net) или просто .NET - многоцелевая многоязычная среда исполнения; созданный и принадлежащий [Microsoft](https://www.microsoft.com) .

Он имеет большую библиотеку классов, поддерживает языковую интерполируемость; и работает в основном на Windows.

Microsoft начала разработку .NET Framework в конце 1990-х годов и к 2000-м годам; выпущена бета-версия. Помимо разработки графического интерфейса Windows и разработки приложений для Windows Phone; .NET framework используется в веб-разработке.

Он работает без проблем с облачной платформой [Microsoft Azure](https://azure.microsoft.com/en-us/) и [базой](https://www.microsoft.com/en-us/server-cloud/products/sql-server/) данных [SQL Server](https://www.microsoft.com/en-us/server-cloud/products/sql-server/) . Он позволяет программировать серверную сторону на нескольких разных языках; преимущественно в [C #](https://msdn.microsoft.com/en-us/library/67ef8sbd.aspx) .

С [MSDN](https://msdn.microsoft.com/en-us/default.aspx) в качестве сообщества, богатого контентом, существует множество возможностей и преимуществ использования .NET для веб-разработки.
