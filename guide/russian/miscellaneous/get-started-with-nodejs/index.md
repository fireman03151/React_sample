[Node.js](https://nodejs.org/) - это среда выполнения JavaScript, встроенная [в механизм](https://developers.google.com/v8/ "Двигатель Chrome V8") JavaScript [V8 JavaScript,](https://developers.google.com/v8/ "Двигатель Chrome V8") позволяющий использовать JavaScript для серверного программирования.

Node.js стал ключевой частью «JavaScript везде», поощряя разработку веб-приложений для унификации вокруг одного языка программирования, а не полагаться на другой язык для написания сценариев на стороне сервера.
