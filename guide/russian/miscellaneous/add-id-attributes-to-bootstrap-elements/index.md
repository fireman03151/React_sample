---
title: Add Id Attributes to Bootstrap Elements
localeTitle: Добавить атрибуты идентификатора в элементы Bootstrap
---
Напомним, что помимо атрибутов класса вы можете указать каждому из ваших элементов атрибут id.

Каждый идентификатор должен быть уникальным для определенного элемента. Помните, что вы можете указать элемент id следующим образом:
```
<div class="well" id="center-well"> 

```
