## OpenID Connect Implicit Flow

Предоставляет механизм для браузеров приложений для получения токенов OAuth для взаимодействия с бэкэнд-услугами.

В частности, ссылка на: http://openid.net/specs/openid-connect-implicit-1\_0.html
