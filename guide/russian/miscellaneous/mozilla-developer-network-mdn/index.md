## Сеть разработчиков Mozilla (MDN)

[MDN](https://developer.mozilla.org/) - это обучающая платформа для веб-технологий и программного обеспечения, которое поддерживает Интернет.

Цель MDN - предоставить разработчикам информацию, необходимую им для легкого создания проектов в открытой сети. Если это открытая технология, открытая в Интернете, MDN хочет ее документировать.
