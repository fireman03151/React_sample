Пара-программирование - это то, где два человека соединяются друг с другом на одном компьютере.

![](//discourse-user-assets.s3.amazonaws.com/original/2X/f/f68cff8b5354f548f9d680369891ece79b8a154f.jpg)

Паровое программирование побуждает вас обсуждать различные подходы к решению проблем и поддерживать друг друга другим. Результат - лучший код, который вы могли бы написать сами.

Из-за своих преимуществ многие инженеры парируют программу на полный рабочий день. И это лучший способ научиться кодированию. Благодаря инструментам, позволяющим двум людям делиться входом мыши и клавиатуры, вы можете спрятать программу с другом, не находясь в одной комнате.

Вы можете испытать эти преимущества с помощью парного программирования с другими студентами из Free Code Camp в наших проблемах с кодированием. В конце концов, вы будете работать с людьми в некоммерческих организациях для создания реальных программных решений.
