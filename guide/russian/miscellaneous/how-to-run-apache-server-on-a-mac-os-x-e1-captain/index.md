---
title: How to Run Apache Server on a Mac Os X E1 Captain
localeTitle: Как запустить Apache Server на Mac Os X E1 Captain
---
Сервер [Apache](http://www.apache.org/) предварительно построен на Mac OS X, нет необходимости устанавливать сторонние инструменты WAMP, LAMP, MAMP и XAMPP для запуска сервера Apache на Mac.

Когда вы рассматриваете запуск сервера [Apache](http://www.apache.org/) в Mac OS X E1 Captain / OS X Yosemite, он сильно отличается от предыдущих версий. Здесь вы должны это сделать из окон терминала, ранее использовались параметры GUI для тика, поскольку веб-сервер из окна системного управления взлетел.

Введите в окне терминала:
```
httpd -v 
```

Он дает версию сервера и дату сборки.

Здесь http означает протокол гипертекстовой передачи d, предназначенный для Daemon, который является программным обеспечением для использования многозадачности, которое также используется в Mac OS X. `httpd` - это программа сервера Apache HyperText Transfer Protocol (HTTP). Он предназначен для запуска как отдельный процесс демон. Тип
```
sudo apachectl start 
```

в окне терминала и зайдите в свой браузер и введите `http://localhost` вы получите, `It works!` в вашем браузере.
