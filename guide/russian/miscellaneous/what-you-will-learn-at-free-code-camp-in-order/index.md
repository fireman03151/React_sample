---
title: What You Will Learn at Free Code Camp in Order
localeTitle: Что вы узнаете в свободном кодовом лагере в порядке
---
*   [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) - Структура веб-страниц.
*   [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) - визуальный стиль веб-страниц.
*   [Bootstrap](http://getbootstrap.com/) - инструмент «отзывчивого дизайна», который помогает вашим веб-сайтам отлично выглядеть на планшетах и ​​телефонах.
*   [jQuery](https://jquery.com/) и [Ajax](https://developer.mozilla.org/en-US/docs/AJAX) - простые инструменты для управления контентом в браузере.
*   [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - единственный язык программирования, который используют все веб-браузеры.
*   [Алгоритмы](https://en.wikipedia.org/wiki/Algorithm) - пошаговые рецепты для того, чтобы все было сделано.
*   [Git](https://git-scm.com/) - система контроля версий для сохранения и совместного использования ваших проектов.
*   [MongoDB](https://www.mongodb.org/) - популярная нереляционная база данных.
*   [D3.js](http://d3js.org/) - библиотека JavaScript для создания динамических интерактивных визуализаций данных в веб-браузерах.
*   [Express.js](http://expressjs.com/) - мощная структура веб-разработки.
*   [Node.js](https://nodejs.org/) - платформа для создания масштабируемых сетевых приложений.
*   [SASS](http://sass-lang.com/) - язык сценариев, который интерпретируется в каскадных таблицах стилей (CSS).
*   [API Development](https://en.wikipedia.org/wiki/Application_programming_interface) - набор подпрограмм, протоколов и инструментов для создания программного обеспечения и приложений.
*   [Agile](https://en.wikipedia.org/wiki/Agile_software_development) - набор принципов разработки программного обеспечения, которые фокусируют проектирование и производство проекта на потребностях его пользователей
