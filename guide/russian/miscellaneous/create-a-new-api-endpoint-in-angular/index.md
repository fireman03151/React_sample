> > yo angular-fullstack: конечная точка whatsit

Используя эту строку, вы получаете Yoman для автоматического создания другой конечной точки API и нового типа коллекции для вашей базы данных. Теперь у нас есть _вещи,_ а также _whatsits!_ Не стесняйтесь открыть **/server/api/whatsit/whatsit.model.js** и определите свой whatsit, как вам нравится.
