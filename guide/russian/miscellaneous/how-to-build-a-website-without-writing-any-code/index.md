---
title: How to Build a Website Without Writing Any Code
localeTitle: Как создать сайт без написания кода
---
В течение нескольких минут можно создавать динамичные мобильные мобильные сайты без написания каких-либо кодов.

Вот технологии, которые мы использовали:

*   [atom.io](http://www.atom.io/) бесплатный редактор кода
*   [startbootstrap.com](http://www.startbootstrap.com/) коллекция бесплатных [настраиваемых](http://www.startbootstrap.com/) шаблонов (Bootstrap)
*   [powr.io](http://www.powr.io/) коллекция бесплатных плагинов JavaScript
*   [bitballoon.com](http://www.bitballoon.com/) инструмент для перетаскивания веб-сайта

Вы быстро достигнете пределов того, что вы можете сделать, не кодируя, но приятно иметь возможность быстро создавать рабочие прототипы, подобные этим.
