Мы покажем вам, как установить различные программы langauges, devtools и веб-браузеры на Ubuntu.

## Широко используемые инструменты:

### Гит
```
sudo apt-get install git 
```

### Узел JS
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.0/install.sh | bash 
 nvm install stable 
```

### Рубин
```
curl -sSL https://get.rvm.io | bash -s stable 
 rvm install 2.3.1 
```

### питон

`Python 2` и `Python 3` устанавливаются на Ubuntu Desktop.

### Современные веб-браузеры

Наиболее популярными браузерами, используемыми разработчиками, являются [Chrome](https://www.google.com/chrome/) / [Chromium](https://www.chromium.org/) и [Firefox Developer Edition](https://www.mozilla.org/firefox/developer/) .

**Установка хрома:**
```
sudo apt-get install chromium-browser 
```

**Установка Firefox Developer Edition:**
```
sudo add-apt-repository ppa:ubuntu-mozilla-daily/firefox-aurora 
 sudo apt-get update 
 sudo apt-get install firefox 
```
