## Открыть данные

Открытые данные - это данные, которые свободно доступны, и каждый может _получить доступ, повторно использовать и поделиться_ им.

Для открытых данных характерны следующие моменты:

*   Данные становятся доступными благодаря доступности в общем понятном формате.
*   Данные должны быть доступны по открытой лицензии. Это позволяет любому использовать, распространять и изменять его.

Открытые данные могут помочь правительствам, предприятиям и отдельным лицам. Он может улучшить услуги, внести социально-экономические изменения и защитить планету.

Одним из способов использования открытых данных является использование Civic Hacking в вашем локальном коде для американской бригады.
