## логика

Логика в области информатики охватывает совпадение между полем логики и информатикой. Тема может быть разделена на три основные области:

Теоретические основы и анализ Использование компьютерной техники для помощи логикам Использование понятий из логики для компьютерных приложений

В этом разделе мы будем иметь руководства по широкому кругу логических головоломок, которые важны для математики и программирования.
