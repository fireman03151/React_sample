## Основное использование VIM

### Открыть файл

*   Запустите vim или vi и откройте данное имя файла.

### Режим вставки

*   После открытия страницы введите I, и вы увидите текст «Режим вставки» внизу экрана. Отсюда вы можете внести любые изменения в текст своего файла.

### Сохранить файл

*   : ш

### Сохранить и выйти

*   :Икс
*   SHIFT ZZ
*   : WQ

### Выйти из файла, если никаких изменений не было сделано

*   : д

### Выйти из файла и отменить все сделанные изменения

*   : Д!

### Отображать номера строк

*   : set nu

### Не отображать номера строк

*   : set nonu

### Добавить цвет синтаксиса на основе языка prog

*   : синтаксис

## О компании Vim

Vim - текстовый редактор, который в основном используется в режиме CLI. Но теперь редактор также доступен в различных версиях. Там есть также GVIM, который является графической версией VIM. vi был главным редактором, затем его улучшил и назвал его VI улучшенным Vim.
