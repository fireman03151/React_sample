## Основное использование VIM

### Открыть файл

*   Напечатайте vim или vi в терминале вместе с именем файла. 
```shell
$ vim my_file.txt 
```

### Режим ввода


*   После открытия файла вы не можете сразу напечатать текст в файл. Для этого нажимайте на клавишу ``i``, и вы увидите текст ``-- INSERT --``«Режим ввода» слева внизу. Теперь вы можете внести любые изменения в текст вашего файла.

### Сохранить файл

*   Если хотите сохранить файл, и вы находитесь в режиме ввода (см. выше), вам придется перейти в режим команды нажимая на клавишу ``Esc``. Для записи (write) файла нажимайте на ``: w``

### Сохранить и выйти

*   Перейдя в режим команды ``:Esc``
*   ``: WQ``, т.е. записать (write) и закрывать "выходить" (quit).
* Так же можно сразу записать и закрывать файл следующей командой
* ``:x``


### Закрыть файл, если никаких изменений не было сделано


*   В режиме команды нажимайте ``:q``.

### Выйти из файла и отменить все сделанные изменения

*   В режиме команды нажимайте ``:q!``. 
Внимание! если вы изменили файл и закрывайте файл таким образом, ваши изменения будут потеряны. Для сохранения файла см. выше.


### Отображать номера строк

*   :set nu

### Не отображать номера строк

*   :set nonu

### Добавить цвет синтаксиса на основе языка prog


*   : syntax on


## О Vim


Vim - текстовый редактор, который в основном используется в режиме CLI, т.е. прямо на эмуляторе терминала. Но теперь редактор также доступен в различных версиях. Там есть также GVIM, который является графической версией VIM. vi был главным редактором, затем Vim, что является "улучшенной версией" "**V**i **im**proved".
