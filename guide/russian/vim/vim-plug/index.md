## Вим-Штекер

Vim-Plug минималистский менеджер плагинов Vim.

### Плюсы:

*   Легче настроить: один файл. Не требуется код шаблона.
*   Легче использовать: краткий, интуитивный синтаксис.
*   Супер-быстрая параллельная установка / обновление (с любым `+job` `+python` , `+python` , `+python3` , `+ruby` или Neovim).
*   Создает мелкие клоны, чтобы минимизировать использование дискового пространства и время загрузки.
*   Загрузка по требованию для более быстрого запуска.
*   Может просматривать и откатывать обновления.
*   Поддержка Branch / tag / commit.
*   Пост-обновление крючков.
*   Поддержка плагинов, управляемых извне.

Вы можете найти руководство по установке [здесь](https://github.com/junegunn/vim-plug#installation) .

Некоторые полезные плагины для начала работы:

*   [nerdtree](https://github.com/scrooloose/nerdtree) - [Проводник](https://github.com/scrooloose/nerdtree) файловой системы для Vim.
*   [Emmet-vim](https://github.com/mattn/emmet-vim/) - поддержка расширений аббревиатур, похожих на [emmet](https://emmet.io/) .
*   [vim-polygot](https://github.com/sheerun/vim-polyglot) - твердый языковой пакет для Vim.
*   [Автопарки](https://github.com/jiangmiao/auto-pairs) - вставлять или удалять скобки, парсеры, котировки в паре.
*   [fugitive.vim](https://github.com/tpope/vim-fugitive) - обертка git для Vim.

Вы можете добавить дополнительные плагины для установки Vim.
