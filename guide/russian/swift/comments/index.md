## Комментарии

Комментарии позволяют документировать ваш код. Комментарии игнорируются компилятором Swift и полезны только для людей, читающих вашу базу кода.

Однострочные комментарии начинаются с двух перекосов `//` .

#### Пример:

```swift
    // Hello, single-line comment! 
```

Многострочные комментарии начинаются с прямой косой черты, за которой следует звездочка `/*` и заканчивается звездочкой, за которой следует прямая `*/`

#### Пример:

```swift
   /* Hello, 
   multiline-line comment! */ 
```
