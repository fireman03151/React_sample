## Комментарии

Комментарии позволяют документировать ваш код. Комментарии игнорируются компилятором Swift и полезны только для людей, читающих ваш код.

Однострочные комментарии начинаются с двух прямых слешей `//` .

#### Пример:

```swift
    // Привет, однострочный комментарий! 
```

Многострочные комментарии начинаются с прямого слеша, за которым следует звездочка `/*` и заканчиваются звездочкой, за которой следует прямой слеш `*/`

#### Пример:

```swift
   /* Привет, 
   многострочный комментарий! */ 
```
