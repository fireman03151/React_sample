## Планирование функций

**Планирование на основе функций** - это методология планирования, которая может быть использована для принятия решения о выпуске программного обеспечения на основе функций, которые будут доставляться клиентам, а не произвольного срока, основанного на выпуске.

В этом методе планирования выпуска команды определяют, какие функции / функции должны быть приоритетными. Исходя из объема этих функций, команда может предсказать, когда можно будет развернуть следующий выпуск.
