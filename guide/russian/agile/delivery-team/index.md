## Команда доставки

Команда схватки состоит из владельца продукта (PO), Scrum Master (SM) и команды доставки.

Команда доставки - это все, кроме ПО и СМ; разработчики, тестеры, системные аналитики, аналитики баз данных, аналитики UI / UX и т. д.

Для того чтобы команда по сбору навыков была эффективной, она должна быть проворной (или подвижной!), Достаточной для того, чтобы решения были согласованы быстро, в то время как они были достаточно разнообразными, поэтому подавляющее большинство спецификаций, необходимых для разработки программного обеспечения, может быть охвачено всей командой. Idealy, команда от 3 до 9 человек, как правило, хорошо работает для команды разработчиков схватки.

Команда также должна способствовать взаимному уважению всех членов команды. Эффективная команда схватки сможет разделить рабочую нагрузку и поставить успехи команды перед достижениями личности.
