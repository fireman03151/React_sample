## Bash command: unzip

**unzip will list, test, or extract files from a ZIP archive, commonly found on MS-DOS systems.**
```
unzip [-Z] [-cflptTuvz[abjnoqsCDKLMUVWX$/:^]] file[.zip] [file(s) ...] [-x xfile(s) ...] [-d exdir]
```
Example:

Easiest way to extract in current folder
```
unzip file.zip
```
