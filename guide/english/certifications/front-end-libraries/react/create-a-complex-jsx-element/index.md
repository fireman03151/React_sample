# Create a Complex JSX Element

## Hint
- nested JSX must return a single, top level element
- Ensure your solution has a parent `div` containing an `h1`, a `p`, and an unordered list (`ul`) that contains three `li` items

## Solution  

```html
const JSX = <div>
  <h1>Heading.</h1>
  <p>Paragraph</p>
 <ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
</div>;  
```
