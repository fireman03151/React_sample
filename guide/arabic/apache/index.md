## أباتشي

يعد خادم Apache HTTP ، المعروف باسم Apache ، خادم ويب مجانيًا ومفتوح المصدر عبر الويب ، تم إصداره بموجب شروط [Apache License 2.0](https://en.wikipedia.org/wiki/Apache_License) . تم تطوير Apache وصيانته بواسطة مجتمع مفتوح من المطورين تحت رعاية مؤسسة Apache Software Foundation.
