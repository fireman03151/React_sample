## Terraform

Terraform هي بنية تحتية برمجية برمجية بواسطة HashiCorp. يتيح للمستخدمين تحديد مركز البيانات البنية التحتية في لغة تكوين عالية المستوى ، والتي من خلالها يمكن إنشاء خطة التنفيذ ل بناء البنية التحتية في مزود خدمة مثل OpenStack أو في مزود خدمة مثل IBM Cloud (Bluemix سابقًا) أو AWS أو Microsoft Azure أو Google Cloud Platform. البنية التحتية محددة في صيغة جمل HCL Terraform أو تنسيق JSON.
