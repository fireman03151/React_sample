Clementine.js هي عبارة عن مؤشر نصفي مصمم خصيصًا لمُعسكرات FCC. يمكن العثور على إرشادات التثبيت الحديثة لبرامج التشغيل الرئيسية الثلاثة (Windows و OS X و Linux) على:
