---
title: Add Id Attributes to Bootstrap Elements
localeTitle: إضافة سمات الهوية إلى عناصر Bootstrap
---
تذكر أنه بالإضافة إلى سمات الصف ، يمكنك إعطاء كل عنصر من عناصرك خاصية المعرف.

يجب أن يكون كل معرف فريدًا لعنصر محدد. تذكر أنه يمكنك إعطاء عنصر معرف مثل هذا:

 `<div class="well" id="center-well"> 
`
