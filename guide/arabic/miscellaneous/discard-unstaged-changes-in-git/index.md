---
title: Discard Unstaged Changes in Git
localeTitle: تجاهل التغييرات غير المنتظمة في جيت
---
عند استخدام Git ، من الشائع إجراء التغييرات التي تريد إزالتها تمامًا قبل مرحلة التدريج. على سبيل المثال ، بعد العمل على بعض الملفات ، فإنك تدرك أنك تريد التراجع عن التغييرات التي تم إجراؤها على ملف معين. لإلغاء التغييرات قبل التدريج والالتزام ، استخدم أمر `$ git checkout` .

## لإفراغ ملف واحد:

`$ git checkout <path-to-file>`

_تذكر استبدال `<path-to-file>` باسم الملف الفعلي._

## لإلغاء تثبيت كافة الملفات:

`$ git checkout -- .`
