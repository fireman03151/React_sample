---
title: Change Text Inside an Element Using jQuery
localeTitle: تغيير النص داخل عنصر باستخدام jQuery
---
باستخدام jQuery ، يمكنك تغيير النص بين علامات البداية والنهاية للعنصر. يمكنك حتى تغيير ترميز HTML.

يحتوي jQuery على وظيفة تسمى `.html()` تسمح لك بإضافة علامات HTML ونص داخل عنصر. سيتم استبدال أي محتوى سابق داخل العنصر تمامًا بالمحتوى الذي تقدمه باستخدام هذه الوظيفة.

في ما يلي كيفية كتابة نص العنوان وتعيينه بخط مائل:

 `$("h3").html("<em>jQuery Funhouse</em>"); 
` 

يحتوي jQuery أيضًا على دالة مشابهة تُسمى `.text()` تقوم فقط بتغيير النص دون إضافة علامات. لذلك عند استخدام `.html()` يرجى تذكر أنك ستقوم بتحرير الترميز بالكامل وليس النص فقط.
