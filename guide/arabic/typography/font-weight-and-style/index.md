## وزن الخط والنمط

يمكن كتابة خط الوزن على هيئة قيم نصية:

 `font-weight: normal; 
 font-weight: bold; 
` 

أو كقيمة رقمية من `100` إلى `900` (بمضاعفات 100):

 `font-weight: 400;  /* equal to 'normal' above */ 
 font-weight: 700; /* equal to 'bold' above */ 
` 

القيمة العددية ووصفها المشترك

| القيمة | وصف عام | | ----- | ---------------------------- | | 100 | رقيقة (شعري) | | 200 | اكسترا لايت (الترا لايت) | | 300 | ضوء | | 400 | عادي | | 500 | متوسطة | | 600 | نصف بولد (ديمي بولد) | | 700 | جريئة | | 800 | جريئة اضافية (الترا بولد) | | 900 | أسود (ثقيل) |

ليست كل الأوزان متوفرة لجميع الخطوط. قد لا تتوفر بعض الخطوط المتخصصة إلا في وزن واحد ( `400` `normal` بشكل عام).

يمكن أيضًا تحديد وزن الخط بالنسبة إلى أصل أحد العناصر (إذا كان الخط يحتوي على أكثر من وزن):

 `font-weight: lighter; 
 font-weight: bolder; 
`
