## SQL

SQL لتقف على لغة الاستعلام الهيكلية. إنها الأداة الأكثر شيوعًا المستخدمة لمعالجة البيانات وإدارتها في قاعدة بيانات علائقية (يشار إليها غالبًا باسم "قاعدة بيانات SQL").

SQL هو واضح بشكل عام "تتمة". أكثر أشكاله شيوعًا هي MySQL و PostgreSQL و SQLite - وهي نسخة من SQL تستخدم عادةً في تصميم النماذج. وقدم مفهوم الوصول إلى العديد من السجلات باستخدام أمر واحد ، باستخدام استعلامات SQL.
