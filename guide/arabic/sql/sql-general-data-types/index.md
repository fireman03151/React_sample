## أنواع البيانات العامة SQL

# أنواع بيانات SQL

كل عمود في جدول قاعدة البيانات مطلوب أن يكون له اسم ونوع بيانات.

يجب على مطور SQL تحديد نوع البيانات التي سيتم تخزينها داخل كل عمود عند إنشاء جدول. يعتبر نوع البيانات دليلاً لـ SQL لفهم نوع البيانات المتوقع داخل كل عمود ، كما أنه يحدد كيفية تفاعل SQL مع البيانات المخزنة.

# MySQL Data Types

تتكون أنواع البيانات العامة لـ SQL مما يلي:

1.  قيمة تستند إلى نص و / أو عددي ، يشار إليها غالبًا باسم STRING
2.  قيمة رقمية فقط ، يشار إليها غالبًا باسم INTEGER
3.  قيمة تستند إلى التقويم و / أو الساعة ، وغالبًا ما يتم التعبير عنها إلى DATE أو TIME
4.  قيمة خاصة بقاعدة البيانات مثل علامة منطقية (خيارين) ، صفيف يقوم بتخزين قيم متعددة في نقطة بيانات واحدة
