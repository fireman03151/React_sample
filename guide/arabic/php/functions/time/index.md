## زمن

ترجع الدالة `time()` الطابع الزمني unix الحالي (عدد الثواني منذ تاريخ Unix - يناير 1 1970 00:00:00 GMT).

### مثال

```php
<?php
echo time();
``` 

**انتاج:**

```text
1511732226
```
