## يموت والخروج

وظائف `die()` و `exit()` متطابقة. يأخذ كل منهم وسيطة واحدة (سلسلة) تحتوي على رسالة خطأ. عند تشغيلها يقومون بإخراج الرسالة ووقف تنفيذ البرنامج النصي فوراً.

 `<?php 
 die('Die() function was run'); 
` 

 `<?php 
 exit('Exit() function was run'); 
`
