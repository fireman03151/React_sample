## MongoDB

MongoDB هو قاعدة بيانات غير علائقية مفتوحة المصدر تستخدم بنية JSON مثل لتخزين البيانات. ويستخدم نموذج مستند لتخزين البيانات واستردادها بدلاً من نموذج الجدول المستخدم في قواعد البيانات المتسلسلة مثل SQL أو Oracle.

### NOSQL Vs RDBMS

| شروط ومفاهيم MongoDB | شروط ومفاهيم SQL | --- | --- | | قاعدة البيانات | قاعدة البيانات | | جمع | الجدول | وثيقة الصف | | المجال العمود | الفهرس | الفهرس | | المستندات المدمجة | الجدول ينضم |

### خصائص MongoDB

1.  قاعدة بيانات الجيل التالي
2.  لا تنضم
3.  تجمع
4.  مفتوحة المصدر
5.  مخطط أقل
6.  لا توجد علاقات

### ملامح MongoDB

1.  قاعدة بيانات الوثائق
2.  أداء عالي
3.  لغة الاستعلام الغنية
4.  توافر عالية
5.  قابلية التوسع الأفقي
