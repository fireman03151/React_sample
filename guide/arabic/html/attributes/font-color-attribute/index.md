## سمة لون الخط

يتم استخدام هذه السمة لتعيين لون للنص المحاط بعلامة `<font>` .

### بناء الجملة:

\`\` أتش تي أم أل \`\` \`

### مهم:

هذه السمة غير مدعومة في HTML5. بدلاً من ذلك ، [تحدد](https://guide.freecodecamp.org/css/colors) هذه [المقالة freeCodeCamp](https://guide.freecodecamp.org/css/colors) طريقة CSS ، والتي يمكن استخدامها.

### ملحوظة:

يمكن أيضًا تحديد لون باستخدام "رمز hex" أو "رمز rgb" ، بدلاً من استخدام اسم.

### مثال:

1.  سمة اسم اللون

 `
<html> 
  <body> 
   <font color="green">Font color example using color attribute</font> 
 </body> 
 </html> 
` 

2.  سمة سداسي عشري

```html
<html>
  <body>
   <font color="#00FF00">Font color example using color attribute</font>
  </body>
</html>
``` 

3.  سمة RGB

```html
<html>
  <body>
    <font color="rgb(0,255,0)">Font color example using color attribute</font>
  </body>
</html>
```
