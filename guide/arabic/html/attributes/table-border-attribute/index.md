## سمة حدود الجدول

سمة علامة علامة `<table>` هي طريقة لترتيب حد إلى جدول HTML. القيم المدعومة هي `0` (بدون حد) و `1` (حد). سمة الحدود غير مدعومة في HTML 5 أحدث إصدار من HTML. يوصى باستخدام CSS لإضافة حد إلى جدول في HTML 5.
