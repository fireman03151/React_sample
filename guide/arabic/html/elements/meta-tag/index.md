## علامة متغيرة

توفر العلامة `<meta>` البيانات التعريفية حول مستند HTML.

يتم استخدام البيانات الوصفية هذه لتحديد مجموعة الحروف والوصف والكلمات الرئيسية والمؤلف وإطار العرض الخاص بالصفحة.

لن تظهر هذه البيانات الوصفية على موقع الويب نفسه ، ولكن سيتم استخدامها من قبل المستعرضين ومحركات البحث لتحديد كيفية عرض الصفحة للمحتوى وتقييم تحسين محرك البحث (SEO).

يتم تعيين بيانات التعريف إلى من وثيقة HTML:

 `
<head> 
  <meta charset="UTF-8"> 
  <meta name="description" content="Short description of website content here"> 
  <meta name="keywords" content="HTML,CSS,XML,JavaScript"> 
  <meta name="author" content="Jane Smith"> 
  <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
 <!-- HTML5 introduced a method to let web designers take control over the viewport, through the <meta> tag. The viewport is the user's visible area of a web page. A <meta> viewport element gives the browser instructions on how to control the page's dimensions and scaling. --> 
 </head> 
`
