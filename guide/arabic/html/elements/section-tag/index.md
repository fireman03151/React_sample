## الجزء

يعرّف عنصر HTML `<section>` قسمًا داخل صفحة HTML يتم استخدامها عندما لا يكون هناك عنصر HTML لغوي محدد أكثر لتمثيله. عادة ، سيتضمن عنصر `<section>` عنصر عنوان ( `<h1>` - `<h6>` ) كعنصر فرعي.

على سبيل المثال ، يمكن تقسيم صفحة الويب إلى أقسام مختلفة مثل أقسام الترحيب والمحتوى والاتصال.

لا ينبغي استخدام عنصر `<section>` بدلاً من عنصر `<div>` إذا كانت هناك حاجة إلى حاوية عامة. يجب استخدامه لتعريف الأقسام في صفحة HTML.

 `
<html> 
 <head> 
  <title>Section Example</title> 
 </head> 
 <body> 
  <section> 
    <h1>Heading</h1> 
    <p>Bunch of awesome content</p> 
  </section> 
 </body> 
 </html> 
`
