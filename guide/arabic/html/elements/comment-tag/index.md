## علامة التعليق

التعليقات هي علامات تُستخدم لمغادرة الملاحظات في المستند. يحق للمطور فقط الوصول إلى التعليقات ، ولا يرى المستخدم النهائي ما لم يتحقق مع Inspector Element الخاص بالمتصفح.

عندما تعمل في بعض الأكواد ، من المفيد ترك نصائح للمطورين الآخرين لفهم طبيعة العمل.

### مثال

```html
<!-- This is a comment -->
``` 

يمكن أيضًا استخدام التعليقات لجعل الشفرة غير نشطة دون الحاجة إلى حذفها بالكامل.

### مثال

```html
<!--
<h1>Hello Campers</h1>
-->
<h2>Hello Friends</h2>
<!--
<p>Hello Paragraph</p>
-->
```
