## أنواع المدخلات

يمكن تخصيص عناصر HTML لإدخال سمة تسمى "type". يحدد نوع عنصر الإدخال نوع البيانات التي يمكن للمستخدم إدخالها. الإدخال الافتراضي هو النص ، ولكن الخيارات الأخرى تحتوي على مربعات الاختيار أو الأزرار البسيطة أو اللون (باستخدام الرموز السداسية).
