## أضف تعليقات في JSX

يمكنك التعليق كما اعتدت القيام به مع كتل التعليمات البرمجية في جافا سكريبت `/* some JS code */` ولكنها تحتاج إلى ملفوفة بواسطة الأقواس المتعرجة لإضافة تعليقات في JSX: فمثلا:

*   تعليق سطر واحد:

 `{/*<h1>Hanoi University of Science</h1>*/} 
` 

*   تعليقات متعددة الخطوط:

 `{/* 
 <h1>Hanoi University of Science</h1> 
 <p>Falculty of Mathematics, Mechanics and Informatics</p> 
 */} 
` 

ملاحظة: لا يمكنك استخدام تعليق HTML في JSX على النحو التالي:

 `
<!-- not working --> 
`
