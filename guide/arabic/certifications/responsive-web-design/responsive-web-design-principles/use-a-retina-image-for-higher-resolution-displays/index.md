## استخدم صورة شبكية العين لعرض أعلى دقة

اتباع التعليمات:

اضبط عرض وارتفاع علامة img على نصف قيمها الأصلية. في هذه الحالة ، يبلغ طول كل من الارتفاع الأصلي والعرض الأصلي 200 بكسل.

يصبح النمط 4:

 `  <style> 
  img { height: 100px; width: 100px; } 
  </style> 
` 

لاحظ أن أبسط طريقة لجعل صورك تظهر "شبكية العين" (وتحسينها لشاشات عرض الشبكية) هي تحديد قيم العرض والارتفاع الخاصة بها كنصف ما هو الملف الأصلي فقط.
