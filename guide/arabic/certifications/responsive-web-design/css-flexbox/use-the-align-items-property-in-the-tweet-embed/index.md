## استخدم الخاصية align-items في Tweet Tweet

على [غرار التحدي السابق](https://github.com/freecodecamp/guides/tree/master/src/pages/certifications/responsive-web-design/css-flexbox/align-elements-using-the-align-items-property/index.md) ، ستحتاج إلى تعيين قيمة الخاصية إلى `center` على اليمين.
