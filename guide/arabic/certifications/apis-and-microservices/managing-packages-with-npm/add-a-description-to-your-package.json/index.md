## أضف وصفًا إلى الحزمة الخاصة بك. json

لإكمال التحدي ، اتبع الخطوات التالية:

*   انتقل إلى الرابط الوارد في الصفحة التمهيدية لتدريس المناهج الدراسية. ![freecodecamp apis-and-microservices intro](https://user-images.githubusercontent.com/15084301/45616545-65732480-ba8d-11e8-9893-412ff6c68be5.png) [رابط المشروع خلل بداية](https://glitch.com/#!/import/github/freeCodeCamp/boilerplate-npm)
    
*   قم بتحرير ملف package.json وتحديث المؤلف
    
*   ثم انقر فوق الارتباط (التطبيق) الموجود في الأعلى الموجود في مشروع Glitch للحصول على الرابط العام ![ارتباط المشروع العام خلل](https://user-images.githubusercontent.com/15084301/45616579-8176c600-ba8d-11e8-8a46-4ac930578e74.png)
    
*   يقدم الرابط في الحل وهذا كل شيء. ![صفحة نجاح](https://user-images.githubusercontent.com/15084301/45616757-019d2b80-ba8e-11e8-8b5b-98dc6d02c235.png)
