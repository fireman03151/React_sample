## تكرار مع JavaScript Do… While Loops

*   `Do...While` الحلقات بالتأكد من تنفيذ الشفرة مرة واحدة على الأقل ، وبعد التنفيذ ، إذا كان الشرط الموجود داخل `while()` **صحيحًا** ، فإنه يستمر بالحلقة ، وإلا فإنه يتوقف.

## حل

```javascript
var myArray = [];
var i = 10;

do {
  myArray.push(i);
  i++;
} while(i <= 10);
```
