## استخدم تدرج قوس لإيجاد الحرف الأول في سلسلة

تذكر أن الحرف الأول لسلسلة ما يكون في الموضع الصفري. فمثلا:

 `var str = "Hello"; 
 var letter = str[0]; // This equals "H" 
`
