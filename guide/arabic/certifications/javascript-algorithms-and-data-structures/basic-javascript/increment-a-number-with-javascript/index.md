## زيادة رقم باستخدام JavaScript

يمكنك بسهولة زيادة (إضافة واحد) إلى متغير أرقام باستخدام عامل الزيادة "++". فمثلا:

 `var a = 6; 
 a++; // Now, 'a' is equal to 7 -- post-fixing 
 ++a; // Now, 'a' is equal to 8 -- pre-fixing 
`
