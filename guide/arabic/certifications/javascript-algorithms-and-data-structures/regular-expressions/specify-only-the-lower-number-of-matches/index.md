المشكلة تغيير haRegex regex لتتطابق مع الكلمة "Hazzah" فقط عندما يحتوي على أربعة أحرف z أو أكثر.

حل let haStr = "Hazzzzah"؛ let haRegex = / Haz {4،30} ah /؛ // قم بتغيير هذا الخط اسمحوا النتيجة = haRegex.test (haStr) ؛
