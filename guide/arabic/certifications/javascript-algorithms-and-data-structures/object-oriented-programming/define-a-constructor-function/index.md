## تحديد وظيفة منشئ

### طريقة:

يجب كتابة الدالة `Dog()` بالتنسيق نفسه تمامًا مثل الدالة `Bird()` المعطاة في المثال. ببساطة استبدال `Bird` مع `Dog` لتمرير جميع حالات الاختبار.

### حل:

```javascript
function Dog() {
  this.name = "Geogre",
  this.color = "White",
  this.numLegs = 4;
}
```
