---
title: Free Code Camp Deployment Workflow
localeTitle: مدونة مجانية كامب نشر العمل
---
1.  يتم كتابة جميع الرموز ودفعها إلى فرع على شوكة المطورين الخاصة في Free Code Camp.
2.  يقوم المطورون بإنشاء طلبات سحب ، وسحب رمز من فروعهم إلى فرع التدبير في Free Code Camp.
3.  شخص آخر غير الشخص الذي قام بتقديم طلب السحب لطلب ضمان الجودة ويقبله إذا كان كل شيء على ما يرام. وإلا ، فإنها تغلق طلب السحب مع توضيح سبب عدم كونه مثاليًا ، ويفتح المطور طلب سحب آخر بمجرد إصلاح الأشياء.
4.  يتم نشر Master إلى freecodecamp.com بشكل دوري.

## ملاحظات:

*   هذا قابل للتغيير بمجرد حصولنا على تغطية إختبار أفضل وعمل تكامل مستمر.
