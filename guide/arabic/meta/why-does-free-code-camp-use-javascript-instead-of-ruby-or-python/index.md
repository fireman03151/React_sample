---
title: Why Does Free Code Camp Use JavaScript Instead of Ruby or Python
localeTitle: لماذا يستخدم Free Code Camp استخدام JavaScript بدلاً من Ruby أو Python
---
مثل JavaScript ، فإن Ruby و Python هي لغات برمجة نصية عالية المستوى والتي يمكن استخدامها لتطوير موقع الويب بالكامل.

ولكن حتى لو تعلمت هذه اللغات ، فستظل بحاجة إلى تعلم JavaScript. ذلك لأن JavaScript هي اللغة الوحيدة التي تعمل في متصفحات الويب. ظلت جافا سكريبت موجودة منذ 20 عامًا ، ولا تزال تتزايد شعبيتها.

ولهذا السبب ، تحتوي JavaScript على أدوات وموارد تعلم عبر الإنترنت أكثر من أي لغة أخرى.
