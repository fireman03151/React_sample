## Map.prototype.size

لعرض عدد العناصر في كائن `Map` .

## بناء الجملة

```javascript
myMap.size();
``` 

## مثال

```javascript
const myMap = new Map();
myMap.set('foo',1);
myMap.set('bar',2);
myMap.set('baz',3);


myMap.size(); // 3
```
