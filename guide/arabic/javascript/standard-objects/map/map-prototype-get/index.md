## Map.prototype.get

لعرض قيمة المفتاح المحدد من كائن `Map` .

## بناء الجملة

```javascript
myMap.get(key);
``` 

## المعلمات

**المفتاح** المطلوب.

## مثال

```javascript
const myMap = new Map();
myMap.set('foo',1);
myMap.set('bar',2);
myMap.set('baz',3);

myMap.get('foo');   // returns 1
myMap.get('baz');   // returns 3
myMap.get('hihi');  // return undefined
```
