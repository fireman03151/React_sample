## التاريخ الآن

إرجاع الأسلوب Date.now () عدد الميلي ثانية انقضاء منذ 1 يناير 1970 00:00:00 UTC.

### بناء الجملة

 `var timeInMs = Date.now(); 
` 

### مثال

 `Date.now(); 
 // 1508783660969 
`
