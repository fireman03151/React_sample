## Date.prototype.getFullYear

إرجاع الأسلوب Date.getFullYear () السنة (أربعة أرقام) من التاريخ المحدد وفقًا للوقت المحلي.

### بناء الجملة

 `dateObj.getFullYear() 
` 

### مثال

 `var date = new Date(); 
 // creates a new Date() object with current date and time. 
 date.getFullYear() 
 // 2018 
`
