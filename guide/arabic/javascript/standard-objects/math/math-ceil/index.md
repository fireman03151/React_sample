## الرياضيات Ceil

تعد `Math.ceil()` طريقة للكائن القياسي في Math تقريب رقم محدد لأعلى إلى العدد الصحيح التالي. لاحظ أنه بالنسبة للأرقام السالبة فهذا يعني أن الرقم سيتم تقريبه "نحو 0" بدلاً من عدد القيمة المطلقة الأكبر (انظر الأمثلة).

### أمثلة

 `Math.ceil(0.1)  //  1 
 Math.ceil(1.3)  //  2 
 Math.ceil(-0.9) // -0 
 Math.ceil(-1.5) // -1 
`
