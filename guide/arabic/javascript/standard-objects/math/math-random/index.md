## الرياضيات عشوائي

إرجاع `Math.random()` رقم بين 0 (ضمناً) و 1 (خاص).

#### بناء الجملة

`Math.random()`

#### مثال

 `function randomInRange(min, max) { 
  return Math.random() * (max - min) + min; 
 } 
`
