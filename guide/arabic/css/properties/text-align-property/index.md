## نص محاذاة الخاصية

تحدد خاصية محاذاة النص المحاذاة الأفقية للنص في عنصر.

 `    .text-align { 
        /* List of keyword values of text-align property*/ 
        text-align: left|right|center|justify|initial|inherit; 
    } 
` 

#### معلومات اكثر:

المستندات: [MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/text-align)
