## تحوم المختار

يعد CSS `:hover` selector واحدًا من العديد من الفئات الزائفة التي يتم استخدامها لعناصر النمط.

يتم استخدام محدد hover: select لتحديد العناصر عند الفأرة فوقها.

يمكن استخدام محدد hover على جميع العناصر ، وليس فقط على الروابط.

استخدم: محدد الرابط إلى ارتباطات النمط إلى الصفحات غير المتوقعة ، والمنتقي: زيارة للانتقال إلى الارتباطات للصفحات التي تمت زيارتها ، والمحدد النشط لأسلوب الارتباط النشط.

ملاحظة:: يجب أن يكون تمرير الماوس بعد: الرابط و: زيارة (إذا كانت موجودة) في تعريف CSS ، لكي تكون فعالة!

CSS البناء :يحوم { اعلانات css }

يطبق محدد التمرير فقط الأنماط المتوفرة في القاعدة عندما يدخل العنصر في حالة التمرير. هذا هو عندما يمرر المستخدم فوق العنصر باستخدام الماوس الخاص به.

```css
button {
  color: white;
  background-color: green;
}

button:hover {
  background-color: white;
  border: 2px solid green;
  color: green;
}
``` 

في المثال أعلاه ، يكون التصميم العادي للزر عبارة عن نص أبيض على زر أخضر. عندما يمرر المستخدم فوق الزر باستخدام الماوس الخاص به ، فإن القاعدة مع `:hover` سوف تصبح نشطة ، وسوف يتغير نمط الزر.
