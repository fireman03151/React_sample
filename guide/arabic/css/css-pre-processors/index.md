## معالجات CSS المسبقة

البريبروسيسور (المعالج المسبق) هو برنامج حاسوب يسمح باستخدام وظائف CSS موسعة تتضمت إمكانية استخدام المتغيرات, الدوال البرمجية, المشغلات (معاملات رياضية) بالإضافه إلى الكثير من الأصول التي تفتقدها CSS

## معالجات مسبقة شعبية

*   [لٍس](http://lesscss.org)
*   [مرقم](http://stylus-lang.com)
