## نوع

يحدد Selector Selector العناصر حسب اسم العقدة (tag).

#### كود النحو

 `* { 
    css declarations; 
 } 
` 

#### مثال الكود

 `h1 { 
    background-color: red; 
 } 
` 

يختار كل عنصر h1 وتعيين لون الخلفية باللون الأحمر.

 `div { 
    width: 100px; 
 } 
` 

يختار كل عنصر div وعرض عنصر العرض 100 بكسل.
