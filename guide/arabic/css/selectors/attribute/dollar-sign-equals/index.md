## علامة الدولار تساوي

علامة الدولار محدد CSS محدد أو $ يستخدم لتحديد السمات التي تنتهي بقيمة محددة.

#### مثال

إذا كنت تريد البحث في جميع روابط الربط بنوع الملف .pdf وجعلها حمراء ، فيمكنك القيام بذلك على النحو التالي:

```css
a[href$=".pdf"] {
   color: red;
}
```
