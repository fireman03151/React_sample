#W3 المغلق الحاويات يوفر W3.css فئة الحاوية التي تضيف 16 بيكسل قيمة الحشو اليسرى واليمنى لأي عنصر html و 0.01em من الحشو العلوية والسفلية. وهذا يجعله مثاليًا لعناصر html التي تحتوي على عناصر مثل `<header>, <div>, <footer>, <section>` إلخ. ولكن يمكن استخدامها أيضًا مع تأثير كبير على عناصر مثل `<img> and <p>.`

### مثال بسيط

 `<header class="w3-container"> 
  <h1>Header With W3 Container</h1> 
 </header> 
` 

 `<header> 
  <h1>Header Without W3 Container</h1> 
 </header> 
`

### لاستخدام حاويات W3.Css

ما عليك سوى إضافة رابط ورقة أنماط عادية إلى رأس صفحة الويب مثل:

 `<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css"> 
` 

أو قم بتنزيل أحدث ملف css إلى جهازك المحلي. هنا: (https://www.w3schools.com/w3css/4/w3.css)
