## أمر Bash: mv

**ينقل الملفات والمجلدات.**

```
mv source target
mv source ... directory
``` 

الوسيطة الأولى هي الملف الذي تريد نقله ، والثاني هو الموقع الذي تريد نقله إليه.

الخيارات الشائعة الاستخدام:

*   `-f` لفرض نقلها والكتابة فوق الملفات دون التحقق مع المستخدم.
*   `-i` لتأكيد التأكيد قبل الكتابة فوق الملفات.
