## تخفيض السعر

Markdown هي لغة ترميز خفيفة الوزن مع صيغة تنسيق النص العادي. وهي مصممة بحيث يمكن تحويلها إلى HTML والعديد من الأشكال الأخرى باستخدام أداة بنفس الاسم. وكثيرا ما يستخدم Markdown لتهيئة ملفات README ، لكتابة الرسائل في منتديات المناقشة عبر الإنترنت ، ولإنشاء نص منسق باستخدام محرر نص عادي. يمكنك أيضًا كتابة Markdown في محرر النص (مثل Sublime Text أو Visual Studio Code). يكتسب Markdown في الرؤية والشعبية من خلال ظهور محررات نصية مبسطة "خالية من الانحرافات".
