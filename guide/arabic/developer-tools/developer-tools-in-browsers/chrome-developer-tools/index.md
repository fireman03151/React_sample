## أدوات تطوير فايرفوكس كروم

تساعد أدوات مطوري Chrome و Firefox مطوري الويب في تحليل HTML و CSS وجافا سكريبت على صفحة الويب. فهي مفيدة لتصحيح الأخطاء ، وتحسين الشفرة ، وفحص العناصر ، وأكثر من ذلك بكثير.

**ميزات:**

*   **مفتش الصفحة:** عرض وتحرير محتوى الصفحة والتخطيط.
*   **وحدة تحكم الويب:** انظر javascript console.log () الرسائل و erros
*   **مصحح جافا سكريبت Javascript:** عرض وتعديل JavaScript يعمل على صفحة
*   **مراقبة الشبكة:** راجع طلبات الشبكة التي يتم إجراؤها عند تحميل الصفحات
*   **أدوات الأداء:** تحليل موارد الكمبيوتر المستخدمة من قبل الموقع

### الوصول إلى أدوات التطوير

1.  في شريط القائمة بأعلى الشاشة ، حدد View> Developer> Developer Tools. ستظهر الأدوات في الجزء السفلي أو الجانبي أو كنافذة منبثقة.
2.  على صفحة الويب ، انقر بزر الماوس الأيمن على عنصر. عندما تظهر القائمة اختر "فحص".
