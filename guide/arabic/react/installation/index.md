## تثبيت رد الفعل

### إنشاء مشروع React جديد

يمكنك فقط تضمين مكتبة React في صفحتك على الويب مثل 2 :

 `
<script src="https://cdnjs.cloudflare.com/ajax/libs/react/16.0.0/cjs/react.production.min.js"></script> 
` 

يريد المبرمجون الأذكياء أن يتخذوا طريقة أكثر عملية وإنتاجية: [إنشاء تطبيق React](https://github.com/facebookincubator/create-react-app)

 `npm install -g create-react-app 
 create-react-app my-app 
 
 cd my-app 
 npm start 
` 

سيؤدي ذلك إلى إعداد بيئة التطوير الخاصة بك بحيث يمكنك استخدام أحدث ميزات JavaScript ، وتوفير تجربة رائعة للمطورين ، وتحسين تطبيقك للإنتاج.

`npm start` خادم تطوير يسمح `npm start` التحميل المباشر 3 .

بعد الانتهاء من المشروع وتكون جاهزًا لنشر تطبيقك على الإنتاج ، يمكنك استخدامه فقط `npm run build` لخلق بناء الأمثل تطبيقك في `build` المجلد.
