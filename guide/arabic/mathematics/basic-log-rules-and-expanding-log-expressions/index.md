## قواعد السجل الأساسية وتوسيع تعبيرات السجل

فيما يلي قواعد السجل الأساسية شائعة الاستخدام:

###### اللوغاريتم المنتج القاعدة

log b (x \* y) = log b (x) + log b (y)

###### قاعدة اللوغاريتم المتساهلة

log b (x / y) = log b (x) - log b (y)

###### لوغاريتم سلطة القاعدة

log b (x y ) = y \* log b (x)

###### لوغاريتم الرقم السالب

log b (x) غير معرّف عند x <0

###### لوغاريتم 0

log b (0) غير معرف

lim x → 0+ log b (x) = -infinity

###### لوغاريتم 1

log b (1) = 0

###### لوغاريتم للقاعدة

log b (b) = 1

###### لوغاريتم انفينيتي

lim x → infinity log b (x) = اللانهاية

###### لوغاريثم قاعدة تبديل القاعدة

log b (c) = 1 / log c (b)

###### قاعدة تغيير قاعدة لوغاريتم

log b (x) = log c (x) / log c (b)
