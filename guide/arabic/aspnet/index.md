## ASP.NET

# نظرة عامة

ASP.Net هي عبارة عن منصة لتطوير الويب مقدمة من Microsoft. يتم استخدامه لإنشاء تطبيقات ومواقع ويب تستند إلى الويب باستخدام HTML و CSS وجافا سكريبت كواجهة أمامية. يمكن استخدام لغات البرمجة من جانب الخادم مثل C # و VB .NET لرمز النهاية الخلفية.

تقدم ASP.NET أطر عمل مختلفة لإنشاء تطبيقات الويب: على سبيل المثال نماذج الويب و ASP.NET MVC.
