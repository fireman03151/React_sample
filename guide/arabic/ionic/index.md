## الإطار الأيوني

أيوني هو إطار تطوير تطبيقات الجوال HTML5 الذي يستهدف بناء تطبيقات الجوال المختلطة.

تتمتع التطبيقات المختلطة بالعديد من الفوائد على التطبيقات الأصلية الخالصة ، خاصة فيما يتعلق بدعم النظام الأساسي وسرعة التطوير والوصول إلى رمز الطرف الثالث.

### بناء تطبيقات الهجين مع الأيونية

أولئك الذين يعرفون تطوير الويب سيجدون بنية التطبيق الأيوني بشكل مباشر. في جوهرها ، إنها مجرد صفحة ويب تعمل في غلاف التطبيق الأصلي! وهذا يعني أنه يمكننا استخدام أي نوع من أنواع HTML و CSS وجافا سكريبت التي نريدها.

ستتم كتابة الجزء الأكبر من التطبيق الأيوني بلغة HTML و جافا سكريبت و CSS. يستخدم Ionic AngularJS أيضًا للعديد من الوظائف الأساسية للإطار.

### قبل أن نبدأ اللعب مع أيوني. دعونا نعد نظامنا أولا:

 `  nodeJS and npm 
  Ionic 2 
  Cordova 
  Android Environment (or iOS if you're working on a MacOS) 
` 

### أول شيء أولاً ، نحن بحاجة إلى Node.js و npm.

 `  sudo apt-get update 
  sudo apt-get install nodejs 
`
