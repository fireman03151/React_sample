title: Python Numeric Types
localeTitle: بايثون أنواع رقمية
---
[الأنواع الرقمية](https://docs.python.org/3/library/stdtypes.html#numeric-types-int-float-complex) لبيثون هي:

*   الأعداد الصحيحة ( `int` )

#### أمثلة:

 `print(1) 
 print(100) 
 print(8435367) 
` 

*   أرقام النقطة العائمة ( `float` )
    *   [البناء](https://docs.python.org/3/library/functions.html#float)

#### أمثلة:

 `print(1.5) 
 print(46.84) 
 print(84357.435) 
` 

*   ارقام مركبة
    *   [البناء](https://docs.python.org/3/library/functions.html#complex)

تضيف المكتبة القياسية أنواعًا رقمية لـ

*   [الكسور](https://docs.python.org/3/library/fractions.html#module-fractions)
*   [الكسور العشرية](https://docs.python.org/3/library/decimal.html#module-decimal)

يتم إنشاء كائنات رقمية من القيم الحرفية أو كنتيجة للوظائف والمشغلين. يتم [توثيق](https://docs.python.org/3/reference/lexical_analysis.html#numeric-literals) بناء الجملة من أجل القيم الحرفية الرقمية بشكل جيد.
