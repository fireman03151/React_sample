## الخطوات التي يجب اتباعها لإرسال الملف.

1.  تأكد من أجهزة الكمبيوتر المتصلة من خلال نفس الشبكة عبر الشبكة المحلية أو WIFI.
2.  افتح المحطة الطرفية في Ubuntu وتأكد من تثبيت python على جهاز الكمبيوتر الخاص بك.
3.  إذا لم يتم تثبيت ثم تثبيته عن طريق كتابة في "sudo apt-get python install" الطرفية دون الاقتباس.
4.  انتقل إلى الدليل الذي تريد مشاركة الملف الخاص به باستخدام الأمر cd (تغيير الدليل).
5.  اكتب هذا الأمر "python -m simpleHTTPserver" بدون علامات اقتباس.
6.  افتح محطة جديدة واكتب ifconfig واعثر على عنوان IP الخاص بك.

## الآن في الكمبيوتر الثاني

1.  افتح المتصفح واكتب عنوان IP الأول.
2.  لا تنس أن تضيف رقم المنفذ في نهاية عنوان IP .. الذي هو افتراضيا هو: 8000

ستفتح صفحة تعرض بنية نوع الدليل وستعرض جميع الملفات من أجهزة الكمبيوتر المصدر.  
الآن يمكنك الوصول إلى جميع الملفات.
