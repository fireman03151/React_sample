## Comando Bash: cd

El nombre del comando cd viene dado por **c**hange **d**irectory
Con el comando cd podemos navegar entre directorios en la terminal.

El comando cd tiene algunos argumentos realmente útiles para ayudar a moverse por los diferentes directorios:

*   `.` se refiere al directorio actual, como `./projects`
*   `..` se usa para subir un nivel en el directorio, `cd ..` , se puede combinar para subir múltiples niveles `../../my_folder`
*   `/` es la raíz de tu sistema para llegar a las carpetas principales, como `system` , `users` , etc.
*   `~` es el directorio de inicio, generalmente la ruta `/users/username` . Vuelve a una carpeta relativa incluyéndola al comienzo de su ruta, por ejemplo `~/projects` .
