## Comando Bash: mv

**Mueve archivos y carpetas.**
```
mv source target 
 mv source ... directory 
```

El primer argumento es el archivo que desea mover, y el segundo es la ubicación para moverlo.

Opciones de uso común:

*   `-f` forzar moverlos y sobrescribir archivos sin consultar con el usuario.
*   `-i` para solicitar confirmación antes de sobrescribir archivos.
