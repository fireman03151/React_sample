## Base de fuego

![Logo de Firebase](https://firebase.google.com/_static/558bc0d91d/images/firebase/lockup.png)

### Visión de conjunto

Firebase, en su forma más básica, fue adquirida por Google en octubre de 2014. Desde entonces, Google ha adquirido compañías adicionales que complementaron el producto Firebase original. Esta selección de herramientas de software ahora constituye la selección actual de herramientas de Firebase que se ofrecen hoy.

### Características principales de Firebase

Las características principales de Firebase se agrupan en 3 categorías:

1.  **Desarrollar**
    
    *   Autenticación
    *   Base de datos
    *   Almacenamiento
    *   Alojamiento
    *   Función
    *   Laboratorio de pruebas
    *   Reporte de Accidentes
    *   Actuación
2.  **Crecer**
    
    *   Notificaciones
    *   Configuración remota
    *   Enlaces dinámicos
3.  **Ganar**
    
    *   AdMob
