## Amazon Web Services

Amazon Web Services, o AWS para abreviar, es una subsidiaria de Amazon.com que proporciona plataformas de computación en la nube bajo demanda a individuos, empresas y gobiernos en una base de suscripción de pago con una opción de nivel libre disponible durante 12 meses. Junto con Azure de Microsoft y Cloud Platform de Google se consideran los tres principales Proveedores de nube pública.

La tecnología permite a los suscriptores tener a su disposición un grupo de computadoras virtuales de pleno derecho, disponibles todo el tiempo, a través de Internet. Versión de AWS de las computadoras virtuales tienen la mayoría de los atributos de una computadora real que incluye hardware (CPU (s) y GPU (s) para procesamiento, memoria local / RAM, almacenamiento en disco duro / SSD); una elección de sistemas operativos; redes; y software de aplicación precargado, como servidores web, bases de datos, CRM, etc. Cada sistema AWS también virtualiza su E / S de consola (teclado, pantalla, y el mouse), permitiendo a los suscriptores de AWS conectarse a su sistema AWS desde cualquier lugar del mundo usando un navegador moderno. El navegador actúa como una ventana hacia el computadora virtual, permitiendo a los suscriptores iniciar sesión, configurar y usar sus sistemas virtuales como lo harían con una computadora física real.

Los suscriptores pueden optar por implementar sus sistemas AWS para proporcionar servicios basados ​​en Internet para su propio beneficio y el de sus clientes. Desde individuos u organizaciones al usar AWS no son físicamente propietarios de la infraestructura de AWS, no tienen que lidiar con algunas de las tareas de mantenimiento asociadas con la posesión de un centro de datos físico, como operando equipos de red.

AWS, en particular, también se ha expandido en sus ofertas de nube. Ofrecen servicios innovadores, como AWS Lambda, en los que los usuarios solo necesitan centrarse en escribir código y no en los servidores que ejecutarán su código (dando lugar al nuevo movimiento _sin servidor_ ). Como resultado de estos beneficios, la computación en la nube. plataformas como AWS han sido populares entre las nuevas empresas que inicialmente podrían tener menos capital para invertir en tecnología de la información o TI. En su lugar, pueden centrarse en la construcción de excelentes productos para sus usuarios y el crecimiento de su negocio a lo largo del tiempo con el modelo a pedido de AWS.

Los servicios populares de AWS incluyen:

*   Máquinas virtuales a través de la plataforma EC2 (Elastic Compute Cloud)
*   Bases de datos que incluyen tanto SQL como NoSQL
*   S3 (Servicio de almacenamiento simple)
*   SQS (Servicio de Cola Simple)
*   Desarrollo de juegos a través del motor de desarrollo de juegos de Amazon Lumberyard y la posterior plataforma de alojamiento.
*   DynamoDB (Base de datos de Amazon Dynamo)
*   CloudWatch (Métrica y Alarmas)
*   CodeDeploy (Implementaciones)
*   Lambda (sin servidor)
*   Herramientas de monitoreo de aplicaciones
*   Herramientas y plataformas de desarrollo móvil.
*   Área de desarrollo de Amazon Alexa.

#### Certificaciones AWS

AWS ofrece muchas certificaciones diferentes para sus practicantes. Existen diferentes niveles para los certificados de AWS, así como certificaciones basadas en roles. Los niveles son:

*   Fundacional
*   Asociar
*   Profesional Para cada rol hay esos tres niveles.

Estos roles incluyen:

*   Arquitecto
*   Desarrollador
*   Operaciones

AWS también ofrece un requisito previo de certificación de AWS Cloud Practitioner Essentials \* relevante para cualquier persona, sin importar su rol. Fuera de los roles estándar, hay certificaciones especializadas para redes avanzadas, Big Data y seguridad.
