\## Rackspace Cloud

Rackspace Cloud es un conjunto de productos y servicios de computación en la nube facturados en una base de computación de utilidad de la compañía estadounidense Rackspace. Las ofertas incluyen el alojamiento de aplicaciones web o la plataforma como un servicio ("Sitios en la nube"), Almacenamiento en la nube ("Archivos en la nube"), servidor privado virtual ("Servidores en la nube"), balanceadores de carga, bases de datos, copias de seguridad y monitoreo. #### Más información:
