## Marcos de aplicación

Un marco de aplicación es un software listo para usar para ejecutar aplicaciones. Los marcos de aplicaciones manejan tareas comunes que la mayoría de las aplicaciones hacen. Los marcos de aplicaciones ahorran tiempo de desarrollo y mantenimiento, ya que varios proveedores de aplicaciones agrupan los recursos por una causa común. Un marco de aplicación es más que una colección de sus componentes nombrados. Incluye software único que integra sus componentes en un paradigma de sistemas que simplifica las tareas comunes, difíciles o repetitivas mediante el uso de bibliotecas de código reutilizable. Cada marco de aplicación resuelve problemas con un conjunto único de componentes o de una manera única, por ejemplo, la administración de recursos.
