#### ¿Qué es TensorFlow?

"TensorFlow es una biblioteca de aprendizaje automático de código abierto para investigación y producción. TensorFlow ofrece API para principiantes y expertos para desarrollar para computadoras de escritorio, móviles, web y en la nube".

Tensorflow permite a los usuarios crear gráficos de flujo de datos. Estas son estructuras que describen cómo un dato se mueve a través de un gráfico. Estos nodos están conectados entre sí, con nodos que representan las operaciones matemáticas y las conexiones que representan las matrices multidimensionales o tensores.

Tensorflow permite a los desarrolladores concentrarse más en la lógica de la aplicación que quedarse atascado en los algoritmos complejos o descubrir la forma más óptima de implementar esos algoritmos. Al crear una Red profunda, también puede resultar tedioso determinar qué nodo debe conectarse a cuál. Tensorflow hace que sea más fácil hacer conexiones entre las capas. Por lo tanto, se vuelve más fácil para los desarrolladores concentrarse más en mejorar la aplicación.
