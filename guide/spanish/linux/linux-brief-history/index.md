## Linux en breve

GNU / Linux (comúnmente conocido como Linux) es un sistema operativo gratuito, de código abierto y personalizable basado en el kernel de Linux y en las utilidades y bibliotecas de GNU. Está desarrollado por miles de desarrolladores y es utilizado por millones de personas, empresas y gobiernos de todo el mundo. Es una solución alternativa a otros sistemas operativos, como Windows y Mac OSX, y tiene muchas distribuciones o "sabores" diferentes. Algunas de las distribuciones más populares son Ubuntu, Red Hat Enterprise Linux (RHEL), Debian, Fedora, CentOS, OpenSuSE, Arch, Mint y Solus. Incluye software que se encuentra en otros sistemas operativos para tareas comunes, como ejecutar aplicaciones de oficina, multimedia, gráficos y juegos. También proporciona potentes herramientas para la programación y desarrollo web. Linux está diseñado para ejecutarse en computadoras de escritorio, servidores, dispositivos de Internet de las cosas y dispositivos móviles (Android).

## ¿Quién inventó Linux?

Linux germinó como una idea en la mente del joven y brillante **Linus Torvalds** cuando era un estudiante de informática. Solía ​​trabajar en el sistema operativo UNIX (software propietario) y pensaba que necesitaba mejoras. Sin embargo, cuando los diseñadores de UNIX rechazaron sus sugerencias, pensó en lanzar un sistema operativo que sea receptivo a los cambios, modificaciones sugeridas por sus usuarios. Así que Linus ideó un Kernel llamado Linux en 1991. A pesar de que necesitaría programas como File Manager, Document Editors, Audio-Video, para ejecutarlo. A medida que pasaba el tiempo, colaboró ​​con otros programadores en lugares como MIT y comenzaron a aparecer aplicaciones para Linux. Entonces, alrededor de 1991, se lanzó oficialmente un sistema operativo Linux con algunas aplicaciones, y este fue el comienzo de una de las opciones de SO de código abierto más queridas y disponibles en la actualidad. Las versiones anteriores de Linux no eran tan fáciles de usar como lo usaban los programadores de computadoras y Linus Torvalds nunca tuvo en mente comercializar su producto. Esto definitivamente frenó la popularidad de Linux como otro sistema operativo orientado comercialmente que Windows se hizo famoso. No obstante, el aspecto de código abierto del sistema operativo Linux lo hizo más robusto.
