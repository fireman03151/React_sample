## Sistema operativo BSD

![Sistema operativo BSD, sobre todo freeBSD: el poder de servir](https://www.freebsd.org/layout/images/logo-red.png)

### Visión de conjunto

Berkeley Software Distribution (BSD) es un grupo de sistemas operativos (SO) similares a código abierto (open-source) relacionados con Unix, con orígenes en las primeras versiones de Research Unix en Bell Labs. FreeBSD es el miembro más popular.

BSD está configurado para el alojamiento de Internet, el alojamiento web y el alojamiento de muchos servidores en un sistema. Es el primer sistema operativo que ha añadido un protocolo de internet. Los sistemas operativos BSD tienen un sistema de tiempo compartido muy fuertemente implementado, lo que los hace más útiles cuando los recursos se comparten entre procesos. Como comparación, se sabe que el sistema operativo Linux es preferido para los procesos de una sola tarea, como las supercomputadoras y los equipos de escritorio. La parte de BSD efectiva de multitarea de interrupción forzada del tiempo compartido se interpone en el camino de procesos únicos dedicados.

BSD incluye un sistema 'Jails' que es algo similar a los contenedores de Linux, excepto con seguridad y flexibilidad adicionales en la implementación.
