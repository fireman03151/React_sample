### Funcionalidad de captura de pantalla de DevTools

Al usar las herramientas de desarrollo de Chrome, puede ser útil capturar una captura de pantalla de toda la página. Y si bien hay complementos para lograr esto, Google Chrome tiene su propia funcionalidad integrada.

## Pasos

1.  Abrir Chrome Devtools
2.  en Devtools Ctrl + P (Cmd + P en Mac)
3.  Comience a escribir> captura de pantalla
4.  Elija entre las funciones disponibles (la captura de pantalla a tamaño completo ha sido, con mucho, la más útil)
5.  Disfruta tu nueva captura de pantalla
