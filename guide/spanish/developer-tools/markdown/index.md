## Reducción

Markdown es un lenguaje de marcado ligero con sintaxis de formato de texto plano. Está diseñado para que pueda convertirse a HTML y muchos otros formatos utilizando una herramienta con el mismo nombre. Markdown se usa a menudo para formatear archivos README, para escribir mensajes en foros de discusión en línea y para crear texto enriquecido usando un editor de texto simple. También puedes escribir Markdown en tu editor de texto (como Sublime Text o Visual Studio Code). Markdown está ganando visibilidad y popularidad gracias a la aparición de editores de texto simplificados y "sin distracciones".
