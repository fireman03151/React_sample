## Proxies

Los servidores proxy son servidores o una red de computadoras que actúan como intermediarios entre un cliente y un servidor.

Hay diferentes tipos de proxies que incluyen pero no se limitan a:

*   Proxies hacia adelante
*   Proxies inversos
*   Proxies transparentes
*   Proxies web

Los más conocidos hoy en día son los servidores web que actúan como intermediarios entre la computadora de un usuario e Internet.

Las características comunes de tales proxies son:

*   Para ocultar la dirección IP de un usuario proporcionando anonimato mientras navega por la web.
*   Permitir a los usuarios acceder a sitios con direcciones IP bloqueadas.
