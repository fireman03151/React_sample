# GraphQL
Desarrollado por Facebook y lanzado en 2015, GraphQL es un estándar API definido por su uso de una solicitud declarativa que devuelve sólo los datos que el cliente solicitó, en la forma que ellos desean. A diferencia de las API REST tradicionales, GraphQL permite que el lado del cliente de una aplicación solicite los datos exactos que requieren de forma legible.

Una alternativa a los puntos finales fijos de REST, su enfoque flexible hace que la agregación de datos de múltiples fuentes sea más sencilla, los análisis son más refinados y minimizar la recolección de datos pone menos presión en las redes más lentas, lo que acelera las respuestas para de los usuarios.

Los esquemas GraphQL a menudo se combinan con marcos como [Relay](https://facebook.github.io/relay/) o [Apollo](https://www.apollographql.com/) para realizar estas solicitudes desde el cliente. GraphQL potencia cómo [GatsbyJS](https://www.gatsbyjs.org) obtiene datos.

## Instalación
Desde la línea de comandos, ejecute:

```sh
npm init
npm install graphql --save
```
