# Genómica computacional

Un nuevo campo que se especializa en el desarrollo de nuevos algoritmos para estudiar el genoma humano.

Esta sección se centrará en las diversas estructuras de datos y algoritmos que se utilizan y crean para este propósito desde matrices de sufijos hasta alineación de secuencias.

## Libros para leer

Introducción a la genómica computacional por Matthew W. Hahn

Un gran libro de iniciación para principiantes.
