## Que es julia

[Julia](https://julialang.org) es una Lenguaje de programación de alto rendimiento, opcionalmente escrito, destinado a la computación científica y numérica. Cuenta con bibliotecas para álgebra lineal, optimización numérica, procesamiento de señales y aprendizaje automático, entre otros. Utiliza el envío múltiple y tiene un backend LLVM y un compilador JIT. Julia es seleccionada para ser utilizada en el desarrollo del [sistema de prevención de colisiones aerotransportadas de próxima generación](https://juliacomputing.com/case-studies/lincoln-labs.html) para críticos de seguridad [.](https://juliacomputing.com/case-studies/lincoln-labs.html)
