Free Code Camp se compromete a respetar la privacidad de los visitantes de nuestros sitios web y aplicaciones web. Las siguientes pautas explican cómo protegemos la privacidad de los visitantes de FreeCodeCamp.com y sus características.

## Información de identificación personal

Free Code Camp protege la identidad de los visitantes de FreeCodeCamp.com al limitar la recopilación de información de identificación personal.

Free Code Camp no recopila ni solicita, a sabiendas, información de identificación personal de o sobre niños menores de 13 años, excepto lo permitido por la ley. Si descubrimos que hemos recibido información de un niño menor de 13 años que viola esta política, la eliminaremos de inmediato. Si cree que Free Code Camp tiene información de o sobre cualquier persona menor de 13 años, envíenos un correo electrónico a [team@freecodecamp.com](mailto:team@freecodecamp.com) .

Toda la información de identificación personal que nos proporciona es utilizada por Free Code Camp y su equipo para procesar y administrar su cuenta, analizar la demografía de nuestros usuarios o brindar servicios a través del sitio.

Si elige proporcionarnos información de identificación personal, puede recibir correos electrónicos ocasionales de nuestra parte que sean relevantes para Free Code Camp, obtener un trabajo o aprender a codificar en general. Free Code Camp también puede utilizar otros proveedores externos para facilitar la entrega de los servicios descritos anteriormente, y estos proveedores externos pueden recibir o tener acceso a información de identificación personal con el único propósito de proporcionarle estos servicios, nombre de Free Code Camp.

Free Code Camp también puede divulgar información de identificación personal en circunstancias legales especiales. Por ejemplo, dicha información se puede utilizar cuando sea necesario para proteger nuestros derechos de autor o derechos de propiedad intelectual, o si la ley así lo exige.

## Información anónima

Se pueden proporcionar datos agregados anónimos a otras organizaciones con las que nos asociamos con fines estadísticos. Por ejemplo, podemos informar a una organización que un cierto porcentaje de los visitantes de nuestro sitio son adultos entre las edades de 25 y 35 años.

## Cookies y balizas - Uso por Free Code Camp; Opting Out

Utilizamos cookies y registros de software para monitorear el uso de FreeCodeCamp.com y para recopilar información no personal sobre los visitantes del sitio. Las cookies son pequeños archivos que Free Code Camp transfiere a los discos duros de los visitantes con fines de mantenimiento de registros. Estos sistemas de monitoreo nos permiten rastrear información general sobre nuestros visitantes, como el tipo de exploradores (por ejemplo, Firefox o Internet Explorer), los sistemas operativos (por ejemplo, Windows o Macintosh) o los proveedores de Internet (por ejemplo, Comcast). ) ellos usan. Esta información se utiliza con fines estadísticos y de investigación de mercado para adaptar el contenido a los patrones de uso y para proporcionar los servicios solicitados por nuestros clientes. Para eliminar estas cookies, consulte la configuración de privacidad de su navegador.

Una baliza es un objeto de archivo electrónico (normalmente una imagen transparente) que se coloca en el código de una página web. Utilizamos balizas de terceros para monitorear los patrones de tráfico de los visitantes de una página de Free Code Camp.com a otra y para mejorar el rendimiento del sitio. Ninguna de la información que recopilamos de esta manera puede usarse para identificar a cualquier persona que visite nuestro sitio.

## Seguridad

Cualquier información de identificación personal recopilada a través de este sitio se almacena en servidores de acceso limitado. Mantendremos medidas de seguridad para proteger estos servidores y la información que almacenan.

## Encuestas

Ocasionalmente podemos realizar encuestas en línea. Todas las encuestas son voluntarias y usted puede negarse a participar.

## Derechos de autor

Todo el contenido en FreeCodeCamp.com está protegido por derechos de autor de Free Code Camp. Si desea redistribuirlo más allá de simplemente compartirlo a través de las redes sociales, contáctenos en [team@freecodecamp.com](mailto:team@freecodecamp.com) .

## Contactando con nosotros

Si tiene preguntas sobre Free Code Camp, o para corregir, actualizar o eliminar información de identificación personal, envíenos un correo electrónico a [team@freecodecamp.com](mailto:team@freecodecamp.com) .

## Enlaces a otros sitios web

Los sitios de Free Code Camp contienen enlaces a otros sitios web. Free Code Camp no es responsable de las prácticas de privacidad ni del contenido de estos sitios web de terceros. Instamos a todos los visitantes de FreeCodeCamp.com a seguir prácticas seguras de Internet: no proporcione información de identificación personal a estos sitios web a menos que haya verificado sus políticas de seguridad y privacidad.

## Retención de datos

Retenemos su información durante el tiempo que sea necesario para permitirnos utilizarla para los fines que le hemos comunicado y cumplir con las leyes o regulaciones aplicables.

## Transferencias de negocios

A medida que continuamos desarrollando nuestro negocio, podríamos vender o comprar subsidiarias o unidades de negocios. En tales transacciones, la información del cliente generalmente es uno de los activos comerciales transferidos, pero permanece sujeto a las promesas hechas en cualquier Política de privacidad preexistente (a menos que, por supuesto, el cliente acepte lo contrario). Además, en el improbable caso de que Free Code Camp, o prácticamente todos sus activos sean adquiridos, la información del cliente será uno de los activos transferidos y seguirá sujeta a nuestra Política de privacidad.

## Sus derechos de privacidad en California

Si usted es residente de California, tiene derecho a evitar el intercambio de su información personal con terceros para sus propios fines de marketing a través de medios gratuitos. Si envía una solicitud a la dirección anterior, Free Code Camp le proporcionará un Aviso de elección del cliente de California que puede utilizar para optar por no compartir dicha información. Para recibir este aviso, envíe una solicitud por escrito a [team@freecodecamp.com](mailto:team@freecodecamp.com) , especificando que busca su "Aviso de elección del cliente de California". Por favor permita por lo menos treinta (30) días para una respuesta.

## Aceptación de los términos y condiciones de la política de privacidad

Al utilizar este sitio, usted expresa su acuerdo con los términos y condiciones de esta Política de privacidad de FreeCodeCamp.com. Si no está de acuerdo con estos términos, no utilice este sitio. Nos reservamos el derecho, a nuestra entera discreción, de cambiar, modificar, agregar o eliminar partes de esta política en cualquier momento. Todos los términos modificados entrarán en vigencia automáticamente 30 días después de su publicación inicial en el sitio. Por favor revise esta página periódicamente para cualquier modificación. Su uso continuado de FreeCodeCamp.com después de la publicación de cualquier cambio a estos términos significará que ha aceptado esos cambios.

Si tiene alguna pregunta o inquietud, envíe un correo electrónico a [team@freecodecamp.com](mailto:team@freecodecamp.com) .
