Twitch es un popular servicio de transmisión en vivo del cual Free Code Camp se alejó de

Por mucho que disfrutáramos usando este servicio, las transmisiones eran pocas y distantes y, por lo tanto, no estaba cumpliendo su propósito.

Hemos decidido centrarnos en el uso de [YouTube](https://www.youtube.com/freecodecamp) para transmitir nuestras cumbres mensuales y la programación en parejas, así como para cargar tutoriales y excursiones.

Puedes encontrar nuestra sala de chat de YouTube aquí: [FreeCodeCamp / YouTube](https://gitter.im/FreeCodeCamp/YouTube)

Puede encontrar el antiguo canal de contracción aquí: [twitch.tv/freecodecamp](https://www.twitch.tv/freecodecamp)
