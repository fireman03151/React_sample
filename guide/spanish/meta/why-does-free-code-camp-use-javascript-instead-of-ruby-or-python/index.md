---
title: Why Does Free Code Camp Use JavaScript Instead of Ruby or Python
localeTitle: ¿Por qué Free Code Camp usa JavaScript en lugar de Ruby o Python?
---
Al igual que JavaScript, Ruby y Python son lenguajes de script de alto nivel que se pueden usar para el desarrollo web de pila completa.

Pero incluso si aprendieras estos idiomas, necesitarías aprender JavaScript. Eso es porque JavaScript es el único idioma que se ejecuta en los navegadores web. JavaScript ha existido durante 20 años, y sigue creciendo en popularidad.

Debido a esto, JavaScript tiene más herramientas y recursos de aprendizaje en línea que cualquier otro idioma.
