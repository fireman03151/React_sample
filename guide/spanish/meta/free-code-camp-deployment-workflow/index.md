---
title: Free Code Camp Deployment Workflow
localeTitle: Flujo de trabajo de implementación de Code Camp gratis
---
1.  Todo el código se escribe y se envía a una rama en la propia bifurcación de desarrolladores de Free Code Camp.
2.  Los desarrolladores crean solicitudes de extracción, extrayendo el código de su sucursal a la sucursal provisional de Free Code Camp.
3.  Alguien que no sea la persona que presentó la solicitud de extracción, el control de calidad es la solicitud de extracción y la acepta si todo es perfecto. De lo contrario, cierran la solicitud de extracción con una explicación de por qué no es perfecta, y el desarrollador abre otra solicitud de extracción una vez que han solucionado el problema.
4.  El maestro se implementa en freecodecamp.com periódicamente.

## Notas:

*   Esto está sujeto a cambios una vez que obtengamos una mejor cobertura de prueba y el trabajo de integración continua.
