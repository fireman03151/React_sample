freeCodeCamp tiene varias teclas de acceso rápido, por lo que los usuarios avanzados pueden moverse rápidamente.

## Cómo utilizar las teclas de acceso rápido

Para usar estas teclas de acceso rápido, debe presionar las teclas en el orden en que aparecen. Es importante tener en cuenta que no debe presionarlos al mismo tiempo.

### Redireccionar teclas de acceso rápido

g n n Redirige al siguiente reto.  
g n a Redirige a la página acerca de  
g n m Redirecciona al mapa.  
g n s Redirecciona a la tienda.  
g n o Redirige a la configuración  
g n r Redirige al repositorio de Free Code Camp GitHub  
g n w Redirecciona al foro

### Teclas de acceso directo del editor

Estas teclas de acceso rápido deben presionarse al mismo tiempo, no en orden.

Ctrl + Enter Ejecuta las pruebas. Si la vez anterior las pruebas fueron exitosas, esto envía el desafío y pasa a la siguiente  
Ctrl + / Comentarios o descomenta la línea actual o el código seleccionado

### Otras teclas de acceso rápido

g m Alternar el cajón del mapa  
g t n Conmutar modo nocturno  
g c Alternar el chat principal (Gitter)

Esc Si presiona esto mientras la entrada de búsqueda está enfocada, vacía la entrada
