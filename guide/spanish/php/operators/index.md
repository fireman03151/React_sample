## Los operadores

PHP contiene todos los operadores normales que uno esperaría encontrar en un lenguaje de programación.

Se utiliza un solo "=" como operador de asignación y un doble "==" o triple "===" se usa para la comparación.

Los "<" y ">" habituales también se pueden usar para la comparación y "+ =" se puede usar para agregar un valor y asignarlo al mismo tiempo.

Lo más notable es el uso de "." Para concatenar cadenas y ". =" Para agregar una cadena al final de otra.
