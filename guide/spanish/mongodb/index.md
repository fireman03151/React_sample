## MongoDB

MongoDB es una base de datos no relacional de código abierto que utiliza una estructura similar a JSON para almacenar datos. Utiliza un modelo de documento para almacenar y recuperar los datos en lugar del modelo de tabla utilizado por las bases de datos secuenciales, como SQL u Oracle.

### NOSQL Vs RDBMS

### Características de MongoDB

1.  Base de datos de próxima generación
2.  No se une
3.  Agrupación
4.  Fuente abierta
5.  Esquema menos
6.  No relaciones

### Características de MongoDB

1.  Base de datos de documentos
2.  Alto rendimiento
3.  Rich Query Language
4.  Alta disponibilidad
5.  Escalabilidad horizontal
