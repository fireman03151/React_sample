## Bases de datos de grafos

Una base de datos gráfica es una base de datos que utiliza estructuras gráficas para consultas semánticas con nodos, bordes y propiedades para representar y almacenar datos. Un concepto clave del sistema es el gráfico (o borde o relación), que relaciona directamente los elementos de datos en la tienda. Las relaciones permiten que los datos en la tienda se vinculen directamente y, en muchos casos, se recuperen con una sola operación.
