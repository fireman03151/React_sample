## Ionicons

Ionicons es un conjunto de iconos de código abierto creado para aplicaciones web, iOS, Android y de escritorio. Ionicons se creó para Ionic Framework, por lo que los iconos tienen versiones de Material Design y iOS. Cuando se usa con Ionic, el componente `ion-icon` usará automáticamente la versión correcta según su plataforma. Además, cuando se utiliza fuera de Ionic, la aplicación puede elegir plataformas `ios` y `md` .

## Ejemplo
```
<ion-icon name="logo-github"></ion-icon>
```
