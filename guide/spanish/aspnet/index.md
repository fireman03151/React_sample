## ASP.NET

# Visión de conjunto

ASP.Net es una plataforma de desarrollo web proporcionada por Microsoft. Se utiliza para crear aplicaciones basadas en web y sitios web utilizando HTML, CSS y JavaScript como front-end. Los lenguajes de programación del lado del servidor como C # y VB .NET pueden usarse para codificar el back-end.

ASP.NET ofrece diferentes marcos para crear aplicaciones web: por ejemplo, Web Forms y ASP.NET MVC.
