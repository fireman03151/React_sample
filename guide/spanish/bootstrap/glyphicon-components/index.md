## Componentes de Glyphicon

Bootstrap 3 incluye 260 glyphicons. Los glifos normalmente no son gratuitos, pero su creador los ha puesto a disposición de Bootstrap de forma gratuita. Como agradecimiento, debe incluir un enlace a Glyphicons siempre que sea posible.

Use glyphicons en texto, navegación, botones, formularios o barras de herramientas.

## Ejemplos

```html

<p>Star icon: <span class="glyphicon glyphicon-star"></span></p> 
 <p>Refresh icon: <span class="glyphicon glyphicon-refresh"></span></p> 
 <p>Envelope icon: <span class="glyphicon glyphicon-envelope"></span></p> 
```

## Nota

Bootstrap 4 cayó Glyphicons. Puedes usar [Font Awesome Icons en su](http://fontawesome.io/icons/) lugar.
