## Definición de modo

En estadística, el término _modo se_ refiere al valor o valores en un conjunto de datos que aparece con más frecuencia. Debido a que se produce con mayor frecuencia en el conjunto de datos, el _modo_ es el valor más probable de muestreo aleatorio.

El modo de un conjunto de números indica el valor que ocurre con más frecuencia en el conjunto. Dada una distribución de datos, el modo puede describirse como el máximo de la función de masa de probabilidad si los datos son discretos, o la función de distribución de probabilidad si los datos son continuos.

#### Un conjunto de datos puede tener uno o más modos.

Considere el conjunto de datos 14, 16, 21, 33, 37, 16, 39, 33 Aquí, 16 y 33 cada uno ocurre dos veces. Todos los demás números se producen una sola vez. Así que 16 y 33 tienen la frecuencia máxima de ocurrencia que es 2. Por lo tanto, hay dos modos para un conjunto de datos dado, es decir, 16 y 33. Un conjunto de números en el que hay dos números que ocurren con mayor frecuencia puede ser descrito como bimodal , y cualquier número mayor que dos se dice que es multimodal.

#### También es posible tener un conjunto de datos sin modo

Considere el conjunto de datos 10, 15, 24, 34, 38, 16, 41. Aquí, todos los números aparecen solo una vez. Así que ningún número tiene la frecuencia máxima de ocurrencia. Por lo tanto, no hay modo para este conjunto de datos dado.

## Reglas para encontrar el modo de un conjunto de datos.

*   Examine todos los números del conjunto de datos y encuentre si uno o más números están ocurriendo más de una vez.
*   El número con la frecuencia máxima de ocurrencia es el modo.
*   Si más de un número tiene una frecuencia máxima, entonces hay más de un modo para el conjunto de datos.
*   Si todos los números del conjunto de datos ocurren solo una vez, entonces no hay modo para el conjunto de datos.

#### Ejemplo

El modo de (1, 2, 3, 2, 4, 5) es **2** :

_Para encontrar el modo, debe contar el número de ocurrencias de cada valor en el conjunto de datos. El valor 2 aparece dos veces en el conjunto de datos, y el resto de los valores aparecen solo una vez, lo que hace que el 2º modo._

El modo de (7, 7, 8, 8, 9) es 7 **y** 8:

_Si hay un empate para los valores que ocurren con más frecuencia en el conjunto de datos, el modo es cada uno de los valores con esta frecuencia._
