## Área de un triángulo

Un triángulo es un polígono de tres caras y dos dimensiones. Hay dos métodos para determinar el área.

### Método 1

El área de un triángulo se puede calcular multiplicando 0.5 a la longitud de la base a la altura perpendicular.

![Triángulo-1](https://github.com/uuykay/misc/blob/master/area-triangle-image-2.jpg)

### Método 2

Si no se conoce la altura perpendicular, puede usar un método diferente para calcular el área. Si conoce la longitud de dos lados y el tamaño del ángulo entre ellos, entonces se puede encontrar el área.

![Triángulo 2](https://github.com/uuykay/misc/blob/master/area-triangle-image-3.jpg)

Matemáticamente esto se expresa como:

![Fórmula 2](https://github.com/uuykay/misc/blob/master/triangle-equation-area-2.jpg)

Tenga en cuenta que la fórmula expresada anteriormente utiliza los lados a y b y el ángulo C entre ellos. Estos pueden ser sustituidos por otros dos lados y el ángulo entre ellos.

### Terminología

Base - El lado inferior. Seleccione un lado que se conoce.

Altura perpendicular: la altura del triángulo, medida perpendicular a la base, hasta el punto más alto del triángulo.

Pecado - La expresión trigonométrica del seno.

### Ejemplos

1.  Encuentra el área de un triángulo con una longitud de base de 4 unidades y una altura perpendicular de 12 unidades
    
    Área = 0.5 x 4 x 12 = 24 unidades 2
    
2.  Si se sabe que dos lados de un triángulo son 3 y 6, y el ángulo entre ellos es de 30 grados, ¿cuál es el ángulo del triángulo?
    
    Área = 0.5 x 3 x 6 x sin (30 grados) = 4.5 unidades 2
