## Reglas de registro básicas y expresiones de registro ampliadas

Las siguientes son reglas de registro básicas comúnmente utilizadas:

###### Regla del producto logaritmo

log b (x \* y) = log b (x) + log b (y)

###### Regla de cociente de logaritmo

log b (x / y) = log b (x) - log b (y)

###### Regla de potencia de logaritmo

log b (x y ) = y \* log b (x)

###### Logaritmo de número negativo

el registro b (x) no está definido cuando x <0

###### Logaritmo de 0

log b (0) no está definido

lim x → 0+ log b (x) = -infinito

###### Logaritmo de 1

log b (1) = 0

###### Logaritmo de la Base

log b (b) = 1

###### Logaritmo del infinito

lim x → registro de infinito b (x) = infinito

###### Regla de conmutación de base logarítmica

log b (c) = 1 / log c (b)

###### Regla de cambio de base de logaritmo

log b (x) = log c (x) / log c (b)
