## Reglas de integración

Al igual que con la diferenciación, utilizamos una variedad de reglas al integrar funciones. A continuación se muestran algunos de los más comunes.

**La regla constante:** ∫k dx = kx + C donde k es una constante

**La Regla de Poder:** ∫ x n dx = + x n + 1 ⁄ n + 1 + C cuando n ≠ 1

**Regla exponencial:** ∫ e kx dx = 1 ⁄ kx e kx + C donde k es una constante

**Reglas trigonométricas:**

∫ cos (x) dx = sin (x) + C

∫ sin (x) dx = -cos (x) + C

**Reglas de suma / diferencia:**

∫ \[f (x) + g (x)\] dx = ∫f (x) dx + ∫g (x) dx

∫ \[f (x) - g (x)\] dx = ∫f (x) dx - ∫g (x) dx
