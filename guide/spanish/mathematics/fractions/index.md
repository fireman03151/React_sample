## Fracciones

Un número fraccional representa parte de un número entero, un valor entre 0 y 1. Se pueden representar utilizando decimales (por ejemplo, 0.1), porcentajes (por ejemplo, 10%), índices negativos (por ejemplo, 10 ^ -1) o fracciones (por ejemplo, 1 / 10).

Una fracción normalmente tiene dos partes que se dividen por una línea:

*   Numerador: representa el número de partes iguales que marcan la fracción. (sobre la línea)
    
*   Denominador: representa el número por el cual el numerador se divide para hacer la fracción. (debajo de la linea)
    

El numerador y el denominador son ambos números enteros (números enteros).

Una fracción apropiada es aquella en la que el numerador es menor que el denominador.

Las fracciones también se pueden agregar después de un número entero para aumentar su precisión.

### Ejemplos

1½ representa 1 + 1/2 o 1.5.

No todas las fracciones tienen representaciones decimales:

⅓ o 1/3 representa 1 dividido por 3
