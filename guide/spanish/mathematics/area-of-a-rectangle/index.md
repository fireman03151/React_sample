## Área de un rectángulo

El área de un rectángulo se puede calcular multiplicando las longitudes de los dos lados perpendiculares entre sí.

`Area = a * b`

**Por ejemplo** Para un rectángulo donde _a = 3_ y _b = 4_

`Area = a * b = 3 x 4 = 12`
