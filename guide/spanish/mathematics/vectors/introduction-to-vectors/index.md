# Introducción a los vectores.

Un vector es una cantidad física que se puede medir (tiene una magnitud en unidades) y también funciona en una dirección particular, a diferencia de un escalar que solo se puede medir. Ejemplos de vectores incluyen velocidad, aceleración, fuerza y ​​momento.

Un vector se puede representar mediante una línea dibujada, de manera que:

*   la longitud de la línea representa la _**magnitud**_ del vector en una unidad dada;
*   la dirección de la línea representa la _**dirección**_ en la que opera (o actúa) el vector.

![representación vectorial](https://github.com/xeroxism/myImages/blob/master/FCC_guides/vector-representation.png?raw=true)

En la figura anterior, el desplazamiento de A a B se escribe como **AB** ( **ā** o **a** ). el punto A es el origen o el punto inicial y el punto B es la cabeza o el punto terminal. Este desplazamiento es equivalente a mover una distancia _x_ horizontalmente seguida de _y_ verticalmente.

### El modulo de un vector

El módulo de un vector es un número escalar positivo que da la magnitud del vector. Está escrito así:

| **AB** | o | **un** | o simplemente **AB** o **a** .

Dónde; | AB | = √ ( _x_ 2 + _y_ 2 )
