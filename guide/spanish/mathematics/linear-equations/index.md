## Ecuaciones lineales

Una ecuación lineal es una línea recta de un punto a otro, que se puede expresar a través de una ecuación o una gráfica.

La ecuación para una función lineal es **y = mx + b** .

*   _'y'_ son las coordenadas y en la pendiente
*   _'mx'_ es la pendiente o tasa a la que se dirige la ecuación. Esto se puede calcular utilizando **(y2-y1 / x2-x1)** , donde el numerador es el desplazamiento vertical, mientras que el denominador es el desplazamiento horizontal.
*   _'b'_ es el intercepto y, o donde la función lineal se encuentra con el eje y.
