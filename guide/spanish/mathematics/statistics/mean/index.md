## Media

La medida más común de la tendencia central, o un valor utilizado para describir un conjunto de números, es el promedio aritmético, o media. Para encontrar la media ( _A_ ) de un conjunto de números, agregue sus valores y divida por el número de valores ( _n_ ).

![Fórmula media](https://wikimedia.org/api/rest_v1/media/math/render/svg/90330653b40adf032ea8e144f84d7eec1a88054d)

### Ejemplos

*   **1, 4, 5, 2, 3**
    
    (1 + 4 + 5 + 2 +3) / 5 = 3
    
*   **20, 14, 23**
    
    (20 + 14 + 23) / 3 = 19
    
*   **\-12, -20, -16, -21**
    
    (-12, -20, -16, -21) / 4 = -17.25 \\
    

La media puede ser una excelente manera de describir un conjunto de números. Sin embargo, es importante darse cuenta de que un valor que es mucho más alto o más bajo que los otros valores, al que se hace referencia como un valor atípico, en el conjunto puede hacer que la media represente incorrectamente el conjunto de números. Por ejemplo, si tiene un conjunto de números compuesto principalmente por números pequeños que contienen un número grande, el promedio terminará siendo mayor que la mayoría de los números, presentando de manera inexacta el conjunto de números como más altos de lo que realmente son.

### Ejemplos

*   **2, 7, -3, 1, 0, 2, 857**
    
    (2 + 7 + (-3) + 1 + 0 + 2 + 857) / 7 = 123.71
    
*   **1276, 1141, 1271, 1053, 1**
    
    (1276 + 1141 + 1271 + 1053 + 1) / 5 = 948.4
