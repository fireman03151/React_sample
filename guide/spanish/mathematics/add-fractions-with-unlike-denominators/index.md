## Añadir fracciones con denominadores diferentes

Para agregar fracciones con denominadores diferentes, debe volver a escribir las fracciones para que todas tengan el mismo denominador. Una forma de hacer esto es multiplicar todos los denominadores diferentes para crear un denominador común.
