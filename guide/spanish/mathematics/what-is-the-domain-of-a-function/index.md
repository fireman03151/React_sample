## ¿Cuál es el dominio de una función?

El dominio de una función es el conjunto completo de valores posibles de la variable independiente. En otras palabras, el dominio es el conjunto de todos los valores x posibles que harán que la función "funcione" y generará valores y reales.

Por ejemplo, para y = sqrt (x + 2) , el dominio de esta función es x> = -2 .
