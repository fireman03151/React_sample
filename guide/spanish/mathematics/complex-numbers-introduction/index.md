## Introducción a los números complejos

Un número complejo es cualquier número que se puede expresar en la forma de z = a + ib.

![yo](https://www.codeproject.com/KB/cpp/882975/SquareRoot1.png)

a se llama la parte real del número complejo, y b se llama la parte imaginaria.

#### Nota

Un número imaginario puro o un número real puro también es un número complejo.

#### Ejemplos

1.  ¿Cuál es la parte real de 2i + 1? A. Parte real: 1
    
2.  ¿Cuál es la parte imaginaria de 2 - 4i? A. Parte imaginaria: -4
    
3.  ¿Es un número complejo z = 2i? A. Sí, porque podemos escribir este número como z = 0 + 2i.
    
4.  ¿Es un número complejo z = 4? A. Sí, porque podemos escribir este número como z = 4 + 0i.
