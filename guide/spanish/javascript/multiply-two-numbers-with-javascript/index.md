---
title: Multiply Two Numbers with JavaScript
localeTitle: Multiplica dos números con JavaScript
---
JavaScript utiliza el símbolo `*` para la multiplicación.
```
var product = 8 * 10; 
```

el 'producto' tendría el valor '80' después de esta línea.

Al realizar cálculos, es posible usar paréntesis para priorizar qué números deben multiplicarse juntos.
