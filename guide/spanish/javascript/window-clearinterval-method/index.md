## Método clearInterval de ventana

El método 'clearInterval ()' se utiliza para borrar un conjunto de temporizador con el método 'setInterval ()'.

```js
    clearInterval(setInteval_ID); 
```

Para poder usar el método 'clearInterval ()', debes usar una variable global.

```js
    myID = setInterval(function, milliseconds); 
```
