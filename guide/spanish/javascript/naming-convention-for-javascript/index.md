Aquí obtendrá una visión general de los diferentes casos de código que se utilizan ampliamente.

## el caso de Carmel

En la programación, el formato de camelCase para nombres de variables se ve así:
```
var camelCase = "lower-case first word, capitalize each subsequent word"; 
```

## PascalCase

PascalCase (o CamelCase) es una variante de camelCase. Se diferencia de camelCase al capitalizar cada palabra, _incluida_ la primera palabra:
```
var PascalCase = "upper-case every word"; 
```

## snake\_case

Otro caso popular llamado snake\_case separa cada palabra con guiones bajos de esta manera:
```
var snake_case = "lower-case everything, but separate words with underscores"; 

```
