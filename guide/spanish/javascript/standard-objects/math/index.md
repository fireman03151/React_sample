## Mates

`Math` es un objeto incorporado estándar a JavaScript que contiene constantes matemáticas y funciona como propiedades y métodos. En particular, contiene las constantes π y la constante de Euler y funciones como `floor()` , `round()` , `ceil()` y muchas más.

### Ejemplo

El siguiente ejemplo muestra cómo usar el objeto `Math` para escribir una función que calcula el área de un círculo:

```javascript
function calculateCircleArea(radius) { 
  return Math.PI * Math.pow(radius, 2); 
 } 
 calculateCircleArea(1); // 3.141592653589793 
```
