## Piso de matematicas

`Math.floor()` es un método del objeto estándar matemático que redondea un número dado hacia abajo al siguiente entero. Tenga en cuenta que para los números negativos, esto significa que el número se redondeará "fuera de 0" en lugar de al número de valor absoluto menor, ya que `Math.floor()` devuelve el número entero más grande menor o igual que el número dado.

### Ejemplos

```javascript
Math.floor(0.9)  //  0 
 Math.floor(1.3)  //  1 
 Math.floor(0.5)  //  0 
 Math.floor(-0.9) // -1 
 Math.floor(-1.3) // -2 
```
