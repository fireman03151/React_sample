## JSON Arrays

Las matrices JSON no son diferentes del objeto de matriz normal que utiliza en Javascript. Es un objeto de matriz que contiene varios `JSON Objects` .

Aquí hay un ejemplo de una matriz JSON:

```Javascript
    var aMyPlaylist = [{ 
        artist: "Ed Sheeran", 
        track: "Supermarket flowers", 
        myRating: 10 
    }, { 
        artist: "Tracy Chapman", 
        track: "Fastcar", 
        myRating: 9 
    }]; 
```

Esta es una matriz JSON llamada como una lista de `aMyPlaylist` . Todos los métodos de matriz como `map` , `filter` , `sort` , etc. pueden aplicarse también a cualquier matriz JSON.
