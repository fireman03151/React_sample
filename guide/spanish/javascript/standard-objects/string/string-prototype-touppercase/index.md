El método de JavaScript `.toUpperCase()` devuelve la misma cadena a la que se llamó, pero en mayúsculas.

**Sintaxis**
```
str.toUpperCase() 
```

## Ejemplos
```
console.log("hello world".toUpperCase()); // Console will output "HELLO WORLD" 
```
