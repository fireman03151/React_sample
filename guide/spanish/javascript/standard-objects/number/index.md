## Número

El objeto `Number` Javacript representa cualquier tipo de número. JavaScript no distingue tipos separados de números, como enteros o flotantes. Todos se almacenan en un formato unificado de punto flotante de 64 bits, de acuerdo con el estándar internacional IEEE 754 1 .

Los nuevos objetos de `Number` se crean así: `var num = new Number(value)` aunque generalmente se pueden crear simplemente asignando un valor numérico a una variable: `var num = 1.616;` . En un contexto que no es de constructor (es decir, sin el `new` operador), se puede usar `Number` para realizar una conversión de tipo. Si el argumento no se puede convertir en un número, devuelve [`NaN`](https://guide.freecodecamp.org/javascript/standard-objects/number/number-nan) .
