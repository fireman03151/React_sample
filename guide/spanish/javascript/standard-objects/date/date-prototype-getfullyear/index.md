## Date.prototype.getFullYear

El método Date.getFullYear () devuelve el año (cuatro dígitos) de la fecha especificada según la hora local.

### Sintaxis

```javascript
dateObj.getFullYear() 
```

### Ejemplo

```javascript
var date = new Date(); 
 // creates a new Date() object with current date and time. 
 date.getFullYear() 
 // 2018 
```
