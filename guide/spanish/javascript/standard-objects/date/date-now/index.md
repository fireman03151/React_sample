## Fecha ahora

El método Date.now () devuelve el número de milisegundos transcurridos desde el 1 de enero de 1970 a las 00:00:00 UTC.

### Sintaxis

```js
var timeInMs = Date.now(); 
```

### Ejemplo

```js
Date.now(); 
 // 1508783660969 
```
