Linting es el proceso de ejecutar un programa que analiza su código en busca de errores programáticos y de estilo. Una herramienta de alineación, o una guía, marca o marca cualquier error potencial en su código, como errores de sintaxis o nombres de variables incorrectamente deletreados. Esto puede ahorrar tiempo y ayudarlo a escribir mejor código.

Los linters se pueden instalar usando `npm` u otro administrador de paquetes. Los linters se pueden usar desde la línea de comando pasando archivos. Los linters también están disponibles como complementos para herramientas y, a veces, se integran directamente en los editores.

Aquí hay algunos Linters de Javascript populares:

[JSLint](http://www.javascriptlint.com/online_lint.php)

[JSHint](http://jshint.com)

[JSCS](http://jscs.info)

[ESLint](http://eslint.org)

#### Linters para editores:

*   Código visual: para instalar la extensión eslint, abra la paleta de comandos (Ver -> Comando Paleta .. o cmd + shift + p) y ejecute el comando: ext instalar eslint
