localeTitle: Preguntas de la entrevista para desarrolladores web frontales junior

El propósito de este artículo es familiarizar a los nuevos Desarrolladores Web Front-End con preguntas sobre libros de texto. Si recién estás ingresando al mundo de los desarrolladores, es muy probable que te sientas nervioso al entrevistar para conocer la posición de un desarrollador. Debido a esto, algunas preguntas relativamente básicas podrían sentirse más difíciles de lo que realmente son. Un buen contador para este escenario es estar bien preparado.

## Preguntas

Sin más preámbulos, aquí hay algunas preguntas que podría encontrar:

### HTML / CSS

1.  En el contexto del estilo CSS, ¿puede explicar qué significa `display-style: inline-block` ?
2.  Si tiene un elemento `<ul>` con varios elementos `<li>` , ¿cómo podría configurar estos elementos en una fila utilizando HTML y CSS?
3.  En el estilo CSS, ¿cuándo usarías `float` s? ¿Cómo trabajan?
4.  ¿Cuándo usarías: `clear: both;` ?
5.  ¿Qué es Bootstrap? ¿Cuándo lo usarías? ¿Cómo funciona su sistema de red?
6.  ¿Explica la noción de diseño responsivo y cómo funciona? Usa un ejemplo si quieres.
7.  ¿Qué es un DOM? ¿Qué sabe usted al respecto?
8.  ¿Qué es SASS? ¿Cuáles son las ventajas de usarlo?

### JavaScript

1.  ¿Qué es ECMAScript?
2.  ¿Qué es jQuery y para qué se usa comúnmente?
3.  ¿Cuál es la diferencia entre `==` y `===` en JS?
4.  ¿Puedes enumerar todos los tipos de datos de JavaScript?
5.  ¿Qué es una función de devolución de llamada en JS y cuándo utilizaría una?
6.  ¿Cuál es la diferencia entre `undefined` y `null` ?
7.  ¿Qué es una respuesta asíncrona?
8.  ¿Qué significa JSON? ¿Para qué se usa esto?
9.  ¿Cómo explicarías una solicitud de API?
10.  ¿Con qué API has trabajado?
11.  ¿Cómo enviarías / convertirías al formato JSON?
12.  ¿Cuál es la diferencia entre `GET` y `POST` cuando se especifica una solicitud AJAX?

### Software / Herramientas

1.  ¿Has trabajado con Photoshop como desarrollador frontal? ¿Con qué propósito?
2.  ¿Qué sistema de versionamiento de software has usado? ¿Podría describir cómo se vería un flujo de trabajo en este sistema?

### Acerca de ti

1.  ¿Cuál fue el proyecto más difícil en el que has trabajado? ¿Por qué fue difícil? Si es una aplicación / algoritmo, ¿puedes explicar cómo funciona?
2.  ¿Cómo te mantienes al día con las noticias de tecnología?
3.  ¿En qué proyectos laterales has estado trabajando?
4.  ¿Hay idiomas o marcos que le gustaría aprender en el futuro? ¿Cómo harías para enseñarte a ti mismo? ¿Por qué escogiste esos lenguajes / marcos?
5.  ¿Cuáles son las habilidades y cualidades más importantes que debe tener un desarrollador de front-end junior?
