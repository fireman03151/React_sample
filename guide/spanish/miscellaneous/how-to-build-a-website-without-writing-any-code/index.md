---
title: How to Build a Website Without Writing Any Code
localeTitle: Cómo construir un sitio web sin escribir ningún código
---
Es posible crear sitios web dinámicos y sensibles a dispositivos móviles sin necesidad de escribir ningún código, en tan solo unos minutos.

Aquí están las tecnologías que utilizamos:

*   [atom.io](http://www.atom.io/) un editor de código libre
*   [startbootstrap.com](http://www.startbootstrap.com/) una colección de plantillas gratuitas de respuesta (Bootstrap)
*   [powr.io](http://www.powr.io/) una colección de complementos de JavaScript gratuitos
*   [bitballoon.com](http://www.bitballoon.com/) una herramienta para el despliegue de sitios web de arrastrar y soltar

Alcanzará rápidamente los límites de lo que puede hacer sin realmente codificar, pero es bueno poder construir rápidamente sitios web de prototipos de trabajo como este.
