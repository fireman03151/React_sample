Hay ciertos caracteres en HTML que están reservados por el idioma, por ejemplo, `<` y `>` . Si los usa en un archivo HTML para otra cosa que no sean etiquetas, se pueden producir errores, ya que su navegador puede interpretarlos como etiquetas. Una forma de solucionar este problema es usar entidades HTML:

`<` se puede escribir como `&lt;` mientras que `>` se puede escribir como `&gt;` y se renderizarán como se espera en el navegador.

Puede encontrar una lista completa de las entidades HTML para otros caracteres reservados y no en su teclado como © y ∑ en [https://dev.w3.org/html5/html-author/charref](https://dev.w3.org/html5/html-author/charref) .
