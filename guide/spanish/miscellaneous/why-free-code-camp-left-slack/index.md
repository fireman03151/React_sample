---
title: Why Free Code Camp Left Slack
localeTitle: ¿Por qué Free Code Camp Left Slack?
---
Slack es un servicio de sala de chat popular que Free Code Camp superó

![Estadísticas de uso de Code Camp Slack gratis](//discourse-user-assets.s3.amazonaws.com/original/2X/d/da98adf7a036ae5bd8410b09c64f394d014694a0.png)

Por mucho que disfrutamos usando este servicio, es un mal ajuste para las grandes comunidades de código abierto. Dicho esto, muchas comunidades de código abierto todavía usan Slack, y Slack no hace nada para desalentar esta práctica.

Si tiene curiosidad, puede leer sobre [la publicidad falsa de Slack y la debacle que esto causó en junio de 2015](http://blog.freecodecamp.com/2015/06/so-yeah-we-tried-slack-and-we-deeply-regretted-it.html) .

Free Code Camp ahora usa [Gitter](https://github.com/FreeCodeCamp/freecodecamp/wiki/Gitter) .
