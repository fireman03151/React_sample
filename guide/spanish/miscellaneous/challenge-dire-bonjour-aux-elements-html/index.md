localeTitle: Desafío saludar a los elementos HTML

Los elementos HTML se escriben con una etiqueta de inicio, una etiqueta de finalización y contenido entre:

*   `<nom-de-la-balise>contenu</nom-de-la-balise>`

El elemento HTML incluye todo lo que se incluye desde la etiqueta inicial hasta la etiqueta final:

*   `<p>Mon premier paragraphe HTML.</p>`

Las etiquetas de apertura tienen este aspecto: `<h1>` , y las etiquetas de cierre tienen este aspecto: `</h1>`

*   Tenga en cuenta que la única diferencia entre una etiqueta de apertura y una etiqueta de cierre es que la etiqueta de cierre tiene una barra diagonal después del chevron de apertura. La barra diagonal en la etiqueta de cierre es muy importante porque indica que la etiqueta es el final de este elemento.
