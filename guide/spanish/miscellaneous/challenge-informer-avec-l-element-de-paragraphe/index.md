---
title: Challenge Informer Avec L Element De Paragraphe
localeTitle: El desafío informa con el elemento del párrafo
---
El elemento `p` es el elemento preferido para texto de párrafo de tamaño normal en sitios web. `p` es la abreviatura de "párrafo".

Puede crear un elemento `p` como este: `<p>Je suis un élément p !</p>` .
```
<h1>Hello World</h1> 
 <h2>ApplicationDePhotosDeChats</h2> 
 <p>Bonjour Paragraphe</p> 

```
