Este artículo está lleno de alternativas al software popular en Windows y Mac, que no está disponible en Linux.

## Alternativas:

> Siéntase libre de agregar software a esta lista en la forma de una solicitud de extracción.

[http://alternativeto.net/](http://alternativeto.net/) es un sitio muy útil para encontrar alternativas al software.

### [Adobe Photoshop](http://www.adobe.com/products/photoshop)

_Descripción:_ Photoshop es el software de edición de fotos, creación de imágenes y diseño gráfico de Adobe. Photoshop es muy útil para crear wireframes y maquetas de sitios web.

_Alternativas:_

*   [CAÑUTILLO](https://www.gimp.org/)
*   [Inkscape](https://inkscape.org)

### [Stata](https://www.stata.com)

_Descripción:_ Stata es un paquete de software de análisis estadístico comercial distribuido y mantenido por StataCorp que puede usarse tanto para el análisis estadístico basado en script como en GUI. Los paquetes o bibliotecas oficiales de StataCorp pueden complementarse con un sólido ecosistema de scripts escritos por el usuario llamado _SSC_ (el archivo de componentes de software estadístico de Boston College) alojado por [RePEC](http://repec.org) (el sitio web Research Papers in Economics). [StataList](http://www.statalist.org/) proporciona apoyo entre pares y oficial en formato de tablero de mensajes; El [Stata Journal](http://www.stata-journal.com/) oficial es una publicación trimestral que presenta artículos revisados ​​por pares sobre "estadísticas, análisis de datos, métodos de enseñanza y uso efectivo del lenguaje de Stata".

*   [PSPP](https://www.gnu.org/software/pspp/) y [sus alternativas](https://alternativeto.net/software/pspp/)
*   [RKWard](https://rkward.kde.org) y [sus alternativas](https://alternativeto.net/software/rkward/)
