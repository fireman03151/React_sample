# Gitter es un sistema de sala de chat popular entre los proyectos de código abierto.

![Una captura de pantalla de Gitter.](//discourse-user-assets.s3.amazonaws.com/original/2X/9/901b6ff18a83b746b1b7e96cb6907440d531d977.png)

[Gitter.im](https://gitter.im) es una startup de rápido crecimiento con sede en Londres.

Free Code Camp tiene cientos de habitaciones Gitter. Estos son accesibles a través del sitio web de Gitter, así como a [aplicaciones de escritorio y móviles](https://gitter.im/apps) .

Puede ver [todas nuestras habitaciones](https://github.com/FreeCodeCamp/FreeCodeCamp/wiki/Official-Free-Code-Camp-Chat-Rooms) y unirse a las que le gustaría.

Gitter te permite compartir [código formateado](https://github.com/FreeCodeCamp/freecodecamp/wiki/Pasting-Code-in-Gitter) con otros campistas.
