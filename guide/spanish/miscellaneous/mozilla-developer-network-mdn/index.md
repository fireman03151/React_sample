## Red de Desarrolladores de Mozilla (MDN)

[MDN](https://developer.mozilla.org/) es una plataforma de aprendizaje para las tecnologías web y el software que impulsa la web.

El objetivo de MDN es proporcionar a los desarrolladores la información que necesitan para construir fácilmente proyectos en la Web abierta. Si se trata de una tecnología abierta expuesta a la web, MDN desea documentarla.
