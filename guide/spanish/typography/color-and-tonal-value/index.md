#### Más información:

En la teoría del color, un valor tonal se produce al agregar blanco, gris o negro a un color seleccionado. Esto no cambia el tono, pero altera el colorido, también conocido como saturación. Cuando se habla de valor tonal, hay tres términos principales que se deben discutir: Tinte, Tono y Sombra.

Tinte es la adición de blanco a un color. El tinte se puede usar para resaltar un área y comenzar a crear la ilusión de profundidad en un objeto.

El tono es la adición de gris a un color. El color tonal crea un color más apagado y menos saturado.

Sombra es la adición de negro a un color. La sombra se puede usar para oscurecer y el área para crear la ilusión de profundidad en un objeto.

Al modificar el valor tonal de los colores, puede crear la ilusión de profundidad en las imágenes, así como modificar el nivel de saturación para aplicar mejor el color a una emoción o estado de ánimo deseado.
