## Alineación

La alineación tipográfica altera el flujo de texto en un documento. Los diseñadores pueden usar esta poderosa herramienta para mejorar sutilmente la presentación del texto al darle forma y contexto visuales.

Experimente con la alineación para encontrar nuevas combinaciones visuales emocionantes que hacen que su contenido se destaque. Asegúrese de probar los errores de alineación en diferentes dispositivos y navegadores.

Usa CSS para alinear el texto siempre que sea posible. Puede encontrar etiquetas de alineación HTML (por ejemplo, `<p align="right"></p>` o `<center></center>` ), pero están en desuso en HTML 5.

### Texto justificado a la izquierda

Por defecto, `text-align: left;`

En caso de duda, el tipo predeterminado de justificación de texto suele ser la mejor opción. Especialmente para artículos o bloques largos de texto, el texto justificado a la izquierda es más fácil de leer, ya que los bordes irregulares de la derecha proporcionan un contexto visual para ayudar a los lectores a mantener su lugar mientras sus ojos cambian de línea en línea.

Usar para:

*   Cuerpo de texto
*   Casi todo lo demás

### Texto justificado a la derecha

`text-align: right;`

El texto justificado a la derecha e irregular puede ser útil como una especia visual, pero es más difícil de leer en largos períodos de tiempo que el texto justificado a la izquierda.

Puede ser utilizado para:

*   Subtítulos
*   Navegación
*   Barras laterales
*   Cotizaciones

### Texto centrado

`text-align: center;`

Use el texto centrado para atraer la atención de los lectores a los puntos de énfasis (por ejemplo, encabezados, títulos de página, pies de página). No utilice texto centrado para bloques largos de texto, ya que es extremadamente agotador de leer.

Puede ser utilizado para:

*   Encabezados
*   Título de la página
*   Navegación
*   Barras laterales
*   Subtítulos
*   Cotizaciones
*   Pies de página

### Texto justificado

`text-align: justify;`

El texto con justificación completa es difícil de recomendar para el texto del cuerpo, ya que pueden aparecer grandes espacios entre las palabras en algunas líneas. Sin embargo, los diseñadores expertos pueden hacer un buen uso de él.

Puede ser utilizado para:

*   Subtítulos
*   Sidenotes
