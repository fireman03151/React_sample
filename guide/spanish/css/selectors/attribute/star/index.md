## Estrella

El selector de atributos Star CSS o \* se utiliza para seleccionar atributos que contienen un valor específico.

#### Ejemplo

Si desea buscar todos los enlaces de anclaje que contienen "código" en cualquier lugar del valor de la URL y hacerlos amarillos, puede hacerlo de la siguiente manera:

```css
a[href*="code"] { 
   color: yellow; 
 } 

```
