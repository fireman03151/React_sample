## Enlazar

La pseudo-clase CSS `:link` se usa para apuntar a todos los elementos no visitados (por ejemplo, elementos `<a>` delimitadores) que tienen un atributo `href` , incluso si el `href` tiene un valor vacío.

Sintaxis general:

```css
target:link 
```

## Ejemplo

```css
/* Selects any <a> that has not been visited yet */ 
 a:link { 
    color: red; 
 } 
```
