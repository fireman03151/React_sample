## Propiedad de visibilidad

La propiedad de visibilidad se puede utilizar para controlar la visibilidad de cualquier elemento. El valor predeterminado es `visibility: visible;`

#### Valores posibles:

La propiedad de visibilidad puede tener 5 valores posibles.

1.  `visible` : es el valor predeterminado que muestra un elemento.
2.  `hidden` : este valor oculta un elemento pero conserva el diseño o el espacio ocupado por el elemento.
3.  `collapse` : este valor es similar a oculto pero es aplicable para filas o columnas de tablas. Oculta el contenido de la fila o columna.
4.  `initial` : esto establece la propiedad a su valor predeterminado (visible).
5.  `inherit` : esto hereda el valor de la propiedad de su elemento padre.

#### Similitudes y diferencias con otras propiedades:

1.  opacidad: la diferencia entre la propiedad de `visibility` y la propiedad de `opacity` es la opacidad que se puede utilizar para el desvanecimiento y el elemento. `visibility: hidden;` y `opacity: 0;` tiene un efecto similar en un elemento.
    
2.  pantalla: La única diferencia entre visibilidad: oculta; y pantalla: ninguno; es que, este último elimina el diseño o el espacio ocupado por el elemento también.
    

#### Compatibilidad del navegador:

Esta es una propiedad CSS2 y es compatible con todos los principales navegadores.
