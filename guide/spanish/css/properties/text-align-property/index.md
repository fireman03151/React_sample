## Propiedad de alineación de texto

La propiedad text-align especifica la alineación horizontal del texto en un elemento.

```css
    .text-align { 
        /* List of keyword values of text-align property*/ 
        text-align: left|right|center|justify|initial|inherit; 
    } 
```

Docs: [MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/text-align)
