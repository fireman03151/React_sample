## Propiedad de ancho

La propiedad de ancho establece el ancho de un elemento. Puede ser un valor de palabra clave, un valor de longitud (cualquier valor seguido de una unidad CSS como px o em), porcentaje, o puede ser heredado por su padre. Por defecto, su valor es automático.

**Nota** : la propiedad de ancho no incluye relleno, bordes o márgenes; establece el ancho del área dentro del relleno, el borde y el margen del elemento!

```css
    .length-value {width: 100px;} 
    .percentage {width: 50%;} 
    .auto {width: auto;} 
    .inherit {width: inherit;} 
    .initial {width: initial;} 
```

**Nota** : Las propiedades de ancho mínimo y ancho máximo anulan el ancho.
