## Fuentes Web Safe

Las fuentes seguras para la Web son fuentes que se incluyen con la mayoría de los sistemas operativos. La implicación de una disponibilidad tan alta es que un diseñador puede esperar que la tipografía de las fuentes seguras para la web aparezca exactamente como está destinada a la mayoría de los usuarios. A continuación, se incluyen listas no exhaustivas de algunas fuentes que se consideran seguras para la Web en el momento de la escritura, clasificadas por familias de fuentes CSS genéricas.

Fuentes de serif seguras para la web:

*   Georgia
*   Times New Roman

Fuentes seguras web sans-serif:

*   Arial
*   Tahoma
*   Trebuchet MS
*   Verdana

Fuentes seguras monoespaciadas web:

*   Mensajero nuevo

Vale la pena tener en cuenta que las pilas de fuentes con opciones de respaldo, incluida una familia de fuentes genérica, aún deben usarse, incluso si su diseño utiliza solo fuentes seguras para la web. Por ejemplo:

```css
p { 
  font-family: Tahoma, Arial, sans-serif; 
 } 
```

#### Una nota sobre fuentes web

El hecho de que algunas fuentes sean más seguras que otras no significa que deba limitar sus diseños a usar solo fuentes seguras para la Web. Los diseños modernos con CSS también pueden aprovechar las fuentes web para garantizar una tipografía consistente en todos los sistemas operativos.
