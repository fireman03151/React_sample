## La propiedad de posición

La propiedad de posición especifica el tipo de método de posicionamiento utilizado para un elemento (estático, relativo, fijo, absoluto o adhesivo). La propiedad de posición especifica el tipo de método de posicionamiento utilizado para un elemento. La propiedad de posición no se usa generalmente para crear diseños, sino que se usa para colocar elementos que de alguna manera se destacan del flujo de páginas.

Hay cinco valores de posición diferentes:

*   estático
*   relativo
*   fijo
*   absoluto
*   pegajoso Los elementos se colocan utilizando las propiedades superior, inferior, izquierda y derecha. Sin embargo, estas propiedades no funcionarán a menos que la propiedad de posición se establezca primero. También funcionan de manera diferente dependiendo del valor de posición.
