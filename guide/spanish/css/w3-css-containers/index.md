#Contenedores CSS W3 W3.css proporciona una clase contenedora que agrega 16 píxeles por valor de relleno izquierdo y derecho a cualquier elemento html, y 0.01em de Acolchado superior e inferior. Esto lo hace ideal para html que contiene elementos como `<header>, <div>, <footer>, <section>` etc. Pero también se puede usar con gran efecto en elementos como `<img> and <p>.`

### Ejemplo simple
```
<header class="w3-container"> 
  <h1>Header With W3 Container</h1> 
 </header> 
```

```
<header> 
  <h1>Header Without W3 Container</h1> 
 </header> 
```

Produciría el siguiente resultado.

![Encabezado img](https://github.com/OcelotDive/randomImages/blob/master/images/WithWithout.PNG)

### Para utilizar los contenedores W3.Css

Solo agregue un enlace de hoja de estilo regular al encabezado de su página web así:
```
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css"> 
```

o descargue el archivo css más reciente en su máquina local. Aquí: (https://www.w3schools.com/w3css/4/w3.css)
