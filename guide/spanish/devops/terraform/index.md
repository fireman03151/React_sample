## Terraform

Terraform es una infraestructura como software de código de HashiCorp. Permite a los usuarios definir un centro de datos. infraestructura en un lenguaje de configuración de alto nivel, desde el cual se puede crear un plan de ejecución para cree la infraestructura en un proveedor de servicios como OpenStack o en un proveedor de servicios como IBM Cloud (anteriormente Bluemix), AWS, Microsoft Azure o Google Cloud Platform. Se define la infraestructura en una sintaxis HCL Terraform o en formato JSON.
