## Git verificación de compromisos

Cuando está creando software con personas de todo el mundo, a veces es importante validar que las confirmaciones y las etiquetas provienen de una fuente identificada. Git admite la firma de confirmaciones y etiquetas con GPG. GitHub muestra cuando se comprometen y las etiquetas están firmadas.

![Compromisos Verificados](https://cloud.githubusercontent.com/assets/25792/14253743/87b504be-fa41-11e5-9140-6dc8b7203c31.png)

Cuando vea una confirmación o etiqueta firmada, verá una credencial que indica si la firma se pudo verificar mediante alguna de las claves GPG del contribuyente cargadas en GitHub. Puede cargar sus claves GPG visitando la página de configuración de claves.

Muchos proyectos y compañías de código abierto quieren estar seguros de que un compromiso proviene de una fuente verificada. La verificación de firma GPG en confirmaciones y etiquetas hace que sea fácil ver cuándo una confirmación o etiqueta está firmada por una clave verificada que GitHub conoce.

![Firma verificada](https://cloud.githubusercontent.com/assets/25792/14290042/5b27dab2-fb12-11e5-9ff9-44116a7780ea.png)
