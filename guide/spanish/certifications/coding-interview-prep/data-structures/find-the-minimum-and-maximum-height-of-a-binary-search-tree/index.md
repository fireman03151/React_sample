## Encuentra la altura mínima y máxima de un árbol de búsqueda binario

Cada navegador web moderno incluye un potente conjunto de herramientas para desarrolladores. Estas herramientas hacen una serie de cosas, desde la inspección de HTML, CSS y JavaScript cargados actualmente, hasta mostrar qué activos ha solicitado la página y cuánto tardaron en cargarse. Algunas de las características principales se enumeran a continuación:

*   Acceso a una consola del navegador.
*   Visores sensibles a la prueba y específicos del dispositivo
*   Editar el DOM
*   Depuración
*   Analizar la actividad de la red.
*   Comprender los problemas de seguridad

y mucho más…

## Cómo abrir herramientas de desarrollo

### Teclado
```
Ctrl + Shift + I 
```

### Barra de menús

#### Firefox
```
Tools ➤ Web Developer ➤ Toggle Tools 
```

#### Cromo / cromo
```
Tools ➤ Developer Tools 
```

#### Safari
```
Develop ➤ Show Web Inspector. 
```

Si no puede ver el menú Desarrollar, vaya a `Safari ➤ Preferences ➤ Advanced` y marque la casilla `Show Develop menu` en la barra de menús.

#### Ópera
```
Developer ➤ Web Inspector 
```

### Menú de contexto

`Right-click` un elemento de una página web (presione Ctrl y haga clic en Mac) y elija `Inspect Element` en el menú contextual que aparece.

Este método resalta el código del elemento en el que hizo clic derecho.
