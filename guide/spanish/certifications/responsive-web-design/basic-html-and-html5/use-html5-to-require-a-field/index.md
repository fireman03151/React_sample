## Usa HTML5 para requerir un campo

El atributo `required` funciona con casi todos los tipos posibles de la etiqueta de `input` (no funciona porque no tiene sentido para los tipos tipo botones o los que tienen un valor predeterminado, por ejemplo, `type=range` ) y obliga al usuario a abandonar al menos una marca mínima de la interacción con la entrada requerida al enviar (seleccione _al menos_ una casilla de verificación, escriba _al menos_ un carácter en un cuadro de texto, etc.)

Su uso bloquea el envío del formulario hasta `required` no se cumpla la condición `required`

Puede usar el atributo `required` con una sintaxis particular: ya que es un valor booleano que puede escribir solo es `required` sin ningún valor.

En este desafío ya tiene un formulario y un elemento de entrada: le pide que agregue el atributo `required` al elemento de entrada; recuerde que si algo del código original se modifica o elimina por accidente, siempre puede reiniciar con un código limpio usando el botón de `reset all code` .
