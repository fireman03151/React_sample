## Crear una lista ordenada

A veces las personas piensan demasiado en este tipo de desafío: las listas ordenadas tienen el mismo patrón exacto de las desordenadas, simplemente cambia la etiqueta: `<ul>` para los desordenados, `<ol>` para los ordenados.

El efecto diferente es que uno tiene las viñetas, el otro tiene números (u otras marcas progresivas):

tipo | código | efecto - | - | - desordenado | `<ul><li>Bulleted item</li></ul>` |

*   Elemento con viñetas

ordenado | `<ol><li>Numbered item</li></ol>` |

1.  Artículo numerado

Si no recuerda cómo funcionan las listas, puede revisar el desafío anterior: [Cree una lista desordenada con viñetas](https://learn.freecodecamp.org/responsive-web-design/basic-html-and-html5/create-a-bulleted-unordered-list) .
