## Definir la cabeza y el cuerpo de un documento HTML

La `head` elemento se utiliza para agrupar todos los metadatos del documento, el `body` para visualizar el contenido del documento.

En la `head` puede encontrar cualquier tipo de etiqueta utilizada para _describir_ el documento: qué idioma utilizará, qué reglas de estilo (a través del enlace de la hoja de estilo), su título, una descripción literal, ...  
De todos modos, no hay nada dentro de la `head` elemento se representará en la página: se puede ver el título y el icono por lo general en la pestaña del navegador o en la barra de marcadores, pero esa no es la _página,_ es el elemento navegador vinculado a él.

El `body` por el contrario, contiene todo lo que se representa en la página: tal vez no solo eso, sino también todo eso.

El desafío le proporciona una página html simple y le pide que inserte la `head` y los elementos html del `body` en el lugar correcto; en otras palabras, debe envolver con la etiqueta `<body></body>` todos los elementos que puede ver representados en el lado derecho de la página y envolver con la etiqueta `<head></head>` todo descriptivo.

Recuerde que tanto la `head` como el `body` deben ser hijos del elemento `<html>` .
