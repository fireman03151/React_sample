## Crear un conjunto de botones de radio

Para resolver este desafío, puede dividirlo en pedazos para comprender mejor los pasos:

*   Cree los 'contenedores': coloque dos elementos de `label` y bríndeles el contenido del texto que el desafío le pide (uno debe tener el texto 'Indoor', uno 'Outdoor', mayúscula);
    
*   Cree el 'contenido': un botón de radio es solo un elemento de `input` con `type=radio` (cree dos de ellos);
    
*   Cree un _grupo_ de botones de radio: asigne a ambos botones de radio el `name` del atributo establecido en el mismo valor (las instrucciones de desafío indican que el grupo debe llamarse `indoor-outdoor` );
    
*   Anida cada botón de radio en una etiqueta;
    
    ¡Buena suerte!
