## Convertir una imagen en un enlace

En un desafío anterior, ya logró la tarea de anidar un elemento HTML en otro (por ejemplo, anidar un elemento de [anclaje dentro de un párrafo](https://learn.freecodecamp.org/responsive-web-design/basic-html-and-html5/nest-an-anchor-element-within-a-paragraph) )

Ahora se le pide que resuelva el mismo problema con diferentes etiquetas: simplemente coloque la imagen (tal como está) dentro del elemento de anclaje y se convertirá en el contenido seleccionable de ese anclaje.

¡Buena suerte!
