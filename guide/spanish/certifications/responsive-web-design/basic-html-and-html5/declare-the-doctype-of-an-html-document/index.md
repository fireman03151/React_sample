## Declarar el doctype de un documento HTML

La declaración del Doctype debe estar en la primera línea ya que proporciona instrucciones al navegador sobre cómo interpretar las siguientes líneas de código.

Puede ir seguido de un elemento `<html>` , incluso si no es obligatorio; Este es el elemento raíz del documento y, si no lo escribe, la mayoría del navegador lo deducirá. Sin embargo, es una buena práctica escribirlo explícitamente.

Todos sus elementos html deben estar anidados dentro del `<html>` uno: en este desafío se le solicita que use un elemento `<h1>` con el contenido de su elección.
