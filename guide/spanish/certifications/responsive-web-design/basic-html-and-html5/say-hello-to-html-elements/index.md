## Diga hola a los elementos HTML

Considere el siguiente ejemplo:  

`<p>This is a paragraph</p>`  

Puede referirse como un **_'elemento p'_** ; desglosado en pedazos puedes encontrar los siguientes subelementos:

subelemento | descripción ----------- | ---------- `<p>` | Esta es la etiqueta de apertura `This is a paragraph` | Este es el texto del elemento. `</p>` | Esta es la etiqueta de cierre

Recuerda la solicitud de desafío:

> cambia el texto de tu elemento h1 para que diga "Hola mundo"

¡Buena suerte!
