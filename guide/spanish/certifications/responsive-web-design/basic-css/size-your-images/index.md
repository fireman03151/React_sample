## Tamaño de sus imágenes

Para dimensionar sus imágenes, primero cree su clase en su etiqueta de estilo.

Un ejemplo: `<style> .fixed-image { width: 500px; height: auto; }`

A continuación, puede agregar la clase a su imagen: `<img class="fixed-image" src="http://www.example.com/picture"/>`
