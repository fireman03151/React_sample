## Ajustar el margen de un elemento

Ajustar el `Margin` de un `Element` significa aumentar o disminuir la cantidad de espacio entre los bordes del elemento y los elementos circundantes.

• Los márgenes se ajustan con enteros positivos y negativos por unidades de píxeles ( `px` ).

• Los márgenes se pueden crear desde todas las direcciones (arriba, abajo, izquierda, derecha).

Este es un ejemplo de cómo está escrito:

```css
margin: 20px; 
```

Puede cambiar o ajustar fácilmente el `Margin` de un `Element` cambiando el valor numérico:

**Antes de**

margen: `20` px;

**Despues**

margen: `50` px;

También puede crear una `class` personalizada con un `margin` que usará repetidamente.

```css
  .style-margin { 
    margin: 20px; 
  } 
```
