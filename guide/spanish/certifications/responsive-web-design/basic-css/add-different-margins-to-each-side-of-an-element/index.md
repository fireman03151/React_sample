## Añadir diferentes márgenes a cada lado de un elemento

Para ajustar los márgenes de un elemento use:

```css
.example { 
  margin: 10px; 
 } 
```

Para especificar el tamaño de los márgenes en un elemento por lados individuales, podemos usar 'margen superior', 'margen derecho', 'margen inferior' y 'margen izquierdo'. Podemos usar cualquiera de estos en combinación y en cualquier orden. Por ejemplo:

```css
.example { 
  margin-top: 5px; 
  margin-bottom: 0px; 
 } 
```

O:

```css
.example { 
  margin-top: 20px; 
  margin-left: 25px; 
  margin-right: 5px; 
 } 

```
