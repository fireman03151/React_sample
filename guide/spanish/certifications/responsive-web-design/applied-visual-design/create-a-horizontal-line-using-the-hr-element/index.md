## Crear una línea horizontal utilizando el elemento hr

La etiqueta `hr` es uno de los elementos HTML más simples. Crea una delgada línea horizontal a través de su elemento padre.

Esta lección utiliza la etiqueta `hr` para separar el título y el contenido de una "tarjeta". Entonces, ¿cómo crear un elemento `hr` ?

```html

<!-- It's this easy --> 
 <hr> 
```

Este elemento es de cierre automático, pero para ser amigable con los analizadores y verificadores de estilo, debe agregar un marcador de cierre explícito.

```html

<hr /> 
```

Sea cual sea el formato que elija, se creará una línea horizontal como en [esta lección](https://learn.freecodecamp.org/responsive-web-design/applied-visual-design/create-a-horizontal-line-using-the-hr-element/) en [freeCodeCamp](https://www.freecodecamp.org/) .
