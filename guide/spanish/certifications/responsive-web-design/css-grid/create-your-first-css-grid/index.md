## Crea tu primera cuadrícula CSS

### Sugerencia 1

Para cambiar la visualización de cualquier elemento, se utiliza la siguiente sintaxis:

```css
display: propertyName; 
```

### Sugerencia 2

Cambiar la propiedad de visualización del elemento .container

### Sugerencia 3

Establecer la visualización del elemento .container a la cuadrícula

### Solución

Para configurar la visualización del elemento .container a la cuadrícula:

```css
display: grid; 

```
