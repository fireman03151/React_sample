## Crear un espacio en la fila usando la rejilla-fila-espacio

Este desafío requiere que agregue un espacio de 20 `20px` entre todas las filas de la Cuadrícula de CSS en .container.

### Insinuación

Para agregar una brecha entre las filas en una cuadrícula de CSS, usamos la propiedad CSS de `grid-row-gap` .

### Solución

Dado que el desafío tiene que agregar un espacio de `5px` , agregaría la siguiente línea al bloque de código CSS de `.container` .

```css
grid-row-gap: 5px; 

```
