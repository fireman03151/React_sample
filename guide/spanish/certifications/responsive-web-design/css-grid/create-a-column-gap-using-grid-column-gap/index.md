## Crear una brecha de columna usando grid-column-gap

Este desafío requiere que agregue un espacio de 20px entre todas las columnas de la Cuadrícula de CSS en .container.

### Insinuación

Para agregar el espacio entre las columnas de una cuadrícula, usamos:

```css
grid-column-gap: parameter; 
```

donde parámetro es la brecha requerida.

### Solución

Dado que el desafío tiene que agregar un espacio de 20 px, agregaría la siguiente línea al bloque de código CSS de .container.

```css
grid-column-gap: 20px; 

```
