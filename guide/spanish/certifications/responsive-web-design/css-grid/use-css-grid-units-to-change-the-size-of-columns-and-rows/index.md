## Utilice las unidades de cuadrícula de CSS para cambiar el tamaño de las columnas y las filas

Este desafío requiere que establezca el ancho de las columnas del contenedor a las que se especifican en la descripción del desafío.

### Insinuación

Cambia la propiedad `grid-template-columns` .

### Solución

Ya que el desafío que requiere para establecer el ancho de `1fr` , `100px` , y `2fr` , cambiar el `css grid-template-columns` propiedad de `.container` a:

```css
grid-template-columns: 1fr 100px 2fr; 
```
