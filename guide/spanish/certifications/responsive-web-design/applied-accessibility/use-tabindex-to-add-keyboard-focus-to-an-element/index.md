## Use tabindex para agregar el foco de teclado a un elemento

Siguiendo las instrucciones:

Agregue un atributo de tabindex a la etiqueta p y establezca su valor en "0".

la linea 16 \`\` \`css

Instrucciones: Complete toda su información y luego haga clic en **Enviar**
```
becomes: 
 ```css 
    <p tabindex="0">Instructions: Fill in ALL your information then click <b>Submit</b></p> 
```

De esta manera, recibirá un enfoque de teclado cuando un usuario haga clic en una página.
