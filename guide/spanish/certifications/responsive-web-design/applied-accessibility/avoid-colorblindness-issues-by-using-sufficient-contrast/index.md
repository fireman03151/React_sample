## Evite los problemas de ceguera al color usando un contraste suficiente

Siguiendo las instrucciones:

Aumente el valor de luminosidad del color de fondo del 35% al ​​55% y disminuya el valor de luminosidad del color del 20% al 15%.

Las líneas 4 y 5 se convierten en:

```css
    color: hsl(0, 55%, 15%); 
    background-color: hsl(120, 25%, 55%); 
```

Esto mejora el contraste a 5.9: 1.
