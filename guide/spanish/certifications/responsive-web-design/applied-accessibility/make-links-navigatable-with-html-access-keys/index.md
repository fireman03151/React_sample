## Haga que los enlaces sean navegables con claves de acceso HTML

Siguiendo las instrucciones: Agregue un atributo de clave de acceso a ambos enlaces y establezca el primero en "g" (para Garfield) y el segundo en "c" (para Chuck Norris). Las líneas 8 y 16 se convierten en:

```css
<h2><a id="first" accesskey="g" href="">The Garfield Files: Lasagna as Training Fuel?</a></h2> 
```

```css
<h2><a id="second" accesskey="c" href="">Is Chuck Norris a Cat Person?</a></h2> 
```

De esta manera, los enlaces en torno a los dos títulos de artículos de blog tienen atajos de teclado para que los usuarios de su sitio puedan navegar rápidamente a la historia completa.
