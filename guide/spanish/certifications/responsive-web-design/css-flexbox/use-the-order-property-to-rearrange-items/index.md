## Utilice la propiedad de orden para reorganizar artículos

Antes de realizar cualquier cambio, preste atención al orden de los colores. Obtienes el azul a la izquierda y el rojo a la derecha. Después de establecer la propiedad del `order` , podrá elegir cuál se debe mostrar primero sin tener que cambiar el código html.

\`\` \`css

# caja-1 {
```
background-color: dodgerblue; 
 order: 2; 
 height: 200px; 
 width: 200px; 
```

} \`\` \`
