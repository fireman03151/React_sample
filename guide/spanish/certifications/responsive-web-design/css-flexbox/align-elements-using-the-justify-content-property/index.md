## Alinear elementos usando la propiedad justify-content

Como se indica en el desafío, para superarlo, debe establecer el `justify-content` en el centro. Sin embargo, el objetivo del desafío es experimentar y ver cómo las diferentes opciones afectan el contenido.

`center` : asegurará que los elementos secundarios estén centrados en el contenedor principal.  
`flex-start` : empujará todo hacia el lado izquierdo.  
`flex-end` : empujará todo hacia el lado derecho.  
`space-between` : empujará los primeros y últimos elementos a la izquierda y la derecha respectivamente y agregará un espacio par entre ellos. Si tiene un número impar de elementos, los agregará al centro y continuará presionando hacia la izquierda y hacia la derecha según el orden.  
`space-around` : hará un espacio similar al espacio intermedio, pero habrá un espacio igual alrededor de los elementos, por lo que el primero y el último no estarán directamente en el borde.
