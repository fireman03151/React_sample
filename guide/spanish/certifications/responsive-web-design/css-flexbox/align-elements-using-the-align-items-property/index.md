## Alinear elementos usando la propiedad align-items

Si bien se recomienda probar las diferentes opciones para `align-items` , establecer el valor para `center` es la única forma de superar este desafío.

Mientras juega con las diferentes opciones, se dará cuenta de que `strech` es el valor predeterminado para la propiedad.
