## Utilice la propiedad flex-wrap para envolver una fila o columna

La configuración de la propiedad `flex-wrap` para `wrap` hará que las columnas vuelvan a su anchura especificada y las columnas grises y negras se enrollarán al principio del marco.

Tenga en cuenta que configurarlo para `wrap-reverse` no hará que el espacio vacío gris del `#box-container` permanezca en la parte inferior mientras cambia el orden de las columnas.
