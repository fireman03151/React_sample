## Usa la propiedad de dirección flexible para hacer una columna

Para apilar los elementos secundarios de su contenedor flexible uno encima del otro, cambiaría la _dirección flexible de_ la siguiente manera:

```CSS
#main-container { 
    display: flex; 
    flex-direction: column; 
 } 

```
