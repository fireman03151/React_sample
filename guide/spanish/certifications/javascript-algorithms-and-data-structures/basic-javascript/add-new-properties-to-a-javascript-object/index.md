## Agregar nuevas propiedades a un objeto de JavaScript

Aquí está el ejemplo:

```js
var ourDog = { 
  "name": "Camper", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["everything!"] 
 }; 
 
 ourDog.bark = "bow-wow"; 
```

Aquí hay una solución:

```js
var myDog = { 
  "name": "Happy Coder", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["freeCodeCamp Campers"] 
 }; 
 
 // Only change code below this line. 
 myDog.bark = "woof"; 

```
