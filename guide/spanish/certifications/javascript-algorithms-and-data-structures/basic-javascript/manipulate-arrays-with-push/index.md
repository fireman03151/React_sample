## Manipular matrices con empuje ()

El método `push()` permite agregar (agregar al final) un elemento a una matriz. Al igual que…

```javascript
var arr = [1, 2, 3, 4]; 
 arr.push(5); // Now, the array is [1, 2, 3, 4, 5] 

```
