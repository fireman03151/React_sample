## Construir una máquina de cotización aleatoria

El proyecto consta de tres partes distintas:

1.  Identificando los componentes necesarios para completar la tarea, ¿qué componentes hay? ¿Se pueden usar algunos componentes más de una vez? Por ejemplo, los botones, ¿son los mismos solo con diferentes eventos onClick?
2.  ¿Qué componente debe ser responsable de mantener el estado y cómo se deben pasar los cambios de estado a otros componentes?
3.  ¿Cómo se debe buscar una nueva cita? ¿Dónde deben almacenarse la nueva cita y el autor?
