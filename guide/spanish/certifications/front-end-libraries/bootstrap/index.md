## Oreja

[Bootstrap](https://www.getbootstrap.com/) es una biblioteca de aplicaciones para usuario que proporciona un marco o diseño básico que un usuario puede usar o modificar directamente y luego implementar. Su objetivo principal es hacer un marco compatible con dispositivos móviles, pero Bootstrap funciona también con dispositivos de escritorio.

Citado en el sitio web de Bootstrap:
```
Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery. 
```

El marco encuentra la empleabilidad en muchos proyectos de aplicaciones para usuario y no es calificado como el marco HTML y CSS más famoso.
