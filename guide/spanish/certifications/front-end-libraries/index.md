## Bibliotecas front-end

Una biblioteca de front-end es un archivo o un fragmento de código escrito previamente que es útil para el usuario al reducir el tiempo consumido.

Estas bibliotecas de código ayudan a sus usuarios a minimizar su tiempo en la escritura de largas líneas de código y aumentan su productividad. Por ejemplo, las bibliotecas front-end como [Bootstrap](https://getbootstrap.com/) proporcionan un marco para que los usuarios desarrollen y reduzcan el tiempo consumido, al tiempo que permiten flexibilidad para el usuario.

Las bibliotecas pueden ser de opinión o no opionadas, es decir, dejar algunas opciones al usuario y no dejar ninguna opción para el usuario respectivamente.

Dado que se trata de bibliotecas front-end, funcionan con HTML, CSS y JavaScript y modifican el diseño, el estilo y el comportamiento de la página en un navegador, respectivamente.
