## Crear un componente funcional sin estado

### Sugerencia 1

*   Las funciones sin estado son una de las formas de crear un componente React.
*   Estas funciones pueden recibir datos y procesarlos, pero no administran ni rastrean los cambios en esos datos.
*   Para hacer el componente, un componente sin estado, una función de retorno es una de las cosas necesarias y eso es lo que falta.

### Sugerencia 2

Una función de retorno se agrega así

```jsx
return( 
    <return elements> 
 ); 
```
