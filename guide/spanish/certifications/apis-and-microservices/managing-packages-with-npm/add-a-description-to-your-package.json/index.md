## Agregue una descripción a su package.json

Para completar el desafío sigue los siguientes pasos:

*   Vaya al enlace que se encuentra en la página de introducción de currículum. ![Introducción de apis-and-microservices de freecodecamp](https://user-images.githubusercontent.com/15084301/45616545-65732480-ba8d-11e8-9893-412ff6c68be5.png) [Enlace al proyecto de inicio de Glitch](https://glitch.com/#!/import/github/freeCodeCamp/boilerplate-npm)
    
*   Edita el archivo package.json y actualiza el autor.
    
*   Luego haga clic en el enlace del programa (su aplicación) en la parte superior del proyecto Glitch para obtener el enlace público ![Enlace público del proyecto de Glitch](https://user-images.githubusercontent.com/15084301/45616579-8176c600-ba8d-11e8-8a46-4ac930578e74.png)
    
*   Presentar el enlace en la solución y eso es todo. ![página de éxito](https://user-images.githubusercontent.com/15084301/45616757-019d2b80-ba8e-11e8-8b5b-98dc6d02c235.png)
