## Use el analizador de cuerpo para analizar las solicitudes POST

El analizador de cuerpo ya debería estar agregado a su proyecto si usó la placa de caldera provista, pero si no, debería estar allí como:

```json
"dependencies": { 
    "body-parser": "^1.4.3", 
    ... 
```

Lo que hay que hacer para este desafío es pasar el middleware a app.use (). Asegúrese de que venga antes de las rutas en las que se debe utilizar.
