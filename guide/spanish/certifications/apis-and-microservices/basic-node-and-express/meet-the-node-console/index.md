## Conozca la consola Node

Asegúrese de haber clonado el repositorio correcto / el proyecto de error y está escribiendo su código en myapp.js.

Hay muchos métodos disponibles desde la consola del nodo. Puede encontrar una lista de estos métodos con la documentación de Node.js [aqui](https://nodejs.org/dist/latest-v10.x/docs/api/console.html) Un uso común sería registrar errores como:

```javascript
    console.log("An error happened"); 
```

### Si usas Glitch

*   Glitch había cambiado el botón 'Registros' al botón 'Estado', aún en la parte superior izquierda, debajo del nombre de la aplicación
*   No envíe la página 'Editar' directamente. Haga clic en el botón 'Mostrar' (en la parte superior) y envíe la aplicación a la url de la página de presentación (el formato de la url es como https://appname.glitch.me/)
