## ¿Qué es AB Testing?

Las pruebas A / B (a veces llamadas pruebas divididas) comparan dos versiones de una página web o elemento para ver cuál funciona mejor. Comparas dos páginas web o elementos mostrando las dos variantes (llamémoslas A y B) a visitantes similares al mismo tiempo. ¡El que da una mejor tasa de conversión, gana!

Cada sitio web quiere que los visitantes se conviertan de solo visitantes a otra cosa (conversiones, suscripciones o simplemente actúen por algo). La tasa a la que un sitio web puede hacer esto es su "tasa de conversión". Medir el rendimiento de una variación (A o B) significa medir la velocidad a la que convierte a los visitantes en objetivos. Los científicos de datos utilizan las pruebas A / B para ver qué versión (s) de su producto conduce a la meta deseada más rápido. A veces, las grandes empresas realizan cientos o miles de pruebas A / B para tomar decisiones sobre los productos. Estas pruebas son casi siempre aleatorias, grandes y ciegas. Esto significa que el sujeto o usuario no conoce la prueba A / B.
