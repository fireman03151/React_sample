# Instalar Go

## Instalar Go en Ubuntu

*   [Instalación de Go utilizando apt-get](https://guide.freecodecamp.org/go/installing-go/ubuntu-apt-get) . Este es el método más sencillo para instalar go ubuntu.
*   [Instalación de Go usando tarball](https://guide.freecodecamp.org/go/installing-go/ubuntu-tarball) . Utilice este método si desea instalar la última versión estable de go.

## Instalar Go en Mac OS X

*   [Instalar Go utilizando el instalador de paquetes](https://guide.freecodecamp.org/go/installing-go/mac-package-installer) . El método más sencillo para instalar la última versión estable de go.
*   [Instalación de Go usando tarball](https://guide.freecodecamp.org/go/installing-go/mac-tarball) . Para aquellos que deseen utilizar el terminal.

## Instalar Go en Windows

*   [Instalar Go usando el instalador MSI](https://guide.freecodecamp.org/go/installing-go/windows-installer)

## Instalar Go from source

*   Para instalar Go from source reffer a la [instalación desde la documentación fuente](https://golang.org/doc/install/source)

Referencia: [documentación de instalación de golang](https://golang.org/doc/install#install)
