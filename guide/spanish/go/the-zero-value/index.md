## El valor cero

Entendiendo algunos valores cero dentro de Golang:

1.  falso para los booleanos
2.  0 para enteros
3.  0.0 para flotadores
4.  "" para cuerdas
5.  nulo para

*   punteros
*   funciones
*   interfaces
*   rebanadas
*   canales
*   mapas

Por favor, utilice operador de declaración corta tanto como sea posible.

*   Use var para:
    
*   valor cero
    
*   alcance del paquete
