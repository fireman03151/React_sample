# Funciones integradas

Java también tiene muchas funciones integradas o predefinidas que generalmente se almacenan en los paquetes java.lang y java.io, que se importan automáticamente en editores como BlueJ o se pueden importar usando la siguiente Sintaxis -

```java
import java.lang.*; 
 import java.io.*; 
```

Estas comprenden funciones que hacen que una tarea larga y difícil de lo contrario sea más fácil de realizar.
