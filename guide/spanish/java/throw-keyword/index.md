## lanzar

La palabra clave de lanzamiento de Java se usa para lanzar explícitamente una excepción. Puede lanzar la excepción marcada o no buscada en la palabra clave de lanzamiento de java. La palabra clave throw se utiliza principalmente para lanzar una excepción personalizada.

**_Ejemplo:_**

```java
throw new ArithmeticException("/ by zero not permitted"); 
```
