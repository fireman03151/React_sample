## Etiqueta de comentario

Los comentarios son etiquetas que se usan para dejar notas en el documento. Solo el desarrollador tiene acceso a los comentarios, el usuario final no lo ve a menos que verifique con el elemento de inspección del navegador.

Cuando está trabajando en algún código, es útil dejar sugerencias a otros desarrolladores para comprender de qué se trata el trabajo.

### Ejemplo

```html

<!-- This is a comment --> 
```

Los comentarios también se pueden utilizar para desactivar el código sin tener que eliminarlo por completo.

### Ejemplo

```html

<!-- 
 <h1>Hello Campers</h1> 
 --> 
 <h2>Hello Friends</h2> 
 <!-- 
 <p>Hello Paragraph</p> 
 --> 

```
