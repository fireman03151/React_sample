## Párrafo

El HTML

El elemento representa un párrafo de texto.

Generalmente representa un bloque de texto que se separa de otros bloques por espacio en blanco vertical.

### Ejemplo

\`\` \`html

Este es un parrafo

Este es otro parrafo

\`\` \`
