## Etiqueta div

La etiqueta `<div>` es un contenedor genérico que define una sección en su documento HTML. Un elemento `<div>` se utiliza para agrupar secciones de elementos HTML y formatearlos con CSS. Un `<div>` es un elemento de nivel de bloque. Esto significa que toma su propia línea en la pantalla. Los elementos justo después de `<div>` se empujarán hacia abajo a la línea de abajo. Para una agrupación y estilo similares que no estén a nivel de bloque, sino en línea, utilizaría la etiqueta `<span>` lugar. La etiqueta se utiliza para agrupar elementos en línea en un documento.

### Ejemplo

Aquí hay un ejemplo de cómo mostrar una sección en el mismo color:

```html

<div style="color:#ff0000"> 
  <h3>my heading</h3> 
  <p>my paragraph</p> 
 </div> 
```
