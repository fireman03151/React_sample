# Etiqueta fuerte

La etiqueta HTML < **_strong_** > es una etiqueta de énfasis de texto que, cuando se utiliza, da como resultado una presentación de texto en negrita, colocada dentro de las etiquetas.

### Uso
```
<strong> Hello World! </strong> 
```

El código anterior resulta en

**Hola Mundo!**

Esta etiqueta también se conoce comúnmente como el **elemento fuerte** . Cuando se trata de atributos, solo los Atributos globales se aplican a **fuertes** etiqueta, es decir, no hay atributos específicos para la etiqueta < **strong** >.

# [https://www.w3schools.com/tags/tag\_strong.asp](https://www.w3schools.com/tags/tag_strong.asp)

La etiqueta `<strong>` se utiliza para dar importancia al texto en un documento HTML. Esto se puede hacer envolviendo el texto que desea enfatizar en una etiqueta `<strong>` . La mayoría de los navegadores mostrarán el texto envuelto en una etiqueta `<strong>` en negrita.

Nota: la etiqueta `<strong>` no debe usarse para el texto en negrita estilística.

### Ejemplo:
```
<body> 
  <p> 
    <strong> This </strong> is important. 
  </p> 
 </body> 
```
