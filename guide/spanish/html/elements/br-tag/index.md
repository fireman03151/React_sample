## Br etiqueta

La etiqueta `<br>` produce un salto de línea en un texto. Esto es útil para poemas y direcciones.

### Ejemplo

```html

<!DOCTYPE html> 
 <html> 
  <body> 
    Mozilla Foundation<br> 
    1981 Landings Drive<br> 
    Building K<br> 
    Mountain View, CA 94043-0801<br> 
    USA 
  </body> 
 </html> 
```
