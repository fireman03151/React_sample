## Atributo de marcador de posición | HTML5

Especifica una sugerencia corta que describe el valor esperado de un control de formulario `<input>` o `<textarea>` .

El atributo de marcador de posición no debe contener retornos de carro o avances de línea.

NOTA: No utilice el atributo de marcador de posición en lugar de un elemento <label>, sus propósitos son diferentes. El atributo <label> describe la función del elemento de formulario (es decir, indica qué tipo de información se espera), y el atributo de marcador de posición es una sugerencia sobre el formato que debe tomar el contenido.

### Ejemplo

Ejemplo de entrada

```html

<form> 
    <input type="text" name="fname" placeholder="First Name"> 
    <input type="text" name="lname" placeholder="Last Name"> 
 </form> 
```

Ejemplo de Textarea

```html

<textarea placeholder="Describe yourself here..."></textarea> 
```

### Compatibilidad

Este es un atributo HTML5.
