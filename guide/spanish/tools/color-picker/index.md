## Selector de color

Un selector de color es una herramienta para experimentar con colores para la web. Puede usarlo para elegir entre cómo se combinan los distintos colores y tonos y encontrar las diferencias entre ellos. Los selectores de color a menudo muestran los colores y su información en varios formatos, incluidos Hexadecimal y RGB.

### Sitios web en línea

Los sitios web de selección de color ofrecen la forma más sencilla de elegir rápidamente entre colores y obtener información sobre ellos. Son útiles para cuando no está seguro de usar un tono de color u otro o simplemente desea obtener el color que está usando en un formato diferente.

### Extensiones de navegador

La instalación de una extensión del navegador de selección de colores le permite obtener información sobre los colores utilizados en las páginas web. Es útil cuando ve un color en algún lugar y desea obtener el código para ello. Simplemente haga clic o coloque el puntero sobre el lugar donde está el color y la extensión le mostrará el código para que pueda usarlo en otro lugar.

### Editor de código de paquetes / extensiones

Otra forma práctica de tener un selector de color es instalarlo como un paquete o extensión en su editor favorito. Te da la posibilidad de visualizar y cambiar colores simplemente haciendo clic en ellos, sin tener que salir de tu editor. Además de los estilos CSS más utilizados, también son compatibles con las variables y colores SASS y LESS en varios formatos.
