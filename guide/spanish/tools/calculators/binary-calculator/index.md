## Calculadora binaria

Una calculadora binaria se usa para preformar la suma, resta, multiplicación o división de dos números binarios. Una calculadora binaria también puede convertir valores binarios a valores decimales, y viceversa.

Por ejemplo: 10101010 + 11001100 = 0101110110 (binario) o 374 (decimal)

O, 10101010 x 11001100 = 01000011101111000 (binario) o 34680 (decimal)
