## Calculadora de subred IP

Las calculadoras de subred IP ayudan a dividir las redes IP en subredes mediante el cálculo de las direcciones de red adecuadas, las máscaras de subred, las direcciones de transmisión y los rangos de IP del host. Para redes simples (como una LAN doméstica), puede ser muy fácil identificar los valores apropiados, pero para subredes más complejas, una calculadora de subred IP es una herramienta excelente.

Aquí hay algunas calculadoras de subred IP en línea:

*   http://www.subnet-calculator.com/
*   http://jodies.de/ipcalc
*   https://www.iplocation.net/subnet-calculator
*   http://www.calculator.net/ip-subnet-calculator.html
