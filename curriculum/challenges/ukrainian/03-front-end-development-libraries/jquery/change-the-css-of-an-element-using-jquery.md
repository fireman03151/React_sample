# --description--

CSS HTML-елемента можна змінювати безпосередньо за допомогою jQuery.

jQuery має функцію під назвою `.css()`, яка дозволяє змінити CSS елемента.

Ось як можна змінити колір елемента на синій:

```js
$("#target1").css("color", "blue");
```

Це дещо відрізняється від звичайного оголошення CSS, оскільки властивість CSS та її значення знаходяться в лапках й розділені комою, а не двокрапкою.

Видаліть селектори jQuery, залишивши `document ready function` порожньою.

Оберіть `target1` та змініть колір на червоний.

# --hints--

Елемент `target1` повинен мати червоний текст.

```js
assert($('#target1').css('color') === 'rgb(255, 0, 0)');
```

Для додавання цих класів використовуйте лише jQuery.

```js
assert(!code.match(/class.*animated/g));
```
