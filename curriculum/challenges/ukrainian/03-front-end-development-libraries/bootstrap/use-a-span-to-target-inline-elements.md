# --description--

Елементи `span` можна використовувати для створення вбудованих елементів. Пам’ятаєте, як ми використали клас `btn-block`, щоб кнопка заповнювала весь ряд?

<button class='btn' style='background-color: rgb(0, 100, 0);  color: rgb(255, 255, 255);'>звичайна кнопка</button>

<button class='btn btn-block' style='background-color: rgb(0, 100, 0);  color: rgb(255, 255, 255);'>кнопка btn-block</button>

Ось і різниця між «вбудованими елементами» та «елементами на рівні блоку».

Використовуючи вбудований елемент `span`, ви можете розташувати декілька елементів в одному ряді та стилізувати його елементи по-різному.

Використовуючи елемент `span`, вкладіть слово `love` в елемент `p`, який наразі має текст `Things cats love`. Потім надайте `span` клас `text-danger`, щоб зробити текст червоним.

Ось так ви б зробили для елемента `p`, який має текст `Top 3 things cats hate`:

```html
<p>Top 3 things cats <span class="text-danger">hate:</span></p>
```

# --hints--

Елемент `span` має знаходитись всередині елемента `p`.

```js
assert($('p span') && $('p span').length > 0);
```

Елемент `span` повинен мати лише текст `love`.

```js
assert(
  $('p span') &&
    $('p span').text().match(/love/i) &&
    !$('p span')
      .text()
      .match(/Things cats/i)
);
```

Елемент `span` повинен мати клас `text-danger`.

```js
assert($('span').hasClass('text-danger'));
```

Елемент `span` повинен мати кінцевий тег.

```js
assert(
  code.match(/<\/span>/g) &&
    code.match(/<span/g) &&
    code.match(/<\/span>/g).length === code.match(/<span/g).length
);
```
