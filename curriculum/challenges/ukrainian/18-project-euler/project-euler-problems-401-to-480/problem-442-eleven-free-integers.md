# --description--

Ціле число називають «без-11», якщо його десяткове представлення не містить жодного рядка, яке є степенем числа 11, окрім 1.

Наприклад, 2404 та 13431 є числами «без-11», а 911 та 4121331 — ні.

Нехай $E(n)$ буде $n$-ним натуральним числом «без-11». Наприклад, $E(3) = 3$, $E(200) = 213$ та $E(500\\,000) = 531\\,563$.

Знайдіть $E({10}^{18})$.
