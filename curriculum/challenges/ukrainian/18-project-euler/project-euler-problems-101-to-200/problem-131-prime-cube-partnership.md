# --description--

Для деяких простих значень $p$ існують натуральні числа $n$, за яких вираз $n^3 + n^{2}p$ є ідеальним кубом.

Наприклад, якщо $p = 19,\\ 8^3 + 8^2 × 19 = {12}^3$.

Мабуть, найдивовижніше те, що значення $n$ є унікальним для кожного простого числа з цією властивістю, і таких простих чисел, що менші за сто, всього чотири.

Скільки простих чисел, що менші за один мільйон, мають цю дивовижну властивість?
