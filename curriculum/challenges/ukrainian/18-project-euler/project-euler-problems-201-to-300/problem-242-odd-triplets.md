# --description--

Дано множину {1,2,..., $n$}. Визначимо $f(n, k)$ як кількість її підмножин з $k$ елементів, що утворюють непарну суму. Наприклад, $f(5,3) = 4$, оскільки множина {1,2,3,4,5} містить чотири підмножини з 3 елементів, суми яких непарні: {1,2,4}, {1,3,5}, {2,3,4} та {2,4,5}.

Якщо усі три значення $n$, $k$ та $f(n, k)$ непарні, вони утворюють непарну трійку $[n, k, f(n, k)]$.

Існує лише п’ять непарних трійок за умови $n ≤ 10$, а саме: $[1, 1, f(1, 1) = 1]$, $[5, 1, f(5, 1) = 3]$, $[5, 5, f(5, 5) = 1]$, $[9, 1, f(9, 1) = 5]$ та $[9, 9, f(9, 9) = 1]$.

Скільки існує непарних трійок за умови $n ≤ {10}^{12}$?
