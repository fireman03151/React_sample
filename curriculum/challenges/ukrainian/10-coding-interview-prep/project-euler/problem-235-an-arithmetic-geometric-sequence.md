# --description--

Дано арифметично-геометричну прогресію $u(k) = (900 - 3k)r^{k - 1}$.

Нехай $s(n) = \sum_{k=1 \ldots n} u(k)$.

Знайдіть значення $r$, при якому $s(5000) = -600\\,000\\,000\\,000\\,000$.

Дайте відповідь, округлену до 12 знаків після коми.
