# --description--

Додатне ціле число $n$ називається безквадратним, якщо воно не ділиться на квадрат числа $n$. Таким чином, 1, 2, 3, 5, 6, 7, 10, 11 - безквадратні, а 4, 8, 9, 12 - квадратні.

Скільки існує безквадратних чисел менших за $2^{50}$?

# --hints--
