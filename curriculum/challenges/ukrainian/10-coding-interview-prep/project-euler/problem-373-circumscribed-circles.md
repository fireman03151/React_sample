# --description--

Кожен трикутник має обмежене коло, яке проходить через три вершини. Розглянемо всі трикутники, сторони яких вимірюються цілими числами, для яких радіус обмеженого кола також вимірюється цілими числами.

$S(n)$ - це сума радіусів обмежених кіл усіх таких трикутників, радіус яких не перевищує $n$.

$S(100) = 4\\,950$ and $S(1\\,200) = 1\\,653\\,605$.

Знайдіть $S({10}^7)$.
