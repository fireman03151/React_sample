# --description--

Нехай $S_m = (x_1, x_2, \ldots, x_m)$ буде $m$-tuple додатних чисел з $x_1 + x_2 + \cdots + x_m = m$, де $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ максимізовано.

Наприклад, можливо перевірити, що $[P_{10}] = 4112$ ([ ] це функція цілочисельної частини).

Знайдіть $\суму {[P_m]}$ для $2 ≤ m ≤ 15$.
