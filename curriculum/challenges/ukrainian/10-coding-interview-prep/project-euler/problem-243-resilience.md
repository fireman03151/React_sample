# --description--

Додатній дріб, чий чисельник менший за знаменник, називається правильним дробом.

Для будь-якого знаменника, $d$, існує $d−1$ правильних дробів, наприклад, для $d = 12$:

$$\frac{1}{12}, \frac{2}{12}, \frac{3}{12}, \frac{4}{12}, \frac{5}{12}, \frac{6}{12}, \frac{7}{12}, \frac{8}{12}, \frac{9}{12}, \frac{10}{12}, \frac{11}{12}$$

Дріб, який неможливо скоротити назвемо стійким дробом.

Також визначимо стійкість знаменника, $R(d)$ як відношення стійких правильних дробів до загальної кількості правильних дробів із цим знаменником, наприклад, $R(12) = \frac{4}{11}$.

Фактично, $d = 12$ є найменшим знаменником зі стійкістю $R(d) &lt; \frac{4}{10}$.

Знайдіть найменший чисельник $d$ зі стійкістю $R(d) &lt; \frac{15\\,499}{94\\,744}$.
