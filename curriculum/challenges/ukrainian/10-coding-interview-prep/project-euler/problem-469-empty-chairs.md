# --description--

У кімнаті $N$ стільців розміщені навколо круглого столу.

Лицарі входять до кімнати по одному і навмання обирають порожній стілець.

Щоб мати достатньо місця для ліктів, лицарі завжди залишають принаймні один порожній стілець між собою.

Коли не залишилося доступних стільців, визначається частина $C$ порожніх стільців. Ми також визначаємо $E(N)$ як очікувану величину $C$.

Можливо перевірити, що $E(4) = \frac{1}{2}$ та $E(6) = \frac{5}{9}$.

Знайдіть $E({10}^{18})$. Дайте відповідь, округлену до чотирнадцяти знаків після коми, у вигляді 0.abcdefghiklmn.
