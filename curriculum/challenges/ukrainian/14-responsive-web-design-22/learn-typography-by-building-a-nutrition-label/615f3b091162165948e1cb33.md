# --description--

Якщо ви перевірите свій елемент `.label` з допомогою інструментів веброзробки, ви можете помітити, що насправді його ширина 288 пікселів, а не 270. Це пояснюється тим, що за замовчуванням браузер враховує кордони та відступи під час визначення розміру елемента.

Щоб розв'язати цю проблему, скиньте блокову модель, створивши селектор `*` та надавши йому властивість `box-sizing` зі значенням `border-box`.

# --hints--

Ви повинні створити селектор `*`.

```js
assert(new __helpers.CSSHelp(document).getStyle('*'));
```

Ваш селектор `*` повинен мати властивість `box-sizing` зі значенням `border-box`.

```js
assert(new __helpers.CSSHelp(document).getStyle('*')?.boxSizing === 'border-box');
```
