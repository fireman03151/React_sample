# --description--

Кольори та форми вашої картини надто різкі, щоб прийняти їх за Ротко.

Використайте властивість `filter`, щоб надати картині `blur` на `2px` в елементі `.canvas`.

Ось приклад правила, яке додає 3px `blur`:

```css
p {
  filter: blur(3px);
}
```

# --hints--

Ви повинні встановити властивість `filter` на `blur(2px)`.

```js
const hasFilter = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style.filter === 'blur(2px)');
assert(hasFilter);
```

Ваш елемент `.canvas` повинен мати `filter` зі значенням `blur(2px)`.

```js
const canvasFilter = new __helpers.CSSHelp(document).getStyle('.canvas')?.getPropertyValue('filter');
assert(canvasFilter === 'blur(2px)');
```
