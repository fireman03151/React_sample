# --description--

В попередньому кроці ви розмістили елементи `h1`, `h2`, коментар та `p` в межах елемента `main`. Це називається *вкладеність*. Вкладені елементи потрібно розміщувати на два пробіли праворуч від елемента, в який вони вкладені. Цей інтервал називається відступом та використовується для полегшення читання HTML.

Елемент `h1`, елемент `h2` та коментар мають відступ на два пробіли більше, ніж елемент `main` в коді нижче. Скористайтеся пробілом на клавіатурі, щоб додати ще два пробіли перед елементом `p`, щоб він також мав належний відступ.
