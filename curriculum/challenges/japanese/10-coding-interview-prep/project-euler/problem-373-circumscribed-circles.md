# --description--

あらゆる三角形において、3 つの頂点を通る外接円を描くことができます。 辺の長さが整数で、外接円の半径も整数であるすべての三角形について考えます。

そのような三角形のうち、半径が $n$ を超えないすべての三角形の外接円の半径の和を $S(n)$ とします。

$S(100) = 4\\,950$, $S(1\\,200) = 1\\,653\\,605$ です。

$S({10}^7)$ を求めなさい。
