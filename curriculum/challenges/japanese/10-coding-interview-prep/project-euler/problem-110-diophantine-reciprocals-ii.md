# --description--

次の式の x, y, n は正の整数です。

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

`n` = 1260 の場合に 113 個の相異なる解があり、1260 は相異なる解の総数が 100 を超える `n` の最小値です。

相異なる解の数が 400 万を超える最小の `n` を求めなさい。

**注:** これは、問題 108 をかなり難しくした問題です。総当たりで解ける限界をはるかに超えているので、賢い方法が必要です。
