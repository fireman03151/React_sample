# --description--

辺 $a ≤ b ≤ c$ を持つ整数辺三角形で、その辺が幾何数列を形成するもの、すなわち $b^2 = a \times c$ であるものを、幾何三角形と定義します。

このような幾何三角形の例は、長さ $a = 144$, $b = 156$, $c = 169$ の 3 辺を持つ三角形です。

$\text{周長} ≤ {10}^6$ を満たす幾何三角形は $861\\,805$ 個あります。

$\text{周長} ≤ 2.5 \times {10}^{13}$ を満たす幾何三角形はいくつありますか。
