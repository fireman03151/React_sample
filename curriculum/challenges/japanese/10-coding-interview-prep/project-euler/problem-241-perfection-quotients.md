# --description--

正の整数 $n$ について、$n$ の約数の総和を $σ(n)$ とします。例えば、$σ(6) = 1 + 2 + 3 + 6 = 12$ です。

周知のとおり、完全数とは $σ(n) = 2n$ が成り立つ数です。

正の整数の完全商 (perfect quotient) を $p(n) = \frac{s(n)}{n} $ と定義します。

$p(n)$ が $k + \frac{1}{2}$ (ここで $k$ は整数) となる正の整数 $n ≤ {10}^{18}$ の総和を求めなさい。
