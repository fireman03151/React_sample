# --description--

正の整数 $n$ について、$1 < x < n$ かつ $x^3 \equiv 1\bmod n$ であるような整数 $x$ の和を $S(n)$ とします。

$n = 91$ のとき、考えられる $x$ の値は 9, 16, 22, 29, 53, 74, 79, 81 の 8 つです。 したがって、$S(91) = 9 + 16 + 22 + 29 + 53 + 74 + 79 + 81 = 363$ です。

$S(13\\,082\\,761\\,331\\,670\\,030)$ を求めなさい。
