# --description--

$\\{1, 2, \ldots, n\\}$ の要素のうち互いに素な要素の集合の最大和を、$Co(n)$ と定義します。 例えば、 $Co(10)$ は 30 であり、最大の部分集合は $\\{1, 5, 7, 8, 9\\}$ になります。

$Co(30) = 193$ と $Co(100) = 1356$ が与えられます。

$Co(200\\,000)$ を求めなさい。
