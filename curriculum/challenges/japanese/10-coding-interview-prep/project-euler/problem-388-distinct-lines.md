# --description--

$0 ≤ a, b, c ≤ N$ を満たすすべての格子点 ($a$, $b$, $c$) について考えます。

すべての線は、原点 $O(0, 0, 0)$ から他の格子点まで引かれます。 相異なるこのような線の数を $D(N)$ とします。

$D(1\\,000\\,000) = 831\\,909\\,254\\,469\\,114\\,121$ です。

$D({10}^{10})$ を求めなさい。 回答は、上位 9 桁に続けて下位 9 桁を示すこと。
