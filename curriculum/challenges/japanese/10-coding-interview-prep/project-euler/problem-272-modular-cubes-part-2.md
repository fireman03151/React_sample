# --description--

正の整数 $n$ について、$1 < x < n$ かつ $x^3 \equiv 1\bmod n$ であるような整数 $x$ の個数を $C(n)$ とします。

$n = 91$ のとき、考えられる $x$ の値は 9, 16, 22, 29, 53, 74, 79, 81 の 8 つです。 したがって、$C(91) = 8$ です。

$C(n)=242$ となる正の整数 $n ≤ {10}^{11}$ の和を求めなさい。
