# --description--

次の式の $x$, $y$, $n$ は正の整数です。

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

上限が $L$ のとき、$x &lt; y ≤ L$ を満たす解の個数を $F(L)$ とします。

$F(15) = 4$, $F(1000) = 1069$ であることを確認できます。

$F({10}^{12})$ を求めなさい。
