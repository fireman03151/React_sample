## --text--

次の XML で何が間違っていますか？

```xml
<person>
  <name>Chuck</name>
  <phone type="intl">
    +1 734 303 4456
  <email hide="yes" />
</person>
```

## --answers--

email タグに終了タグがない。

---

スペースの記述が原因で XML が無効になる。

---

phone タグに終了タグがない。

---

プレーンテキストは UTF-8 を使用してエンコードする必要がある。
