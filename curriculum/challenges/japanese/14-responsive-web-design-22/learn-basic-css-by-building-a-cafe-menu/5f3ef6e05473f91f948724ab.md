# --description--

テキストの `font-family` を変更して、ブラウザのデフォルトのフォントと違う見た目にすることができます。 各ブラウザで一般的なフォントが利用できるようになっています。

`font-family` プロパティと値 `sans-serif` を追加して、`body` 内のすべてのテキストを変更してください。 これはとても読みやすく一般的なフォントです。

# --hints--

`font-family` プロパティを `sans-serif` に設定する必要があります。

```js
const hasFontFamily = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style['font-family'] === 'sans-serif');
```

`body` には値が `sans-serif` の `font-family` が必要です。

```js
const bodyFontFamily = new __helpers.CSSHelp(document).getStyle('body')?.getPropertyValue('font-family');
assert(bodyFontFamily === 'sans-serif');
```
