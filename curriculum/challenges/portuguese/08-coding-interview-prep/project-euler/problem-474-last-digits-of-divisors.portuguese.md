## Description
<section id="description"> Para um inteiro positivo n e dígitos d, definimos F (n, d) como o número dos divisores de n cujos últimos dígitos são iguais a d. Por exemplo, F (84, 4) = 3. Entre os divisores de 84 (1, 2, 3, 4, 6, 7, 12, 14, 21, 28, 42, 84), três deles (4, 14 , 84) tem o último dígito 4. <p> Podemos também verificar que F (12 !, 12) = 11 e F (50 !, 123) = 17888. </p><p> Encontre o módulo F (106 !, 65432) (1016 + 61). </p></section>
