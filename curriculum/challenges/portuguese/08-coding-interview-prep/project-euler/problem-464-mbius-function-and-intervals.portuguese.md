## Description
<section id="description"> A função de Möbius, denotada μ (n), é definida como: μ (n) = (-1) ω (n) se n é quadrado livre (onde ω (n) é o número de fatores primos distintos de n) μ (n ) = 0 se n não for quadrado. <p> Seja P (a, b) o número de inteiros n no intervalo [a, b] tal que μ (n) = 1. Seja N (a, b) o número de inteiros n no intervalo [a, b ] tal que μ (n) = -1. Por exemplo, P (2,10) = 2 e N (2,10) = 4. </p><p> Seja C (n) o número de pares inteiros (a, b) tal que: 1 ≤ a ≤ b ≤ n, 99 · N (a, b) ≤ 100 · P (a, b) e 99 · P ( a, b) ≤ 100 · N (a, b). </p><p> Por exemplo, C (10) = 13, C (500) = 16676 e C (10 000) = 20155319. </p><p> Encontre C (20 000 000). </p></section>
