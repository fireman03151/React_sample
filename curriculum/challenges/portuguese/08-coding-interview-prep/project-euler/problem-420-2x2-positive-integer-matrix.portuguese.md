## Description
<section id="description"> Uma matriz inteira positiva é uma matriz cujos elementos são todos inteiros positivos. Algumas matrizes de inteiros positivos podem ser expressas como um quadrado de uma matriz inteira positiva de duas maneiras diferentes. Aqui está um exemplo: <p> Definimos F (N) como o número de matrizes de inteiros positivos 2x2 que têm um traço menor que N e que pode ser expresso como um quadrado de uma matriz inteira positiva de duas maneiras diferentes. Podemos verificar que F (50) = 7 e F (1000) = 1019. </p><p> Encontre F (107). </p></section>
