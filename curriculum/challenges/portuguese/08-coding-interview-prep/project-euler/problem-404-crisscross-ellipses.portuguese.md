## Description
<section id="description"> Ea é uma elipse com uma equação da forma x2 + 4y2 = 4a2. Ea &#39;é a imagem girada de Ea por θ graus no sentido anti-horário em torno da origem O (0, 0) para 0 ° &lt;θ &lt;90 °. <p> b é a distância até a origem dos dois pontos de interseção mais próximos da origem ec é a distância dos dois outros pontos de interseção. Nós chamamos um trinca ordenado (a, b, c) um tripleto canônico elipsoidal se a, b e c são inteiros positivos. Por exemplo, (209, 247, 286) é um tripleto canônico elipsoidal. </p><p> Seja C (N) o número de tripletos canônicos elipsoidais distintos (a, b, c) para um ≤ N. Pode-se verificar que C (103) = 7, C (104) = 106 e C (106) = 11845 . </p><p> Encontre C (1017). </p></section>
