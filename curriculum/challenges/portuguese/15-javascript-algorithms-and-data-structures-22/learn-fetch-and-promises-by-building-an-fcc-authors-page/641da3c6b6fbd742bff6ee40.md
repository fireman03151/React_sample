# --description--

A <dfn>Fetch API</dfn> é uma interface integrada do JavaScript que serve para fazer solicitações de rede a um servidor. Ela tem um método `fetch()`, que você pode usar para fazer solicitações de `GET`, `POST`, `PUT` ou `PATCH`. Neste projeto, você fará uma solicitação de `GET` a um URL, pedindo um arquivo JSON com informações sobre autores do editorial do freeCodeCamp.

Você pode fazer uma solicitação de `GET` com o método `fetch()` assim:

```js
fetch("url-goes-here")
```

Faça uma solicitação de `GET` para este URL: `https://cdn.freecodecamp.org/curriculum/news-author-page/authors.json`. Não termine o código com um ponto-e-vírgula ainda.
