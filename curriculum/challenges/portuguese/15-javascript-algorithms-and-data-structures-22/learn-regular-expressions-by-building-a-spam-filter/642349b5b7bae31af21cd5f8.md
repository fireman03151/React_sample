# --description--

Para juntar tudo o que aprendemos, atualize `dearRegex` para que mapeie as vogais para os números correspondentes (observe que `i` deve corresponder a `1` e corresponder ao símbolo de tubo `|`), bem como corresponder a palavras inteiras.

Com isso, o projeto de filtro de spam está completo.

# --hints--

`dearRegex` deve usar uma classe de caractere que corresponda a `e` ou `3`.

```js
assert.match(dearRegex.source, /\[(e3|3e)\]/);
```

`dearRegex` deve usar uma classe de caractere que corresponda a `a`,`@` ou `4`.

```js
assert.match(dearRegex.source, /\[(a@4|a4@|4a@|4@a|@a4|@4a)\]/);
```

`dearRegex` deve usar uma classe de caractere que corresponda a `i`,`1` ou `|`.

```js
assert.match(dearRegex.source, /\[(i1\||i\|1|1i\||1\|i|\|1i|\|i1)\]/);
```

`dearRegex` deve usar um grupo de não captura.

```js
assert.match(dearRegex.source, /\(\?:/);
```

`dearRegex` deve usar um grupo de não captura que corresponda a `\s` ou `^`.

```js
assert.match(dearRegex.source, /\(\?:(\^\|\\s|\\s\|\^)\)/);
```

`dearRegex` deve usar um segundo grupo de não captura.

```js
assert.lengthOf(dearRegex.source.match(/\(\?:/g), 2);
```

`dearRegex` deve usar um grupo de não captura que corresponda a `\s` ou `$`.

```js
assert.match(dearRegex.source, /\(\?:(\$\|\\s|\\s\|\$)\)/);
```

assert.match('dear friend', dearRegex);
