# --description--

Um seletor de tipo (ou seletor de elemento) selecionará todos os elementos do tipo de elemento dado. A sintaxe é apenas o nome do elemento:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Aqui, todos os três elementos `<div>` seriam selecionados, enquanto o elemento `<p>` não seria.

# --question--

## --text--

Qual das seguintes opções melhor descreve o código CSS fornecido acima?

## --answers--

O código aplica uma cor `white` a todos os elementos do arquivo HTML.

---

O código aplica uma cor `white` a todos os elementos `div` do arquivo HTML.

---

O código aplica uma cor `white` a todos os elementos `p` do arquivo HTML.
