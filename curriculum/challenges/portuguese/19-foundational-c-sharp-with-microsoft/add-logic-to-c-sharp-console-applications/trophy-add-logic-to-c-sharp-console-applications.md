Agora que você completou todos os módulos de "Adicione lógica a aplicações de console em C#" na plataforma de aprendizagem da Microsoft, envie abaixo o URL do seu troféu.

Siga estas instruções para encontrar o URL do seu troféu:

1. Acesse <a href="https://learn.microsoft.com/en-us/users/me/achievements#badges-section" target="_blank">https://learn.microsoft.com/en-us/users/me/achievements#badges-section</a> usando o navegador pelo qual você está conectado à Microsoft
1. Encontre o troféu de "Adicione lógica a aplicações simples em C#" e clique no ícone "Compartilhar" ao lado dele
1. Clique no botão "Copiar URL"
1. Cole o URL na entrada abaixo

O URL deve ser semelhante a: `https://learn.microsoft.com/en-us/training/achievements/learn.wwl.get-started-c-sharp-part-3.trophy?username=your-username&sharingId=your-sharing-id`

Este troféu é necessário para se qualificar para fazer o exame de certificação.
