# --description--

As orelhas devem sempre ser colocadas acima da parte da cabeça a que se sobrepõem. Você pode fazer isso com a propriedade `z-index`.

`z-index` é uma propriedade que você pode usar para definir a ordem de sobreposição dos elementos do HTML. Qualquer elemento com um `z-index` maior sempre será posicionado sobre um elemento com um `z-index` menor.

Para ver o `z-index` em ação, defina a propriedade `z-index`da orelha esquerda como `-1`.

# --hints--

O seletor `.cat-left-ear` deve ter a propriedade `z-index` definida como `-1`. Não se esqueça de adicionar um ponto e vírgula.

```js
assert(new __helpers.CSSHelp(document).getStyle('.cat-left-ear')?.zIndex === '-1')
```
