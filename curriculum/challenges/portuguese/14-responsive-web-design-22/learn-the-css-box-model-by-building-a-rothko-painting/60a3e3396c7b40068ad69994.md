# --description--

A propriedade `border-radius` aceita até quatro valores para arredondar os cantos superior esquerdo, superior direito, inferior direito e inferior esquerdo.

Arredonde o canto superior esquerdo de `.three` em 30 pixels, o canto superior direito em 25 pixels, o canto inferior direito em 60 pixels e o canto inferior esquerdo em 12 pixels.

# --hints--

Você deve definir a propriedade `border-radius` como `30px 25px 60px 12px`.

```js
const hasBorderRadius = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style['border-radius'] === '30px 25px 60px 12px');
assert(hasBorderRadius);
```

O elemento `.three` deve ter um valor de `border-radius` de `30px 25px 60px 12px`.

```js
const threeBorderRadius = new __helpers.CSSHelp(document).getStyle('.three')?.getPropertyValue('border-radius');
assert(threeBorderRadius === '30px 25px 60px 12px');
```
