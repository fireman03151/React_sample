# --description--

Um número primo $p$ é chamado de primo de Panaitopol se $p = \frac{x^4 - y^4}{x^3 + y^3}$ para alguns inteiros positivos $x$ e $y$.

Encontre quantos primos de Panaitopol são menores que $5 × {10}^{15}$.
