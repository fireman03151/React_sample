# --description--

Crie um aplicativo full stack em JavaScript que seja funcionalmente semelhante a este: <a href="https://yoyo44.herokuapp.com/" target="_blank" rel="noopener noreferrer nofollow">https://yoyo44.herokuapp.com/</a>. Use um construtor de site de sua escolha para completar o projeto.

Aqui estão as histórias de usuário que você deve implementar para este projeto:

**História de usuário:** como usuário não autenticado, você pode visualizar todas as barras de sua área.

**História de usuário:** como usuário autenticado, você pode se adicionar a um bar para indicar que estará lá esta noite.

**História de usuário:** como usuário autenticado, você pode se remover de um bar para indicar que não quer mais ir lá.

**História de usuário:** como um usuário não autenticado, quando você faz o login, você não deve ter que pesquisar novamente.

**Dica:** tente usar a API Yelp para encontrar espaços nas cidades que seus usuários procuram. Se você usar a API da Yelp, certifique-se de mencionar isso na aplicação.

Quando terminar, certifique-se de que uma demonstração funcional do seu projeto está hospedada em algum lugar público. Em seguida, envie o URL para a solução no campo Solution Link. Como opção, envie também um link para o código-fonte do projeto no campo GitHub Link.
