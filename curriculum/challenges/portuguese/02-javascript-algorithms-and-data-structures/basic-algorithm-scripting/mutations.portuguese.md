## Description
<section id="description"> Retorna true se a string no primeiro elemento da matriz contiver todas as letras da string no segundo elemento da matriz. Por exemplo, <code>[&quot;hello&quot;, &quot;Hello&quot;]</code> , deve retornar true porque todas as letras na segunda string estão presentes na primeira, ignorando a diferença entre maiúsculas e minúsculas. Os argumentos <code>[&quot;hello&quot;, &quot;hey&quot;]</code> devem retornar false porque a string &quot;hello&quot; não contém um &quot;y&quot;. Por último, <code>[&quot;Alien&quot;, &quot;line&quot;]</code> , deve retornar true porque todas as letras em &quot;line&quot; estão presentes em &quot;Alien&quot;.
