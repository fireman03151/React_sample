# --description--

Unaweza kubadilisha `font-family` ya maandishi, ili kuifanya ionekane tofauti na fonti chaguomsingi ya kivinjari chako. Kila kivinjari kina fonti za kawaida zinazopatikana kwake.

Badilisha maandishi yote katika `body` yako, kwa kuongeza kipengele cha `font-family` chenye thamani `sans-serif`. Hii ni fonti ya kawaida ambayo inasomeka kwa urahisi.

# --hints--

Unapaswa kuweka sifa ya `font-family` kuwa `sans-serif`.

```js
const hasFontFamily = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style['font-family'] === 'sans-serif');
```

`body` yako inapaswa kuwa na `font-family` ya `sans-serif`.

```js
const bodyFontFamily = new __helpers.CSSHelp(document).getStyle('body')?.getPropertyValue('font-family');
assert(bodyFontFamily === 'sans-serif');
```
