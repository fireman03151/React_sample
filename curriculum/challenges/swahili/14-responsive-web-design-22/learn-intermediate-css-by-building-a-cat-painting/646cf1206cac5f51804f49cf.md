# --description--

Ili kuweka sikio la kushoto vizuri, unaweza kutumia CSS transform ili kulizungusha kwa kiwango fulani.

Sifa ya `transform` hukuruhusu kurekebisha umbo, nafasi, na ukubwa wa kipengele bila kubadilisha mpangilio au kuathiri vipengele vinavyozunguka. Ina vitendaji kama vile `translate()`, `rotate()`, `scale()`, `skew()` na `matrix()`.

Weka sifa ya `transform` kuwa `rotate(-45deg)` na uone kitakachotokea.

# --hints--

Unapaswa kuweka sifa ya `transform` ya kipengele cha `.cat-left-ear` kuwa `rotate(-45deg)`. Usisahau kuongeza nusu koloni.

```js
assert(new __helpers.CSSHelp(document).getStyle('.cat-left-ear')?.transform === 'rotate(-45deg)')
```
