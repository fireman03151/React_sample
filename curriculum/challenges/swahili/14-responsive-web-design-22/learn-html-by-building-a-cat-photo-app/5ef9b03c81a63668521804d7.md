# --description--

Sifa ya `action` inaonyesha ambapo data ya fomu inapaswa kutumwa. Kwa mfano, `<form action="/submit-url"></form>` huambia kivinjari kwamba data ya fomu inapaswa kutumwa kwa njia `/submit-url`.

Ongeza sifa ya `action` yenye thamani `https://freecatphotoapp.com/submit-cat-photo` kwenye kipengele cha `form`.

```js
const noSpaces = code.replace(/\s/g, '');
assert(
  document.querySelector('form') &&
    code.match(/<\/form>/g) &&
    noSpaces.indexOf('<form') < noSpaces.indexOf('</form>')
);
```
