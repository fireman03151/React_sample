# --description--

Rangi na maumbo ya uchoraji wako ni mkali sana kupita kama Rothko.

Tumia sifa ya `filter` kutia `blur` kwa mchoro kwa `2px` katika kipengele cha `.canvas`.

Huu hapa ni mfano wa sheria inayoongeza 3px ya `blur`:

```css
p {
  filter: blur(3px);
}
```
