## Description
<section id='description'>
与<code>header</code>和<code>nav</code>类似，<code>footer</code>也具有语义化特性，可以使辅助设备快速定位到它。它位于页面底部，用于呈现版权信息或者相关文档链接。
</section>

## Instructions
<section id='instructions'>

</section>
