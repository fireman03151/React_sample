## Description
<section id='description'>
CSS 里面的 <code>font-size</code> 属性不只限于标题，它可以应用于任何包含文字的元素内。
</section>

## Instructions
<section id='instructions'>
把段落的 <code>font-size</code> 设置为 16px 让它看起来更清晰。
</section>
