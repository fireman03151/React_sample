# --instructions--

在这项挑战中，使用 Python 创建一个多功能计算器，计算器可以接受输入并实现以下功能：

- 以三种方式获取 .csv 文件
  - 从计算机本地上传
  - 从用户输入获取 url
  - 在代码中输入 url
- 使用 Pandas 库将 .csv 保存为数据帧
- 打印标题和前两行
- 将列名存储为列表
- 选择一列或两列，将数据转换为 Numpy 数组
- 用散点图或线形图展示数据
- 选择不同列时均可实现以上内容，并解释数据图
