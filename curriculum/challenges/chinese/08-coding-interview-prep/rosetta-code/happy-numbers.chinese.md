## Description
<section id="description"><p>幸福的数字由以下过程定义： </p><p>从任何正整数开始，将数字替换为其数字的平方和，并重复该过程直到数字等于1（它将保持不变），或者它在一个不包括1的循环中无休止地循环。这些数字这个过程在1结束的是幸福的数字，而那些不以1结尾的数字是不愉快的数字。 </p><p>实现一个函数，如果数字是满意的，则返回true，否则返回false。 </p></section>
