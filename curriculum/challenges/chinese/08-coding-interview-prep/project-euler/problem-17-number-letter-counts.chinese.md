## Description
<section id="description">如果数字1到5用文字写出：一，二，三，四，五，则总共使用3 + 3 + 5 + 4 + 4 = 19个字母。如果从1到包含<code>limit</code>所有数字都用文字写出，那么会使用多少个字母？ <b>注意：</b>不要计算空格或连字符。例如，342（三百四十二）包含23个字母，115（一百一十五）包含20个字母。在写出数字时使用“和”符合英国的用法。 </section>
