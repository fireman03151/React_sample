## Description
<section id="description">
爱丽丝和鲍勃玩游戏Nim Square。
Nim Square就像普通的三堆普通游戏Nim一样，但是玩家只能从堆中取出一平方的石头。
三个堆中的石头数量由有序三元组（a，b，c）表示。
如果0≤a≤b≤c≤29，则下一个玩家的失落位数为1160。


如果0≤a≤b≤c≤100000，则查找下一个玩家的失落位置数。
