# --description--

如果存在一对m> 0​​且n≥k的整数，则让我们称正整数ka square-pivot，使得直到k的（m + 1）个连续正方形的总和等于m个连续正方形的总和从（n + 1）开始：

（km）2 + ... + k2 =（n + 1）2 + ... +（n + m）2。

一些小的方形枢轴是4：32 + 42 = 52 21：202 + 212 = 292 24：212 + 222 + 232 + 242 = 252 + 262 + 272 110：1082 + 1092 + 1102 = 1332 + 1342找到所有的总和不同的方形枢轴≤1010。
