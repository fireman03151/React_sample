# --description--

令P（m，n）为m×n乘法表中的不同项的数量。

例如，3×4乘法表如下所示：

×12341 12342 24683 36912

有8个不同的术语{1,2,3,4,6,8,9,12}，因此P（3,4）= 8。

给出：P（64,64）= 1263，P（12,345）= 1998，P（32,1015）= 13826382602124302。

求P（64,1016）。
