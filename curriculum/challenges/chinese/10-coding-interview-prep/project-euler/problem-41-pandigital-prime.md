# --description--

我们将说*n*数字是pandigital，如果它使用所有数字1到*n*恰好一次。例如，2143是一个4位数的pandigital，也是素数。什么是最大的*n长*数字pandigital素数？

```js
assert(pandigitalPrime(4) == 4231);
```
