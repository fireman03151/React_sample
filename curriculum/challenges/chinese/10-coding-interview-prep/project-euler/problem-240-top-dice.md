# --description--

有1111种方式可以滚动五个6面骰子（编号为1到6的边），使前三个总和为15.一些例子是：

D1，D2，D3，D4，D5 = 4,3,6,3,5

D1，D2，D3，D4，D5 = 4,3,3,5,6

D1，D2，D3，D4，D5 = 3,3,3,6,6

D1，D2，D3，D4，D5 = 6,6,3,3,3

在多少个方面可以滚动20个12面骰子（编号为1到12的边），使前十个骰子总和达到70？
