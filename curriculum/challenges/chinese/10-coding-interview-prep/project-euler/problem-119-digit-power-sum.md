# --description--

数字 512 很有趣，因为它等于其各位数字之和的某个幂：$5 + 1 + 2 = 8$，$8^3 = 512$。 Another example of a number with this property is $614656 = 28^4$.

我们定义 $a_n$ 为这种序列中的第 $n-th$ 项，并要求这种序列至少包含两位数字。

已知 $a_2 = 512$ 而 $a_{10} = 614656$。

求 $a_{30}$。
