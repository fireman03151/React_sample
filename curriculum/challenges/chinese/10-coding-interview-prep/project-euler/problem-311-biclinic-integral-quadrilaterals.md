# --description--

ABCD是凸的整数边四边形，其1≤AB

例如，以下四边形是双斜积分四边形： AB = 19，BC = 29，CD = 37，AD = 43，BD = 48和AO = CO = 23。

令B（N）为满足AB2 + BC2 + CD2 + AD2≤N的不同双斜积分四边形ABCD的数量。 我们可以验证B（10000）= 49和B（1 000 000）= 38239。

求B（10000000000000）。
