# --description--

如果添加自然数（大于零的整数）的数字的平方，则始终以1或89结尾：

```
 15  - > 26  - > 40  - > 16  - > 37  - > 58  - > 89
7  - > 49  - > 97  - > 130  - > 10  - > 1 
```

编写一个函数，该函数将数字作为参数，并在执行上述过程后返回1或89。

```js
function iteratedSquare(n) {
    var total;
    while (n != 89 && n != 1) {
        total = 0;
        while (n > 0) {
            total += Math.pow(n % 10, 2);
            n = Math.floor(n/10);
        }
        n = total;
    }
    return n;
}
```
