# --description--

将每个 `input` 链接到相应的 `label` 元素很重要。 这为辅助技术用户提供了对输入的视觉参考。

这是通过给 `label` 一个 `for` 属性来实现的，该属性包含 `input` 的 `id`。

这个部分将填写学生的名字、电子邮件地址和出生日期。 为 `label` 元素提供适当的 `for` 属性以及文本内容。 然后，将 `input` 元素链接到相应的 `label` 元素。
