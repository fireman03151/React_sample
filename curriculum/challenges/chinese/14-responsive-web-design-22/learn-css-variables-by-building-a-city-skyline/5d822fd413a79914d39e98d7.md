# --description--

要使用变量，请将变量名放在括号中，并在它们前面加上 `var`，例如：`var(--variable-name)`。 无论你给出了什么值，变量都将应用到你使用的属性上。

添加在上一步中创建的变量 `--building-color1` 作为 `.bb1a` 类的 `background-color` 属性的值。
