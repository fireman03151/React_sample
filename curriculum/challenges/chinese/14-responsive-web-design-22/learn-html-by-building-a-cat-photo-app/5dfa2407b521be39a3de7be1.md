# --description--

向锚点元素（`a`）的起始标签中添加值为 `_blank` 的 `target` 属性，用以在新的标签页中打开链接。

# --hints--

你的 `p` 元素应该嵌套着一个写着文本 `cat photos` 的锚点元素（`a`）。 你可能删除了它或有拼写错误。

```js
const anchor = $('p > a');
assert(
  anchor.length &&
    anchor[0].innerText.toLowerCase().replace(/\s+/g, ' ') === 'cat photos'
);
```

你的锚点元素（`a`）缺少一个 `target` 属性。 请检查在起始标签的名称后面要有一个空格，且所有的属性名称前面也要有一个空格。

```js
assert(document.querySelector('a').hasAttribute('target'));
```

`target` 属性的值应为 `_blank`。 你可能省略了这个值或者有拼写错误。 别忘了应该用引号包裹属性的值。

```js
assert(document.querySelector('a').getAttribute('target') === '_blank');
```
