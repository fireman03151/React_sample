# --description--

`:last-of-type` 伪选择器的作用正好相反——它以匹配选择器的最后一个元素为目标。 创建一个 `h1 .flex span:last-of-type` 选择器以定位 flex 容器中的最后一个 `span`，并给它一个 `font-size` 属性设置为 `1.2em`，使其看起来像一个标题。

# --hints--

应该有一个 `h1 .flex span:last-of-type` 选择器。

```js
assert(new __helpers.CSSHelp(document).getStyle('h1 .flex span:last-of-type'));
```

`h1 .flex span:last-of-type` 选择器应该有一个 `font-size` 属性设置为 `1.2em`。

```js
assert(new __helpers.CSSHelp(document).getStyle('h1 .flex span:last-of-type')?.getPropertyValue('font-size') === '1.2em');
```
