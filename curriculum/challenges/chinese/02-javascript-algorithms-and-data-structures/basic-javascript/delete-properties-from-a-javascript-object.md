## Description
<section id='description'>
我们同样可以删除对象的属性，例如：
<code>delete ourDog.bark;</code>
</section>

## Instructions
<section id='instructions'>
删除<code>myDog</code>对象的<code>"tails"</code>属性。
</section>

<section id='tests'>

## Solution
<section id='solution'>


```js
var ourDog = {
  "name": "Camper",
  "legs": 4,
  "tails": 1,
  "friends": ["everything!"],
  "bark": "bow-wow"
};
var myDog = {
  "name": "Happy Coder",
  "legs": 4,
  "tails": 1,
  "friends": ["freeCodeCamp Campers"],
  "bark": "woof"
};
delete myDog.tails;
```
