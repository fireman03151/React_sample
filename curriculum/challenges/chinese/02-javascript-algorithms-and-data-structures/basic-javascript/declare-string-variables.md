# --description--

之前我们写过这样的代码：

`var myName = "your name";`

`"your name"`被称作<dfn>字符串变量</dfn>。字符串是用单引号或双引号包裹起来的一连串的零个或多个字符。

# --instructions--

创建两个新的`字符串变量`：`myFirstName`和`myLastName`，并用你的姓和名分别为它们赋值。

# --hints--

assert(
