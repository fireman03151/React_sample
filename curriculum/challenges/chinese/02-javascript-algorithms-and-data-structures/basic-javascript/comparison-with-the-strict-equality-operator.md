# --description--

严格相等运算符（`===`）是相对相等操作符（`==`）的另一种比较操作符。与相等操作符不同的是，它会同时比较元素的值和`数据类型`。

如果比较的值类型不同，那么在严格相等运算符比较下它们是不相等的，会返回 false 。

**示例**

```js
3 ===  3   // true
3 === '3'  // false
```

`3`是一个`数字`类型的，而`'3'`是一个`字符串`类型的，所以 3 不全等于 '3'。

# --instructions--
