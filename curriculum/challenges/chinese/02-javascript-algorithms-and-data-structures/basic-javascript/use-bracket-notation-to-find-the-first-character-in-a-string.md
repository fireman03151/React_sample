# --description--

方括号表示法是一种在字符串中的特定`index`（索引）处获取字符的方法。

大多数现代编程语言，如JavaScript，不同于人类从 1 开始计数。它们是从 0 开始计数，这被称为 <dfn>基于零</dfn> 的索引。

例如, 在单词 "Charles" 中索引 0 上的字符为 "C"，所以在`var firstName = "Charles"`中，你可以使用`firstName[0]`来获得第一个位置上的字符。
