## Description
<section id='description'>
<code>Number</code>是 JavaScript 中的一种数据类型，表示数值。
现在让我们来尝试在 JavaScript 中做加法运算。
JavaScript 中使用<code>+</code>号进行加法运算。
<strong>示例</strong>

```js
myVar = 5 + 10; // assigned 15
```

</section>
