## Description
<section id='description'>
创建一个名叫<code>myList</code>的购物清单，清单的数据格式就是多维数组。
每个子数组中的第一个元素应该是购买的物品名称，第二个元素应该是物品的数量，类似于：
<code>["Chocolate Bar", 15]</code>
任务：你的购物清单至少应该有 5 个子数组。
</section>
