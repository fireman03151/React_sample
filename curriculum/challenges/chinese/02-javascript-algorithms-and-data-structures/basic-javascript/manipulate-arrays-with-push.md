# --description--

一个简单的方法将数据添加到一个数组的末尾是通过`push()`函数。

`.push()`接受一个或多个参数，并把它“推”入到数组的末尾。

```js
var arr = [1,2,3];
arr.push(4);
// arr is now [1,2,3,4]
```
