## Description
<section id='description'>
要获取字符串的最后一个字符，可以用字符串的长度减 1 的索引值。
例如，在<code>var firstName = "Charles"</code>中，你可以这样操作<code>firstName[firstName.length - 1]</code>来得到字符串的最后的一个字符。
</section>

## Instructions
<section id='instructions'>
