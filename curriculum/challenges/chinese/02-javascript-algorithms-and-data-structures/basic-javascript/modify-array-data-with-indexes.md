# --description--

与字符串的数据不可变不同，数组的数据是可变的，并且可以自由地改变。

**示例**

```js
var ourArray = [50,40,30];
ourArray[0] = 15; // equals [15,40,30]
```

**提示**  
数组名称和方括号之间不应有任何空格，如`array [0]`尽管 JavaScript 能够正确处理，但可能会让看你代码的其他程序员感到困惑。

# --instructions--

修改数组`myArray`中索引0上的值为`45`。
