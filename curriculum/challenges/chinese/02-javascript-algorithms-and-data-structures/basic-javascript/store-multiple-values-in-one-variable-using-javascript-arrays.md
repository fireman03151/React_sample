## Description
<section id='description'>
使用<code>数组</code>，我们可以在一个地方存储多个数据。
以左方括号<code>[</code>开始定义一个数组，以右方括号<code>]</code>结束，里面每个元素之间用逗号隔开，例如：
<code>var sandwich = ["peanut butter", "jelly", "bread"]</code>.
</section>
