## Description
<section id='description'>
与<code>+=</code>操作符类似，<code>-=</code>操作符用来对一个变量进行减法赋值操作。
<code>myVar = myVar - 5;</code>
变量<code>myVar</code>等于自身减去<code>5</code>的值。也可以写成这种形式：
<code>myVar -= 5;</code>
</section>

## Instructions
<section id='instructions'>
使用<code>-=</code>操作符实现同样的效果。
</section>
