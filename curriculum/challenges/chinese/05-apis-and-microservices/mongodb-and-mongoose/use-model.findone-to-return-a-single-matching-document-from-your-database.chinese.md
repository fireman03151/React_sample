## Description
<section id='description'> 
Model.findOne（）的行为类似于.find（），但它只返回一个文档（不是数组），即使有多个项目也是如此。在按声明为唯一的属性进行搜索时，它尤其有用。使用Model.findOne（） - &gt; Person，找到一个在她的收藏夹中有某种食物的人。使用函数参数food作为搜索键。
