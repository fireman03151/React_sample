## Description
<section id="description"> <code>grid-gap</code> - это сокращенное свойство для <code>grid-row-gap</code> и <code>grid-column-gap</code> из двух предыдущих упражнений, которое более удобно в использовании. Если <code>grid-gap</code> имеет одно значение, это создаст разрыв между всеми строками и столбцами. Однако, если есть два значения, он будет использовать первое, чтобы установить разрыв между строками, и второе значение - между столбцами. </section>

## Instructions
<section id="instructions"> Используйте <code>grid-gap</code> чтобы задать <code>10px</code> промежуток между строками и <code>20px</code> промежуток между столбцами. </section>
