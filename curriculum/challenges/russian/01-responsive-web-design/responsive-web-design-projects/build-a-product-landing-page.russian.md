## Description
<section id="description"> <strong>Цель:</strong> создать приложение <a href="https://codepen.io" target="_blank">CodePen.io</a> , функционально похожее на это: <a href="https://codepen.io/freeCodeCamp/full/RKRbwL" target="_blank">https://codepen.io/freeCodeCamp/full/RKRbwL</a> . Выполните приведенные ниже <a href="https://en.wikipedia.org/wiki/User_story" target="_blank">истории пользователей</a> и получите все тесты для прохождения. Дайте ему свой личный стиль. Вы можете использовать HTML, JavaScript и CSS для завершения этого проекта. Рекомендуется использовать простой CSS, потому что это то, что до сих пор изучали уроки, и вы должны получить некоторую практику с простым CSS. Вы можете использовать Bootstrap или SASS, если вы выберете. Дополнительные технологии (например, jQuery, React, Angular или Vue) не рекомендуются для этого проекта, и использование их на свой страх и риск. Другие проекты дадут вам возможность работать с различными технологическими пакетами, такими как React. Мы примем и попытаемся исправить все отчеты о проблемах, которые используют предлагаемый стек технологий для этого проекта. Счастливое кодирование! <strong>User Story # 1:</strong> Моя целевая страница продукта должна иметь элемент <code>header</code> с соответствующим <code>id=&quot;header&quot;</code> . <strong>User Story # 2:</strong> Я вижу изображение внутри элемента <code>header</code> с соответствующим <code>id=&quot;header-img&quot;</code> . Логотип компании сделает хороший образ здесь. <strong>User Story # 3:</strong> В элементе <code>#header</code> я вижу элемент <code>nav</code> с соответствующим <code>id=&quot;nav-bar&quot;</code> . <strong>User Story # 4:</strong> Я вижу как минимум три элемента, которые можно щелкнуть внутри элемента <code>nav</code> , каждый из которых <code>nav-link</code> класс <code>nav-link</code> . <strong>User Story # 5:</strong> Когда я нажимаю кнопку <code>.nav-link</code> в элементе <code>nav</code> , я <code>.nav-link</code> в соответствующий раздел целевой страницы. <strong>User Story # 6:</strong> Я могу посмотреть видео с встроенным продуктом с <code>id=&quot;video&quot;</code> . <strong>User Story # 7:</strong> Моя целевая страница имеет элемент <code>form</code> с соответствующей <code>id=&quot;form&quot;</code> . <strong>User Story # 8:</strong> В форме есть поле <code>input</code> с <code>id=&quot;email&quot;</code> где я могу ввести адрес электронной почты. <strong>User Story # 9:</strong> <code>#email</code> ввода <code>#email</code> должно содержать текст заполнителя, чтобы пользователь знал, для чего это поле. <strong>User Story # 10:</strong> <code>#email</code> ввода <code>#email</code> использует проверку HTML5, чтобы подтвердить, что введенный текст является адресом электронной почты. <strong>User Story # 11:</strong> В форме есть <code>input</code> с соответствующим <code>id=&quot;submit&quot;</code> . <strong>User Story # 12:</strong> Когда я <code>#submit</code> элемент <code>#submit</code> , электронная почта отправляется на статическую страницу (используйте этот макет URL: <a href="https://www.freecodecamp.com/email-submit" target="_blank">https://www.freecodecamp.com/email-submit</a> ), который подтверждает, что адрес электронной почты был введен и что он опубликован успешно. <strong>User Story # 13:</strong> навигационная панель всегда должна находиться в верхней части окна просмотра. <strong>User Story # 14: На</strong> целевой странице моего продукта должен быть хотя бы один мультимедийный запрос. <strong>User Story # 15:</strong> Моя целевая страница продукта должна использовать CSS flexbox хотя бы один раз. Вы можете создать свой проект, <a href="http://codepen.io/freeCodeCamp/full/MJjpwO" target="_blank">нажимая эту ручку CodePen</a> . Или вы можете использовать эту ссылку CDN для запуска тестов в любой среде: <code>https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js</code> Как только вы закончите, отправьте URL-адрес своей рабочей проект с прохождением всех его тестов. Не забудьте использовать метод <a href="https://forum.freecodecamp.org/t/how-to-get-help-when-you-are-stuck/19514" target="_blank">Read-Search-Ask,</a> если вы застряли. </section>
