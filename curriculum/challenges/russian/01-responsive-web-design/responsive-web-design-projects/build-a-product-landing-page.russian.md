## Description
<section id="description"> <strong>Цель:</strong> Создать приложение <a href="https://codepen.io" target="_blank">CodePen.io</a> , функционально похожее на это: <a href="https://codepen.io/freeCodeCamp/full/RKRbwL" target="_blank">https://codepen.io/freeCodeCamp/full/RKRbwL</a> . Выполните приведенные ниже <a href="https://ru.wikipedia.org/wiki/%D0%9F%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B5_%D0%B8%D1%81%D1%82%D0%BE%D1%80%D0%B8%D0%B8" target="_blank">пользовательские истории</a> и пройдите все тесты. Сделайте приложение в своем стиле. Вы можете использовать HTML, JavaScript и CSS для завершения этого проекта. Рекомендуется использовать чистый CSS, потому что он использовался в упражнениях, и вы должны получить некоторую практику работы с чистым CSS. Вы можете использовать Bootstrap или SASS, если хотите. Дополнительные технологии (например, jQuery, React, Angular или Vue) не рекомендуются для этого проекта, но вы можете рискнуть их использовать. Другие проекты дадут вам возможность работать с различными пакетами, такими как React. Мы примем и попытаемся исправить все отчеты о проблемах, которые используют предлагаемый стек технологий для этого проекта. Счастливого кодирования! <strong>Пользовательская история # 1:</strong> Моя целевая страница продукта должна иметь элемент <code>header</code> с соответствующим <code>id=&quot;header&quot;</code> . <strong>Пользовательская история # 2:</strong> Я вижу изображение внутри элемента <code>header</code> с соответствующим <code>id=&quot;header-img&quot;</code> . У компании должен быть хороший логотип. <strong>Пользовательская история # 3:</strong> В элементе <code>#header</code> я вижу элемент <code>nav</code> с соответствующим <code>id=&quot;nav-bar&quot;</code> . <strong>Пользовательская история # 4:</strong> Я вижу как минимум три элемента, которые можно щелкнуть внутри элемента <code>nav</code> , каждый с классом <code>nav-link</code> . <strong>Пользовательская история # 5:</strong> Когда я нажимаю кнопку <code>.nav-link</code> в элементе <code>nav</code> , я перемещаюсь в соответствующий раздел целевой страницы. <strong>Пользовательская история # 6:</strong> Я могу посмотреть встроенное видео с продуктом, с <code>id=&quot;video&quot;</code> . <strong>Пользовательская история # 7:</strong> Моя целевая страница имеет элемент <code>form</code> с соответствующим <code>id=&quot;form&quot;</code> . <strong>Пользовательская история # 8:</strong> В форме есть поле <code>input</code> с <code>id=&quot;email&quot;</code> , где я могу ввести адрес электронной почты. <strong>Пользовательская история # 9:</strong> Поле ввода <code>#email</code> должно содержать плейсхолдер (текст заполнителя), чтобы пользователь знал, для чего это поле. <strong>Пользовательская история # 10:</strong> Поле ввода <code>#email</code> использует проверку HTML5, чтобы подтвердить, что введенный текст является адресом электронной почты. <strong>Пользовательская история # 11:</strong> В форме есть <code>input</code> с соответствующим <code>id=&quot;submit&quot;</code> . <strong>Пользовательская история # 12:</strong> Когда я нажимаю на элемент <code>#submit</code> , электронная почта отправляется на статическую страницу (используйте этот фиктивный URL: <a href="https://www.freecodecamp.com/email-submit" target="_blank">https://www.freecodecamp.com/email-submit</a> ), который подтверждает что адрес электронной почты был введен и отправлен успешно. <strong>Пользовательская история # 13:</strong> Навигационная панель всегда должна находиться в верхней части окна просмотра. <strong>Пользовательская история # 14:</strong> На целевой странице моего продукта должен быть хотя бы один мультимедийный запрос. <strong>Пользовательская история # 15:</strong> Моя целевая страница продукта должна использовать CSS flexbox хотя бы один раз. Вы можете создать свой проект от данного шаблона <a href="http://codepen.io/freeCodeCamp/full/MJjpwO" target="_blank">CodePen</a> . Или вы можете использовать эту ссылку CDN для запуска тестов в любой среде: <code>https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js</code> Как только вы закончите, отправьте URL-адрес своего рабочего проекта с прохождением всех тестов. Не забудьте использовать <a href="https://forum.freecodecamp.org/t/how-to-get-help-when-you-are-stuck/19514" target="_blank">Read-Search-Ask</a> , если вы зашли в тупик. </section>
