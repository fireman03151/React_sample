## Description
<section id="description"> <strong>Цель.</strong> Построить полное приложение JavaScript для стека, которое функционально похоже на это: <a href="http://whatsgoinontonight.herokuapp.com/" target="_blank">http://whatsgoinontonight.herokuapp.com/</a> и развернуть его в Heroku. Обратите внимание, что для каждого проекта вам необходимо создать новый репозиторий GitHub и новый проект Heroku. Если вы не можете вспомнить, как это сделать, перейдите на страницу <a href="/challenges/get-set-for-our-dynamic-web-application-projects">https://freecodecamp.com/challenges/get-set-for-our-dynamic-web-application-projects</a> . Вот конкретные истории пользователей, которые вы должны реализовать для этого проекта: <strong>User Story:</strong> как пользователь, не прошедший проверку, я могу просмотреть все бары в своей области. <strong>User Story:</strong> Как аутентифицированный пользователь, я могу добавить себя в бар, чтобы указать, что я собираюсь туда сегодня вечером. <strong>User Story:</strong> Как аутентифицированный пользователь, я могу удалить себя из бара, если я больше не хочу туда идти. <strong>User Story:</strong> Как пользователь, не прошедший проверку подлинности, когда я вхожу в систему, мне больше не нужно искать. <strong>Подсказка:</strong> попробуйте использовать <a href="https://www.yelp.com/developers/documentation/v2/overview" target="_blank">Yelp API</a> для поиска мест в городах, в которых ваши пользователи ищут. Если вы используете API Yelp, обязательно укажите это в своем приложении. После того, как вы закончите реализацию этих историй пользователей, нажмите кнопку «Я завершил эту задачу» и введите URL-адреса как для вашего репозитория GitHub, так и для вашего живого приложения, работающего на Heroku. Вы можете получить отзывы о своем проекте, поделившись с друзьями на Facebook. </section>
