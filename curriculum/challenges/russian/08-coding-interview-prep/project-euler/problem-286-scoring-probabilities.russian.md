## Description
<section id='description'>
Барбара - математик и баскетболист. Она обнаружила, что вероятность забивания точки при стрельбе с расстояния х точно равна (1 - x / q), где q - действительная константа больше 50. <p> Во время каждой тренировки она снимает выстрелы с расстояний x = 1, x = 2, ..., x = 50 и, согласно ее записям, у нее есть ровно 2% шанс набрать всего ровно 20 очков. </p><p> Найдите q и дайте ответ округленным до десяти знаков после запятой. </p>
