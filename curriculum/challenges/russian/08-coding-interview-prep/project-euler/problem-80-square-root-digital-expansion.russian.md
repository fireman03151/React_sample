## Description
<section id='description'>
Хорошо известно, что если квадратный корень из натурального числа не является целым числом, то он иррационален. Десятичное разложение таких квадратных корней бесконечно без каких-либо повторяющихся закономерностей. Квадратный корень из двух равен 1.41421356237309504880 ..., а цифровая сумма первой сто десятичной цифры - 475. Для первых сотен натуральных чисел найдите общее количество цифровых сумм первой сто десятичной цифры для всех иррациональные квадратные корни.
