## Description
<section id='description'>
Удивительно, что есть только три числа, которые могут быть записаны как сумма четвертых степеней их цифр: 1634 = 1 <sup>4</sup> + 6 <sup>4</sup> + 3 <sup>4</sup> + 4 <sup>4</sup> 8208 = 8 <sup>4</sup> + 2 <sup>4</sup> + 0 <sup>4</sup> + 8 <sup>4</sup> 9474 = 9 <sup>4</sup> + 4 <sup>4</sup> + 7 <sup>4</sup> + 4 <sup>4</sup> Поскольку 1 = 1 <sup>4</sup> не является суммой, она не включена. Сумма этих чисел равна 1634 + 8208 + 9474 = 19316. Найдите сумму всех чисел, которые могут быть записаны как сумма n степеней их цифр.
