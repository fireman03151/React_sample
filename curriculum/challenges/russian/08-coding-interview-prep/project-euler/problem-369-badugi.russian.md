## Description
<section id='description'>
В стандартной 52 колоде карт игральных карт набор из 4 карт - это Бадуги, если он содержит 4 карты без пар и две карты одного и того же костюма. <p> Пусть f (n) - количество способов выбора n карт с 4-карточным подмножеством, которое является Badugi. Например, существует 2598960 способов выбрать пять карт из стандартной колоды из 52 карт, из которых 514800 содержат 4 подмножества карт, которые являются Badugi, поэтому f (5) = 514800. </p><p> Найти Σf (n) для 4 ≤ n ≤ 13. </p>
</section>
