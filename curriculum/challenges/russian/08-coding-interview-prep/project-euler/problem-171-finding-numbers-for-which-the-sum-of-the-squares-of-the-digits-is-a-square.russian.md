## Description
<section id="description"> Для положительного целого числа n пусть f (n) - сумма квадратов цифр (в базе 10) n, например f (3) = 32 = 9, f (25) = 22 + 52 = 4 + 25 = 29, f (442) = 42 + 42 + 22 = 16 + 16 + 4 = 36 Найдите последние девять цифр суммы всех n, 0 &lt;n &lt;1020, так что f (n) - идеальный квадрат. </section>
