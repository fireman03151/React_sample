## Description
<section id="description"> Для некоторых натуральных чисел k существует целочисленное разбиение вида 4t = 2t + k, где 4t, 2t и k - все положительные целые числа, а t - действительное число. <p> Первые два таких раздела: 41 = 21 + 2 и 41.5849625 ... = 21.5849625 ... + 6. </p><p> Разделы, где t также целое число, называются совершенными. Для любого m ≥ 1 пусть P (m) - доля таких разбиений, которые являются совершенными с k ≤ m. Таким образом, P (6) = 1/2. </p><p> В следующей таблице перечислены некоторые значения P (m) P (5) = 1/1 P (10) = 1/2 P (15) = 2/3 P (20) = 1/2 P (25) = 1/2 P (30) = 2/5 ... P (180) = 1/4 P (185) = 3/13 </p><p> Найдите наименьшее m, для которого P (m) &lt;1/12345 </p></section>
