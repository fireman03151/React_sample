## Description
<section id='description'>
Определим геометрический треугольник как целочисленный односторонний треугольник со сторонами a ≤ b ≤ c, так что его стороны образуют геометрическую прогрессию, т. Е. B2 = a · c. <p> Примером такого геометрического треугольника является треугольник со сторонами a = 144, b = 156 и c = 169. </p><p> Есть 861805 геометрических треугольников с периметром ≤ 106. </p><p> Сколько геометрических треугольников существует с периметром ≤ 2,5 · 1013? </p>
