## Description
<section id='description'>
Алиса и Боб играют в игру Nim Square. Площадь Нима - это как обычная трехкратная нормальная игра Нима, но игроки могут удалять только квадратное число камней из кучи. Количество камней в трех кучах представлено упорядоченной тройкой (a, b, c). Если 0≤a≤b≤c≤29, то количество проигрывающих позиций для следующего игрока равно 1160. <p> Найдите количество проигрывающих позиций для следующего игрока, если 0≤a≤b≤c≤100 000. </p>
