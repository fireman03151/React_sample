## Description
<section id='description'>
Назовем положительный целочисленный квадрат квадрата ka, если существует пара целых чисел m&gt; 0 и n ≥ k, так что сумма (m + 1) последовательных квадратов до k равна сумме m последовательных квадратов от (n + 1): <p> (км) 2 + ... + k2 = (n + 1) 2 + ... + (n + m) 2. </p><p> Некоторые небольшие квадратные точки: 4: 32 + 42 = 52 21: 202 + 212 = 292 24: 212 + 222 + 232 + 242 = 252 + 262 + 272 110: 1082 + 1092 + 1102 = 1332 + 1342Найти сумму всех четкие квадратные точки ≤ 1010. </p>
