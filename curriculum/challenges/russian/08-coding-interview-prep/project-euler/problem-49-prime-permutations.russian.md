## Description
<section id="description"> Арифметическая последовательность, 1487, 4817, 8147, в которой каждое из членов увеличивается на 3330, необычна двумя способами: (i) каждое из трех членов является простым, и (ii) каждое из 4-значных чисел перестановки друг друга. Нет арифметических последовательностей, состоящих из трех 1-, 2- или 3-значных простых чисел, обладающих этим свойством, но есть еще одна 4-значная возрастающая последовательность. Какое 12-значное число вы формируете, объединяя три термина в этой последовательности? </section>
