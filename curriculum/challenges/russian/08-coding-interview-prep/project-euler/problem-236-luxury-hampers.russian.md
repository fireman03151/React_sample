## Description
<section id='description'>
Поставщики «A» и «B» предоставили следующие номера продуктов для рынка роскоши: <p> Product&#39;A &quot;B&#39;Beluga Caviar5248640Christmas Cake13121888Gammon Joint26243776Vintage Port57603776Champagne Truffles39365664 </p><p> Хотя поставщики очень стараются поставлять свои товары в отличном состоянии, неизбежно происходит некоторая порча - то есть продукты ухудшаются. </p><p> Поставщики сравнивают свою эффективность с использованием двух типов статистики: пять коэффициентов порчи продукта для каждого поставщика равны количеству продуктов, которые были плохими, поделенными на количество поставляемых продуктов, по каждому из пяти продуктов в свою очередь. Общий коэффициент порчи для каждого поставщика равен общему количеству продуктов, которые были плохими, деленным на общее количество продуктов, предоставляемых этим поставщиком. К их удивлению, поставщики обнаружили, что каждый из пяти коэффициентов порчи продукта был хуже (выше ) для «B», чем для «A» тем же коэффициентом (отношение скоростей порчи), m&gt; 1; и все же, как ни парадоксально, общий коэффициент порчи был хуже для «А», чем для «В», а также в м. </p><p> Есть тридцать пять м&gt; 1, для которых этот неожиданный результат мог произойти, наименьшим из которых является 1476/1475. </p><p> Какова максимальная возможная величина m? Дайте ваш ответ как фракцию, сведенную к ее наименьшим терминам, в форме u / v. </p>
