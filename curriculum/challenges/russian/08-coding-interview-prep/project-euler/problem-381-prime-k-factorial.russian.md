## Description
<section id='description'>
Для простого p пусть S (p) = (Σ (pk)!) Mod (p) для 1 ≤ k ≤ 5. <p> Например, если p = 7, (7-1)! + (7-2)! + (7-3)! + (7-4)! + (7-5)! = 6! + 5! + 4! + 3! + 2! = 720 + 120 + 24 + 6 + 2 = 872. Так как 872 mod (7) = 4, S (7) = 4. </p><p> Можно проверить, что ΣS (p) = 480 для 5 ≤ p &lt;100. </p><p> Найти ΣS (p) для 5 ≤ p &lt;108. </p>
</section>
