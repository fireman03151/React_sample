## Description
<section id='description'>
Число Хэмминга - это положительное число, не имеющее простого коэффициента больше 5. Таким образом, первые несколько чисел Хэмминга - 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15. Есть 1105 чисел Хэмминга не более 108. <p> Мы будем называть положительное число обобщенным числом Хэмминга типа n, если он не имеет простого множителя, большего, чем n. Следовательно, числа Хэмминга являются обобщенными числами Хэмминга типа 5. </p><p> Сколько обобщенных чисел Хемминга типа 100 существует, которые не превышают 109? </p>
