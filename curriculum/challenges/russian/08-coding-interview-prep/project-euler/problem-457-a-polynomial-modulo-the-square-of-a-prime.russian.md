## Description
<section id="description"> Пусть f (n) = n2 - 3n - 1. Пусть p - простое число. Пусть R (p) - наименьшее натуральное число n такое, что f (n) mod p2 = 0, если такое целое число n существует, в противном случае R (p) = 0. <p> Пусть SR (L) - ΣR (p) для всех простых чисел, не превосходящих L. </p><p> Найдите SR (107). </p></section>
