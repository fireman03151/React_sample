## Description
<section id='description'>
Пусть R (a, b, c) - максимальная площадь, покрытая тремя неперекрывающимися окружностями внутри треугольника с длинами ребер a, b и c. Пусть S (n) - среднее значение R (a, b, c) по всем целочисленным триплетам (a, b, c), для которых 1 ≤ a ≤ b ≤ c &lt;a + b ≤ n Вы даны S (2 ) = R (1, 1, 1) ≈ 0.31998, S (5) ≈ 1.25899. Найдите S (1803), округленное до 5 знаков после запятой.
