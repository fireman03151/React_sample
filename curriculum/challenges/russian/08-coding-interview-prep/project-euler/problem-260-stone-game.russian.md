## Description
<section id='description'>
В игру играют три груды камней и два игрока. В свою очередь игрок удаляет один или несколько камней из свай. Однако, если она берет камни из более чем одной кучи, она должна удалить столько же камней из каждой из выбранных свай. <p> Другими словами, игрок выбирает некоторое N&gt; 0 и удаляет: N камней из любой одной сваи; или N камней из каждой из двух свай (всего 2N); или N камней от каждой из трех свай (всего 3N). Игрок, принимающий последний камень (-ы), выигрывает игру. </p><p> Победившая конфигурация - это та, где первый игрок может заставить выиграть. Например, (0,0,13), (0,11,11) и (5,5,5) являются выигрышными конфигурациями, потому что первый игрок может сразу удалить все камни. </p><p> Потерянная конфигурация - это та, где второй игрок может заставить победу, независимо от того, что делает первый игрок. Например, (0,1,2) и (1,3,3) теряют конфигурации: любой юридический ход оставляет выигрышную конфигурацию для второго игрока. </p><p> Рассмотрим все потери конфигураций (xi, yi, zi), где xi ≤ yi ≤ zi ≤ 100. Мы можем проверить, что для них Σ (xi + yi + zi) = 173895. </p><p> Найдите Σ (xi + yi + zi), где (xi, yi, zi) пробегает потеряющие конфигурации с xi ≤ yi ≤ zi ≤ 1000. </p>
</section>
