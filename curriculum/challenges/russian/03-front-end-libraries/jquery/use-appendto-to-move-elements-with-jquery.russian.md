## Description
<section id="description"> Теперь давайте попробуем переместить элементы из одного <code>div</code> в другой. Функция jQuery имеет функцию <code>appendTo()</code> которая позволяет вам выбирать элементы HTML и добавлять их к другому элементу. Например, если мы хотим перенести <code>target4</code> с нашей правой скважины на нашу левую скважину, мы будем использовать: <code>$(&quot;#target4&quot;).appendTo(&quot;#left-well&quot;);</code> Переместите элемент <code>target2</code> с <code>left-well</code> на <code>right-well</code> . </section>
