# --description--

El texto `.sr-only` todavía es visible. Hay un patrón común para ocultar texto visualmente para que sólo los lectores de pantalla lean.

Este patrón es establecer las siguientes propiedades de CSS:

```css
position: absolute;
width: 1px;
height: 1px;
padding: 0;
margin: -1px;
overflow: hidden;
clip: rect(0, 0, 0, 0);
white-space: nowrap;
border: 0;
```

Utilice lo anterior para definir la clase `sr-only`.
