# --description--

Der `header` und der `footer` der Tweet-Einbettung verwendeten die `flex-direction` Eigenschaft früher mit einem row-Wert. In ähnlicher Weise würden die Elemente innerhalb von `.profile-name` gut übereinander funktionieren.

# --instructions--

Füge die CSS-Eigenschaft `flex-direction` zum `.profile-name` Element des Headers hinzu und setze den Wert auf `column`.

# --hints--

Dein `.follow-btn` sollte auf der Seite gerendert werden. Stelle sicher, dass Add-ons wie Ad-Blocker deaktiviert sind.

```js
assert($('.follow-btn').length > 0 && $('.follow-btn').css('display') !== 'none');
```

Das `.profile-name` Element sollte eine `flex-direction` Eigenschaft mit Wert `column` haben.

```js
assert($('.profile-name').css('flex-direction') == 'column');
```
