# --description--

Du solltest nun den Katzenkopf zentrieren.

Weise dem `.cat-head`-Element eine Positionseigenschaft zu, die auf `absolute` gesetzt ist. Lege für die `right`-, `left`-, `top`- und `bottom`-Eigenschaften einen Wert von `0` fest und setze dann ihre `margin`-Eigenschaft an allen Seiten auf `auto`. Dies ist eine Methode, ein Element vertikal und horizontal mittels CSS-Positionierung zu zentrieren.
