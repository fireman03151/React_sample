# --description--

Bisher hast du Typ- und ID-Selektoren für verschiedene Style-Elemente verwendet. However, it is more common to use a different selector to style your elements.

Ein <dfn>Klassenselektor</dfn> ist durch einen Namen mit einem Punkt definiert, der sich direkt vor ihm befindet, wie folgendes Beispiel:

```css
.class-name {
  styles
}
```

Ändere den vorhandenen `#menu`-Selektor in einen Klassenselektor, indem `#menu` durch eine Klasse namens `.menu` ersetzt wird.

# --hints--

Du solltest einen `.menu`-Klassenselektor haben.

```js
const hasMenu = new __helpers.CSSHelp(document).getStyle('.menu');
assert(hasMenu);
```

Du solltest keinen `#menu`-Selektor haben.

```js
const hasDiv = new __helpers.CSSHelp(document).getStyle('#menu');
assert(!hasDiv);
```
