# --description--

Füge für die beiden `p`-Elemente, die du gerade hinzugefügt hast, `dessert` als Wert des ersten `p`-Elements und dessen `class`-Attribut hinzu und den Wert `price` für das zweite `p`-Element und dessen `class`-Attributs hinzu.

# --hints--

Du solltest ein `p`-Element mit der `dessert`-Klasse haben.

```js
assert($('.dessert').length === 1);
```

Dein `p`-Element mit dem Text `Donut` sollte die `dessert`-Klasse haben.

```js
assert($('.dessert')[0].innerText.match(/donut/i));
```

Dein `p`-Element mit dem Text `1.50` sollte die `price`-Klasse haben.

```js
assert($('.price').last().text().match(/1\.50/));
```

Du solltest keine Lücken zwischen deinen `p`-Elementen haben.

```js
assert(!code.match(/<\/p>\s+<p/));
```
