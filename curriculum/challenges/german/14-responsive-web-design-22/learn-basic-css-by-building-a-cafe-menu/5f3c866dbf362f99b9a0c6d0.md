# --description--

Die `p`-Elemente sind innerhalb eines `article`-Elements mit dem Klassenattribut `item` verschachtelt. Du kannst all die `p`-Elemente formatieren, die irgendwo in ein Element mit einer Klasse namens `item` verschachtelt sind, und zwar wie folgt:

```css
.item p { }
```

Verwende den obigen Selektor, fÃ¼ge eine `display`-Eigenschaft mit dem Wert `inline-block` hinzu, so dass die `p`-Elemente sich mehr wie Inline-Elemente verhalten.
