# --description--

Für einen positiven Integer $n$ lasse $σ(n)$ die Summe aller Teiler von $n$, also z.B. $σ(6) = 1 + 2 + 3 + 6 = 12$ sein.

Eine perfekte Zahl ist, wie du wahrscheinlich weißt, eine Zahl mit $σ(n) = 2n$.

Definieren wir den Perfektionsquotienten einer positiven ganzen Zahl als $p(n) = \frac{σ(n)}{n}$.

Finde die Summe aller positiven ganzen Zahlen $n ≤ {10}^{18}$, für die $p(n)$ die Form $k + \frac{1}{2}$ hat, wobei $k$ ein Integer ist.
