## --text--

Was ist falsch mit dem folgenden XML?:

```xml
<person>
  <name>Chuck</name>
  <phone type="intl">
    +1 734 303 4456
  <email hide="yes" />
</person>
```

## --answers--

E-Mail-Tag fehlt ein abschließendes Tag.

---

Durch Leerzeichen wird XML ungültig.

---

Telefon-Tag fehlt ein abschließendes Tag.

---

Einfacher Text sollte mit UTF-8 kodiert werden.
