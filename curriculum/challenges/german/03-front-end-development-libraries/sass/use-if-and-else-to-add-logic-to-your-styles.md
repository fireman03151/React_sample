# --description--

Die `@if`-Direktive in Sass ist nützlich, um auf einen bestimmten Fall zu testen - sie funktioniert genau wie die `if`-Anweisung in JavaScript.

```scss
@mixin make-bold($bool) {
  @if $bool == true {
    font-weight: bold;
  }
}
```

And just like in JavaScript, the `@else if` and `@else` directives test for more conditions:

```scss
@mixin text-effect($val) {
  @if $val == danger {
    color: red;
  }
  @else if $val == alert {
    color: yellow;
  }
  @else if $val == success {
    color: green;
  }
  @else {
    color: black;
  }
}
```

# --instructions--

Erstelle ein Mixin namens `border-stroke`, das einen Parameter `$val` erhält. The mixin should check for the following conditions using `@if`, `@else if`, and `@else` directives:

```scss
light - 1px solid black
medium - 3px solid black
heavy - 6px solid black
```

If the `$val` parameter value is not `light`, `medium`, or `heavy`, then the `border` property should be set to `none`.
