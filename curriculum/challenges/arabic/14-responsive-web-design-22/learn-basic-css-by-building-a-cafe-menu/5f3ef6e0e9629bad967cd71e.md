# --description--

يمكنك إضافة قيمة <dfn>احتياطية</dfn> لمجموعة الخطوط (font-family) عن طريق إضافة اسم خط آخر مفصولاً بفاصلة. يتم استخدام الخطوط الاحتياطية في الحالات التي لا يتم فيها العثور على الخط الأول أو في حال عدم توافره.

أضف الخط احتياطي `serif` بعد خط `Impact`.

# --hints--

يجب عليك إضافة `serif` كنسخة احتياطية للخط `Impact`.

```js
const fontFamily = new __helpers.CSSHelp(document).getStyle('h1, h2')?.getPropertyValue('font-family');
assert(fontFamily === 'Impact, serif');
```
