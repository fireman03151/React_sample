# --description--

الخاصية `animation-iteration-count` تحدد عدد مرات تكرار الرسم المتحرك. يمكن تعيين هذا إلى عدد، أو إلى `infinite` لتكرار الرسوم المتحركة إلى أجل غير مسمى. لا ينبغي Ferris wheel الخاصة بك أن تتوقف أبدا، لذا عيّن منتقي `.wheel` ليكون لديه `animation-iteration-count` بقيمة `infinite`.

تحدد الخاصية `animation-timing-function` كيفية تقدم الحركة بمرور الوقت. هناك بعض القيم المختلفة لهذه الخاصية، ولكنك تريد تشغيل الرسوم المتحركة إلى Ferris wheel بنفس المعدل من البداية إلى النهاية. قم بتعيين `animation-timing-function` إلى `linear` في منتقي `.wheel` الخاص بك.

# --hints--

يجب أن يكون لمنتقي `.wheel` الخاص بك الخاصية `animation-iteration-count` بقيمة `infinite`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.wheel')?.animationIterationCount === 'infinite');
```

يجب أن يكون لمنتقي `.wheel` الخاص بك الخاصية `animation-timing-function` بقيمة `linear`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.wheel')?.animationTimingFunction === 'linear');
```
