# --description--

لاحظ أن حواف الظلال حادة. هذا لأن هناك قيمة إختيارية `blurRadius` للخاصية `box-shadow`:

```css
box-shadow: offsetX offsetY blurRadius color;
```

إذا لم يتم تضمين قيمة `blurRadius`، فتصبح بشكل افتراضي `0` وتنتج حواف حادة. كلما ارتفعت قيمة `blurRadius`، كلما زاد التأثير الضبابي (blurring effect).

في قاعدة CSS مسمى `.green`، أضف خاصية `box-shadow` مع القيم `5px` ألى `offsetX`, و `5px` ألى `offsetY`, و `5px` ألى `blurRadius`, و `green` ألى `color`.

# --hints--

يجب أن تحتوي قاعدة CSS مسمى `.green` علي خاصية `box-shadow` مع القيمة `5px 5px 5px green`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.green')?.boxShadow === 'green 5px 5px 5px');
```
