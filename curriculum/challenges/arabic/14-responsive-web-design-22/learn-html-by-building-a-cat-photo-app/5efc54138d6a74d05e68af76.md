# --hints--

يجب أن يحتوي مربع الاختيار الخاص بك على السمة `id`. تحقق من وجود مسافة بعد اسم الـ opening tag و/أو ان هناك مسافات قبل جميع أسماء السمات.

```js
assert($('input[type="checkbox"]')[0].hasAttribute('id'));
```

يجب أن يكون لمربع للاختيار الخاص بك سمة `id` مع القيمة `loving`. إما أنك حذفت القيمة أو لديك خطأ إملائي. تذكر أن قيم السمة يجب أن تكون محاطة بعلامات اقتباس.

```js
assert($('input[type="checkbox"]')[0].id.match(/^loving$/));
```
