## Description
<section id="description"> Considere el número 6. Los divisores de 6 son: 1,2,3 y 6. Cada número desde 1 hasta e incluyendo 6 puede escribirse como una suma de divisores distintos de 6: 1 = 1, 2 = 2, 3 = 1 + 2, 4 = 1 + 3, 5 = 2 + 3, 6 = 6. Un número n se llama un número práctico si cada número desde 1 hasta e incluyendo n puede expresarse como una suma de divisores distintos de n. <p> Un par de números primos consecutivos con una diferencia de seis se llama un par sexy (ya que &quot;sexo&quot; es la palabra latina para &quot;seis&quot;). La primera pareja sexy es (23, 29). </p><p> Ocasionalmente podemos encontrar un par triple, lo que significa tres pares primos sexy consecutivos, de modo que el segundo miembro de cada par es el primer miembro del siguiente par. </p><p> Llamaremos a un número n tal que: (n-9, n-3), (n-3, n + 3), (n + 3, n + 9) forman un par triple, y los números n-8 , n-4, n, n + 4 y n + 8 son todos prácticos, </p><p> Un paraíso para los ingenieros. </p><p> Encuentra la suma de los primeros cuatro paraísos de los ingenieros. </p></section>
