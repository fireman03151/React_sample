## Description
<section id="description"> Tome el número 192 y multiplíquelo por cada uno de 1, 2 y 3: 192 × 1 = 192 192 × 2 = 384 192 × 3 = 576 Al concatenar cada producto obtenemos el pandigital del 1 al 9, 192384576. Llamaremos al 192384576. El producto concatenado de 192 y (1, 2, 3). Lo mismo se puede lograr comenzando con 9 y multiplicando por 1, 2, 3, 4 y 5, dando el pandigital, 918273645, que es el producto concatenado de 9 y (1, 2, 3, 4, 5). ¿Cuál es el número pandigital de 9 dígitos más grande que se puede formar como producto concatenado de un número entero con (1, 2, ..., <var>n</var> ) donde <var>n</var> &gt; 1? </section>
