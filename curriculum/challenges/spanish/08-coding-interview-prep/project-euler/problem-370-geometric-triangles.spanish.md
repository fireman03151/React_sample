## Description
<section id="description"> Definamos un triángulo geométrico como un triángulo de lados enteros con lados a ≤ b ≤ c, de modo que sus lados formen una progresión geométrica, es decir, b2 = a · c. <p> Un ejemplo de tal triángulo geométrico es el triángulo con lados a = 144, b = 156 y c = 169. </p><p> Hay 861805 triángulos geométricos con un perímetro ≤ 106. </p><p> ¿Cuántos triángulos geométricos existen con el perímetro ≤ 2.5 · 1013? </p></section>
