## Description
<section id="description"> Definiremos un sqube como un número de la forma, p2q3, donde p y q son primos distintos. Por ejemplo, 200 = 5223 o 120072949 = 232613. <p> Los primeros cinco squbes son 72, 108, 200, 392 y 500. </p><p> Curiosamente, 200 también es el primer número para el que no puede cambiar ningún dígito para hacer un número primo; Llamaremos a esos números, prueba principal. El siguiente sqube de prueba principal que contiene la subcadena contigua &quot;200&quot; es 1992008. </p><p> Encuentre el número 200 de prueba principal que contiene la subcadena contigua &quot;200&quot;. </p></section>
