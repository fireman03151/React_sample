## Description
<section id="description"> Sea x un número real. Una mejor aproximación a x para el denominador enlazado d es un número racional r / s en forma reducida, con s ≤ d, de modo que cualquier número racional que esté más cerca de x que r / s tenga un denominador mayor que d: <p> | p / qx | &lt;| r / sx | ⇒ q&gt; d </p><p> Por ejemplo, la mejor aproximación a √13 para el límite del denominador 20 es 18/5 y la mejor aproximación a √13 para el límite del denominador 30 es 101/28. </p><p> Encuentre la suma de todos los denominadores de las mejores aproximaciones a √n para el denominador unido 1012, donde n no es un cuadrado perfecto y 1 &lt;n ≤ 100000. </p></section>
