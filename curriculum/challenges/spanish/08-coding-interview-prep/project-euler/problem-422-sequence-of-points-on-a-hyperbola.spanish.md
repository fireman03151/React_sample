## Description
<section id="description"> Sea H la hipérbola definida por la ecuación 12x2 + 7xy - 12y2 = 625. <p> A continuación, defina X como el punto (7, 1). Se puede ver que X está en H. </p><p> Ahora definimos una secuencia de puntos en H, {Pi: i ≥ 1}, como: P1 = (13, 61/4). P2 = (-43/6, -4). Para i&gt; 2, Pi es el único punto en H que es diferente de Pi-1 y, por lo tanto, la línea PiPi-1 es paralela a la línea Pi-2X. Se puede demostrar que Pi está bien definido y que sus coordenadas son siempre racionales. Se le da que P3 = (-19/2, -229/24), P4 = (1267/144, -37/12) y P7 = (17194218091/143327232, 274748766781/1719926784). </p><p> Encuentre Pn para n = 1114 en el siguiente formato: Si Pn = (a / b, c / d) donde las fracciones están en los términos más bajos y los denominadores son positivos, entonces la respuesta es (a + b + c + d) mod 1 000 000 007. </p><p> Para n = 7, la respuesta habría sido: 806236837. </p></section>
