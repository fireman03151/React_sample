## Description
<section id="description"> La secuencia de buscar y decir va 1, 11, 21, 1211, 111221, 312211, 13112221, 1113213211, ... La secuencia comienza con 1 y todos los demás miembros se obtienen describiendo el miembro anterior en términos de dígitos consecutivos. Ayuda hacer esto en voz alta: 1 es &#39;uno uno&#39; → 11 11 es &#39;dos ​​unos&#39; → 21 21 es &#39;uno dos y uno&#39; → 1211 1211 es &#39;uno uno, uno dos y dos unos&#39; → 111221 111221 es &#39;tres unos, dos dos y uno&#39; → 312211 ... <p> Defina A (n), B (n) y C (n) como el número de unos, dos y tres en el elemento n &#39;ésimo de la secuencia, respectivamente. Se puede verificar que A (40) = 31254, B (40) = 20259 y C (40) = 11625. </p><p> Encuentre A (n), B (n) y C (n) para n = 1012. Dé su respuesta módulo 230 y separe sus valores para A, B y C con una coma. Ej. Para n = 40 la respuesta sería 31254,20259,11625 </p></section>
