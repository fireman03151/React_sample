## Description
<section id="description"> Sea n un entero y S (n) el conjunto de factores de n. <p> Un subconjunto A de S (n) se denomina anticadena de S (n) si A contiene solo un elemento o si ninguno de los elementos de A divide alguno de los otros elementos de A. </p><p> Por ejemplo: S (30) = {1, 2, 3, 5, 6, 10, 15, 30} {2, 5, 6} no es un antigén de S (30). {2, 3, 5} es un anticadeno de S (30). </p><p> Sea N (n) la longitud máxima de un anticadeno de S (n). </p><p> Encuentra ΣN (n) para 1 ≤ n ≤ 108 </p></section>
