## Description
<section id="description"> Sea C el círculo con el radio r, x2 + y2 = r2. Elegimos dos puntos P (a, b) y Q (-a, c) para que la línea que pasa por P y Q sea tangente a C. <p> Por ejemplo, el cuadrupleto (r, a, b, c) = (2, 6, 2, -7) satisface esta propiedad. </p><p> Sea F (R, X) el número de los cuadrupletes de enteros (r, a, b, c) con esta propiedad, y con 0 &lt;r ≤ R y 0 &lt;a ≤ X. </p><p> Podemos verificar que F (1, 5) = 10, F (2, 10) = 52 y F (10, 100) = 3384. Encuentre F (108, 109) + F (109, 108). </p></section>
