## Description
<section id="description"> Peter tiene nueve dados de cuatro lados (piramidales), cada uno con caras numeradas 1, 2, 3, 4. Colin tiene seis dados de seis lados (cúbicos), cada uno con caras numeradas 1, 2, 3, 4, 5, 6. <p> Peter y Colin lanzan sus dados y comparan los totales: las ganancias totales más altas. El resultado es un empate si los totales son iguales. </p><p> ¿Cuál es la probabilidad de que Pyramidal Pete supere a Cubic Colin? Da tu respuesta redondeada a siete lugares decimales en la forma 0.abcdefg </p></section>
