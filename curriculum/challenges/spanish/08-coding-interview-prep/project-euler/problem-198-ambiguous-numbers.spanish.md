## Description
<section id="description"> Una mejor aproximación a un número real x para el denominador enlazado d es un número racional r / s (en forma reducida) con s ≤ d, de modo que cualquier número racional p / q que esté más cerca de x que r / s tenga q&gt; re. <p> Por lo general, la mejor aproximación a un número real se determina de forma única para todos los límites del denominador. Sin embargo, hay algunas excepciones, por ejemplo, 9/40 tiene las dos mejores aproximaciones 1/4 y 1/5 para el denominador enlazado 6. Llamaremos un número real x ambiguo, si hay al menos un denominador enlazado para el cual x posee Dos mejores aproximaciones. Claramente, un número ambiguo es necesariamente racional. </p><p> ¿Cuántos números ambiguos x = p / q, 0 &lt;x &lt;1/100, hay cuyo denominador q no excede de 108? </p></section>
