## Description
<section id="description"> Una fracción unitaria contiene 1 en el numerador. La representación decimal de las fracciones unitarias con denominadores 2 a 10 se da: <p> 1/2 = 0.5 1/3 = 0. (3) 1/4 = 0.25 1/5 = 0.2 1/6 = 0.1 (6) 1/7 = 0. (142857) 1/8 = 0.125 1/9 = 0. (1) 1/10 = 0.1 </p><p> Donde 0.1 (6) significa 0.166666 ..., y tiene un ciclo recurrente de 1 dígito. Se puede ver que 1/7 tiene un ciclo recurrente de 6 dígitos. </p><p> Las fracciones unitarias cuyo denominador no tiene otros factores primos distintos de 2 y / o 5 no se considera que tengan un ciclo recurrente. Definimos la longitud del ciclo recurrente de esas fracciones unitarias como 0. </p><p> Sea L (n) la duración del ciclo recurrente de 1 / n. Se le da que ∑L (n) para 3 ≤ n ≤ 1 000 000 es igual a 55535191115. </p><p> Encuentre ∑L (n) para 3 ≤ n ≤ 100 000 000 </p></section>
