## Description
<section id="description"> Una secuencia de enteros S = {si} se llama una secuencia n si tiene n elementos y cada elemento si satisface 1 ≤ si ≤ n. Así, hay nn secuencias n distintas en total. Por ejemplo, la secuencia S = {1, 5, 5, 10, 7, 7, 7, 2, 3, 7} es una secuencia de 10. <p> Para cualquier secuencia S, sea L (S) la longitud de la subsecuencia contigua más larga de S con el mismo valor. Por ejemplo, para la secuencia S anterior, L (S) = 3, debido a los tres 7 consecutivos. </p><p> Sea f (n) = ∑ L (S) para todas las n-secuencias S. </p><p> Por ejemplo, f (3) = 45, f (7) = 1403689 y f (11) = 481496895121. </p><p> Encontrar f (7 500 000) mod 1 000 000 009. </p></section>
