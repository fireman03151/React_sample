## Description
<section id="description"> Dados los valores de los enteros 1 &lt;a1 &lt;a2 &lt;... &lt;an, considere la combinación lineal q1a1 + q2a2 + ... + qnan = b, usando solo valores enteros qk ≥ 0. <p> Tenga en cuenta que para un conjunto dado de ak, puede ser que no todos los valores de b sean posibles. Por ejemplo, si a1 = 5 y a2 = 7, no hay q1 ≥ 0 y q2 ≥ 0, por lo que b podría ser 1, 2, 3, 4, 6, 8, 9, 11, 13, 16, 18 o 23 . </p><p> De hecho, 23 es el mayor valor imposible de b para a1 = 5 y a2 = 7. Por lo tanto, llamamos f (5, 7) = 23. De forma similar, se puede mostrar que f (6, 10, 15) = 29 y f (14, 22, 77) = 195. </p><p> Encuentre ∑ f (p <em>q, p</em> r, q * r), donde p, qyr son números primos yp &lt;q &lt;r &lt;5000. </p></section>
