## Description
<section id="description"> Al usar cada uno de los dígitos del conjunto, {1, 2, 3, 4}, exactamente una vez, y haciendo uso de las cuatro operaciones aritméticas (+, -, *, /) y paréntesis / paréntesis, es posible formar Diferentes objetivos enteros positivos. Por ejemplo, 8 = (4 * (1 + 3)) / 2 14 = 4 * (3 + 1/2) 19 = 4 * (2 + 3) - 1 36 = 3 * 4 * (2 + 1) Nota que no se permiten las concatenaciones de los dígitos, como 12 + 34. Usando el conjunto, {1, 2, 3, 4}, es posible obtener treinta y un números objetivo diferentes, de los cuales 36 es el máximo, y cada uno de los números 1 a 28 se puede obtener antes de encontrar el primer no expresable número. Encuentre el conjunto de cuatro dígitos distintos, a &lt;b &lt;c &lt;d, para el cual se puede obtener el conjunto más largo de enteros positivos consecutivos, 1 a n, dando su respuesta como una cadena: abcd. </section>
