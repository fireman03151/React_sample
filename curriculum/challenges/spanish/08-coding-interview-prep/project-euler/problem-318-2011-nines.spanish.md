## Description
<section id="description"> Considera el número real √2 + √3. Cuando calculamos las potencias pares de √2 + √3 obtenemos: (√2 + √3) 2 = 9.898979485566356 ... (√2 + √3) 4 = 97.98979485566356 ... (√2 + √3) 6 = 969.998969071069263 ... (√2 + √3) 8 = 9601.99989585502907 ... (√2 + √3) 10 = 95049.999989479221 ... (√2 + √3) 12 = 940897.9999989371855 ... (√2 + √3) 14 = 9313929.99999989263 ... (√2 + √3) 16 = 92198401.99999998915 ... <p> Parece que el número de nueves consecutivas al comienzo de la parte fraccionaria de estas potencias no disminuye. De hecho, se puede probar que la parte fraccionaria de (√2 + √3) 2n se acerca a 1 para n grande. </p><p> Considere todos los números reales de la forma √p + √q con p y q enteros positivos yp &lt;q, de modo que la parte fraccional de (√p + √q) 2n se aproxime a 1 para n grande. </p><p> Sea C (p, q, n) el número de nueves consecutivas al comienzo de la parte fraccionaria de (√p + √q) 2n. </p><p> Sea N (p, q) el valor mínimo de n tal que C (p, q, n) ≥ 2011. </p><p> Encuentre ∑N (p, q) para p + q ≤ 2011. </p></section>
