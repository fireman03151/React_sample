## Description
<section id="description"> <strong>Objetivo:</strong> crear una aplicación de JavaScript de pila completa que sea funcionalmente similar a esta: <a href="http://whatsgoinontonight.herokuapp.com/" target="_blank">http://whatsgoinontonight.herokuapp.com/</a> y desplegarla en Heroku. Tenga en cuenta que para cada proyecto, debe crear un nuevo repositorio GitHub y un nuevo proyecto Heroku. Si no puede recordar cómo hacerlo, visite <a href="/challenges/get-set-for-our-dynamic-web-application-projects">https://freecodecamp.com/challenges/get-set-for-our-dynamic-web-application-projects</a> . Estas son las historias de usuario específicas que debe implementar para este proyecto: <strong>Historia de usuario:</strong> como usuario no autenticado, puedo ver todas las barras en mi área. <strong>Historia de usuario:</strong> como usuario autenticado, puedo agregarme a una barra para indicar que voy allí esta noche. <strong>Historia de usuario:</strong> como usuario autenticado, puedo retirarme de una barra si ya no quiero ir allí. <strong>Historia de usuario:</strong> como usuario no autenticado, cuando inicio sesión no debería tener que buscar nuevamente. <strong>Sugerencia:</strong> intente utilizar la <a href="https://www.yelp.com/developers/documentation/v2/overview" target="_blank">API de Yelp</a> para buscar lugares en las ciudades que buscan sus usuarios. Si usa la API de Yelp, asegúrese de mencionarlo en su aplicación. Una vez que haya terminado de implementar estas historias de usuario, haga clic en el botón &quot;He completado este desafío&quot; e ingrese las URL para el repositorio de GitHub y la aplicación en vivo que se ejecuta en Heroku. Puedes obtener comentarios sobre tu proyecto compartiéndolo con tus amigos en Facebook. </section>
