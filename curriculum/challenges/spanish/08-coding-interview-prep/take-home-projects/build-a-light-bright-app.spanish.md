## Description
<section id="description"> <strong>Objetivo:</strong> crear una aplicación <a href="https://codepen.io" target="_blank">CodePen.io</a> que sea funcionalmente similar a esta: <a href="https://codepen.io/freeCodeCamp/full/eyLYXE" target="_blank">https://codepen.io/freeCodeCamp/full/eyLYXE</a> .
  
  <strong>Regla # 1:</strong> No mires el código del proyecto de ejemplo. Descúbrelo por ti mismo. 
  
  <strong>Regla # 2:</strong> Cumple lo siguiente <a href="https://es.wikipedia.org/wiki/Historias_de_usuario" target="_blank">historias de usuario</a> . Utiliza las bibliotecas o API que necesites. Dale tu propio estilo personal.
  
  <strong>Historia de usuario:</strong> Puedes hacer clic o arrastrar el cursor del mouse para colorear los círculos. 
  
  <strong>Historia de usuario:</strong> Haz doble clic en un círculo coloreado para eliminar el color. 
  
  <strong>Historia de usuario:</strong> Puedes hacer clic en un círculo coloreado para cambiar su color. 
  
  <strong>Historia de usuario:</strong> Debes tener un círculo de color diferente en cada clic. 
  
  <strong>Historia de usuario:</strong> Puedes hacer clic en el botón &#39;Restablecer&#39; para eliminar el color reciente. 
  
  <strong>Historia de usuario:</strong> Puedes hacer clic en el botón &quot;Restablecer todo&quot; para eliminar todos los colores de los círculos. 
  
  Recuerda usar <a href="http://forum.freecodecamp.org/t/how-to-get-help-when-you-are-stuck/19514" target="_blank">Read-Search-Ask</a> si  estas confundido. Cuando hayas terminado, haz clic en el botón &quot;He completado este desafío&quot; e incluye un enlace a tu CodePen. Puedes obtener comentarios sobre tu proyecto compartiéndolo con tus amigos en Facebook. </section>
