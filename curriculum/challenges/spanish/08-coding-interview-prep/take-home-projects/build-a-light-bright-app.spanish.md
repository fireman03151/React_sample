## Description
<section id="description"> <strong>Objetivo:</strong> Crear una aplicación <a href="https://codepen.io" target="_blank">CodePen.io</a> que sea funcionalmente similar a esta: <a href="https://codepen.io/freeCodeCamp/full/eyLYXE" target="_blank">Círculos luminosos que cambian de color</a>. 
 
<strong>Regla # 1:</strong> No mires el código del proyecto de ejemplo. Descúbrelo por ti mismo.

<strong>Regla # 2:</strong> Cumplir las siguientes <a href="https://en.wikipedia.org/wiki/User_story" target="_blank">historias de usuario</a>:

1. Utiliza las bibliotecas o API que necesites. Dale tu propio estilo personal.

2. Puedes hacer clic o arrastrar el cursor del ratón para colorear los círculos.

3. Puedes hacer doble clic en un círculo de color para eliminar el color.

4. Puedes hacer clic en un círculo de color para que cambie de color.

5. Deberías obtener un círculo de color diferente en cada clic.

6. Puedes hacer clic en el botón &#39;Restablecer&#39; para eliminar el último color.

7. Puedes hacer clic en el botón &quot;Restablecer todo&quot; para eliminar todos los colores de los círculos. 

Recuerda usar <a href="http://forum.freecodecamp.org/t/how-to-get-help-when-you-are-stuck/19514" target="_blank">Read-Search-Ask</a> si te atascas. Cuando hayas terminado, haz clic en el botón &quot;He completado este desafío&quot; e incluye un enlace a tu CodePen. Puedes obtener comentarios sobre tu proyecto compartiéndolo con tus amigos en Facebook. </section>
