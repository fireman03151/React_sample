# --description--

# --hints--

`.sun` 的 `position` 應該爲 `absolute`。

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('.sun')?.position, 'absolute');
```

`.sun` 的 `border-radius` 應該爲 `50%`。

```js
assert.equal(new __helpers.CSSHelp(document).getStyle('.sun')?.borderRadius, '50%');
```
