# --description--

現在給 `@keyframes wheel` 規則一個 `100%` 的選擇器。 在這裏面，將 `transform`設置爲 `rotate(360deg)`。 通過這樣做，你的動畫將完成完整的旋轉。

你的 `100%` 選擇器應該有一個 `transform`屬性，設置爲 `rotate(360deg)`。

```js
assert(new __helpers.CSSHelp(document).getCSSRules('keyframes')?.[0]?.cssRules?.[1]?.style?.transform === 'rotate(360deg)')
```
