# --description--

對於這支筆，你將使用十六進制顏色代碼生成漸變色。

使用 `linear-gradient` 函數並將 `gradientDirection` 設置爲 `180deg`。 對於第一個顏色參數，使用十六進制顏色代碼，其值 `55` 表示紅色，`68` 表示綠色，`0D` 表示藍色。

# --hints--

你的 `.green` CSS 規則應該有一個 `background` 屬性，其值爲 `linear-gradient(180deg, #55680D)`。

```js
assert.match(__helpers.removeWhiteSpace(code), /\.green\{.*?background:linear-gradient\(180deg,#55680D\)/i);
```
