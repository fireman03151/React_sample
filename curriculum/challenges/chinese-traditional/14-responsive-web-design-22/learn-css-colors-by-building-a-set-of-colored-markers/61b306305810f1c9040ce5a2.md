# --description--

如你所見，你在筆的周圍添加了一個簡單的紅色陰影，它向右 5 個像素，向下 5 個像素。

但如果你想把陰影定位在對面呢？ 你可以通過對 `offsetX` 和 `offsetY` 使用負值來做到這一點。

更新 `box-shadow` 屬性的值，並將 `offsetX` 設置爲 `-5px`，將 `offsetY` 設置爲 `-5px`。

# --hints--

你的 `.red` CSS 規則應該有一個 `box-shadow` 簡寫屬性，並且其值爲 `-5px -5px red`。

```js
assert(new __helpers.CSSHelp(document).getStyle('.red')?.boxShadow === 'red -5px -5px');
```
