# --description--

在第一個 `tr` 中，添加一個 `th` 元素，其中包含文本 `Cash This is the cash we currently have on hand.`。 將除 `Cash` 之外的所有文本包裝在 `span` 元素中，並將 `class` 設置爲 `description`。

之後，添加三個 `td` 元素和以下文本（按順序）：`$25`、`$30`、`$28`。 給第三個 `td` 元素一個 `class` 屬性設置爲 `current`。
