# --description--

你可能已經注意到當瀏覽器窗口小於 `768px` 時鍵會摺疊。 在第一個 `.keys` 選擇器中將 `overflow` 設置爲 `hidden`，以解決此問題。 此屬性將隱藏任何被推到 `.keys` 的設置 `width` 值之外的元素。

# --hints--

原來的 `.keys` 選擇器應該將 `overflow` 屬性設置爲 `hidden`。

```js
assert(new __helpers.CSSHelp(document).getStyle('.keys')?.overflow === 'hidden');
```
