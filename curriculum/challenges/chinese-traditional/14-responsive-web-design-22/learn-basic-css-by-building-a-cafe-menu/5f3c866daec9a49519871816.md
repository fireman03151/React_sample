# --description--

`article` 元素通常包含多個具有相關信息的元素。 在這個示例裏，它將包含咖啡風味和該風味的價格。 在 `article` 元素中嵌套兩個 `p` 元素。 第一個的文本應該是 `French Vanilla`，第二個的文本應該是 `3.00`。
