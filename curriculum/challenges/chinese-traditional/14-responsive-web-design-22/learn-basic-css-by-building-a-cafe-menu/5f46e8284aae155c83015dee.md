```js
assert($('img').length === 1);
```

```js
assert($('img').attr('src') === 'https://cdn.freecodecamp.org/curriculum/css-cafe/coffee.jpg');
```

你的 `img` 元素應該具有值爲 `coffee icon` 的 `alt` 屬性。

```js
assert($('img').attr('alt').match(/coffee icon/i));
```
