# --description--

關注菜單上的項目和價格，每一行之間有相當大的差距。

定位所有嵌套在 `class` 名爲 `item` 的元素中的 `p` 元素，並將它們的頂部和底部 margin 設置爲 `5px`。

const hasOneSelector = new __helpers.CSSHelp(document).getStyleDeclarations('.item p');
