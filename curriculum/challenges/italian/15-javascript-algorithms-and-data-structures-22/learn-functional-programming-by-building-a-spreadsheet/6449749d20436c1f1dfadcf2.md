# --description--

Nella funzione `window.onload`, devi dire agli elementi `input` di chiamare la funzione `update` quando cambia il valore. Puoi farlo impostando direttamente la proprietà `onchange`.

Imposta la proprietà `onchange` come riferimento per la funzione `update`.

# --hints--

La funzione `window.onload` dovrebbe accedere alla proprietà `onchange` dell'elemento `input`.

```js
assert.match(window.onload.toString(), /input\.onchange/);
```

La funzione `window.onload` dovrebbe impostare la proprietà `onchange` su `update`.

```js
assert.match(window.onload.toString(), /input\.onchange\s*=\s*update/);
```

La funzione `window.onload` non dovrebbe chiamare la funzione `update`.

```js
assert.notMatch(window.onload.toString(), /update\(\s*\)/);
```

Gli elementi `input` dovrebbero avere tutti la funzione `update` come proprietà `onchange`.

```js
const inputs = document.querySelectorAll('input');
inputs.forEach(input => {
  assert.property(input, 'onchange');
  assert.equal(input.onchange, update);
})
```
