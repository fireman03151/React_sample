# --description--

Poiché l'evento `update` è in esecuzione come un event listener `change`, l'argomento `event` sarà un evento change.

La proprietà `target` dell'evento change rappresenta l'elemento che è cambiato. Assegna la proprietà `target` a una nuova variabile chiamata `element`.
