# --description--

Aggiungi un terzo oggetto all'array `locations`. Dagli le stesse proprietà degli altri due oggetti.

Imposta `name` su `cave`. Imposta `button text` su un array con le stringhe `Fight slime`, `Fight fanged beast` e `Go to town square`. Imposta `button functions` su un array con le variabili `fightSlime`, `fightBeast`, `goTown`. Imposta la proprietà `text` su `You enter the cave. You see some monsters.`.
