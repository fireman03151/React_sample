# --description--

Dopo l'ultimo elemento `.divider`, crea un elemento `p` e dagli il testo `Cholesterol 0mg 0%`. Inserisci il testo `Cholesterol` in un elemento `span` e assegna a quest'elemento `span` un attributo `class` con il valore `bold`. Avvolgi il testo `0%` in un altro elemento `span` con l'attributo `class` impostato su `bold`. Infine annida l'elemento `span` `Cholesterol` e il testo `0mg` in un altro elemento `span` per l'allineamento.
