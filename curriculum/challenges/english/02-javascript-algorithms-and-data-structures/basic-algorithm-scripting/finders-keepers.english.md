## Solution
<section id='solution'>


```js
function findElement(arr, func) {
  let num;

  arr.some(e => {
    if (func(e)) {
      num = e;
      return true;
    }
  });

  return num;
}

findElement([1, 2, 3, 4], num => num % 2 === 0);

```
