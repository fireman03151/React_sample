## Tag da Tabela

A tag `<table>` permite exibir uma tabela na sua página da web.

### Exemplo

```html

<table> 
  <tr> 
    <td>Row 1, Cell 1</td> 
    <td>Row 1, Cell 2</td> 
  </tr> 
  <tr> 
    <td>Row 2, Cell 1</td> 
    <td>Row 2, Cell 2</td> 
  </tr> 
 </table> 
```

Este bloco de código produziria a seguinte saída:

Linha 1, célula 1

Linha 1, célula 2

Linha 2, célula 1

Linha 2, célula 2
