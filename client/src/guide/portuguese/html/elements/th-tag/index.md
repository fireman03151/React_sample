## Th Tag

A tag `<th>` permite que você defina uma célula de cabeçalho nas tabelas HTML. Por padrão, a maioria dos navegadores irá negrito e centralizará os dados nessas tags.

### Exemplo

```html

<table> 
 <tr> 
   <th>Movie</th> 
   <th>Rating</th> 
 </tr> 
 <tr> 
   <td>Die Hard</td> 
   <td>5 Stars</td> 
 </tr> 
 </table> 
```

Essa marcação criará a seguinte saída:

Filme

Avaliação

Duro de Matar

5 estrelas
