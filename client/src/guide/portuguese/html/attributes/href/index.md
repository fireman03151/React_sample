## Href

Referência de hipertexto (HREF) é um atributo HTML que você usa para especificar um destino de link ou URL (Uniform Resource Locator). Normalmente você verá o atributo HREF emparelhado com uma tag de âncora `<a>` .

O atributo HREF obtém o significado exato de um link, dependendo do elemento que o está usando. Por exemplo, ao usar a tag `<a>` , ela está fazendo referência ao local de um objeto express como um URL. Ao usar o atributo HREF com a tag `<image>` , o atributo HREF faz referência ao URL da imagem a ser renderizada.

### Exemplos:

Link para a página inicial do Google:

\-> O texto "Visite a página do Google funciona como o link para a página inicial

```html

<a href="https://www.google.com">Visit Google's Homepage</a> 
```

Imagem como um Link:

\-> Logotipo do Google que se refere à página inicial do Google

```html

<a href="https://www.google.com"> 
 <img border="0" alt="Google" src="https://www.google.com/logos/doodles/2015/googles-new-logo-5078286822539264.3-hp2x.gif" width="100" height="100"> 
```

Tags que usam o HREF:

```html

<a> 
 <area>

```
