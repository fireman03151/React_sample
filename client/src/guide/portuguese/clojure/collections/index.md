Um dos principais pontos positivos do Clojure é que ele tem alguns tipos diferentes de sequência: vetores, listas, hashmaps, _oh my!_ E cada tipo diferente de coleção tem suas próprias vantagens, desvantagens e uma série de funções úteis que operam nelas.

Para entender as diferenças entre cada tipo de coleção, quando usar cada uma delas e como converter entre todas elas, pode ser necessário verificar outros tipos de coleção e lembrar-se do que as torna especiais.

Como tal, esta página serve como um índice para que possa ver facilmente qualquer um dos artigos sobre estruturas de dados.

*   [Vetores](http://forum.freecodecamp.com/t/clojure-vectors/18421)
