### Funcionalidade de captura de tela do DevTools

Ao usar as ferramentas de desenvolvedor do Chrome, pode ser útil capturar uma captura de tela de toda a página. E, embora existam complementos para isso, o Google Chrome tem sua própria funcionalidade incorporada.

## Passos

1.  Abra o Devtools do Chrome
2.  em Devtools Ctrl + P (Cmd + P no Mac)
3.  Comece a digitar> captura de tela
4.  Escolha entre as funções disponíveis (capturar captura de tela inteira foi de longe a minha mais útil)
5.  Aproveite sua nova captura de tela
