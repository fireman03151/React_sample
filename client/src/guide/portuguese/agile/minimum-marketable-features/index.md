## Características Mínimas Comerciáveis

Um Recurso Mínimo Comercializável (MMF) é um recurso independente que pode ser desenvolvido rapidamente e dá um valor significativo para o usuário.

**É importante observar: O** Recurso Mínimo Comercializável não é o mesmo que o Produto Mínimo Viável (MVP). Eles são conceitos diferentes. No entanto, ambos são essenciais para apoiar a ideia de que os Desenvolvedores devem buscar funcionalidade mínima para realizar qualquer resultado determinado.

Divida o MMF em seus componentes principais:

**Mínimo** - o menor conjunto absoluto de funcionalidade. Isso é essencial para que um determinado recurso seja comercializado rapidamente

**Comercializável** - Vendendo o usuário final ou organização que o recurso tem um valor significativo

**Recurso** - O valor percebido pelo usuário final. O que isso me dá? Reconhecimento de marca? Mais receita? Ajudar a cortar custos? Isso me dá uma vantagem competitiva?
