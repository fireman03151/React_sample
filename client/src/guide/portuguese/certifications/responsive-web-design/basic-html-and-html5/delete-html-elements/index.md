## Excluir elementos HTML

Para remover um elemento HTML, você deve remover seus subelementos de tag: o conteúdo, seja ele qual for, pode ser mantido, pois não é representativo de nenhum elemento HTML.

Lembre-se: se algum código diferente da parte que você pediu para modificar foi alterado, você pode reiniciar o seu desafio clicando no botão `reset all code` .

Boa sorte!
