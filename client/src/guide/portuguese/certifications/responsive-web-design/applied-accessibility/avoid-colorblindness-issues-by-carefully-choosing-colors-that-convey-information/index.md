## Evite problemas de daltonismo escolhendo cuidadosamente as cores que transmitem informações

Seguindo as instruções:

Alterar a cor do texto para azul escuro (# 003366)

a linha 4 torna-se:

```css
color: #003366; 
```

e a cor do texto ficará mais escura.
