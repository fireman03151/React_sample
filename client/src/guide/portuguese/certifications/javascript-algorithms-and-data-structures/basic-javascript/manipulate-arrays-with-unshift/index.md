## Manipular matrizes com unshift ()

Enquanto `push()` adicionava elementos à parte de trás da matriz, `unshift()` adiciona à frente. Tudo o que tem a fazer é:

```javascript
var arr = [2, 3, 4, 5]; 
 arr.unshift(1); // Now, the array has 1 in the front 

```
