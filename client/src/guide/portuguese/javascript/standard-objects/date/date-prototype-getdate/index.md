## Date.prototype.getDate

O método `getDate()` retorna o dia do mês para a data especificada de acordo com a hora local.

### Sintaxe

```js
dateObject.getDate() 
```

### Exemplo

```js
var Xmas95 = new Date('December 25, 1995 23:15:30'); 
 var day = Xmas95.getDate(); 
 
 console.log(day); // 25 
```
