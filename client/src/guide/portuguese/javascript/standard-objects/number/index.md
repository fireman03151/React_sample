## Número

O objeto `Number` Javacript representa qualquer tipo de número. O JavaScript não distingue tipos separados de números, como inteiros ou flutuantes. Eles são armazenados em um formato unificado de ponto flutuante de 64 bits, de acordo com o padrão internacional IEEE 754 1 .

Objetos New `Number` são criados assim: `var num = new Number(value)` embora geralmente eles possam ser criados simplesmente atribuindo um valor numérico a uma variável: `var num = 1.616;` . Em um contexto não-construtor (ou seja, sem o `new` operador), `Number` pode ser usado para realizar uma conversão de tipo. Se o argumento não puder ser convertido em um número, ele retornará [`NaN`](https://guide.freecodecamp.org/javascript/standard-objects/number/number-nan) .
