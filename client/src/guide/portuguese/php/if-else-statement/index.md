## Introdução

Se / Else for uma declaração condicional onde dependendo da veracidade de uma condição, ações diferentes serão executadas.

> **Observação:** os `{}` colchetes são necessários apenas se a condição tiver mais de uma instrução de ação.

## Se declaração
```
  if (condition){ 
    statement1; 
    statement2; 
  } 
```

> **Nota:** A instrução `else` é opcional.
> 
> ## Se / Else Statement
```
  if (condition){ 
    statement1; 
    statement2; 
  } 
  else{ 
    statement3; 
    statement4; 
  } 
```

## Se / Elseif / Else Statement
```
  if (condition){ 
    statement1; 
    statement2; 
  } 
  elseif{ 
    statement3; 
    statement4; 
  } 
  else 
    statement5; 
```

Para mais informações, confira o seguinte link: [PHP Else](http://php.net/manual/en/control-structures.elseif.php)
