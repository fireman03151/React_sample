## Estrutura CSS UIKit

UIKit é uma estrutura de front-end leve e modular para o desenvolvimento de interfaces web rápidas e poderosas.

### Começando

Aqui está um modelo HTML simples que inclui o último CSS compilado e minimizado para a biblioteca UIKit.

```html

<!DOCTYPE html> 
 <html> 
    <head> 
        <title></title> 
        <link rel="stylesheet" href="css/uikit.min.css" /> 
        <script src="js/jquery.js"></script> 
        <script src="js/uikit.min.js"></script> 
        <script src="js/uikit-icons.min.js"></script> 
    </head> 
    <body> 
    </body> 
 </html> 
```

Nós usamos um CDN neste exemplo, mas você pode verificar outras formas de instalar o UIKit [aqui](https://getuikit.com/docs/introduction) .

### Recursos de aprendizagem

*   A documentação oficial do UIKit está disponível [aqui](https://getuikit.com/docs/introduction) .
*   Confira o repositório GitHub de código-fonte aberto do UIKit [aqui](https://github.com/uikit/uikit) .
