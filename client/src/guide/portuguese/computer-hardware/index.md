## Hardware de computador

Nesta seção, exploramos os vários componentes de hardware dos computadores modernos.

## Partes de um computador pessoal

### Unidade de processamento central (CPU)

Uma Unidade Central de Processamento (CPU) é o circuito eletrônico dentro de um computador que executa as instruções de um programa de computador executando as operações básicas de aritmética, lógica, controle e entrada / saída (E / S) especificadas pelas instruções.

![Intel Core i7](http://ecx.images-amazon.com/images/I/51MOTjmtg2L.jpg)

### Placas de Expansão (por exemplo, GPU)

Uma unidade de processamento gráfico (GPU) é um circuito eletrônico especializado projetado para manipular e alterar rapidamente a memória para acelerar a criação de imagens em um buffer de quadros destinado à saída para um dispositivo de exibição.

![Nvidia Titan XP](https://cdn.arstechnica.net/wp-content/uploads/2017/04/nvidia-titan-x.jpg)

### Armazenamento (SSD ou HDD)

A unidade de estado sólido (SSD) ou a unidade de disco rígido (HDD) são usadas para armazenar informações no computador. Ele mantém a memória mesmo após a perda de energia.

![SSD](https://images10.newegg.com/NeweggImage/ProductImage/20-147-373-09.jpg)

### Memória (RAM)

A memória de acesso aleatório (RAM) é uma forma de armazenamento de dados de computador que armazena os dados e o código de máquina atualmente sendo usado. Um dispositivo de memória de acesso aleatório permite que os itens de dados sejam lidos ou gravados quase na mesma quantidade de tempo, independentemente da localização física dos dados dentro da memória.

![RAM](https://images10.newegg.com/NeweggImage/ProductImage/20-232-476-S01.jpg)

### Caso

Um caso é usado para abrigar todos os componentes de um computador.

![Caso](https://images-na.ssl-images-amazon.com/images/I/41Wq97Jk20L._SL500_AC_SS350_.jpg)

### Fonte de energia

Uma fonte de alimentação é um dispositivo eletrônico que fornece energia elétrica a uma carga elétrica.

![Fonte de energia](https://upload.wikimedia.org/wikipedia/commons/thumb/b/b6/SFX_PC-PSU_Delta_IMG_9760.JPG/620px-SFX_PC-PSU_Delta_IMG_9760.JPG)

### Motherboard

A placa-mãe mantém e permite a comunicação entre muitos dos componentes eletrônicos cruciais de um sistema, como a unidade central de processamento (CPU) e a memória, e fornece conectores para outros periféricos.

![Motherboard](https://fthmb.tqn.com/mmTUdCN6UBGIKlkEggGq4fpV-3k=/768x0/filters:no_upscale()/gig-57c732ed3df78c71b60e7aa5.jpg)

### Dispositivos de entrada e saída (por exemplo, teclado e mouse)

Dispositivo de Entrada / Saída é qualquer hardware usado por um operador humano ou outros sistemas para se comunicar com um computador.

![Dispositivos de E / S](https://www.clasohlson.com/medias/sys_master/9225929949214.jpg)

### Monitor

Um monitor de computador é um dispositivo de saída que exibe informações em forma de figura.

![Monitor](https://assets.pcmag.com/media/images/523934-dell-ultrasharp-34-curved-monitor-u3417w.jpg?thumb=y)
