#### Mais Informações:

Na teoria das cores, um valor tonal é produzido adicionando branco, cinza ou preto a uma cor selecionada. Isso não altera o matiz, mas altera o colorfullness, também conhecido como saturação. Ao discutir o valor tonal, há três termos principais que devem ser discutidos: Matiz, Tom e Sombra.

Matiz é a adição de branco a uma cor. Tint pode ser usado para destacar uma área, bem como começar a criar a ilusão de profundidade em um objeto.

Tom é a adição de cinza a uma cor. A cor tonal cria uma cor mais atenuada e menos saturada.

Sombra é a adição de preto a uma cor. A sombra pode ser usada para escurecer e área para criar a ilusão de profundidade em um objeto.

Ao alterar o valor tonal de uma cor, você pode criar a ilusão de profundidade nas imagens, bem como alterar o nível de saturação para melhor aplicar a cor a uma emoção ou humor desejado.
