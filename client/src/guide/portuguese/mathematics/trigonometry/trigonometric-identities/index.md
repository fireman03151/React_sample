# Identidades trigonométricas

_Identidades trigonométricas_ são igualdades que envolvem funções trigonométricas. As funções trigonométricas são funções de um ângulo que relaciona os ângulos de um triângulo com os comprimentos de seus lados (por exemplo, seno, cosseno e tangente). _Identidades trigonométricas_ são verdadeiras para cada valor das variáveis ​​onde ambos os lados da igualdade são definidos.

Sempre que expressões envolvendo funções trigonométricas precisam ser simplificadas, as _Identidades Trigonométricas_ são muito úteis.
