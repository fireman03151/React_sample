## Adicionando e Subtraindo Números Negativos

Adicionar e subtrair números positivos é bastante direto, mas o que significa adicionar e subtrair números negativos? Vejamos um exemplo do mundo real para ter uma ideia melhor. Suponha que tenhamos uma cesta com alguns balões amarrados a ela, empurrando-a para cima e alguns pesos amarrados a ela, empurrando-a para baixo. Pense na força puxando o balão para cima como um número positivo e a força puxando-o para baixo como um número negativo. Se adicionarmos mais pesos, é como se estivéssemos adicionando um número negativo e a cesta fosse puxada para baixo. Se removermos pesos, estamos subtraindo um número negativo e a cesta é puxada para cima.

### Exemplos

(-1) + 3 = 2

(-3) + (-3) = -6

(-2) - 4 = -6

(-10) + 2 = -8

20 + (-44) = -24
