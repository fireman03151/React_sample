## CSS3 Text Shadow Property

La propiedad de sombra de texto agrega sombra al texto. Cada sombra es la combinación de las compensaciones X e Y del elemento original, el radio de desenfoque y el color. \`\` \`css Sintaxis de css

text-shadow: Color del radio de desenfoque del offset Y del offset X
```
**Example :** 
```

css pag { text-shadow: 2px 2px 8px # FF0000; } \`\` \`
