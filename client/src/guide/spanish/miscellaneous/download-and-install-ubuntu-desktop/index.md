Un tutorial rápido que muestra cómo instalar Ubuntu.

## Pasos:

Puedes descargar la última versión de Ubuntu Desktop [aquí](http://www.ubuntu.com/download/desktop) .  
Si necesita ayuda para instalar el sistema operativo, [este](http://www.ubuntu.com/download/desktop/install-ubuntu-desktop) tutorial de Ubuntu lo ayudará. Si desea iniciar el arranque dual de su computadora con Windows y Ubuntu, consulte [este](https://help.ubuntu.com/community/WindowsDualBoot) artículo útil.

Una vez que haya instalado Ubuntu, abra el terminal y escriba `sudo apt update && sudo apt upgrade` para actualizar sus paquetes.
