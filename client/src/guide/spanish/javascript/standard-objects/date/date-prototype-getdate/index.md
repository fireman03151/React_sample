## Date.prototype.getDate

El método `getDate()` devuelve el día del mes para la fecha especificada según la hora local.

### Sintaxis

```js
dateObject.getDate() 
```

### Ejemplo

```js
var Xmas95 = new Date('December 25, 1995 23:15:30'); 
 var day = Xmas95.getDate(); 
 
 console.log(day); // 25 
```
