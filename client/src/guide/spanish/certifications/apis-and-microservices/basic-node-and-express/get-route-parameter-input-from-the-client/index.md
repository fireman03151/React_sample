## Obtener entrada de parámetros de ruta del cliente

Si alguien te dice que construyas un GET o POST, harías app.get (...) o app.post (...) en consecuencia. La estructura básica del desafío es:

```javascript
app.get("/:word/echo", function(req, res) { 
  // word = req.params.word; 
  // respond with the json object 
 }); 
```
