## تتضمن التعدادات

يمكن للمطورين استخدام `Enums` لتعريف مجموعة من الثوابت المسماة.

هناك نوعان من `Enums`

1.  رقمية
2.  سلسلة القائم

 `// numeric enum 
 enum NumericEnum { 
    numeric1 = 1, 
    numeric2, 
    numeric3, 
    numeric4, 
 } 
 
 // string based enum 
 enum StringBasedEnum { 
    Programming = "is fun", 
    Pizza = "is good" 
 } 
`
