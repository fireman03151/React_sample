## صفيف من

تقوم طريقة 'Array.from ()' بإنشاء مثيل صفيف جديد من كائن شبيه بالمصفوفة أو متكرر.

### بناء الجملة:

'' ' Array.from (arrayLike \[، mapFn \[، thisArg\]\]) '' '

### مثال:

'' ' Array.from ( 'فو')؛ // \["f"، "o"، "o"\] '' '
