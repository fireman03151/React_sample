ترجع طريقة JavaScript `.toUpperCase()` نفس السلسلة التي تم استدعاؤها ، ولكن في كل الحالة العليا.

**بناء الجملة**

 `str.toUpperCase() 
` 

## أمثلة

 `console.log("hello world".toUpperCase()); // Console will output "HELLO WORLD" 
`
