localeTitle: تكررت مع جافا سكريبت في حين الحلقات
---
هناك نوع آخر من حلقات جافا سكريبت يسمى `while loop` تشغيلها `while` يكون هناك شيء حقيقي ، ويتوقف بمجرد أن يصبح شيء ما غير صحيح.

 `var ourArray = []; 
 var i = 0; 
 while(i < 5) { 
  ourArray.push(i); 
  i++; 
 } 
`
