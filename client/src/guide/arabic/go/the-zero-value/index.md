## القيمة صفر

فهم بعض القيم الصفرية داخل Golang:

1.  خاطئة للبوليانيين
2.  0 للأعداد الصحيحة
3.  0.0 للعوامات
4.  "" للسلاسل
5.  لا شيء ل

*   مؤشرات
*   المهام
*   واجهات
*   شرائح
*   قنوات
*   خرائط
