برنامج تعليمي سريع يعرض كيفية تثبيت Ubuntu.

## خطوات:

يمكنك تنزيل أحدث إصدار من Ubuntu Desktop [هنا](http://www.ubuntu.com/download/desktop) .  
إذا كنت بحاجة إلى أي مساعدة في تثبيت نظام التشغيل ، فإن [هذا](http://www.ubuntu.com/download/desktop/install-ubuntu-desktop) البرنامج التعليمي من Ubuntu سيساعدك. إذا كنت ترغب في التمهيد المزدوج لجهاز الكمبيوتر الخاص بك مع Windows و Ubuntu الخروج [هذه](https://help.ubuntu.com/community/WindowsDualBoot) المادة مفيدة.

بمجرد تثبيت Ubuntu ، قم بفتح المحطة الطرفية ونوع `sudo apt update && sudo apt upgrade` لتحديث الحزم الخاصة بك.
