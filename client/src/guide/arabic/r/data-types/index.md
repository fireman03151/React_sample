--- ## سكالارس

يشير العددية إلى كمية ذرية يمكن أن تحتفظ بقيمة واحدة فقط في كل مرة. Scalars هي أنواع البيانات الأساسية. بعض الأنواع الشائعة من scalars:

1.  رقم
    
    > س <- 5 ص <- 5.5 الفئة (س) \[1\] "رقمي" أنيق) \[1\] "رقمي" فئة (+ ص خ) \[1\] "رقمي"
    
2.  قيمة منطقية
    
    > م <- س> ذ # تستخدم للتحقق ، هل إكس أكبر من ذ؟ n <- x <y # تستخدم للتحقق ، هل x أصغر من y؟ م \[1\] خطأ ن \[1\] صحيح فئة (م) \[1\] "منطقي" class (NA) # NA هي قيمة منطقية أخرى: "غير متوفر" / قيم مفقودة \[1\] "منطقي"
    
3.  سلسلة أحرف)
    
    > أ <- "1"؛ ب <- "2.5" و؛ ب \[1\] "1" \[1\] "2.5" أ + ب خطأ في + b: وسيطة غير رقمية لمشغل ثنائي فئة (أ) \[1\] "شخصية" فئة (as.numeric (أ)) \[1\] "رقمي" فئة (as.character (خ)) \[1\] "شخصية"
    

## قوه موجهة

إنه تسلسل لعناصر البيانات من النوع الأساسي نفسه. فمثلا:

 `> o <- c(1,2,5.3,6,-2,4)                                 # Numeric vector 
 > p <- c("one","two","three","four","five","six")        # Character vector 
 > q <- c(TRUE,TRUE,FALSE,TRUE,FALSE,TRUE)                # Logical vector 
 > o;p;q 
 [1]  1.0  2.0  5.3  6.0 -2.0  4.0 
 [1] "one"   "two"   "three" "four"  "five"  "six" 
 [1]  TRUE  TRUE FALSE  TRUE FALSE 
` 

## مصفوفة

إنها مجموعة بيانات مستطيلة ثنائية الأبعاد. المكونات في المصفوفة يجب أن تكون من نفس النوع الأساسي مثل المتجه. فمثلا:

 `> m = matrix( c('a','a','b','c','b','a'), nrow = 2, ncol = 3, byrow = TRUE) 
 > m 
 >[,1] [,2] [,3] 
 [1,] "a"  "a"  "b" 
 [2,] "c"  "b"  "a" 
` 

## إطار البيانات

إنه أكثر عمومية من المصفوفة ، حيث يمكن أن تحتوي الأعمدة المختلفة على أنواع بيانات أساسية مختلفة. فمثلا:

 `> d <- c(1,2,3,4) 
 > e <- c("red", "white", "red", NA) 
 > f <- c(TRUE,TRUE,TRUE,FALSE) 
 > mydata <- data.frame(d,e,f) 
 > names(mydata) <- c("ID","Color","Passed") 
 > mydata 
 
   ID Color Passed 
 1  1   red   TRUE 
 2  2 white   TRUE 
 3  3   red   TRUE 
 4  4  <NA>  FALSE 
` 

## قوائم

إنه كائن R- يمكن أن يحتوي على العديد من أنواع العناصر المختلفة داخله مثل المتجهات والوظائف وقائمة أخرى داخله. فمثلا:

 `> list1 <- list(c(2,5,3),21.3,sin) 
 > list1 
 [[1]] 
 [1] 2 5 3 
 [[2]] 
 [1] 21.3 
 [[3]] 
 function (x)  .Primitive("sin") 
`
