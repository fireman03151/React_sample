## تعويم الملكية

تسمح الخاصية `float` لعنصر يطفو إلى يمين أو يمين عنصر آخر. الاستخدام الأساسي `float` هو الحصول على صورة على يسار أو يمين كتلة النص.

 `.my-element { 
  float: left; 
  float: right; 
  float: inline-start; 
  float: inline-end; 
  float: none; 
 } 
`
