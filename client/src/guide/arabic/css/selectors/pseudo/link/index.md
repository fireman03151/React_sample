## حلقة الوصل

يتم استخدام `:link` فئة CSS الزائفة لاستهداف جميع العناصر غير المرغوب فيها (مثل عناصر `<a>` مرساة) التي لها سمة `href` ، حتى إذا كان `href` يحتوي على قيمة فارغة.

بناء الجملة العام:

 `target:link 
` 

## مثال

 `/* Selects any <a> that has not been visited yet */ 
 a:link { 
    color: red; 
 } 
`
