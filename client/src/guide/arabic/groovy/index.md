## رائع

إن Apache Groovy أو Groovy هي لغة قوية وديناميكية مع إمكانات التجميع والكتابة الثابتة لمنصة Java ، وقد تم تصميمها لزيادة الإنتاجية بفضل تركيبتها المألوفة والموجودة. أنه يدمج مع أي برنامج جافا بكل سهولة.
