## إنشاء أول شبكة CSS

### تلميح 1

لتغيير عرض أي عنصر ، يتم استخدام البنية التالية:

 `display: propertyName; 
` 

### تلميح 2

قم بتغيير خاصية العرض الخاصة بعنصر .container

### تلميح 3

اضبط عرض عنصر .container على الشبكة

### حل

لتعيين عرض عنصر .container على الشبكة:

 `display: grid; 
`
