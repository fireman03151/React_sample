## قم بتعريف عمل الخالق

### تلميح 1

يتم تعريف الدالة باستخدام بناء الجملة التالي:

 `functionName(){ 
  console.log("Do something"); 
 } 
` 

حيث يمكن تغيير console.log حسب الحاجة.

### تلميح 2

يتم إرجاع القيمة باستخدام `return` الكلمة

### حل

 `function actionCreator(){ 
    return action; 
 } 
`
