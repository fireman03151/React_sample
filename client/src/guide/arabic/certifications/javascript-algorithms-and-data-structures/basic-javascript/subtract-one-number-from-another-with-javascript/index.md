## اطرح رقم واحد من آخر باستخدام JavaScript

باستخدام عامل الطرح "-" ، يمكنك الحصول على الفرق بين رقمين ...

 `var diff1 = 30 - 14; // Difference is 16 
 var diff2 = 90 - 60; // Difference is 30 
`
