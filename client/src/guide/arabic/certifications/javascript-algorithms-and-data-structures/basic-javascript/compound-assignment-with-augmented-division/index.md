## تكليف مركب مع قسم معزز

قراءة أجهزة الكمبيوتر من اليسار إلى اليمين. لذا ، فإن استخدام معامل '/ =' يعني أن المتغير مقسوم مع الرقم الموجود على اليمين ، ثم يتم تعيين المتغير إلى الحاصل. مثل ذلك:

 `var d = 10; 
 d /= 5; // Now, 'd' is equal to 2 
`
