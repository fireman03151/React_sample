## إلحاق المتغيرات بالسلاسل

تأكد من صحة التهجئة والمسافات. يمكن إتمام سلاسل الإضافة (المخزنة داخل المتغيرات) كما يلي:

 `var adj = "happy!"; 
 var sent = "Today, I woke up being "; 
 sent += adj; // The result is "Today, I woke up being happy!" 
`
