## تحديث خصائص الكائن

## ملحوظة:

استخدم نقطة \*\*. \*\* تدوين للوصول إلى خاصية الكائن.

## تنبيه المفسد: الحل إلى الأمام!

## حل:

 `// Setup 
 var myDog = { 
  "name": "Coder", 
  "legs": 4, 
  "tails": 1, 
  "friends": ["freeCodeCamp Campers"] 
 }; 
 
 // Only change code below this line. 
 
 myDog.name = "Happy Coder"; // Solution 
`
