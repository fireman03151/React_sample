## مهمة مركبة مع الطرح المعزز

قراءة أجهزة الكمبيوتر من اليسار إلى اليمين. لذا ، فإن استخدام عامل التشغيل '- =' يعني أن المتغير يطرح بواسطة الرقم الموجود على اليمين ، ثم يتم تعيين المتغير إلى الفرق. مثل ذلك:

 `var b = 10; 
 b -= 4; // Now, 'b' is 6 
`
