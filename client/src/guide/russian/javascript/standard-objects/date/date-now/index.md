## Дата сейчас

Метод Date.now () возвращает количество миллисекунд, прошедших с 1 января 1970 года 00:00:00 UTC ..

### Синтаксис

```js
var timeInMs = Date.now(); 
```

### пример

```js
Date.now(); 
 // 1508783660969 
```
