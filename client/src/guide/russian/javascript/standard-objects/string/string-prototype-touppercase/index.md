Метод JavaScript `.toUpperCase()` возвращает ту же строку, на которую он был вызван, но во всех верхних случаях.

**Синтаксис**
```
str.toUpperCase() 
```

## Примеры
```
console.log("hello world".toUpperCase()); // Console will output "HELLO WORLD" 
```
