## Закон Синеса

Закон синусов только для правых треугольников. Когда у вас есть две или несколько сторон и, по крайней мере, один из их соответствующих углов для решения вашей недостающей стороны.

Уравнение:

Sin (A) / a = Sin (B) / b = Sin (C) / c
