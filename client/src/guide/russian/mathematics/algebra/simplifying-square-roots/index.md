### Пример: упростить ![](https://latex.codecogs.com/gif.latex?\sqrt{180} "\ SQRT {180}")

Шаги решения:

*   Перерыв 180 в произведение простых чисел: ![](https://latex.codecogs.com/gif.latex?180&space;=&space;2^2&space;*&space;3^2&space;*&space;5 "180 = 2 ^ 2 * 3 ^ 2 * 5")
*   Использовать свойство продукта радикалов ![](https://latex.codecogs.com/gif.latex?\sqrt{ab}&space;=&space;\sqrt{a}\sqrt{b}&space;\Rightarrow&space;\sqrt{180}&space;=&space;\sqrt{2^2}\sqrt{3^2}\sqrt{5}=2*3\sqrt{5}=6\sqrt{5} "\ sqrt {ab} = \ sqrt {a} \ sqrt {b} \ Rightarrow \ sqrt {180} = \ sqrt {2 ^ 2} \ sqrt {3 ^ 2} \ sqrt {5} = 2 * 3 \ sqrt { 5} = 6 \ SQRT {5}")  
    Мы не можем упростить квадратный корень из простого числа, поэтому окончательный ответ: ![](https://latex.codecogs.com/gif.latex?\sqrt{180}&space;=&space;6\sqrt{5} "\ sqrt {180} = 6 \ sqrt {5}")

### Пример: упростить ![](https://latex.codecogs.com/gif.latex?\sqrt{1750} "\ SQRT {} 1750")

Шаги решения:

*   Перерыв 1750 в продукт простых чисел: ![](https://latex.codecogs.com/gif.latex?1750&space;=&space;2*7*5^3 "1750 = 2 * 7 * 5 ^ 3")
*   Используйте свойство продукта радикалов и продолжайте разрушать результаты: ![](https://latex.codecogs.com/gif.latex?\sqrt{ab}&space;=&space;\sqrt{a}\sqrt{b}&space;\Rightarrow&space;\sqrt{1750}&space;=&space;\sqrt{2}\sqrt{7}\sqrt{5^3}&space;=&space;\sqrt{2}\sqrt{7}\sqrt{5}\sqrt{5^2}&space;=&space;5\sqrt{2}\sqrt{7}\sqrt{5} "\ sqrt {ab} = \ sqrt {a} \ sqrt {b} \ Rightarrow \ sqrt {1750} = \ sqrt {2} \ sqrt {7} \ sqrt {5 ^ 3} = \ sqrt {2} \ sqrt { 7} \ sqrt {5} \ sqrt {5 ^ 2} = 5 \ sqrt {2} \ sqrt {7} \ sqrt {5}")  
    Теперь снова используйте свойство продукта, чтобы объединить радикалы, которые не могут быть упрощены: ![](https://latex.codecogs.com/gif.latex?5\sqrt{2}\sqrt{7}\sqrt{5}&space;=&space;5\sqrt{70} "5 \ sqrt {2} \ sqrt {7} \ sqrt {5} = 5 \ sqrt {70}")

### Терминология:

Используя √49 в качестве примера:

*   «√» называется **радикальным символом** .
*   49 называется **радикалом** .
