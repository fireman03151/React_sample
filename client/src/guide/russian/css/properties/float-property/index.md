## Свойство Float

Свойство `float` позволяет элементу плавать влево или вправо другого элемента. Основное использование `float` - иметь изображение слева или справа от блока текста.

```css
.my-element { 
  float: left; 
  float: right; 
  float: inline-start; 
  float: inline-end; 
  float: none; 
 } 
```
