## Доступ к объектным свойствам с нотами Dot

Вот одно из возможных решений:

```js
var testObj = { 
  "hat": "ballcap", 
  "shirt": "jersey", 
  "shoes": "cleats" 
 }; 
 
 // Only change code below this line 
 
 var hatValue = testObj.hat;      // Change this line 
 var shirtValue = testObj.shirt;    // Change this line 

```
