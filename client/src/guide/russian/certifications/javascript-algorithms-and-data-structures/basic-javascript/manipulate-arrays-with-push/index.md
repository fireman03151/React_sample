## Манипулировать массивами С помощью push ()

Метод `push()` позволяет добавить (добавить к концу) элемент в массив. Вот так…

```javascript
var arr = [1, 2, 3, 4]; 
 arr.push(5); // Now, the array is [1, 2, 3, 4, 5] 

```
