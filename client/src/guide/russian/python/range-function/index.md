# Функция диапазона

Если вам нужно перебирать последовательность чисел, вам может понадобиться встроенный диапазон функций (). Он генерирует арифметические прогрессии:

#### Пример использования

`py for i in range(5): print(i)`

\#### Выход `0 1 2 3 4`
