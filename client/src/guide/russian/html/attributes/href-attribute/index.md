## Атрибут Href

Href сокращен для «Справочника гипертекста» и является атрибутом HTML. Атрибут href в основном используется для тегов `<a>` чтобы указать URL-адрес веб-страницы, к которой ведет ссылка (будь то на другом разделе той же страницы или на совершенно другой веб-странице).

#### Как пользоваться

`<a href="URL"></a>`

#### Примеры

```html

<a href="https://www.freecodecamp.org">This is an absolute URL</a> 
 
 <a href="index.html">This is a relative URL</a> 
```
