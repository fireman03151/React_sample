## Атрибут таблицы границ

Атрибут границы тега `<table>` - это способ упорядочить границу в таблице HTML. Поддерживаемые значения равны `0` (без границ) и `1` (граница). Атрибут border не поддерживается в HTML 5 последней версией HTML. Рекомендуется использовать CSS для добавления границы в таблицу в HTML 5.
