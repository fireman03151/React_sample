## Тег таблицы

Тег `<table>` позволяет вам отображать таблицу на вашей веб-странице.

### пример

```html

<table> 
  <tr> 
    <td>Row 1, Cell 1</td> 
    <td>Row 1, Cell 2</td> 
  </tr> 
  <tr> 
    <td>Row 2, Cell 1</td> 
    <td>Row 2, Cell 2</td> 
  </tr> 
 </table> 
```

Этот кодовый блок создаст следующий результат:

Строка 1, ячейка 1

Строка 1, ячейка 2

Строка 2, ячейка 1

Строка 2, ячейка 2
