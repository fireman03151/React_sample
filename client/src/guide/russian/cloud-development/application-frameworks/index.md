## Рамки приложений

Рамка приложений - это готовое к использованию программное обеспечение для запуска приложений. Приложения Framework обрабатывают общие задачи, которые выполняют большинство приложений. Рамки приложений сохраняют время разработки и обслуживания, поскольку многие поставщики приложений объединяют ресурсы для общей причины. Рамки приложений больше, чем набор его названных компонентов. Он включает уникальное программное обеспечение, которое интегрирует его компоненты в системную парадигму, которая упрощает общие, сложные или повторяющиеся задачи, используя библиотеки многоразового кода. Каждая инфраструктура приложений решает проблемы с уникальным набором компонентов или уникальным способом - например, управление ресурсами.
