# Тип пустоты

Не следует использовать для переменных, используйте `null` или `undefined` .  
Однако используйте его для функций, которые не возвращают значение.

```typescript
const greeting(): void { 
    alert('Hello, and Welcome to My Webpage!'); 
 } 

```
