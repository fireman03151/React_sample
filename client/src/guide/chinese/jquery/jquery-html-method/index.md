---# HTML方法

Jquery `.html()`方法获取HTML元素的内容或设置HTML元素的内容。

## 入门

要返回HTML元素的内容，请使用以下语法：

```javascript
$('selector').html(); 
```

例如：

```javascript
$('#example').html(); 
```

## 设置

要设置HTML元素的内容，请使用以下语法：

```javascript
$('selector').html(content); 
```

例如：

```javascript
$('p').html('Hello World!'); 
```

这会将所有`<p>`元素的内容设置为Hello World！
