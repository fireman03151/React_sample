## 文章标签

`<article>`标记表示文档中的自包含内容。该文章应独立于页面的其余部分，旨在可分发和可重用。

`<article>`标签已添加到HTML5中，并受到主流浏览器的支持。

### 例

以下是如何在网页中使用文章标记的示例：

```html

<article> 
  <h1>FreeCodeCamp</h1> 
  <p> 
    Learn to code with a community of programmers and contribute to open source projects. 
  </p> 
 </article> 
```
