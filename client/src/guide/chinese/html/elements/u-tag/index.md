## U标签

HTML `<u>`标记使用下划线呈现文本。

`<u>`元素在HTML 4.01中已弃用。在HTML5中， `<u>`标签被重新定义为表示与普通文本在风格上不同的文本。这包括拼写错误的单词或中文专有名词。

相反，要为文本加下划线，建议使用`<span>`标记代替`<u>` 。使用带有`underline`值的CSS `text-decoration`属性设置`<span>`标签的样式。
