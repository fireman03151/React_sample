## 表标签

`<table>`标签允许您在网页上显示表格。

### 例

```html

<table> 
  <tr> 
    <td>Row 1, Cell 1</td> 
    <td>Row 1, Cell 2</td> 
  </tr> 
  <tr> 
    <td>Row 2, Cell 1</td> 
    <td>Row 2, Cell 2</td> 
  </tr> 
 </table> 
```

此代码块将生成以下输出：

第1行，单元格1

第1行，第2单元

第2行，单元格1

第2行，第2单元
