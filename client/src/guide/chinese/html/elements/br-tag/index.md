## Br Tag

`<br>`标记在文本中生成换行符。这对诗歌和地址很有用。

### 例

```html

<!DOCTYPE html> 
 <html> 
  <body> 
    Mozilla Foundation<br> 
    1981 Landings Drive<br> 
    Building K<br> 
    Mountain View, CA 94043-0801<br> 
    USA 
  </body> 
 </html> 
```
