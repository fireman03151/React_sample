## 文档存储数据库

文档存储数据库被认为是另一种类型的NoSQL数据库，它与Key Value数据库非常相似。文档存储中的记录表示单个结构化文档，该文档可以与数据库中的所有其他文档不同。与关系数据库不同，没有对象映射到数据库中的各种表。相反，我们可以获取整个对象并将其直接写入文档存储数据库。

例如，我们可以在代码中包含以下对象。

```json
{ 
    "name": "freeCodeCamp", 
    "job": "contributor" 
 } 
```

整个对象可以直接写入文档存储数据库而无需进一步解析。
