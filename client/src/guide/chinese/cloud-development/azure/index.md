## Microsoft Azure

Microsoft Azure是Microsoft的公共云产品，使个人和组织能够创建，部署和运行基于云的应用程序和基础结构服务。 Microsoft Azure公共云平台为全球组织提供IaaS，PaaS和SaaS服务，并支持许多不同的编程语言，工具和框架，包括Microsoft特定的软件和第三方软件和系统。

Microsoft Azure还可以与各种Microsoft解决方案（如Microsoft System Center）结合使用，并且可以结合使用，将组织的当前数据中心扩展为混合云，从而扩展容量并提供超出仅通过以下方式提供的功能。前提立场。

### 用于Microsoft Azure

Microsoft Azure提供了许多服务和资源。 例如，您可以使用Azure虚拟机计算服务来构建虚拟服务器网络，以托管应用程序，数据库或自定义解决方案，这将是基于IaaS的产品。

其他服务可以归类为PaaS，因为您可以在不维护底层操作系统的情况下使用它们。 例如，当您在Azure Web Apps中运行网站或在Azure SQL数据库中运行SQL数据库时，没有必要确保您使用的是最新版本的Internet信息服务（IIS）或SQL并拥有最新的修补程序和更新安装，因为这是Microsoft Azure平台的责任。

Microsoft Azure上的SaaS服务示例是Operations Management Suite，它通过Operational Insights服务进行设置和访问。在这里，您只需设置管理工具并通过Microsoft Azure连接到您希望管理的服务，因此不需要或无需管理本地基础架构。

### 环境

Azure提供了丰富的不同环境类型，包括操作系统平台，如Windows Server和Ubuntu Server，可以选择多种变体，如Ubuntu 14.04或Ubuntu 16.10，可以通过通过首选客户端内置Azure Web CLI或SSH访问。

有各种级别的机器“尺寸”可供选择，从简单的准系统低规格机器到多GPU和基于多CPU的重型打击盒。
