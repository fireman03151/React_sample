## 定义一个Action Creator

### 提示1

使用以下语法定义函数：

```javascript
functionName(){ 
  console.log("Do something"); 
 } 
```

其中console.log可以根据需要进行更改。

### 提示2

使用`return`关键字返回一个值

### 解

```javascript
function actionCreator(){ 
    return action; 
 } 

```
