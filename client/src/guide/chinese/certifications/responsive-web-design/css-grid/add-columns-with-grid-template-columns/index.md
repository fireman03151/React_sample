## 使用grid-template-columns添加列

在此挑战中，您需要为网格定义列。

### 暗示

您可以通过在CSS代码块中声明以下内容来定义列：

```css
grid-template-columns: parameters; 
```

参数是你想要的任何东西。

### 解

由于挑战要求您创建三列，每列100px，请在.container CSS代码块中声明以下内容：

```css
grid-template-columns: 100px 100px 100px; 

```
