## 创建有序列表

有时人们会过度思考这种挑战：有序列表与无序列表具有完全相同的模式，它只是更改标记： `<ul>`表示无序， `<ol>`表示有序。

不同的效果是一个有子弹，另一个有数字（或其他渐进标记）：

类型|代码|影响 - | - | - 无序的| `<ul><li>Bulleted item</li></ul>` |

*   项目符号项目

订购| `<ol><li>Numbered item</li></ol>` |
