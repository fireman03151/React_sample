# 的Joomla

的Joomla！是一个免费的开源内容管理系统（CMS），它建立在模型 - 视图 - 控制器Web应用程序框架之上，可以独立于CMS使用。它由Open Source Matters，Inc。开发。

的Joomla！是用面向对象编程（OOP）PHP编写的。它可以在MySQL，PostgreSQL或MS SQL数据库中存储数据。功能包括页面缓存，RSS源，页面的可打印版本，新闻快讯，博客，搜索和语言国际化支持。
