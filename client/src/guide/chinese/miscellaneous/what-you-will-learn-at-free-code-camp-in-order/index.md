---
title: What You Will Learn at Free Code Camp in Order
localeTitle: 您将在免费代码营中学到什么
---
*   [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) - 网页的结构。
*   [CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) - 网页的视觉风格。
*   [Bootstrap](http://getbootstrap.com/) - 一种“响应式设计”工具，可帮助您的网站在平板电脑和手机上显示出色。
*   [jQuery](https://jquery.com/)和[Ajax](https://developer.mozilla.org/en-US/docs/AJAX) - 用于控制浏览器内容的简单工具。
*   [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - 所有Web浏览器使用的一种编程语言。
*   [算法](https://en.wikipedia.org/wiki/Algorithm) - 完成任务的逐步配方。
*   [Git](https://git-scm.com/) - 用于保存和共享项目的版本控制系统。
*   [MongoDB](https://www.mongodb.org/) - 一种流行的非关系数据库。
*   [D3.js](http://d3js.org/) - 用于在Web浏览器中生成动态交互式数据可视化的JavaScript库。
*   [Express.js](http://expressjs.com/) - 一个功能强大的Web开发框架。
*   [Node.js](https://nodejs.org/) - 用于创建可伸缩网络应用程序的平台。
*   [SASS](http://sass-lang.com/) - 一种脚本语言，被解释为层叠样式表（CSS）。
*   [API开发](https://en.wikipedia.org/wiki/Application_programming_interface) - 用于构建软件和应用程序的一组例程，协议和工具。
*   [敏捷](https://en.wikipedia.org/wiki/Agile_software_development) - 一组软件开发原则，旨在将项目的设计和生产重点放在用户的需求上
