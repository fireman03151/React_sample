---
title: How to Build a Website Without Writing Any Code
localeTitle: 如何在不编写任何代码的情况下构建网站
---
只需几分钟，就可以构建动态，移动响应的网站，而无需编写任何代码。

以下是我们使用的技术：

*   [atom.io](http://www.atom.io/)一个免费的代码编辑器
*   [startbootstrap.com](http://www.startbootstrap.com/)免费响应（Bootstrap）模板的集合
*   [powr.io](http://www.powr.io/)是一系列免费的JavaScript插件
*   [bitballoon.com](http://www.bitballoon.com/)用于拖放网站部署的工具

如果没有实际编码，您将很快达到可以做的极限，但能够快速构建这样的工作原型网站真是太好了。
