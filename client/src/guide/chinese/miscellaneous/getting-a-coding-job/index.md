## 以下是事实：

*   有数十万个未填写的编码工作。
*   雇主和美国政府联合起来推广非常规编码程序，如Free Code Camp。
*   全栈JavaScript和Swift是最好的两种技能组合，并且有大量的职位空缺。
*   您是一名可雇用的开发人员的最佳证据就是充满了真人所使用的真实应用程序的投资组合。

你仍然需要培养基本的面试技巧和软技能（如果你已经从事知识工作者类型的工作几年，你可能已经有了这些）。

[Hackerrank](https://www.hackerrank.com/domains/tutorials/cracking-the-coding-interview)是提高编码技能的绝佳资源

您还需要努力与您所在城市的编码人员建立联系。您可以加入[您所在城市的本地小组](https://github.com/FreeCodeCamp/FreeCodeCamp/wiki/LocalGroups-List)并参与尽可能多的活动。

我发现[这个](https://github.com/cassidoo/getting-a-gig)链接很有用。它在解释过程方面做得很好。

## 专家提示：

与任何其他行业或领域一样，最难的部分实际上是开始。不要太担心并采取行动 - 结果将令人着迷。
