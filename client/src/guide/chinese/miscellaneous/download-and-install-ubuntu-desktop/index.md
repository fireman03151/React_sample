一个快速教程，展示如何安装Ubuntu。

## 脚步：

您可以在[此处](http://www.ubuntu.com/download/desktop)下载最新版本的Ubuntu Desktop。  
如果您需要任何有关安装操作系统的帮助，Ubuntu的[本](http://www.ubuntu.com/download/desktop/install-ubuntu-desktop)教程将帮助您。如果你想用Windows和Ubuntu双重启动计算机，请查看[这篇](https://help.ubuntu.com/community/WindowsDualBoot)有用的文章。

安装Ubuntu后，打开终端并键入`sudo apt update && sudo apt upgrade`以更新软件包。
