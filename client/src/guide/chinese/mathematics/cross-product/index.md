## 交叉产品

在集合论中，两组的叉积是在集合上执行的二元运算，其输出结果集。 它用符号X表示。有时它也被称为数学家Rene Descartes之后的笛卡尔积。
```

 AXB = {(a, b) | a &isin; A and b &isin; B} 
```

### 例子
```
let us have, 
  set A = {a, b} 
 & set B = {1, 2} 
 then AXB = {(a, 1), (a, 2), 
               (b, 1), (b, 2)} 
```

### 使用

它用于在组合数据库中的两个表之后生成所有可能的组合。
