## 数组来自

'Array.from（）'方法从类似数组或可迭代的对象创建一个新的Array实例。

### 句法：

“”” Array.from（arrayLike \[，mapFn \[，thisArg\]\]） “””

### 例：

“”” Array.from（ '富'）; // \[“f”，“o”，“o”\] “””
