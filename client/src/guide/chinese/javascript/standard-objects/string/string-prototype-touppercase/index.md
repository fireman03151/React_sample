localeTitle: String.prototype.toUpperCase

JavaScript方法`.toUpperCase()`返回调用它的相同字符串，但全部大写。

**句法**
```
str.toUpperCase() 
```

## 例子
```
console.log("hello world".toUpperCase()); // Console will output "HELLO WORLD" 
```
