## Date.prototype.getDate

`getDate()`方法根据本地时间返回指定日期的月中某天。

### 句法

```js
dateObject.getDate() 
```

### 例

```js
var Xmas95 = new Date('December 25, 1995 23:15:30'); 
 var day = Xmas95.getDate(); 
 
 console.log(day); // 25 
```
