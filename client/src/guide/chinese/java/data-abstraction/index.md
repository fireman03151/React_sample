## 定义

根据字典，抽象是处理想法而不是事件的质量。

同样在面向对象的编程中，抽象是从用户隐藏实现细节的过程，只有功能将被提供给用户。换句话说，用户将获得关于对象做什么而不是如何做的信息。

在Java中，使用抽象类和接口实现抽象。
