---
title: Free Code Camp Brownie Points
localeTitle: 免费代码营布朗尼积分
---
Free Code Camp上你的照片旁边的数字告诉你你有多少布朗尼点数。

![与布朗尼积分得分相邻的用户个人资料照片](//discourse-user-assets.s3.amazonaws.com/original/2X/a/aa0c6b56b3b0d4f3acd63212b5e737fdcdd75729.png)

您可以通过两种方式获得布朗尼积分：

1.  完成挑战 - 您完成每次挑战时获得一分
2.  帮助其他露营者进入gitter聊天室 - 每次你帮助另一个露营者并且他们感谢你（通过输入“thanks @yourname ”），你会得到一个观点。

> 布朗尼积分可以帮助您看起来像那种经常编码，分享相关链接并帮助人们的人。

它们还会添加到您的[Streak中](http://forum.freecodecamp.com/t/free-code-camp-streaks/18379) 。
