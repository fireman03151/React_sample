## Hora

La función `time()` devuelve la marca de tiempo actual de Unix (número de segundos desde la Epoca Unix - 1 de enero de 1970 a las 00:00:00 GMT).

### Ejemplo

```php
<?php 
 echo time(); 
```

**Salida:**

```text
1511732226 
```
