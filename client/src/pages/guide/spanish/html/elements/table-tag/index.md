## Etiqueta de tabla

La etiqueta `<table>` permite mostrar una tabla en su página web.

### Ejemplo

```html

<table> 
  <tr> 
    <td>Row 1, Cell 1</td> 
    <td>Row 1, Cell 2</td> 
  </tr> 
  <tr> 
    <td>Row 2, Cell 1</td> 
    <td>Row 2, Cell 2</td> 
  </tr> 
 </table> 
```

Este bloque de código produciría la siguiente salida:

Fila 1, Celda 1

Fila 1, celda 2

Fila 2, celda 1

Fila 2, celda 2
