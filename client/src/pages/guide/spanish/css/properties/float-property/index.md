## Propiedad flotante

La propiedad `float` permite que un elemento flote a la izquierda o derecha de otro elemento. Un uso principal de `float` es tener una imagen a la izquierda o derecha de un bloque de texto.

```css
.my-element { 
  float: left; 
  float: right; 
  float: inline-start; 
  float: inline-end; 
  float: none; 
 } 
```
