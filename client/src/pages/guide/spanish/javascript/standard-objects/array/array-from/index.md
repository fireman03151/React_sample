## Matriz de

El método 'Array.from ()' crea una nueva instancia de Array a partir de un objeto similar a un array o iterable.

### Sintaxis:

'' ' Array.from (arrayLike \[, mapFn \[, thisArg\]\]) '' '

### Ejemplo:

'' ' Array.from ('foo'); // \["f", "o", "o"\] '' '
