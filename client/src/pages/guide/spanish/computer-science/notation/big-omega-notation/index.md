Similar a [la](https://guide.freecodecamp.org/computer-science/notation/big-o-notation) notación [O grande](https://guide.freecodecamp.org/computer-science/notation/big-o-notation) , la función de gran Omega (Ω) se usa en ciencias de la computación para describir el rendimiento o la complejidad de un algoritmo.

Si un tiempo de ejecución es Ω (f (n)), entonces para n lo suficientemente grande, el tiempo de ejecución es al menos k⋅f (n) para alguna constante k. Aquí es cómo pensar en un tiempo de ejecución que es Ω (f (n)):  
![gran función omega](https://s3.amazonaws.com/ka-cs-algorithms/Omega_fn.png)  
Decimos que el tiempo de ejecución es "big-Ω of f (n)". Utilizamos la notación big Ω para **los límites inferiores asintóticos** , ya que limita el crecimiento del tiempo de ejecución desde abajo para tamaños de entrada lo suficientemente grandes.

### Diferencia entre Big O y Big Ω

La diferencia entre la notación Big O y la notación Big Ω es que Big O se utiliza para describir el peor tiempo de ejecución de un algoritmo. Pero, por otra parte, la notación Big Ω se utiliza para describir el mejor tiempo de ejecución de un algoritmo dado.
