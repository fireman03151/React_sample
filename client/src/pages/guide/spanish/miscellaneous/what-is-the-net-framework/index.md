[.NET Framework](https://www.microsoft.com/net) o simplemente, .NET es un marco de tiempo de ejecución multi-lenguaje multi-propósito; creado y propiedad de [Microsoft](https://www.microsoft.com) .

Tiene una gran biblioteca de clases, soporta interpolabilidad de idiomas; y se ejecuta principalmente en Windows.

Microsoft comenzó el desarrollo de .NET Framework a fines de la década de 1990 y en la década de 2000; Se lanzó una versión beta. Aparte del desarrollo de GUI de Windows y el desarrollo de aplicaciones de Windows Phone; El framework .NET se utiliza en el desarrollo web.

Funciona a la perfección con la plataforma de hospedaje en la nube de [Microsoft Azure](https://azure.microsoft.com/en-us/) y la base de datos de [SQL Server](https://www.microsoft.com/en-us/server-cloud/products/sql-server/) . Permite programar el lado del servidor en múltiples idiomas diferentes; predominantemente en [C #](https://msdn.microsoft.com/en-us/library/67ef8sbd.aspx) .

Con [MSDN](https://msdn.microsoft.com/en-us/default.aspx) como una comunidad rica en contenido, existen muchos potenciales y beneficios de usar .NET para el desarrollo web.
