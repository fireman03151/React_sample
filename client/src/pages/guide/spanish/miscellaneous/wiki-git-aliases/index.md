Para la mayoría de los desarrolladores, una gran parte de nuestro tiempo lo pasamos en Terminal y la mayoría lo gastamos escribiendo comandos Git. Hemos creado un conjunto de métodos abreviados de teclado con funciones y alias de Bash para acelerar su flujo de trabajo y guardar cientos de pulsaciones de teclas todos los días.

Git le permite establecer alias, pero son limitados y solo le ahorran algunas pulsaciones de tecla (es decir, en lugar de git checkout puede escribir git co, pero aún tiene que escribir git). Dado que Bash es el intérprete de línea de comandos predeterminado de Terminal, también puede configurar los alias de Bash para reducir aún más las pulsaciones de teclas.

Aquí hay una lista de alias y funciones de Git Bash. Para usarlos como propios, simplemente agréguelos al archivo en el que almacena sus alias / funciones. (Es decir, `~/.bash_profile` o `~/.bashrc` )

Al copiar y pegar, es importante mantener el espacio. (es decir, para los alias, no debe haber espacios antes y después de los signos de igual, y para las funciones, debe haber un espacio después del corchete de apertura de la declaración y un punto y coma después del comando. No olvide volver a cargar su archivo ( source `~/.bash_profile` ) o reinicie Terminal después de hacer cambios.
