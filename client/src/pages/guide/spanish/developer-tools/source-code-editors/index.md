## Editores de código fuente

Los editores de código fuente son los programas que permiten la creación de código y  
edición. Cualquier editor de texto puede ser usado para escribir código. Pero código dedicado  
Los editores ofrecen muchas funciones útiles, que incluyen: completar texto,  
coloreado de marcas, y pestañas / paneles para la edición en paralelo. Estas características pueden  
ser útil incluso para los nuevos usuarios que no conocen su camino alrededor de todo el  
opciones avanzadas.

### Elegir un editor

Hay muchos, muchos editores disponibles para descargar o usar  
En línea, tanto de pago como de forma gratuita. Ofrecen muchas de las mismas características básicas, pero  
Cada uno tiene sus propias sutilezas. Más investigación puede ayudar a encontrar el que está  
bien por ti. Los siguientes editores son todos multiplataforma y  
Libre de usar o evaluar.

### Texto sublime

![Icono de texto sublime](https://logo.clearbit.com/sublimetext.com)

Sublime Text es un editor muy popular que ha existido [desde 2008](https://www.sublimetext.com/blog/articles/one-point-oh) . Hay muchas opciones y [extensiones](https://packagecontrol.io/) disponibles, y mucha [ayuda disponible en](http://stackoverflow.com/search?q=sublime+text) línea para ayudar a [configurarlo](http://www.websightdesigns.com/wiki/Setting_up_Sublime_Text_3) . (La licencia para uso continuado cuesta $ 70. Sin embargo, Sublime Text se puede descargar y evaluar de forma gratuita, con una ventana emergente en pantalla).

[sublimetext.com](https://www.sublimetext.com/)

![Captura de pantalla de texto sublime](//discourse-user-assets.s3.amazonaws.com/original/2X/6/63f774c31a7b1a439bcef9d11e95463b6998310e.png)

### Soportes

![Icono de soportes](https://logo.clearbit.com/brackets.io)

Brackets es un editor de código abierto relativamente nuevo de Adobe. Es muy fácil de usar, especialmente para las personas que no están acostumbradas a las interfaces de línea de comandos o configuraciones / preferencias de estilo JSON. Las extensiones y los temas son rápidos y fáciles de encontrar e instalar a través de Extension Manager.

[corchetes.io](http://brackets.io/)

![Brackets Captura de pantalla](//discourse-user-assets.s3.amazonaws.com/original/2X/2/2f395a58d24ed551ed8440a10ae640777f1143b6.png)

### Átomo

![Icono del átomo](https://logo.clearbit.com/atom.io)

Atom es un editor de [código abierto](https://github.com/atom/atom) , desarrollado por [GitHub](https://github.com/) . Como Sublime Text, Atom es bastante popular. Es muy configurable ("hackeable", como dicen) pero las características avanzadas pueden ser abrumadoras para los nuevos usuarios. También hay un montón de [documentación](http://flight-manual.atom.io/) [útil](http://readwrite.com/2014/05/20/github-atom-5-tips-getting-started-tutorial-corey-johnson/) [disponible en](http://stackoverflow.com/search?q=atom) línea.

[atom.io](https://atom.io/)

![Captura de pantalla del átomo](//discourse-user-assets.s3.amazonaws.com/original/2X/8/8016e7c0937e8ce6fcb6104c9aee0026443912f5.png)

### Código de Visual Studio

![Icono de código de Visual Studio](https://logo.clearbit.com/code.visualstudio.com)

Visual Studio Code (también conocido como vsc o vscode) es un editor de código abierto muy popular, desarrollado por [Microsoft](https://www.microsoft.com) y que se está utilizando rápidamente por desarrolladores de todos los idiomas. Al igual que Atom y Sublime, Visual Studio Code tiene una amplia gama de [extensiones](https://marketplace.visualstudio.com/VSCode) y [temas](https://marketplace.visualstudio.com/search?target=VSCode&category=Themes&sortBy=Downloads) disponibles para ayudarlo a personalizar y perfeccionar su experiencia al usarlo, sin embargo, VSC es altamente configurable "listo para usar". También hay un equipo de desarrollo muy activo detrás de VSCode, con nuevas funciones y correcciones de errores que el equipo de Visual Studio Code agrega [cada mes](https://code.visualstudio.com/updates) .

[code.visualstudio.com](https://code.visualstudio.com/)

### Bloc de notas ++

![Icono de Notepad ++](https://upload.wikimedia.org/wikipedia/commons/0/0f/Notepad%2B%2B_Logo.png)

Notepad ++ es un editor de código fuente abierto y gratuito que se ejecuta en el entorno de Microsoft Windows. Fue desarrollado en 2003, por Dan Ho. Está escrito en [C ++](www.cplusplus.com) (el nombre viene de eso). Notepad ++ es compatible con la edición con pestañas, que permite trabajar con múltiples archivos abiertos en una sola ventana y también puede usarse como editor de texto sin formato.

[notepad-plus-plus.org](https://notepad-plus-plus.org)
